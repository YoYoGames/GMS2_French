Scripts de compatibilité


Lorsque vous importez un projet créé avec une version précédente de GameMaker Studio, toutes les fonctions obsolètes utilisées continueront de fonctionner car elles auront été automatiquement transformées par l'EDI en scripts de compatibilité. En GameMaker Studio 2 analyse votre projet pour toutes les fonctions et variables obsolètes que vous avez utilisées, puis crée une série de scripts utilisant ces noms de fonctions. Ces scripts contiendront soit la nouvelle fonction équivalente, soit un court script de code effectuant la même action que la fonction obsolète.

Lorsque vous importez un projet GameMaker Studio 2 pour la première fois, un rapport de compatibilité très similaire à celui-ci s'affiche:

Le corps principal du rapport indique quels objets et scripts ont été affectés et où, à la fin, vous pouvez trouver une liste des scripts créés pour votre projet. Vous pouvez cliquer sur le bouton central de la souris ou appuyez sur sur l'un des scripts répertoriés dans le rapport pour l'ouvrir et voir ce qui a été fait. Notez que si vous voyez des scripts créés avec une double barre inférieure avant leur nom (par exemple: __global_object_depths ), il s’agit de scripts qui ont été créés par l’EDI pour améliorer la compatibilité et ne sont pas des scripts convertis à partir de fonctions obsolètes. Ils ne doivent donc pas être altérés car ils sont nécessaires au bon fonctionnement des autres scripts de compatibilité.

Lorsque vous avez terminé de réviser le rapport de compatibilité, vous pouvez fermer l'espace de travail. Si vous devez modifier quoi que ce soit dans le rapport, vous pourrez le retrouver en consultant la section Notes de l'arborescence des ressources.

Veuillez noter que certaines fonctions introduites dans le langage GameMaker pour assurer la compatibilité avec des projets plus anciens apparaissent dans ces scripts de compatibilité. Cependant, ces fonctions ne doivent jamais être utilisées dans vos propres projets. Vous pouvez trouver une liste de ces fonctions dans la section suivante du manuel:
