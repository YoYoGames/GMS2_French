Micro Web Server


Cette fonction principale est de se connecter et de servir des fichiers sur certaines des plates-formes cibles disponibles pour GameMaker Studio 2 (comme la cible HTML5, par exemple). Lorsque vous lancez un jeu sur une cible nécessitant le serveur Web Micro, une nouvelle fenêtre s'ouvre lorsque le jeu est diffusé sur le périphérique choisi:

NOTE: La première fois que vous lancez un jeu sur une cible nécessitant le Micro Web Server, il est possible que vous receviez une alerte de pare-feu. Vous devez autoriser le serveur Web à communiquer via le pare-feu si cela se produit.


Le serveur Web affichera une certaine quantité d'informations de débogage liées aux fichiers qui ont été servis, et dispose d'un bouton pour effacer cela en bas. Vous ne pouvez pas éditer la plupart des autres informations ici (comme cela est défini par les paramètres du Gestionnaire de cible, mais vous pouvez changer la valeur de la Vitesse du Réseau.) Le régler à autre chose que la valeur maximum ralentira la vitesse de transfert et vous permettra tester des jeux dans un environnement à faible bande passante.
