Préférences - Timelines


Les préférences de timelines sont utilisées pour définir certaines propriétés lorsque vous travaillez avec l' éditeur de montage. Les options disponibles sont:

  • Réponse automatique à la conversion DnD™ au langage GameMaker: lorsque vous utilisez le bouton droit de la souris option de menu sur un moment de la ligne de temps pour convertir les actions glisser et déposer en script (code) vous obtiendrez une invite vous demandant si vous souhaitez continuer, car il s'agit d'une conversion à sens unique. Changer cette option pour "OK" supprimera le message et aller de l'avant avec la conversion comme si vous aviez cliqué sur "OK" dans la boîte de dialogue. Le paramètre par défaut est "Afficher le message".

  • Réponse automatique à l'avertissement d'action permanente: Certaines actions dans une chronologie ne peuvent pas être annulées, et quand cela arrive, vous recevrez un message d'avertissement vous demandant si vous voulez continuer. Changer cette option pour "OK" supprimera le message et ira de l'avant avec l'action comme si vous aviez cliqué sur "OK" dans la boîte de dialogue. Le paramètre par défaut est "Afficher le message".

  • Ouvrir de nouveaux moments pour l'édition à la création: Normalement, lorsque vous ajoutez un nouveau moment à un montage, GameMaker Studio 2 ouvre automatiquement l'éditeur pour que vous ajoutiez votre code ou faites un glisser-déposer. Cependant, si vous décochez cette option, le moment sera ajouté sans ouvrir l'éditeur (vous pouvez l'ouvrir en double-cliquant en ce moment). Par défaut, cette option est activée.
