Préférences - Éditeur de sprites


Les préférences de l' éditeur de sprites sont utilisées pour définir la manière dont l' éditeur de sprite est conçu et fonctionne. L'option suivante existe:

  • Origine de l'image-objet par défaut: cette option vous permet de définir la position d'origine par défaut pour toutes les nouvelles ressources d'image-objet que vous créez. La valeur par défaut est le coin supérieur gauche.

  • Mode d'animation d'image-objet par défaut: cette option vous permet de définir le mode d'animation par défaut pour les images-objets dans l'éditeur d'images-objets. Vous pouvez choisir "Images par seconde" ou "Images par image de jeu", le réglage par défaut étant "Images par seconde".

  • Largeur d'image-objet par défaut: cette option vous permet de définir la largeur par défaut pour tous les nouveaux objets-objets créés dans l'éditeur d'images-objets. La valeur par défaut est 64.

  • Hauteur de sprite par défaut: Cette option vous permet de définir la hauteur par défaut de tous les nouveaux sprites créés dans l'éditeur de sprites. La valeur par défaut est 64.

La sous-catégorie suivante existe également pour l'éditeur de sprites:

Les options disponibles ici contrôlent les messages qui apparaissent lorsque vous devez confirmer une action. Vous pouvez modifier ces paramètres pour toujours afficher le message ou le rejeter avec une action spécifique (Oui ou Non):

  • Réponse automatique à l'importation de fichier: Lorsque vous importez une nouvelle image et qu'une autre existe déjà, vous serez invité par défaut à poursuivre l'importation et à écraser l'image existante ou à annuler l'opération. Cette option vous permet de supprimer ce message et d'annuler l'importation en sélectionnant "Non" ou d'écraser l'image en cours en sélectionnant "Oui". La valeur par défaut est "Afficher le message".

  • Réponse automatique à l'échec du chargement de l'image: Lorsque vous importez une nouvelle image et que l'importation échoue, un message d'avertissement vous informe par défaut de l'existence d'un échec. Cette option vous permet de supprimer ce message en sélectionnant "OK". La valeur par défaut est "Afficher le message".
