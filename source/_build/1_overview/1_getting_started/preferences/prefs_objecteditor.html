Préférences - Éditeur d'objets


Les préférences de l'éditeur d'objets permettent de définir certaines propriétés pour la fenêtre de l' éditeur d'objets. La section principale contient l'option suivante:

  • Ajouter un contenu d'aide par défaut: Par défaut, cette option est activée et tout ce qu'elle fait est d'ajouter un simple commentaire dans l' éditeur de code pour tous les nouveaux événements pour vous rappeler que vous pouvez y ajouter votre code.

  • Contenu de l'événement par défaut: Ici, vous pouvez modifier les commentaires qui sont ajoutés par défaut dans la fenêtre de code pour les nouveaux événements.

La sous-catégorie suivante existe également pour l'éditeur d'objets:

Cette section traite des boîtes de dialogue qui peuvent être affichées lors de l'utilisation de l'éditeur d'objets, avec l'option suivante disponible:

  • Réponse automatique à DnD & trad; conversion en langue de gamemaker: Vous pouvez utiliser le bouton droit de la souris dans l'espace de travail Glisser-Déposer pour sélectionner l'option permettant de transformer tout Glisser-Déposer pour cet événement en code GML. L'action par défaut ici est d'afficher un message d'avertissement vous demandant si vous voulez continuer, mais vous pouvez le régler sur "OK" et la boîte de dialogue ne sera pas affichée avant la conversion.

  • Réponse automatique à la limite vertex de dépassement de physique: Cette option peut être utilisée pour contrôler le message contextuel affiché lorsque vous dépassez le nombre de vertex autorisé dans l'éditeur de forme physique. L'action par défaut ici est d'afficher un message d'avertissement, mais vous pouvez le régler sur "OK" et la boîte de dialogue ne s'affichera pas.
