L'installation


GameMaker Studio 2 est conçu pour être utilisé sur des ordinateurs de bureau avec les spécifications minimales suivantes:

Pour la plate-forme Windows, vous devez disposer de la configuration de base suivante:

  • Processeur Dual Core compatible Intel 64 bits
  • 2 Go de RAM
  • Carte graphique compatible DX11
  • Microsoft Windows 7 64 bits
  • Au moins 3 Go d'espace libre sur le disque dur

Cependant, les spécifications recommandées sont:

  • Quad core Intel 64 bits compatible Intel
  • 8 Go de RAM
  • Carte graphique basée sur DX11
  • Microsoft Windows 10 64 bits
  • Disque dur SSD

Pour la plate-forme Mac OS, vous devez avoir au moins les spécifications minimales suivantes:

  • Mac OSX El Capitan ou supérieur
  • Un processeur Intel Dual Core i3
  • 4 Go ou plus de RAM
  • Xcode 7 ou supérieur

Cependant, les spécifications recommandées sont:

  • macOS
  • Un processeur Intel Quad Core i5
  • 8 Go de RAM
  • Disque dur SSD

Vous pouvez télécharger le GameMaker Studio 2 installation de GameMaker Studio 2 depuis le site Web des YoYo Games. Une fois les mises à jour installées, vous devrez fournir les détails de la licence de votre compte YoYo Games (voir ici ) avant de pouvoir utiliser le produit. En d'autres GameMaker Studio 2 la première fois que vous utiliserez GameMaker Studio 2 vous devez disposer d'une connexion Internet. Après la connexion initiale, vous devriez pouvoir utiliser le produit hors connexion. Toutefois, vous devrez vous connecter régulièrement pour vérifier le compte de connexion.
