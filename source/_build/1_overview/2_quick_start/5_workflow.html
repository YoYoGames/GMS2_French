Workflow


Dans cette section de notre Guide de démarrage rapide, nous allons vous guider dans le processus de création de sprites, d'objets et de pièces, pour vous permettre de compiler le projet et de voir les résultats. GameMaker Studio 2 est innovant de plusieurs façons, comme vous le verrez, nous espérons donc que cette section vous donnera assez de connaissances pour plonger et commencer sans trop de problèmes...

Pour illustrer le flux de travail, nous allons réaliser un projet très simple qui fera rebondir une balle dans une pièce. Dans le cours des choses, vous aurez à utiliser l'éditeur d'image, l'éditeur d'objets, l'éditeur de code et l'éditeur de pièce, où vous passerez le plus clair de votre temps sur n'importe quel projet que vous créerez. Comme il s'agit essentiellement d'un mini-didacticiel pour vous aider à démarrer, nous l'avons divisé en sections auxquelles vous pouvez accéder à partir des liens suivants:

  1. Créer des sprites
  2. Création de jeux de tuiles
  3. Créer des sons
  4. Créer des objets
  5. Créer des chambres
