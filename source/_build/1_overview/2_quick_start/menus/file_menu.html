Le menu Fichier


Le menu Fichier vous permet de modifier, d’enregistrer ou d’importer des projets et de définir les préférences IDE de GameMaker Studio 2. Les options disponibles sont:

  • Nouveau projet: permet de créer un nouveau projet à partir de zéro dans GameMaker Studio 2. Cela ouvrira la boîte de dialogue "Enregistrer" et vous invitera à choisir un emplacement pour enregistrer le nouveau projet. Une fois que vous avez donné un nom au projet et un emplacement pour le sauvegarder, vous serez redirigé vers un nouvel espace de travail vide pour commencer à travailler dessus. Les fichiers de projet sont enregistrés avec le *.yyp extension de fichier.

  • Ouvrir projet - Ceci ouvrira l’explorateur de fichiers GameMaker Studio 2 pour vous permettre de sélectionner un fichier créé précédemment. *.yyp fichier de projet pour le chargement.

  • Importer un projet - Si vous avez un projet réalisé avec le GameMaker: Studio 1.4 de YoYo Games, vous pouvez sélectionner cette option pour importer *.gmx ou la *.gmz fichier.

    REMARQUE: pour importer un projet GameMaker précédent dans GameMaker Studio 2 il doit avoir été créé avec la version 1.4 et même dans ce cas, il se peut que l'opération ne fonctionne pas directement lors de l'importation. Beaucoup de choses diffèrent entre GameMaker Studio 2 et GameMaker: Studio 1.4. Par conséquent, vos projets plus anciens peuvent nécessiter quelques modifications pour fonctionner. Reportez-vous à l'article Portage d'un jeu GM: S 1.4 sur GameMaker Studio 2 dans la base de connaissances de YoYo Games et sur la section du manuel qui couvre les fonctions obsolètes et les scripts de compatibilité.
    IMPORTANT! L'importation d'anciens projets et de projets YYZ n'est pas disponible avec la licence d'évaluation du produit.


  • Projets récents - Vous pouvez utiliser cette option pour afficher les projets récemment utilisés. En le sélectionnant, vous rouvrirez la page de démarrage (si vous l'avez fermée) et vous pouvez sélectionner un projet récemment utilisé dans la liste qui s'y trouve.

  • Enregistrer le projet - Ceci enregistrera votre jeu au même emplacement que celui que vous avez défini lors de la création du projet (ou de l'endroit où vous l'avez ouvert).

  • Enregistrer le projet sous - Comme ci-dessus, vous ne spécifiez que l'emplacement et le nouveau nom du projet.

  • Exporter le projet - Ceci exportera le projet en tant que simple *.yyz fichier. Très utile pour envoyer des copies à des personnes ou pour sauvegarder des sauvegardes.
    IMPORTANT! L'exportation de projets YYZ n'est pas disponible avec la licence d'évaluation du produit.
  • Préférences - Sélectionnez cette GameMaker Studio 2 la GameMaker Studio 2 Préférences où vous pouvez configurer l'apparence et la GameMaker Studio 2.

  • Nouvel IDE - Lorsque vous sélectionnez cette option, une nouvelle instance de l'EDI sera ouverte pour que vous puissiez l'utiliser (elle s'ouvrira sur la page de démarrage).

  • Mon compte - Cette option ouvre votre navigateur Web par défaut et vous amène à la page de votre compte YoYo.

  • Déconnexion - Sélectionnez cette GameMaker vous déconnecter de GameMaker Studio 2. Vous devrez vous reconnecter avec la même licence (ou une autre) pour pouvoir utiliser le produit ultérieurement.
    IMPORTANT! Si vous utilisez une licence d' GameMaker Studio 2 de GameMaker Studio 2, puis achetez l'une des autres licences (Desktop, Mobile, UWP, Web ou Console), vous devrez peut- être vous déconnecter de l'EDI, puis redémarrer GameMaker Studio et vous GameMaker Studio à nouveau pour vous assurer que la nouvelle licence est appliquée correctement (même si cela devrait se produire automatiquement).
  • Quitter: GameMaker Studio 2 sur le bureau.
