Le menu Layouts


Ce menu est lié à l'enregistrement et au chargement de différentes mises en page IDE. Vous pouvez configurer l'EDI comme bon vous semble, avec certains éléments (comme l'Arborescence des ressources, ou les Fenêtres de sortie) disposés différemment par défaut. Cela vous permet de configurer GameMaker Studio 2 en fonction de votre flux de production et avec les options de ce menu, vous pouvez enregistrer les mises en page de projet en cours en tant que *.xml fichier qui peut ensuite être chargé dans un projet pour définir les mises en page pour être le même. Les options de ce menu sont les suivantes:

  • Load Layout - Ceci ouvre un explorateur de fichiers dans lequel vous pouvez sélectionner une mise en page précédemment enregistrée *.xml fichier à charger.

  • Enregistrer la mise en page - Ceci ouvre un explorateur de fichier où vous pouvez nommer et enregistrer la mise en page IDE actuelle en tant que *.xml fichier.

  • Réinitialiser la mise en page - Ceci réinitialisera la mise en page IDE actuelle aux paramètres par défaut.
