Le menu d'aide


Ce menu concerne l'obtention d'aide lors de l'utilisation de GameMaker Studio 2. Il contient les éléments de menu suivants:

  • Page de démarrage - Ceci ouvrira un nouvel espace de travail.

  • Page d'accueil - Ceci ouvre le diaporama initial de la page d'accueil dans un nouvel espace de travail.

  • Ouvrir le manuel - Ouvre le document d'aide. Notez que cliquer sur le bouton d'aide dans la barre de menu ouvre également le document, comme le fera .

  • Base de connaissances - Vous GameMaker Centre d'aide de GameMaker Studio 2 où vous pouvez trouver des articles sur la configuration de la plate-forme, des problèmes connus, l'utilisation de GameMaker Studio 2 et bien d'autres choses encore.

  • Signaler un bug - Si vous trouvez des bugs ou si vous rencontrez des problèmes lors de l'utilisation de GameMaker Studio 2 vous pouvez les signaler ici.

  • YoYo Games.com - Ouvre la page principale de YoYo Games.

  • GameMaker Community - Cela va ouvrir les forums de la communauté GameMaker dans votre navigateur. Notez qu'il s'agit d'un forum géré par la communauté, donc si vous avez un rapport de bug ou si vous avez besoin d'une assistance technique, vous devriez d'abord aller sur les forums de support officiels (voir ci-dessus).

  • Notes de publication - Ceci ouvrira les notes de GameMaker Studio 2 pour GameMaker Studio 2 dans votre navigateur. Si vous devez annuler une mise à jour, vous pouvez obtenir les versions précédentes en cliquant sur le lien du numéro de version sur cette page.

  • Notes de version de l'exécution en cours - Comme le moteur d'exécution peut être mis à jour indépendamment de l'EDI, il possède son propre ensemble de notes de version d'exécution. Sélectionnez cette option pour les ouvrir dans votre navigateur, mais notez que l'exécution en cours n'est peut-être pas celle que vous utilisez. Pour savoir à quelle exécution vous êtes, ou pour savoir comment ajouter, supprimer ou activer des temps d'exécution, veuillez consulter la section Préférences d'exécution.

  • SDK requis - En raison de la nature multiplateforme de GameMaker Studio 2 un certain nombre de SDK tiers sont requis. En cliquant sur ce lien, vous accédez à un article du forum de support qui détaille ce dont vous avez besoin par plate-forme.

  • Ouvrir le projet dans l'explorateur - Cela ouvrira l'emplacement de sauvegarde du projet actuellement chargé dans l'explorateur de fichiers de l'OS.

  • À propos de - Ouvre la fenêtre "à propos de GameMaker Studio 2 " dans laquelle vous pouvez obtenir des informations sur la version du produit (ce qui est essentiel pour signaler tout bogue ou problème).
