iap_status

Vous permet de suivre l'état du système d'achat intégré.

Syntaxe:

iap_status();


Retours: Constant


La description

Cette fonction vous permet de suivre l'état du système d'achat de l'application GameMaker Studio 2 et renvoie l'une des constantes suivantes:

Constant La description
iap_status_uninitialised Les achats in-app n'ont pas été activés pour votre jeu.
iap_status_unavailable Aucune connexion n'est disponible pour le magasin cible.
iap_status_loading Une connexion au magasin cible se termine et est en cours d'activation.
iap_status_available Les achats ont été activés et le magasin cible est disponible.
iap_status_processing Il y a actuellement une communication entre le jeu et le magasin cible, comme quand (par exemple) faire un achat ou lors de la restitution d'un achat, etc...
iap_status_restoring Sur la cible iOS, cela peut être retourné pendant que votre jeu tente de restaurer les achats.


Exemple:

status = iap_status();

Le code ci-dessus demande l'état du magasin pour le jeu en cours et le stocke dans une variable à vérifier.
