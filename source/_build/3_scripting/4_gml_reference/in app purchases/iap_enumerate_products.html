iap_enumerate_products

Remplissez une liste ds_list avec tous les produits IAP actuellement disponibles dans le magasin cible.

Syntaxe:

iap_enumerate_products();


Retours: N / A


La description

Cette fonction récupère la liste des identifiants de produit achetés dans l'application rencontrés. Sur la plupart des plates-formes, cela renvoie simplement l'ensemble des identifiants de produit fournis via iap_activate(), mais sur les plateformes où le magasin offre la possibilité de récupérer une liste complète des produits disponibles pour un titre, cet ensemble d'identifiants de produit inclura tous les identifiants de produit disponibles pour le jeu. La fonction nécessite que vous fournissiez un précédemment fait ds_list (que vous devriez effacer d'abord s'il a été utilisé à d'autres fins) qui sera rempli par l'appel de la fonction. Quelle que soit la plate-forme, si le magasin cible n'est pas disponible, il recevra les ID de produit de ceux fournis lorsque vous avez activé les achats en utilisant iap_activate.


Exemple:

var status = iap_status();
var iap_list = ds_list_create();
if status == iap_status_available
   {
   iap_enumerate_products(iap_list);
   }

Le code ci-dessus vérifie l'état du magasin IAP cible et, s'il est disponible, il remplira la liste créée avec les ID de produit disponibles pour une utilisation ultérieure.
