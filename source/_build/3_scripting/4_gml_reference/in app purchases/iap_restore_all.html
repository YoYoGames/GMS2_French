iap_restore_all

Restaure tous les achats pour le jeu.

Syntaxe:

iap_restore_all();


Retours: N / A


La description

Restaure tous les achats à leur statut par défaut pour le jeu et l'utilisateur. Cela serait applicable si (par exemple) l'utilisateur a changé de périphérique ou supprimé le titre de son périphérique actuel et a réinstallé le titre. Cela déclenchera deux événements IAP, le premier étant du type iap_ev_restore et le second du type iap_ev_purchase. Pour l'événement de restauration, le ds_map iap_data aura la clé supplémentaire suivante qui peut ensuite être vérifiée pour s'assurer que les produits et les achats ont été restaurés avec succès:

  • " result " - Le résultat sous forme de valeur booléenne, où true indique une restauration réussie et false une sorte d'erreur.

Pour l'événement d'achat, le iap_data la carte contiendra les mêmes informations que celles renvoyées lorsque vous appelez iap_acquire.


Exemple:

iap_restore_all();

Cela demanderait au magasin cible de restaurer tous les achats du jeu sur le périphérique qui l'exécutait, et déclencherait un événement IAP avec le résultat de l'appel stocké dans le dossier spécial. iap_data ds_map.
