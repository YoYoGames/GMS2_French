place_snapped

Syntaxe:

place_snapped(hsnap, vsnap);


Argument La description
hsnap L'accrochage horizontal à vérifier.
vsnap L'accrochage vertical à vérifier.


Renvoie: Boolean


La description

Avec cette fonction, vous pouvez vérifier et voir si l'origine d'une instance (sa position x et y) est alignée sur une grille avec les valeurs hsnap et vsnap spécifiées par vous.


Exemple:

with (obj_Pieces)
   {
   if !place_snapped(32, 32)
      {
      move_snap(32, 32);
      }
   }

Le code ci-dessus vérifie toutes les instances de "obj_Pieces" pour voir si elles sont accrochées à une grille de 32x32 pixels, et si elles ne le sont pas, il les accroche à la position la plus proche dans cette grille.
