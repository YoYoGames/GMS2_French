position_meeting


La description

Avec cette fonction, vous pouvez vérifier la position d'une collision avec une autre instance ou toutes les instances d'un objet. Lorsque vous utilisez ceci, vous recherchez un exemple ou un objet dans un seul point de la pièce. Cette vérification sera effectuée sur le cadre de sélection de l'instance ou sur le masque de l'instance si des collisions précises ont été vérifiées pour cette instance. Si vous devez obtenir l' ID d' instance unique si l'objet en collision, vous devez utiliser instance_position. Cette fonction accepte également les mots-clés spéciaux all et other.


Syntaxe:

position_meeting(x, y, obj);

Argument La description
x La position x à vérifier.
y La position y à vérifier.
obj L'objet (ou l'ID d'instance, ou les mots-clés "all" ou "other") avec lesquels rechercher une collision.


Résultats:

Boolean


Exemple:

if mouse_check_button(mb_left)
   {
   if !position_meeting(mouse_x, mouse_y, all) instance_create_layer(mouse_x, mouse_y, "Walls", obj_Wall);
   }

Le code ci-dessus recherche le bouton gauche de la souris et, s’il est appuyé, vérifie la position x / y de la souris en cas de collision avec n’importe quelle instance. S'il n'y en a pas, une instance de "obj_Wall" est créée.


Retour: vérification de la collision
