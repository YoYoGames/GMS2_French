changement de position


La description

Cette fonction vérifie la position d'une collision avec toutes les instances du point donné et, le cas échéant, toutes les occurrences de la collision seront des occurrences de l'objet choisi. Vous pouvez définir l’argument "perf" sur true ce qui obligera GameMaker Studio 2 à exécuter les événements Détruire et Nettoyer de l’instance trouvée et l’événement Créer de la nouvelle instance, ou false ne pas effectuer aucun de ces événements. Veuillez noter que si vous choisissez de ne pas exécuter les événements Détruire, Nettoyer et Créer, toute instance créée utilisant une variable normalement définie dans l'événement create plantera le jeu, car cette variable n'existera plus.


Syntaxe:

position_change(x, y, obj, perf);

Argument La description
x La coordonnée x de l'endroit où modifier les instances en collision.
y La coordonnée y de l'endroit où modifier les instances en collision.
obj Nouvel objet en tant qu'objet appelant.
perf Indique s'il faut exécuter l'événement Create de ce nouvel objet (true) ou non (false).


Résultats:

N/A


Exemple:

position_change(200, 200, obj_Bird, true);

Cela modifiera toutes les instances en collision au point (200, 200) en une instance de "obj_Bird", en effectuant l'événement Create "obj_Bird" pour chacune d'elles dans le processus.


Retour: vérification de la collision
