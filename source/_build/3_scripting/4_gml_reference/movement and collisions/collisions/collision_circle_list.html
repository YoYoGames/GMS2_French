liste_circle_collection


La description

Cette fonction est la même que la collision_circle() Au lieu de simplement détecter une instance en collision à la fois, il détectera plusieurs instances. Vous indiquez la position x / y du centre de la zone circulaire à vérifier, ainsi que le rayon et l'objet à vérifier. Vous pouvez également définir le contrôle de manière précise (auquel cas toutes les occurrences vérifiées devront avoir une collision précise). masques) et si la vérification doit inclure l’instance appelante ou non. Vous fournissez également une liste DS qui sera renseignée avec l'unique id les valeurs de toutes les occurrences de l'objet considérées comme étant en conflit avec l'instance appelante et vous avez la possibilité d'ordonner la liste en fonction de la distance entre l'origine de la zone circulaire et l'origine des occurrences trouvées en conflit. Notez que la fonction accepte également le mot clé spécial. all Dans ce cas, toutes les instances trouvées en conflit seront répertoriées. La fonction renvoie le nombre d'instances trouvées, ou 0 si aucune n'est trouvée.


Syntaxe:

collision_circle_list(x1, y1, rad, obj, prec, notme, list, ordered);

Argument La description
x1 La coordonnée x du centre du cercle à vérifier.
y1 La coordonnée y du centre du cercle à vérifier.
rad Le rayon (distance en pixels de son centre à son bord).
obj L'objet à vérifier pour les collisions d'instance.
prec Si le contrôle est basé sur des collisions précises ( true, qui est plus lent) ou son cadre de sélection en général ( false, plus rapide).
notme Si l'instance appelante, le cas échéant, doit être exclue ( true ) ou pas ( false ).
list La liste DS à utiliser pour stocker les ID des instances en conflit.
ordered Si la liste doit être classée par distance ( true ) ou pas ( false ).


Résultats:

Int (The number of instances found to be in collision)


Exemple:

var _list = ds_list_create();
var _num = collision_circle_list(x, y, 100, obj_Enemy, false, true, _list, false);
if _num > 0
    {
    for (var i = 0; i < _num; ++i;)
        {
        instance_destroy(_list[| i]);
        }
    }
ds_list_destroy(_list);

Le code ci-dessus vérifie les collisions avec des instances de "obj_Enemy" dans une zone circulaire d'un rayon de 100 pixels autour de la position de l'instance appelante. S'il y a des collisions, la liste pré-créée est bouclée et chaque instance qui était dans la collision est détruite.
