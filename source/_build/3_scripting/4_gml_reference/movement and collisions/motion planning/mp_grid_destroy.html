mp_grid_destroy

Syntaxe:

mp_grid_destroy(id);


Argument La description
id Index du mp_grid à détruire


Retours: N / A.


La description

Vous pouvez utiliser cette fonction pour détruire le mp_grid indiqué et libérer la mémoire qui lui est allouée. Il est essentiel d'appeler cette opération chaque fois que mp_grid est terminé, sinon vous risquez une fuite de mémoire, ce qui signifie que votre jeu ralentira avec le temps et finira par planter.

REMARQUE: Utilisation de mp_grid_* les fonctions sur une grille après sa destruction entraîneront une erreur!


Exemple:

if timer = 0
   {
   mp_grid_destroy(grid);
   room_goto(rm_Menu);
   }

Le code ci-dessus détruira le mp_grid indexé dans la variable "grille" si la variable "timer" est égal à 0, puis passera à une autre pièce.
