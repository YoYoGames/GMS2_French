mp_grid_to_ds_grid

Syntaxe:

mp_grid_to_ds_grid(source, destination);


Argument La description
source Index du mp_grid à utiliser
destination Index du ds_grid à utiliser pour copier les données de la grille.


Renvoie: Booléen.


La description

Avec cette fonction, vous pouvez copier la grille MP donnée dans une grille DS. La grille DS doit avoir la même taille que la grille MP, bien qu'elle ne doive pas l'être (les données seront perdues si elles sont plus petites, et si elles sont plus grandes, toutes les cellules supplémentaires seront égales à 0). Les cellules de la grille DS contiendront la valeur -1 si la cellule de la grille MP a été marquée comme occupée, ou 0 si ce n'est pas le cas.


Exemple:

motion_grid = ds_grid_create(room_width / 32, room_height / 32); mp_grid_to_ds_grid(mp_grid, motion_grid);

Le code ci-dessus va créer une nouvelle grille DS, puis copier les données de la grille MP contenues dans la variable "mp_grid" dans la nouvelle grille DS.
