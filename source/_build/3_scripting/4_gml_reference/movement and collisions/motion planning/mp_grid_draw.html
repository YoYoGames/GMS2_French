mp_grid_draw

Syntaxe:

mp_grid_draw(id);


Argument La description
id Index du mp_grid à dessiner


Renvoie: Booléen.


La description

Cette fonction dessine le mp_grid spécifié, en marquant les cellules libres comme des cellules vertes et interdites en rouge. Cette fonction est essentielle en tant qu'outil de débogage, mais il convient de noter qu'elle est très lente et ne fonctionne que lorsqu'elle est utilisée dans l'événement draw de l'instance.


Exemple:

draw_set_alpha(0.3);
mp_grid_draw(grid);
draw_set_alpha(1);

Le code ci-dessus dessine le mp_grid indexé dans la variable "grid" comme une superposition semi-transparente (mais seulement si l'instance exécutant le code a une profondeur inférieure à tout le reste).
