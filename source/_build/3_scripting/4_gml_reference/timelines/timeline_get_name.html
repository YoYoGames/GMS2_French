timeline_get_name

Obtient le nom d'une ligne de temps donnée.

Syntaxe:

timeline_get_name( ind );


Argument La description
ind L'index de la ligne de temps pour vérifier le nom de.


Retourne: Chaîne


La description

Cette fonction peut être utilisée pour obtenir le nom d'une ligne de temps sous forme de chaîne. si la ligne de temps a été créée dynamiquement en utilisant le timeline_add Dans la fonction, le nom retourné aura le format "__newtimeline N " où " N " est le numéro de la ligne de temps (à partir de 0). S'il vous plaît noter que ce n'est qu'une chaîne et ne peut pas être utilisé pour référencer la chronologie directement - pour cela, vous auriez besoin de l' index chronologie. Vous pouvez toutefois utiliser cette chaîne pour obtenir l' index de la chronologie en utilisant la chaîne renvoyée avec la fonction asset_get_index().


Exemple:

str = timeline_get_name(global.tl);

Le code ci-dessus va stocker le nom de la ligne de temps indexée dans la variable "global.tl".
