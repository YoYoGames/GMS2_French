timeline_exists

Vérifie si une ligne de temps existe.

Syntaxe:

timeline_exists(ind);


Argument La description
ind L'index de la ligne de temps à vérifier.


Renvoie: Boolean


La description

Avec cette fonction, vous pouvez vérifier et voir si une ligne de temps existe (retourne true ) ou non (renvoie false ). Ceci est particulièrement utile lors de la création dynamique de timelines en utilisant timeline_add fonction, mais vous devriez noter que si vous recherchez une variable non initialisée (qui serait sinon affectée à l'index d'une ligne de temps), une erreur sera levée.


Exemple:

if timeline_exists(global.tl) timeline_delete(global.tl);

Le code ci-dessus vérifie si une ligne de temps existe et est indexée dans la variable globale "tl" et si c'est le cas, il supprime alors cette ligne de temps.
