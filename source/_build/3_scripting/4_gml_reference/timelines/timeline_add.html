timeline_add

Ajoute une nouvelle ligne de temps.

Syntaxe:

timeline_add();


Retours: Real


La description

Avec cette fonction, vous pouvez ajouter une nouvelle ligne de temps (vide) dans votre jeu. la fonction renvoie l'index de cette nouvelle ligne de temps qui doit être stockée dans une variable pour être utilisée dans tous les autres appels de fonction qui impliquent cette nouvelle ligne de temps. Vous devriez également être sûr d'utiliser la fonction timeline_delete chaque fois que vous ne souhaitez plus utiliser la ligne de temps (comme à la fin de la pièce) afin d'éviter toute fuite de mémoire qui pourrait ralentir ou bloquer votre jeu.

IMPORTANT! Cette fonction n'est pas disponible avec la licence d'évaluation du produit.


Exemple:

global.tl = timeline_add();

Le code ci-dessus crée une nouvelle ligne de temps et stocke son index dans la variable "global.tl".
