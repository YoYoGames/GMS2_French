timeline_delete

Supprime une ligne de temps donnée.

Syntaxe:

timeline_delete( ind );


Argument La description
ind L'index de la ligne de temps à supprimer.


Retours: N / A


La description

Avec cette fonction, vous pouvez supprimer n'importe quelle ligne de temps de votre partie. Si cette ligne de temps a été créée dans l' GameMaker Studio 2 ressources de GameMaker Studio 2, notez qu'elle est complètement supprimée du jeu et que vous essayez d'appeler de nouveau la ligne de temps indexée après avoir utilisé cette fonction provoquera une erreur. Si la ligne de temps a été créée dynamiquement en utilisant timeline_add fonction, alors cette fonction devrait être utilisée au moment où la ligne de temps n'est plus nécessaire pour éviter toute fuite de mémoire qui pourrait ralentir (et finalement planter) votre jeu.

IMPORTANT! Cette fonction n'est pas disponible avec la licence d'évaluation du produit.


Exemple:

if timeline_exists(global.tl) timeline_delete(global.tl);

Le code ci-dessus vérifie si une ligne de temps existe et est indexée dans la variable globale "tl" et si c'est le cas, il supprime alors cette ligne de temps.
