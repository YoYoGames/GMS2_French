draw_enable_swf_aa

Activez ou désactivez l'anticrénelage pour les images vectorielles au format SWF.

Syntaxe:

draw_enable_swf_aa(enable);


Argument La description
enable Activer ( true ) ou désactiver ( false AA pour tous les sprites SWF.


Retours: N / A


La description

Avec cette fonction, vous pouvez activer ou désactiver l'anti-aliasing (AA) pour les sprites vectoriels au format SWF. AA lisse simplement les bords des images vectorielles pour leur donner un aspect plus agréable. La quantité d'AA utilisée dépendra de la valeur définie en utilisant la fonction draw_set_swf_aa_level(). Par défaut, c'est désactivé.

IMPORTANT! Cette fonction n'est pas disponible avec la licence d'évaluation du produit.



Exemple:

if draw_get_swf_aa_level() == 0
   {
   draw_enable_swf_aa(true);
   draw_set_swf_aa_level(0.5);
   }

Le code ci-dessus va vérifier la valeur AA pour les sprites au format SWF, et si elle est 0, elle active AA et définit la valeur à 0.5.


Retour: Dessin de sprites et de carreaux
