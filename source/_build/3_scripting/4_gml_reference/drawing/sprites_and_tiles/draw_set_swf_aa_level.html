draw_set_swf_aa_level

Définissez le niveau d'anti-aliasing pour les images vectorielles au format SWF.

Syntaxe:

draw_set_swf_aa_level(AA);


Argument La description
AA La valeur anti-aliasing à utiliser de 0 à 1.


Retours: N / A


La description

Cette fonction peut être utilisée pour définir le niveau d'anti-aliasing (AA) pour les sprites vectoriels au format SWF. Cela peut être une valeur réelle de 0 à 1 et "lisse" les bords de ces sprites. Notez que pour voir cet effet, vous devez d'abord avoir activé AA en utilisant la fonction draw_enable_swf_aa().

IMPORTANT! Cette fonction n'est pas disponible avec la licence d'évaluation du produit.



Exemple:

if draw_get_swf_aa_level() == 0
   {
   draw_enable_swf_aa(true);
   draw_set_swf_aa_level(0.5);
   }

Le code ci-dessus va vérifier la valeur AA pour les sprites au format SWF, et si elle est 0, elle active AA et définit la valeur à 0.5.
