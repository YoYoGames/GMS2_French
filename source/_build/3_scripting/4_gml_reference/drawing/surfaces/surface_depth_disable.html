surface_depth_disable


Syntaxe:

surface_depth_disable(disable);

Argument La description
disable Si réglé à true le tampon de profondeur sera désactivé pour les surfaces.


Résultats:

N/A


La description

Cette fonction désactive la génération automatique de tampon de profondeur pour toutes les surfaces du jeu. Normalement, toutes les surfaces ont des tampons de profondeur. Par conséquent, si vous dessinez des objets 3D, elles seront triées correctement par profondeur. Toutefois, l'allocation de tampons de profondeur double essentiellement la taille des surfaces, ce qui peut constituer un temps système excessif et inutile, surtout si votre jeu est très mémoire. intensif.

REMARQUE: Cette fonction n'est actuellement valide que pour la plate-forme cible Sony Playstation 4.


Exemple:

if surface_get_depth_disable() == false
    {
    surface_depth_disable(true);
    }

Le code ci-dessus vérifiera l'état actuel du tampon de profondeur de surface et, s'il est activé, le désactivera.
