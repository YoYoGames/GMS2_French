application_surface_draw_enable


Syntaxe:

application_surface_draw_enable(flag);

Argument La description
flag Le drapeau sera soit true (activé, la valeur par défaut) ou false (désactivée).


Résultats:

N/A


La description

Vous pouvez utiliser cette fonction pour activer ou désactiver le dessin automatique de la surface de l'application. Par défaut, cela est activé, mais dans de nombreux cas, vous voudrez prendre en charge quand et comment la surface est dessinée (en utilisant des shaders par exemple), auquel cas vous utiliserez cette fonction pour la définir sur false afin de pouvoir la dessiner vous-même quand et comment vous avez besoin.


Exemple:

application_surface_draw_enable(false);

Le code ci-dessus va désactiver le dessin automatique de la surface de l'application.
