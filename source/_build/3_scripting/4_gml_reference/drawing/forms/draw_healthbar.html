draw_healthbar

Dessine une barre de santé personnalisée.

Syntaxe:

draw_healthbar(x1, y1, x2, y2, amount, backcol, mincol, maxcol, direction, showback, showborder);


Argument La description
x1 La coordonnée x de la gauche de la barre de santé
y1 La coordonnée y du haut de la barre de santé
x2 La coordonnée x de la droite de la barre de santé
y2 La coordonnée y du bas de la barre de santé
amount La variable qui définit la santé totale (entre 0 et 100)
backcol La couleur du fond pour la barre
mincol La couleur de la barre quand à pas de santé (0)
maxcol La couleur de la barre en pleine santé (100)
direction Où la barre est "ancrée" (0 = gauche, 1 = droite, 2 = en haut, 3 = en bas)
showback Si l'arrière-plan de la barre est dessiné ( true ) ou pas ( false ). Si false, l'argument de backcol n'est pas pris en compte.
showborder Si les éléments de la barre ont une bordure noire large de 1 px ( true ) ou pas ( false ).


Retours: N / A


La description

Avec cette fonction, vous pouvez dessiner une barre de couleur pour afficher une valeur constante. Bien que la fonction utilise le mot "healthbar", vous pouvez l'utiliser pour afficher tout ce que vous souhaitez, à condition que le montant à afficher soit un pourcentage compris entre 0 et 100 (un pourcentage de n'importe quelle valeur peut être calculé à l'aide de la formule (CURRENT_Value / MAXIMUM_value ) * 100 ), vous pouvez par exemple utiliser ceci pour afficher la puissance, la santé, le mana, l'heure ou tout ce qui pourrait bénéficier d'un affichage à barres.

Vous pouvez définir différentes choses qui modifient l'aspect visuel de la barre. Celles-ci sont illustrées dans l'image ci-dessous (notez que les couleurs utilisées ici sont les suivantes). c_green pour la valeur maximale et c_red pour la valeur minimale, mais vous pouvez utiliser les couleurs que vous souhaitez):

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes: sommet, texture, couleur. Cependant, lors de l'utilisation de cette fonction, seules les données de sommet et de couleur sont transmises et le shader ne peut donc rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence.



Exemple:

var pc;
pc = (hp / max_hp) * 100;
draw_healthbar(100, 100, 500, 200, pc, c_black, c_red, c_lime, 0, true, true)

Le code ci-dessus utilise la valeur en pourcentage que vous obtenez des variables "hp" et "hp_max" pour dessiner une barre de santé rouge / verte standard.
