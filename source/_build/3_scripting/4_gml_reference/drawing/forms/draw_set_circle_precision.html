draw_set_circle_precision

Définit la précision pour dessiner un cercle ou une ellipse.

Syntaxe:

draw_set_circle_precision(precision);


Argument La description
precision La précision de tous les cercles dessinés après cette fonction est appelée.


Retours: N / A


La description

Pour optimiser les performances lors de la création de cercles, GameMaker Studio 2 dessine une forme de polygone avec suffisamment de côtés pour la faire apparaître circulaire. Toutefois, en fonction de la taille de votre cercle, vous pouvez vous rendre compte que le fait de modifier cette valeur peut augmenter les performances de votre jeu ou améliorer l'aspect des cercles. La valeur de précision que vous pouvez entrer dans cette fonction est le nombre de côtés du polygone de cercle. Ce nombre peut être compris entre 4 et 64, mais doit être un nombre divisible par 4, avec une valeur par défaut de 24.

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes: sommet, texture, couleur. Cependant, lors de l'utilisation de cette fonction, seules les données de sommet et de couleur sont transmises et le shader ne peut donc rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence.



Exemple:

draw_set_circle_precision(12);

Le code ci-dessus dessine tous les cercles en utilisant un polygone à 12 côtés.
