draw_set_circle_precision

Définit la précision pour dessiner un cercle ou une ellipse.

Syntaxe:

draw_set_circle_precision(precision);


Argument La description
precision La précision de tous les cercles dessinés après cette fonction est appelée.


Retours: N / A


La description

Pour optimiser les performances lorsque vous GameMaker cercles, GameMaker Studio 2 dessine essentiellement une forme de polygone avec suffisamment de côtés pour la faire apparaître circulaire. Cependant, en fonction de la taille de vos cercles, vous pouvez constater que modifier cette valeur peut améliorer les performances de votre jeu ou améliorer l'apparence des cercles. La valeur de précision que vous pouvez entrer dans cette fonction est le nombre de côtés du polygone de cercle, et ce nombre peut être compris entre 4 et 64, mais doit être un nombre divisible par 4, avec une valeur par défaut de 24.


Exemple:

draw_set_circle_precision(12);

Le code ci-dessus va dessiner tous les cercles en utilisant un polygone à 12 côtés.
