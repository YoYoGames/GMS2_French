draw_circle

Dessine un cercle.

Syntaxe:

draw_circle(x, y, r, outline);


Argument La description
x La coordonnée x du centre du cercle.
y La coordonnée y du centre du cercle.
r Le rayon du cercle (longueur du centre au bord)
outline Si le cercle est dessiné rempli ( false ) ou sous la forme d’un contour large d’un pixel ( true ).


Retours: N / A


La description

Avec cette fonction, vous pouvez dessiner un contour de cercle ou un cercle rempli. Vous pouvez définir la précision du dessin avec la fonction draw_set_circle_precision.

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes: sommet, texture, couleur. Cependant, lors de l'utilisation de cette fonction, seules les données de sommet et de couleur sont transmises et le shader ne peut donc rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence.



Exemple:

draw_set_colour(c_white);
draw_circle(100, 100, 50, true);

Cela tracera un contour de cercle blanc large d'un pixel avec un rayon de 50 pixels.
