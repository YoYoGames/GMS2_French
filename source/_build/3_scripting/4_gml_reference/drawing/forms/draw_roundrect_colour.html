draw_roundrect_colour

Dessine un rectangle avec des coins arrondis de rayon fixe, en le coloriant avec un dégradé de deux couleurs.

Syntaxe:

draw_roundrect_colour(x1, y1, x2, y2, col1, col2, outline);


Argument La description
x1 La coordonnée x de la gauche du rectangle arrondi.
y1 La coordonnée y du sommet du rectangle arrondi.
x2 La coordonnée x de la droite du rectangle arrondi.
y2 La coordonnée y du bas du rectangle arrondi.
col1 La couleur centrale.
col2 La couleur du bord extérieur.
outline Si le rectangle est un contour (vrai) ou rempli (faux).


Retours: N / A


La description

Avec cette fonction, vous pouvez tracer le contour d’un rectangle arrondi ou d’un rectangle arrondi rempli, la position (x1, y1) étant le coin supérieur gauche et la position (x2, y2) le coin inférieur droit. Si elle est remplie, vous pouvez définir les couleurs individuelles pour le centre et les bords. Si ces couleurs ne sont pas identiques, vous obtiendrez un effet de dégradé de l'un à l'autre (les paramètres de couleur remplaceront le jeu de couleurs de base avec la fonction draw_set_colour ).Vous pouvez définir la précision du dessin des coins avec la fonction draw_set_circle_precision, mais les angles sont toujours dessinés avec un rayon fixe. Si vous avez besoin de changer le rayon, vous devez utiliser la fonction draw_roundrect_colour_ext().

REMARQUE: Si vous souhaitez dessiner une forme à l'aide d'un shader, sachez que la plupart des shaders attendent les entrées suivantes: sommet, texture, couleur. Cependant, lors de l'utilisation de cette fonction, seules les données de sommet et de couleur sont transmises et le shader ne peut donc rien dessiner (ou dessiner quelque chose mais pas correctement). Si vous avez besoin de dessiner des formes de cette manière, le shader doit être personnalisé en conséquence.



Exemple:

draw_roundrect_colour(50, 50, 200, 200, c_black, c_white, false);

Cela dessinerait un carré rempli de coins arrondis et avec un dégradé lisse noir / blanc du centre vers les bords.
