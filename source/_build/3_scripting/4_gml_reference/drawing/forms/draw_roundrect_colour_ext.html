draw_roundrect_colour_ext

Dessine un rectangle avec des coins arrondis du rayon donné, en le coloriant avec un dégradé de deux couleurs.

Syntaxe:

draw_roundrect_colour_ext(x1, y1, x2, y2, xrad, yrad, col1, col2, outline);


Argument La description
x1 La coordonnée x de la gauche du rectangle arrondi.
y1 La coordonnée y du sommet du rectangle arrondi.
x2 La coordonnée x de la droite du rectangle arrondi.
y2 La coordonnée y du bas du rectangle arrondi.
xrad Le rayon de la courbe le long de l'axe x à partir des coins du rectangle.
yrad Le rayon de la courbe le long de l'axe y à partir des coins du rectangle.
col1 La couleur du centre.
col2 La couleur du bord extérieur.
outline Si le rectangle est un contour (vrai) ou rempli (faux).


Retours: N / A


La description

Avec cette fonction, vous pouvez dessiner soit un contour d'un rectangle arrondi ou un rectangle arrondi rempli où la position (x1, y1) est le coin supérieur gauche et la position (x2, y2) est le coin inférieur droit. Si le rectangle est rempli, les arguments de couleur seront utilisés pour générer un dégradé de couleurs du centre vers les bords, où la couleur 1 est la couleur du centre et la couleur 2 la couleur du bord. Vous devez également fournir des valeurs de rayon pour les axes x et y (en pixels) et les coins seront arrondis par ces montants. Vous pouvez définir la précision du dessin des coins avec la fonction draw_set_circle_precision.


Exemple:

var dist = point_distance(x, y, mouse_x, mouse_y) / 10;
var col = make_colour_hsv(clamp(dist, 0, 255), 255, 255);
draw_roundrect_colour_ext(x - 50, y - 50, x + 50, y + 50, dist, dist, col, c_white, 0);

Cela dessinerait un carré rempli avec des coins arrondis, dont le rayon est défini par la distance de l'instance faisant le dessin à la souris. Cette valeur est également utilisée pour calculer la couleur de fusion du centre.
