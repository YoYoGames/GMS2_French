draw_light_get


La description

Cette fonction obtiendra les paramètres de lumière spécifiés en tant que tableau avec les 6 éléments suivants -

  • [0] = activé / désactivé ( true / false )
  • [1] = type de lumière (voir les constantes ci-dessous)
  • [2] = position x
  • [3] = position y
  • [4] = position z
  • [5] = rayon de la lumière (uniquement pour les lumières ponctuelles, sera 0 pour la direction)
  • [6] = couleur claire (un vrai)

Le type de lumière peut être l'une des deux constantes suivantes:

Constant La description
lighttype_dir La lumière est une lumière directionnelle
lighttype_point La lumière est une lumière ponctuelle


Syntaxe:

draw_light_get(ind)


Arguments:

Argument La description
ind Le numéro d'indice de la lumière (de 0 à 7)


Résultats:

Real


Exemple:

light_a = draw_light_get(1);
if light_a[5] < 200
   {
   light_a[5] += 5;
   draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
   }

Le code ci-dessus obtiendra les valeurs utilisées pour définir la lumière indexée comme "1", puis vérifier le rayon et si elle est inférieure à 200, elle sera augmentée et le rayon de la lumière réglé à la nouvelle valeur.
