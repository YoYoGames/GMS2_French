draw_set_lighting


La description

Cette fonction est utilisée pour activer tous les effets d'éclairage. La valeur par défaut est désactivée ( false ).


Syntaxe:

draw_set_lighting(enable)


Arguments:

Argument La description
enable or disable all lighting ( true or false)


Résultats:

N/A


Exemple:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

Le code ci-dessus activera l'éclairage pour toute la scène, puis définira une lumière directionnelle blanche dans l'espace de la pièce, puis allumera la lumière.
