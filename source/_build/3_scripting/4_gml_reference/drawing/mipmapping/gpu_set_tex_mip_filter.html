gpu_set_tex_mip_filter


La description

Avec cette fonction, vous pouvez définir le mode de filtre MIP actuel sur l'un des trois types pris en charge. Vous indiquez la constante correspondant au mode de filtrage du MIP dont vous avez besoin, où vous pouvez choisir entre le filtrage par points (réglage par défaut, sans filtrage), le filtrage linéaire (également appelé filtrage trilinéaire ) ou le filtrage aniostropique (voir le tableau des constantes au dessous de).

Constant La description
tf_point Cela désactivera le filtrage entre les niveaux de mipmap, ce qui peut provoquer des transitions de texture visibles, mais donne les meilleures performances.
tf_linear Cela permettra un filtrage linéaire entre les niveaux de mipmap (également connu sous le nom de filtrage trilinéaire ), ce qui adoucit les transitions de texture, mais donnera un coup mineur aux performances.
tf_anisotropic Cela permettra un filtrage anisotrope entre les niveaux de mipmap, ce qui améliore grandement la qualité de la transition de texture et peut réduire le flou visible avec d'autres modes de filtrage, mais il a le plus grand impact sur les performances.


Syntaxe:

gpu_get_tex_mip_filter(filter);


Argument La description
filter Le mode de filtre de mip à utiliser (une constante, par défaut: tf_point ).


Résultats:

N/A


Exemple:

if keyboard_check(ck_enter)
    {
    switch(gpu_get_tex_mip_filter())
        {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
        }
    }

Le code ci-dessus vérifie le clavier et si la touche spécifiée est pressée, il obtient alors le filtre de mip actuel et fait passer la valeur à la suivante, passant par les différents modes.
