draw_set_alpha


La description

Avec cette fonction, vous pouvez définir l’alpha de base pour le jeu. Cette valeur peut être définie entre 0 et 1, 0 étant totalement transparent et 1, complètement opaque (valeur par défaut). Il affectera tous les dessins ultérieurs, y compris les arrière-plans, les images-objets, les polices de caractères, les primitives et la 3D.

Notez que si vous dessinez l’un des actifs et que vous fournissez un argument alpha dans le cadre de la fonction (par exemple, lors de l’utilisation de draw_sprite_ext() ), la valeur alpha définie ici sera remplacée, donc dessiner un sprite en utilisant draw_sprite_ext avec un argument alpha de 1 dessine l’image-objet totalement opaque, quel que soit l'alpha défini avec draw_set_alpha (cela fonctionnera de la même manière en utilisant draw_self() et mettre le image_alpha de l'instance). Notez également que lorsqu’une instance est un dessin par défaut (c’est-à-dire qu’elle a un sprite mais pas d’événement Draw), le image_alpha variable sera serré à l'ensemble alpha avec draw_set_alpha(), de sorte qu'un image_alpha de 1 et un tirage alpha de 0,5 attirera l’image-objet à alpha 0,5, tandis qu’un image_alpha de 0,25 et un tirage alpha de 0,5 attireront le sprite à 0,25 alpha.


Syntaxe:

draw_set_alpha(alpha);


Argument La description
alpha L'alpha à définir (entre 0 et 1)


Résultats:

N/A


Exemple:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

Le code ci-dessus dessine du texte à la position spécifiée avec un effet d’ombre créé par draw alpha modifié et par couleur.
