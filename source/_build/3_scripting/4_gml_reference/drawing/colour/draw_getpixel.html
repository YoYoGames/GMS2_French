draw_getpixel


La description

Avec cette fonction, vous pouvez obtenir la valeur de couleur de tout pixel dessiné vers la cible de rendu actuelle. Cela signifie que les résultats dépendront de l'événement dans lequel la fonction est appelée, ainsi que de la surface cible utilisée. Notez que cela ne renverra aucune valeur alpha, pour cela vous devriez utiliser draw_getpixel_ext.

NOTE: Cette fonction est incroyablement lente et ne devrait donc être utilisée qu'en cas d'absolue nécessité.


Syntaxe:

draw_getpixel(x, y);


Argument La description
x La coordonnée x du pixel à vérifier
y La coordonnée y du pixel à vérifier


Résultats:

Real


Exemple:

draw_set_colour(draw_getpixel(500,200));

Le code ci-dessus définira la couleur de dessin de tous les textes, formulaires et primitives sur la couleur du pixel situé à (500 200) sur l'écran de jeu.
