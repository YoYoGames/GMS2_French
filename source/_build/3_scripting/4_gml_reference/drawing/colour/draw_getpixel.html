draw_getpixel

Renvoie la couleur du pixel affiché à une coordonnée donnée.

Syntaxe:

draw_getpixel(x, y);


Argument La description
x La coordonnée x du pixel à vérifier
y La coordonnée y du pixel à vérifier


Retours: Real


La description

Avec cette fonction, vous pouvez obtenir la valeur de couleur de tout pixel qui est dessiné sur la cible de rendu en cours. Cela signifie que les résultats dépendront de l'événement dans lequel la fonction est appelée, ainsi que de la surface cible utilisée. Notez que cela ne renverra aucune valeur alpha, pour cela vous devriez utiliser draw_getpixel_ext.

NOTE: Cette fonction est incroyablement lente et ne devrait donc être utilisée que lorsque c'est absolument nécessaire.


Exemple:

draw_set_colour(draw_getpixel(500,200))

Le code ci-dessus définira la couleur de dessin pour tous les textes, les formes et les primitives à la couleur du pixel à (500, 200) sur l'écran de jeu.
