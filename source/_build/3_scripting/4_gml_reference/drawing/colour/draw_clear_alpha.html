draw_clear_alpha


La description

Cette fonction peut être utilisée pour effacer la totalité de l’écran avec une couleur donnée et le composant alpha de la destination est défini sur la valeur que vous avez définie - cette fonction ne crée pas de fusion car elle fonctionne, mais vous pouvez configurer toutes les opérations de fusion suivantes. utilisez l'alpha de destination que vous avez défini. Ceci est uniquement utilisé dans l'événement draw d'une instance (il ne s'affichera pas s'il est utilisé dans un autre événement) et peut également être très utile pour effacer les surfaces lors de leur création.

Il est à noter que lorsque vous utilisez le module HTML5, vous pouvez utiliser cette fonction pour rendre l'arrière-plan de la toile transparent, ce qui permettra ensuite à toutes les images utilisées dans le code HTML hôte d'être vues par le jeu. Pour obtenir cet effet, vous devez d'abord accéder à l'éditeur de salle et supprimer tous les calques d'arrière-plan de l' éditeur de salle, puis désélectionner l'option Effacer le fond de la vue trouvée dans les propriétés de la vue. Enfin, vous devrez créer un nouvel objet, le placer dans la première salle du jeu et lui attribuer une profondeur très élevée (par exemple, 10000). Ensuite, dans l’événement Pré-dessin, utilisez cette fonction pour définir le canevas alpha sur la couleur et la transparence souhaitées.


Syntaxe:

draw_clear_alpha(col, alpha);


Argument La description
col La couleur avec laquelle l'écran sera effacé
alpha La transparence de la couleur avec laquelle l'écran sera effacé


Résultats:

N/A


Exemple:

draw_clear_alpha(c_red, 0.5);

Cela effacera l'écran avec une transparence de 50% (0,5) avec la couleur rouge.
