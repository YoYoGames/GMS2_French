draw_text_ext_transformed_colour

Dessine une chaîne avec les attributs de couleur, d'échelle, d'angle ou d'espacement que vous définissez.

Syntaxe:

draw_text_ext_transformed_colour(x, y, string, sep, w, xscale, yscale, angle, c1, c2, c3, c4, alpha);


Argument La description
x La coordonnée x de la chaîne dessinée.
y La coordonnée y de la chaîne dessinée.
string La chaîne à dessiner.
sep La distance en pixels entre les lignes de texte.
w La largeur maximale en pixels de la chaîne avant un saut de ligne.
xscale L'échelle horizontale
yscale L'échelle verticale
angle L'angle du texte
c1 La couleur en haut à gauche du texte dessiné.
c2 La couleur pour le coin supérieur droit du texte dessiné.
c3 La couleur pour le bas à droite du texte dessiné.
c4 La couleur pour le bas à gauche du texte dessiné.
alpha L'alpha pour le texte.


Retours: N / A


La description

Cette fonction est une combinaison de la base draw_text fonctionner avec le draw_text_ext, draw_text_transformed et draw_text_colour les fonctions. En tant que tel, c'est la plus générale de toutes les fonctions pour dessiner du texte et donne une grande flexibilité lors du dessin. Notez que l'argument "width" est basé sur une échelle de 1, donc si l'échelle est différente, cette valeur doit être modifiée proportionnellement. Par exemple, si la largeur de base pour un saut de ligne est 300 et que vous définissez l'échelle sur 2, le texte apparaîtra incorrect, dépassant la largeur donnée. Au lieu de cela, vous devriez avoir défini la largeur à 150 pour compenser la mise à l'échelle.

REMARQUE: La fusion de dégradés n'est pas disponible pour la cible HTML5, à moins que WebGL ne soit activé, bien que vous puissiez toujours définir les couleurs de fusion et fusionner la police avec la première couleur donnée. Cependant, tout mélange de cette manière crée une police dupliquée qui est ensuite stockée dans le cache et utilisée si nécessaire, ce qui est loin d'être optimal et si vous utilisez plusieurs changements de couleur, cela ralentira les performances de vos jeux. Vous pouvez définir la taille du cache de polices pour essayer de limiter cela si cela s'avère nécessaire font_set_cache_size.


Exemple:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text_ext_transformed_colour(200, 200, "My name is" + global.name +".", 20, 200, 0.5, 0.5, 180, c_blue, c_blue, c_navy, c_navy, 1);

Cela va dessiner le texte donné avec un gradient descendant bleu à bleu foncé, à la position (200,200) dans la pièce, tourné pour être à l'envers et à la moitié de la taille du point normal de la police. Il y aura 20 pixels entre chaque ligne s'il y a plus d'une ligne tracée et chaque ligne aura une largeur maximale de 200 pixels.
