draw_text

Dessinez une chaîne à une position donnée.

Syntaxe:

draw_text(x, y, string);


Argument La description
x La coordonnée x de la chaîne dessinée.
y La coordonnée y de la chaîne dessinée.
string La chaîne à dessiner.


Retours: N / A


La description

Avec cette fonction, vous pouvez dessiner n'importe quelle chaîne à n'importe quelle position dans la pièce (pour dessiner des nombres réels, vous devez utiliser le string fonction pour les convertir en texte). Pour combiner des chaînes, vous pouvez utiliser + (voir l'exemple ci-dessous) et vous pouvez également utiliser / n dans une chaîne pour ajouter un saut de ligne afin qu'il soit dessiné sur plusieurs lignes (pour savoir comment formater correctement une chaîne et quels caractères d'échappement vous pouvez utiliser, s'il vous plaît voir ici ). La couleur du texte et l'alpha sont régies par les valeurs alpha et de couleur de base actuelles définies par draw_set_alpha et draw_set_colour

NOTE: La position réelle du texte sera influencée par les valeurs d'alignement définies par draw_set_halign et draw_set_valign.


Exemple:

draw_text(x, y, "Hello, " + global.Name + "!/nI hope you are well!");

Le code ci-dessus dessine une chaîne à la position x / y de l'instance, qui utilisera la chaîne stockée dans la variable globale "Name" et la divisera sur deux lignes.


Retour: Texte de dessin
