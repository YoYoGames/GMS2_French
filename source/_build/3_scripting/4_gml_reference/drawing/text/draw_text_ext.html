draw_text_ext


La description

Cette fonction dessine le texte de la même manière que draw_text Ce n'est que maintenant que vous pouvez définir l'espace entre chaque ligne de texte - si le texte occupe plus d'une ligne - et limiter la largeur (en pixels) de la chaîne par ligne de sorte que toute ligne dépassant cette valeur dépasse GameMaker Studio 2, elle sera automatiquement divisée. le texte à la ligne suivante à l'espace blanc disponible le plus proche (si le texte n'a pas d'espaces blancs, il dépassera cette valeur de largeur maximale). Notez que tout espace blanc placé au début de la chaîne sera supprimé avant d'être analysé pour le dessin pour cette raison. Notez également qu'une valeur de -1 pour l'argument de séparation de ligne correspond par défaut à une séparation basée sur la hauteur du caractère "M" dans la police choisie.


Syntaxe:

draw_text_ext(x, y, string, sep, w);

Argument La description
x La coordonnée x de la chaîne dessinée.
y La coordonnée y de la chaîne dessinée.
string La ficelle à dessiner.
sep La distance en pixels entre les lignes de texte.
w La largeur maximale en pixels de la chaîne avant un saut de ligne.


Résultats:

N/A


Exemple:

draw_text_ext(100, 50, keyboard_string, 3, 300);

Le code ci-dessus dessine le texte saisi par l'utilisateur sur le clavier, en le divisant en de nouvelles lignes chaque fois que la longueur de la chaîne dépasse 300 pixels. le code maintiendra également une séparation de 3 pixels entre les lignes si cela se produit.
