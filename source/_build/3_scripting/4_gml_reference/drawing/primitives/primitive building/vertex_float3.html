vertex_float3

Ajouter des valeurs à virgule flottante à un sommet.

Syntaxe:

vertex_float3(buffer, float, float, float);


Argument La description
buffer Le tampon pour écrire l'information.
float La première valeur d'entrée.
float La deuxième valeur d'entrée.
float La troisième valeur d'entrée.


Retours: N / A


La description

Cette fonction ajoutera trois valeurs à virgule flottante aux données de sommet. Le vertex doit avoir été formaté correctement pour l'accepter en utilisant le vertex_format_add_custom fonction.


Exemple:

vertex_float3(buff, 0.05, 0.01, room_width / x);

Le code ci-dessus ajoute trois valeurs à virgule flottante aux données de sommet en cours de définition.
