draw_vertex_colour


Syntaxe:

draw_vertex_colour(x, y, col, alpha)


Argument La description
x La coordonnée x du sommet.
y La coordonnée y du sommet.
col La couleur pour dessiner ce sommet avec.
alpha L'alpha pour dessiner ce sommet avec (0-1).


Résultats:

N/A


La description

Cette fonction définit la position d'un sommet pour une primitive, avec sa propre couleur et son propre paramètre alpha. L'aspect final de la primitive dépendra du type primitif choisi pour dessiner et de l'ordre dans lequel vous y ajoutez les vertex (voir draw_primitive_begin pour plus d'informations) et les vertex avec des couleurs différentes et des alphas se fondent en douceur de l'un à l'autre. Pour terminer et dessiner le primitif, vous devez appeler draw_primitive_end.


Exemple:

draw_primitive_begin(pr_trianglelist);
draw_vertex_colour(100, 100, c_blue, 0.1);
draw_vertex_colour(100, 200, c_red, 0.1);
draw_vertex_colour(150, 150, c_green, 1);
draw_primitive_end();

Le code ci-dessus dessine un triangle semi-transparent avec chaque sommet coloré d'une couleur différente.
