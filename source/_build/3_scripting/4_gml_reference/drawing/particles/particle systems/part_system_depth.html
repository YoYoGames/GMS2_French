part_system_depth

Définit la profondeur du système de particules.

Syntaxe:

part_system_depth( ind, depth );


Argument La description
ind L'indice du système de particules à changer.
depth La profondeur à laquelle définir le système de particules.


Retours: N / A


La description

Avec cette fonction, vous pouvez définir la profondeur de tirage pour le système de particules, de la même manière que vous pouvez définir la profondeur de rendu de différentes couches dans la pièce, où une faible profondeur de tirage signifie qu'elle apparaîtra en haut de toutes les profondeur, et une profondeur de tirage élevée le plaçant en dessous de tout avec une profondeur de tirage inférieure.


Exemple:

global.Sname = part_system_create();
part_system_depth(global.Sname, -1000 );

Le code ci-dessus va créer un système de particules et stocker son index dans la variable globale "Sname". ce système reçoit alors une faible profondeur de -1000, ce qui signifie qu'il apparaîtra au-dessus de tout avec une profondeur de tirage plus élevée.
