part_system_draw_order

Définit l'ordre dans lequel les nouvelles particules sont dessinées.

Syntaxe:

part_system_draw_order(ind, oldtonew);


Argument La description
ind L'indice du système de particules à changer.
oldtonew Que les vieilles particules doivent être tirées derrière les plus récentes (vrai) ou vice versa (faux).


Retours: N / A


La description

Avec cette fonction, vous pouvez définir la manière dont les particules sont dessinées lorsqu'elles sont créées à l'écran. Le système par défaut utilise un ancien> nouveau look (la fonction est définie sur true ), où les vieilles particules sont dessinées à une profondeur plus élevée que les plus récentes et ainsi apparaissent "en dessous" de nouvelles particules, mais en réglant cette fonction sur false vous pouvez inverser cet ordre et faire en sorte que les nouvelles particules soient plus hautes et apparaissent ainsi "en dessous" des plus anciennes. Les images ci-dessous illustrent ceci, avec l'image sur la gauche étant la valeur par défaut de true et l'image sur le droit étant false:


REMARQUE: Lorsque les particules sont dessinées avec un mode de mélange additif, l'effet de cette fonction n'est pas toujours évident.


Exemple:

mysystem = part_system_create();
part_system_clear( mysystem, 1 );

Cela va créer un nouveau système de particules avec l'index mysystem. Il fixera ensuite ses particules pour attirer de nouvelles particules sur les plus anciennes.
