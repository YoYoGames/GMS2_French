part_system_drawit

Dessine les particules dans un système donné.

Syntaxe:

part_system_drawit(ind);


Argument La description
ind L'indice du système de particules pour dessiner les particules de.


Retours: N / A


La description

Avec cette fonction, vous pouvez forcer un système de particules à dessiner toutes les particules actuelles sur l'écran. Si part_system_automatic_draw est désactivée, cette fonction affiche les particules lorsqu'elles sont utilisées dans l'événement draw d'une instance, ou elle peut être utilisée lorsque la cible du dessin est définie sur une surface (en utilisant surface_set_target ) pour attirer les particules dans le système vers cette surface.


Exemple:

if surface_exists(surf)
   {
   surface_set_target(surf);
   part_system_drawit();
   surface_reset_target();
   }

Le code ci-dessus vérifie si la surface indexée dans la variable "surf" existe, et si c'est le cas, place la cible de dessin sur la surface, dessine les particules puis réinitialise la cible de dessin pour que tout dessin normal apparaisse une fois plus.
