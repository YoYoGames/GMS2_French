part_system_clear

Efface un système de particules à ses paramètres par défaut.

Syntaxe:

part_system_clear(ind);


Argument La description
ind L'indice du système de particules à effacer.


Retours: N / A


La description

Avec cette fonction, vous pouvez effacer le système indexé à son état par défaut, en supprimant tous les émetteurs et en réinitialisant la profondeur et la position (s'ils ont été modifiés). Faites attention en utilisant cette fonction comme si vous aviez un réglage d'instance ou un éclatement ou toute autre action utilisant un émetteur associé à ce système, vous obtiendrez une erreur à moins que vous n'utilisiez le part_emitter_exists fonction à vérifier. Il n'y a pas non plus besoin d'appeler le part_emitter_destroy fonctionne comme cela est pris en charge automatiquement aussi.

REMARQUE: cette fonction effacera les particules visibles dans la pièce, mais elle ne supprime pas les propriétés des particules et ne les supprime pas de la mémoire. Pour cela, vous devriez utiliser les fonctions part_type_clear et part_type_destroy.


Exemple:

part_system_clear(global.Sname);

Le code ci-dessus effacera le système de particules indexé dans la variable globale "Sname" à son état par défaut.
