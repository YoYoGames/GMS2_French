part_system_get_layer


La description

Cette fonction peut être utilisée pour récupérer la valeur d'ID de couche unique pour le système de particules donné. Vous fournissez la valeur ID du système de particules (telle que renvoyée par la fonction part_system_create_layer ) et la fonction renverra la valeur ID de la couche. Notez que si le système de particules a été marqué comme persistant, il peut avoir été placé sur une couche gérée (si la couche définie lorsque le système a été créé dans une pièce précédente n'existe pas), la valeur retournée sera 0.


Syntaxe:

part_system_get_layer(ind);

Argument La description
ind La valeur ID du système de particules pour obtenir l'ID de couche de


Résultats:

Real (unique layer ID value or 0 if the layer is an internally managed one)


Exemple:

if part_system_get_layer(global.p_sys) != "effects_layer"
    {
    part_system_layer(global.p_sys, "effects_layer";
    }

Le code ci-dessus va vérifier un système de particules pour voir sur quelle couche il est et s'il n'est pas sur la couche donnée, il sera changé.
