part_system_update

Met à jour les positions des particules dans un système donné.

Syntaxe:

part_system_update(ind);


Argument La description
ind L'index du système de particules à mettre à jour.


Retours: N / A


La description

Cette fonction fera avancer le système de particule donné d'un pas dans le temps de jeu et peut être utilisée pour simuler le comportement du système de particules lorsque la mise à jour automatique est désactivée ou peut être utilisée pour avancer un système de particules en une étape. Par exemple, si vous avez un système de particules pour la neige, et que vous voulez qu'il apparaisse que la neige tombe dès le moment où le joueur entre dans la pièce, vous pouvez avancer vos particules de neige pendant quelques secondes dans l'événement de création. une instance de sorte que la première fois qu'ils sont dessinés, il y a des particules partout sur l'écran (l'exemple de code ci-dessous le montre). Vous pouvez également utiliser cette fonction lorsque la fonction de mise à jour automatique du système est désactivée par la fonction part_system_automatic_update pour déplacer le système.


Exemple:

repeat (room_speed * 3)
   {
   part_system_update(global.RainSys);
   }

Le code ci-dessus avancera de 3 secondes le système de particules indexé dans la variable globale "RainSys".
