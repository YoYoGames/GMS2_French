part_type_scale

Définit la mise à l'échelle horizontale et verticale du type de particule.

Syntaxe:

part_type_scale(ind, xscale, yscale);


Argument La description
ind L'indice du type de particule à changer.
xscale L'échelle horizontale de la particule de base.
yscale L'échelle verticale de la particule de base.


Retours: N / A


La description

Cette fonction définit l'échelle horizontale et verticale de la particule avant que d'autres changements de taille ne soient effectués. Cette fonction fonctionne sur le rapport de la taille de l'image-objet, donc une valeur de 1 (la valeur par défaut) est un rapport de 1: 1, ce qui signifie que 0,5 serait moitié et deux fois plus.


Exemple:

part_type_shape(particle1, pt_shape_flare);
part_type_size(particle1, 2, 2, 0.10, 1);
part_type_scale(particle1, 8, 0.50);
part_type_colour3(particle1, 8388608, 4227327, 8454143);
part_type_alpha3(particle1, 0.10, 0.10, 0);
part_type_speed(particle1, 0, 1, 0.18, 20);
part_type_direction(particle1, 0, 359, 0, 0);
part_type_orientation(particle1, 0, 0, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 40);

Le code ci-dessus va définir diverses valeurs de particules, y compris l'échelle qui définit le xscale à huit fois la taille et l'échelle y à la moitié de la taille.
