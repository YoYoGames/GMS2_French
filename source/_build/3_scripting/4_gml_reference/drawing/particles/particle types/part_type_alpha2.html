part_type_alpha2

Définit un type de particule pour commencer à une valeur alpha et passer à un autre tout au long de sa durée de vie.

Syntaxe:

part_type_alpha2(ind, alpha1, alpha2);


Argument La description
ind L'indice du type de particule à changer.
alpha1 Le début alpha de la particule.
alpha2 L'alpha final de la particule.


Retours: N / A


La description

Cette fonction peut être utilisée pour définir un gradient de valeur alpha (transparence) pour chaque particule créée du type donné. Le premier alpha est celui avec lequel toutes les particules vont commencer, et le second alpha est celui avec lequel la particule se terminera, et un léger changement de gradient apparaîtra à l'alpha au cours de la vie des particules de l'une à l'autre. Cela peut aller de 0 (transparent) à 1 (opaque).


Exemple:

part_type_alpha2(global.Snow_Part, 0, 1);

Le code ci-dessus définira toutes les particules créées du type de particule indexées dans la variable globale "Snow_Part" pour avoir une valeur alpha de 0 (transparent) puis passeront à l'avant pour avoir un alpha de 1 (opaque) à la fin de leur vie.
