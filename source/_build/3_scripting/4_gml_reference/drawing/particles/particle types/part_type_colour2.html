part_type_colour2

Définit le type de particule à fusionner entre deux couleurs

Syntaxe:

part_type_colour2(ind, colour1, colour2);


Argument La description
ind L'indice du type de particule à changer.
colour1 La couleur à partir de laquelle la particule commencera.
colour2 La couleur à laquelle la particule finira.


Retours: N / A


La description

Cette fonction peut être utilisée pour définir un gradient de deux couleurs pour chaque particule créée du type donné. La première couleur est celle avec laquelle toutes les particules vont commencer, et la deuxième couleur est celle sur laquelle la particule se terminera, et un changement progressif du gradient apparaîtra à la couleur sur la durée de vie des particules d'une couleur à l'autre.


Exemple:

part_type_colour2( part_Health, c_red, c_white);

Le code ci-dessus fera en sorte que toutes les particules créées du type indexé dans la variable "part_Health" aient un mélange de couleurs du rouge au blanc au cours de leur durée de vie.
