partie_type_shape

Définit la forme des particules créées sur l'une des quatorze formes intégrées.

Syntaxe:

part_type_shape(ind, shape);


Argument La description
ind L'index du type de particule à définir.
shape La forme à choisir.


Retours: N / A


La description

Avec cette fonction, vous pouvez définir la forme du sprite à utiliser pour le type de particule, les constantes suivantes étant acceptables:

Constant La description
ind L'index du type de particule à définir.
pt_shape_pixel Un pixel 1x1. (Ce sont les paramètres par défauts.)
pt_shape_disk Un cercle rempli.
pt_shape_square Un carré rempli.
pt_shape_line Une ligne horizontale de 8 pixels.
pt_shape_star Une étoile remplie de cinq points.
pt_shape_circle Un cercle souligné 3px.
pt_shape_ring Un cercle avec une lueur vers l'intérieur (ressemble à une bulle).
pt_shape_sphere Un cercle avec une lueur vers l'extérieur 'solide au milieu, brillant vers l'extérieur.
pt_shape_flare Un point rougeoyante (ressemble à une véritable étoile dans la nuit).
pt_shape_spark Un effet d'étincelle 'comme une étoile avec de multiples points disparaissant.
pt_shape_explosion Un nuage de fumée carré exige plusieurs couleurs pour ressembler à une explosion.
pt_shape_cloud Un nuage mince nécessite une mise à l'échelle et de multiples particules pour ressembler à un nuage.
pt_shape_smoke Une version lisse de l'effet d'explosion. Utilisez plusieurs pour créer un nuage de fumée.
pt_shape_snow Une forme générique de flocon de neige.




Exemple:

global.p1 = part_type_create();
part_type_shape(global.p1, pt_shape_square);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Le code ci-dessus définit le type de particule indexé dans la variable globale "p1" à la forme d'un carré, puis définit diverses autres propriétés de particules.
