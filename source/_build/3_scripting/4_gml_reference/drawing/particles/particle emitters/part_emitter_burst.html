part_emitter_burst

Éclate des particules sur une seule étape à partir de l'émetteur donné.

Syntaxe:

part_emitter_burst(ps, ind, parttype, number);


Argument La description
ps Le système de particules dans lequel se trouve l'émetteur.
ind L'indice de l'émetteur à éclater.
parttype L'index (type) des particules à créer.
number Le nombre de particules à créer.


Retours: N / A


La description

Cette fonction vous permet de définir un émetteur pour éclater un type spécifique de particule et est généralement utilisé dans les alarmes et les détruire car il s'agit d'un code unique qui crée le nombre de particules spécifiées en une fois suivant la distribution, la forme et la position la fonction part_emitter_region. Si vous avez besoin que les particules apparaissent à chaque étape, vous devriez utiliser la fonction part_emitter_stream plutôt que d'appeler cette fonction à chaque étape.


Exemple:

part_emitter_burst(global. Sname, p_emit1, p1, 30 + irandom(30));

Le code ci-dessus va éclater un nombre aléatoire de particules entre 30 et 60 de l'émetteur choisi.
