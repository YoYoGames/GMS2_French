texture_set_stage


La description

Cette fonction définira la "position" de la scène donnée comme texture à utiliser. Le nombre de "slots" d'étape disponibles dépend de la plate-forme sur laquelle vous compilez, avec un maximum de 8 disponibles pour Windows, Mac et Linux, mais sur Android appareils Android gamme inférieure (par exemple), ce nombre peut être aussi bas que 2. Il est également intéressant de noter que la première étape "slot" est toujours utilisée automatiquement par GameMaker Studio 2.

NOTE: Cette fonction ne fera rien en dehors du contexte d'un shader en cours d'exécution! Voir Shaders pour plus d'informations.


Syntaxe:

texture_set_stage(stage, tex);

Argument La description
stage La texture "slot" à utiliser.
tex La texture à utiliser.


Résultats:

N/A


Exemple:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

Le code ci-dessus obtiendra la poignée de l'échantillonneur dans le shader, indexé comme "GlassShader", puis définira cette constante de shader sur la texture de l'image-objet donnée.
