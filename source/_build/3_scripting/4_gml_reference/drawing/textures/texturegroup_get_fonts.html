texturegroup_get_fonts


La description

Avec cette fonction, vous pouvez récupérer l'index de police de chacune des polices affectées aux pages de texture dans le groupe de texture donné. Vous fournissez la chaîne d'ID de groupe de textures (telle que définie dans l'éditeur de groupes de textures) et la fonction renvoie un tableau 1D dans lequel chaque entrée contient l'index de police pour une ressource de police. Si la fonction échoue - c'est-à-dire qu'un groupe non valide est fourni ou qu'aucune texture ne lui est affectée - le tableau sera vide (longueur 0).


Syntaxe:

texturegroup_get_fonts(tex_id);

Argument La description
tex Le nom du groupe de texture à vérifier (une chaîne)


Résultats:

1D array


Exemple:

var _tex_array = texturegroup_get_fonts( "MainMenu");
for (var i = 0; i < array_length_1d(_tex_array); ++i;)
    {
    show_debug_message("Font " + string(i) + " Index:" + string(tex_array[i]));
    }

Le code ci-dessus récupère les index de police pour le groupe de textures "MainMenu", puis affiche ces identifiants dans la fenêtre de sortie de la console.
