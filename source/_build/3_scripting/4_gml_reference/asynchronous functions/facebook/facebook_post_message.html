facebook_post_message


Syntaxe:

facebook_post_message(title, caption, message, picture_url, redirect_url, actionname, action_url)

Argument La description
title Le "nom" du lien.
caption La légende du lien (apparaît sous le nom du lien).
message Le message à inclure avec le post.
picture_url Un lien vers la photo incluse avec ce post.
redirect_url Le lien attaché à ce post.
actionname Une action à appliquer pour ce poste.
action_url Le lien à utiliser pour l'action ci-dessus.


Résultats:

N/A


La description

Cette fonction publiera un message directement dans le flux utilisateur (par exemple, un message «/ me / feed» dans l’API de graphe de Facebook). Vous donnez le nom et la légende ainsi qu'un message et une image à inclure, puis l'URL vers laquelle vous souhaitez créer un lien. Les deux derniers arguments, "actionname" et "actionurl" sont facultatifs et peuvent être laissés sous forme de chaînes vides ("") ou peuvent être définis sur l'une des actions Facebook autorisées, comme commenter ou aimer, auquel cas vous devriez également fournissez une URL appropriée (pour plus d'informations, consultez Api Post de Facebook Developers ).

REMARQUE: Aucun dialogue Facebook n'est affiché pour cette fonction, car le message est envoyé «de manière transparente» sans intervention de l'utilisateur. Pour cette raison, vous devez avoir demandé les autorisations appropriées pour la connexion ("publish_actions").

REMARQUE: L'utilisateur doit être connecté pour que le message puisse fonctionner.


Exemple:

if mouse_check_button_pressed(mb_left)
   {
   facebook_post_message("Catch The Haggis", "MacSweeneygames.com", "I just got a hi-score playing Catch The Haggis!", "http://MacSweeneygames.com/Clown.jpg", "http://MacSweeneygames.com/CatchTheHaggis.html", "", "");
   }

Le code ci-dessus publiera silencieusement un message dans le flux mural des utilisateurs.
