facebook_accesstoken


Syntaxe:

facebook_accesstoken()


Résultats:

String


La description

Un jeton d'accès est une chaîne aléatoire qui fournit un accès temporaire et sécurisé aux API Facebook, et cette fonction renvoie le jeton d'accès que Facebook vous donne lorsque vous vous connectez pour vérifier qu'il s'agit d'une connexion valide.


Exemple:

token = facebook_accesstoken();

Le code ci-dessus obtient le jeton d'accès que Facebook génère pour l'utilisateur et le stocke comme une chaîne dans la variable "token".
