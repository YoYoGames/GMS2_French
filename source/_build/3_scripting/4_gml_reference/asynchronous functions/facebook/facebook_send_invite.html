facebook_send_invite


Syntaxe:

facebook_send_invite(titlestring, msg, picture_url, redirect_url, ds_map_response);

Argument La description
titlestring Le titre du dialogue (une chaîne)
msg Le message qui apparaît dans la boîte de dialogue (une chaîne).
picture_url l'URL de l'image à inclure dans la boîte de dialogue.
redirect_url Le «SiteURL» de votre application sur la page de paramètres de l'application Développeurs de Facebook.
ds_map_response Ds_map pré-créé utilisé pour stocker les détails de la réponse.


Résultats:

N/A


La description

Avec cette fonction, vous pouvez faire en sorte que votre jeu affiche une boîte de dialogue "Demande" de facebook, qui permettra à votre joueur de sélectionner des amis dans une liste et de leur envoyer un message d'invitation à la partie. Cette fonction nécessite que vous fournissiez un fichier déjà créé. ds_map qui sera ensuite rempli avec les détails de l'utilisateur pertinents. Cette carte doit être interrogée à chaque étape après avoir appelé cette fonction, car la réponse peut arriver à tout moment. À ce moment-là, la carte devrait avoir une clé " to " qui renverra une ds_list contenant tous les ID utilisateur Facebook des personnes invitées. (voir l'exemple ci-dessous pour plus de détails).

REMARQUE: "SiteURL" est ignoré sur Android modules iOS et Android et n'est requis sur le module HTML5 que si vous n'hébergez pas l'application sur "SiteURL" (c'est-à-dire que vous testez localement).

REMARQUE: L'utilisateur n'a pas besoin d'être connecté pour utiliser cette fonction car la boîte de dialogue de connexion apparaîtra dans le cadre de la demande d'invitation.


Exemple étendu:

Vous pouvez envoyer une invitation d'amis de la manière suivante:

if mouse_check_button_pressed(mb_left)
   {
   map = ds_map_create();
   facebook_send_invite("Catch The Haggis!", "Come and play Catch The Haggis with me...", "http://MacSweeneygames.com/Clown.jpg", "", map);
   }

Le code ci-dessus affichera une boîte de dialogue "demande" dans laquelle vous pourrez sélectionner des amis à inviter à jouer. ds_map contenir les identifiants Facebook des utilisateurs invités. Ceci doit ensuite être interrogé dans l'événement Step d'une instance pour recevoir les résultats, comme ceci:

if ds_map_exists(map, "to")
   {
   var list = ds_map_find_value(map, "to");
   var size = ds_list_size(list);
   for(var i = 0; i < size; i++;)
      {
      fb_uid[i] = ds_list_find_value(list, i);
      }
   ds_map_clear(map);
   }

Ce code vérifie la clé "to" dans ds_map et lorsqu'il est ajouté, il accède ensuite à la liste ds que "to" contient et la parcourt en boucle, définissant le tableau "fb_uid" pour contenir les ID utilisateur qui ont été renvoyés.
