réalisation_charge_leader


Syntaxe:

achievement_load_leaderboard(ident, minindex, maxindex, filter)

Argument La description
ident L'ID unique du classement affiché dans le tableau de bord du développeur.
minindex La valeur d'index de départ pour obtenir les données du classement.
maxindex La valeur d'index maximale pour obtenir les données du classement.
filter Que ce soit pour filtrer les résultats sur ceux de votre liste d'amis ou sur tous les joueurs (voir la description ci-dessous)


Résultats:

N/A


La description

Cette fonction enverra une demande au serveur pour obtenir des informations sur le classement donné. Le classement doit avoir été créé précédemment sur le tableau de bord du développeur pour votre jeu et l'ID donné est celui que vous devez fournir comme premier argument de la fonction. Il déclenchera un événement social asynchrone de rappel qui contient le async_load carte remplie avec les paires clé / valeur appropriées. La clé d' identification de cette ds_map est utilisé pour identifier le rappel correct (il peut exister plus d'une fonction de déclenchement pour un événement asynchrone donné) et sera associé à la constante achievement_leaderboard_info ainsi qu'un certain nombre d'autres paires clé / valeur pour chaque joueur. Le contenu exact de la carte est indiqué ci-dessous:

  • " id " - Pour cette fonction, il devrait être achievement_leaderboard_info

  • " leaderboardid " - Identifiant unique du classement défini dans le tableau de bord du fournisseur.

  • " numentries " - Le nombre d'entrées que vous avez reçues dans le classement.

  • " PlayerN " - Le nom du joueur, où "N" est une valeur entière correspondant à sa position dans la liste des entrées au classement.

  • " PlayeridN " - L'identifiant unique du joueur, "N".
  • " RankN " - Le rang du joueur "N" dans le classement.

  • " ScoreN " - Le score du joueur "N".

Vous pouvez spécifier la partie exacte du classement à afficher en donnant une valeur d'index minimale et maximale, et demander que les résultats soient filtrés pour récupérer uniquement les scores de vos amis ou ceux publiés par tous les joueurs à l'aide des constantes de filtrage suivantes:

  • achievement_filter_friends_only - Obtenez uniquement les scores des amis des joueurs.

  • achievement_filter_all_players - Obtenez tous les scores pour tous les joueurs.

REMARQUE: cette fonction est uniquement pour iOS.


Exemple étendu:

Le code suivant sera probablement appelé après que le joueur se soit connecté à son compte de jeu en utilisant achievement_login pour obtenir la liste de tous les postes de classement des amis pour le jeu, sous Android ou iOS:

if os_type == os_ios
   {
   achievement_load_leaderboard("CgkIs9_51u0PEAIQBw", 1, 100, achievement_filter_friends_only);
   }

Ceci enverra une demande d'informations sur le classement donné et générera un rappel asynchrone avec le spécial async_load ds_map contenant les données suivantes:

var ident = ds_map_find_value(async_load, "id");
if ident == achievement_leaderboard_info
   {
   var lbid = ds_map_find_value(async_load,"leaderboardid");
   if lbid == "CgkIs9_51u0PEAIQBw" || lbid == "leaderboard1id"
      {
      global.numentries = ds_map_find_value(async_load,"numentries");
      for(var i = 0; i < numentries; i++;)
         {
         global.playername[i] = ds_map_find_value(async_load, "Player" + string(i));
         global.playerid[i] = ds_map_find_value(async_load, "Playerid" + string(i));
         global.playerrank[i] = ds_map_find_value(async_load, "Rank" + string(i));
         global.playerscore[i] = ds_map_find_value(async_load, "Score" + string(i));
         achievement_get_pic(global.playerid[i]);
         }
      }
   }

Le code ci-dessus vérifie le retour ds_map dans l' événement social asynchrone et si son "id" correspond à la constante en cours de vérification, il vérifie ensuite que le classement envoyé correspond à l'appel de fonction d'origine avant de parcourir la carte et de stocker toutes les différentes valeurs dans un certain nombre de tableaux. Il demande également une image pour chacune des entrées de la carte (voir achievement_get_pic ).
