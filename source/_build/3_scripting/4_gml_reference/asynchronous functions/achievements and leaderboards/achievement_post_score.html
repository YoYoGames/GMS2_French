achievement_post_score


Syntaxe:

achievement_post_score(name, score)

Argument La description
name Le nom de la table de score (ou "Leaderboard").
score Le score à envoyer.


Résultats:

N/A


La description

Vous pouvez utiliser cette fonction pour envoyer votre score au classement et au service de réussite choisis. Vous envoyez le nom de la table de score (le nom de «leaderboard» que vous avez attribué à votre jeu lorsque vous avez configuré le compte) et la valeur de score réelle).


Exemple:

if achievement_available() achievement_post_score("Leaderboard", 1000);

Le code ci-dessus vérifie si le classement et le service de réussite choisis sont disponibles et si c'est le cas, il affiche un score sur la table de score appelée "Leaderboard".
