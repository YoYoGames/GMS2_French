Réalisations et classements


Les jeux conçus pour cibler des appareils exécutant iOS ou Android peuvent également se connecter à l'une des deux API de classement et de réussite. Pour iOS, vous avez le iTunes Game Center, et pour Android vous pouvez utiliser le Amazon Game Circle ou Google Play, tous utilisant l'utilisation des fonctions de cette page. Ces fonctions permettent à votre jeu d'envoyer des informations sur les scores et les réalisations aux classements que vous avez configurés dans le compte des développeurs pour le jeu que vous faites. C'est un excellent moyen d'impliquer les joueurs dans le côté social du jeu. quelles autres personnes ont joué et ont essayé de les améliorer.

Lorsque vous utilisez ces fonctions, vous devez noter qu'elles ne suivent pas les réalisations et les scores automatiquement pour vous. Ils ne montrent que les données que vous leur envoyez à partir de votre jeu, vous devez donc coder vos propres réalisations et votre système hi-score pour garder une trace du statut des individus dans le jeu, puis utiliser ces fonctions pour ajouter les données au réseau choisi. peut demander des informations de progrès pour les réalisations en utilisant le achievement_load_progress fonction).

Certaines de ces fonctions déclencheront un événement social asynchrone de rappel, qui contiendra un ds_map des données relatives à la fonction qui l'a déclenchée. Les fonctions qui déclenchent l'événement détaillent également les paires clé / valeur de la carte renvoyées par le rappel, cependant un rappel de Social Event peut également être déclenché par votre jeu recevant des notifications externes sur les défis etc... Les détails exacts du ds_map renvoyés pour ces événements sont expliqués dans la section Événement social asynchrone.

REMARQUE: Pour que ces fonctions fonctionnent avec iOS, vous devez disposer d'un compte Game Center valide et d'un identifiant d'application iOS pour votre jeu.

REMARQUE: Pour que ces fonctions fonctionnent avec Android vous devez soumettre votre jeu à l' Amazon App Store ou à Google Play. Vous devez également configurer les sections nécessaires et activer l'API dans les Options de jeu Android de votre jeu.

Les fonctions listées ci-dessous sont les fonctions de base utilisées pour intégrer votre jeu à la plate-forme de jeu social cible choisie:

  • achievement_login_status
  • achievement_login
  • achievement_logout
  • réalisation_disponible
  • achievement_post
  • achievement_post_score
  • achievement_increment
  • achievement_show_achievements
  • achievement_show_leaderboards
  • achievement_show_challenge_notifications
  • achievement_get_challenges
  • achievement_get_pic
  • achievement_load_friends
  • achievement_load_leaderboard
  • réalisation_load_progress
  • achievement_send_challenge
  • achievement_reset
