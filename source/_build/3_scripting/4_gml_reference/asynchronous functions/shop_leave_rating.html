shop_leave_rating


La description

Cette fonction ouvre une boîte de dialogue dépendante du système d'exploitation dans laquelle vous pouvez demander à l'utilisateur de publier une note ou un commentaire sur une page particulière. Vous pouvez définir le texte qui doit apparaître dans le dialogue, ainsi que le texte que vous souhaitez afficher sur les deux boutons et l'URL où le commentaire doit être posté.

NOTE: Cette fonction est uniquement pour Android cibles Android et iOS.
IMPORTANT! Sur iOS, lorsque vous utilisez TestFlight, en cours de développement, l'invite s'affiche, mais vous ne pouvez rien faire. Une fois que votre application est dans l'App Store, ce bouton devient actif et l'utilisateur a la possibilité d'écrire une critique ou de laisser une note. Notez que si vous distribuez le jeu via TestFlight, cette fenêtre peut même ne pas s'afficher.


Syntaxe:

shop_leave_rating(text, yes_string, no_string, url)

Argument La description
text Texte qui apparaît dans le dialogue
yes_string Texte qui apparaît sur le bouton "oui".
no_string Texte qui apparaît sur le bouton "non".
url L'URL complète où le commentaire doit être envoyé.


Résultats:

N/A


Exemple:

if timer <= 0
   {
   shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://MacSweeney/comments");
   timer = 10000000;
   }
else
   {
   timer -= 1;
   }

Le code ci-dessus demandera à l'utilisateur de laisser un commentaire si la variable "timer" a compté jusqu'à 0.
