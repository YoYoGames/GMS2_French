string_char_at


La description

Vous pouvez utiliser cette fonction pour renvoyer un caractère spécifique à une position spécifique dans une chaîne, l'index commençant à 1 pour le premier caractère. Si aucun caractère n'est trouvé ou si la chaîne est plus courte que la valeur donnée à index, une chaîne vide "" est renvoyée.


Syntaxe:

string_char_at(str, index);

Argument La description
str La chaîne à vérifier.
index La position pour obtenir le personnage de.


Résultats:

String


Exemple:

str1 = "Hello World";
str2 = string_char_at(str1, 7);

Cela va définir str2 au septième caractère ("H" comptant comme le 1er) dans la chaîne str1, dans ce cas "W".
