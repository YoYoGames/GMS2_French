string_lettersdigits


La description

Cette fonction retournera une copie d'une chaîne donnée avec tout sauf ses lettres et chiffres supprimés, ce qui signifie qu'elle peut être utilisée pour supprimer tous les caractères indésirables (comme "#" ou "?") De, par exemple, un nom de connexion ou un mot de passe.

REMARQUE: Cette fonction renvoie uniquement l'alphabet anglais de 26 lettres de A à Z (majuscules ou minuscules) et les chiffres de 0 à 9.


Syntaxe:

string_lettersdigits(string);

Argument La description
string La chaîne à convertir en lettres et chiffres.


Résultats:

String


Exemple:

if string_length(password) > string_length(string_lettersdigits(password))
   {
   draw_text(32,32,"Invalid Password! Only Letters and numbers please!");
   }

Le code ci-dessus va vérifier la longueur d'une chaîne par rapport à la longueur de la même chaîne, mais avec tous les symboles supprimés. Si ce n'est pas la même chose, un message est affiché sur l'écran.
