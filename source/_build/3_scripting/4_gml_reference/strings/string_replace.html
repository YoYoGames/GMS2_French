string_replace


La description

Vous pouvez utiliser cette fonction pour analyser une chaîne recherchant une partie spécifique, qui peut ensuite être remplacée par la nouvelle chaîne que vous avez spécifiée.


Syntaxe:

string_replace(str, substr, newstr);

Argument La description
str La chaîne à copier.
substr La sous-chaîne dans la chaîne à remplacer.
newstr La nouvelle sous-chaîne pour remplacer la précédente.


Résultats:

String


Exemple:

str1 = "Hello Earth";
str2 = string_replace(str1, "Earth", "World");

Cela va définir str2 à str1, mais avec son instance de "Terre" remplacé par "Monde", ce qui entraîne str2 être "Hello World".
