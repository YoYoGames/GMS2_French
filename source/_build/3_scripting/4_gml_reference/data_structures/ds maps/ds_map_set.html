ds_map_set

Définir une clé de carte sur une valeur

Syntaxe:

ds_map_set(id, key, value)


Argument La description
id L'identifiant de la carte à utiliser.
key La clé à définir.
value La valeur à laquelle définir la clé.


Retours: N / A


La description

Avec cette fonction, vous pouvez définir la valeur d’une clé dans une mappe DS donnée. Vous fournissez la valeur de l'ID de mappe DS (telle que renvoyée par la fonction ds_map_create(), puis donnez la clé que vous souhaitez définir et la valeur à définir. Les clés peuvent être des entiers ou des chaînes, et si la clé donnée n'existe pas, elle sera créée pour vous. Cette fonction revient à utiliser l’ accesseur de carte DS pour définir / créer une paire clé / valeur de carte.


Exemple:

if is_undefined(ds_map_find_value(map, "score"))
    {
    ds_map_set(map, "score", 0);
    }

Le code ci-dessus vérifiera si la clé donnée existe et si elle n’existe pas, elle est créée et définie.
