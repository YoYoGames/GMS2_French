ds_map_find_value

Trouve la clé spécifiée et renvoie la valeur appariée à partir de ds_map.

Syntaxe:

ds_map_find_value(id, key)


Argument La description
id L'identifiant de la carte à utiliser.
key La clé à trouver.


Retours: Réel, chaîne ou indéfini


La description

Avec cette fonction, vous pouvez obtenir la valeur d'une clé spécifiée. Les valeurs d'entrée de la fonction sont le ds_map (créé précédemment) à utiliser et la clé à vérifier.

NOTE: Si cette clé n'existe pas, la fonction retournera <undefined>. Vous devriez toujours vérifier cela en utilisant le is_undefined() une fonction.


Exemple:

amount = ds_map_find_value(inventory, "food");

Ou, en utilisant l'accesseur de carte "?":

amount = inventory[? "food"];

Le code ci-dessus obtiendra la valeur de la clé "food" et la stockera dans la variable "quantité".
