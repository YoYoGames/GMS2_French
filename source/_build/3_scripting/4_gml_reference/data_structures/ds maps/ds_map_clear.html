ds_map_clear

Efface toutes les données d'une carte donnée.

Syntaxe:

ds_map_clear(id);


Argument La description
id L'ID de la structure de données à effacer.


Retours: N / A.


La description

Cette fonction effacera le fichier ds_map (précédemment créé) de toutes les paires clé / valeur, mais il ne supprimera pas le fichier ds_map de la mémoire. Pour cela, vous devriez utiliser la fonction ds_map_destroy.



Exemple:

if global.new_game
   {
   ds_map_clear(inventory);
   }

La fonction ci-dessus vérifiera une variable globale pour voir si elle est vraie et, si c'est le cas, alors la ds_map indexée dans la variable d'instance "inventaire" sera effacée de toutes les paires clé / valeur (mais elle n'est pas détruite).
