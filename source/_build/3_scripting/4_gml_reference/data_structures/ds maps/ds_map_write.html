ds_map_write

Cette fonction écrit les données d'un fichier ds_map sous la forme d'une chaîne.

Syntaxe:

ds_map_write(id);


Argument La description
id L'ID de la structure de données à utiliser


Retourne: Chaîne


La description

Cette fonction transforme les données ds_map de la carte spécifiée en un format de chaîne qui peut ensuite être écrit dans un fichier *.ini ou un fichier *.txt pour un stockage facile. Cette chaîne peut ensuite être relue dans un nouveau fichier ds_map en utilisant ds_map_read().

Remarque: la chaîne renvoyée n'est pas une chaîne lisible par un humain, mais plutôt une sauvegarde du contenu de la structure de données


Exemple:

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

Le code ci-dessus ouvre un fichier ini prêt à être écrit. Il utilise ensuite ds_map_write() pour générer une chaîne stockée dans la variable temporaire "t_string". Enfin, il écrit cette chaîne dans le fichier ini avant de le fermer.
