ds_priority_write

Écrit la structure de données dans une chaîne renvoyée. Cela permet à la structure d'être enregistrée dans quelque chose comme un fichier.ini pour une utilisation future.

Syntaxe:

ds_priority_write(id);


Argument La description
id L'ID de la structure de données à vérifier.


Retourne: Chaîne


La description

Cette fonction renvoie une chaîne qui peut ensuite être stockée ou transférée vers une autre structure de données à ds_priority_read fonction.

Remarque: la chaîne renvoyée n'est pas une chaîne lisible par un humain, mais plutôt une sauvegarde du contenu de la structure de données


Exemple:

var str;
ini_open("save.ini");
str =ds_priority_write(p_queue);
ini_write_string("P_Queues", "0", str);
ds_priority_clear(p_queue);
ini_close();

Le code ci-dessus ouvre un fichier ini puis écrit une chaîne contenant les informations stockées dans la file d'attente ds_priority indexée dans la variable "p_queue". La file d'attente prioritaire est alors effacée et le fichier ini fermé.
