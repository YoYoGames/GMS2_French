ds_list_set

Syntaxe:

ds_list_set(id, pos, val);


Argument La description
id L'identifiant de la liste à ajouter.
pos La position dans la liste à définir.
[val2,... max_val] La valeur à ajouter à la liste.


Retours: N / A


La description

Cette fonction peut être utilisée pour définir une entrée de liste ajoutée précédemment. Vous donnez l'ID de la liste (tel que renvoyé lorsque vous avez créé la liste) et la position dans la liste à définir ainsi que la valeur à définir. Notez que si l'entrée définie est en dehors des limites de la liste (par exemple, vous définissez l'entrée de la liste 20 mais la liste actuelle ne contient que 10 entrées), la liste sera remplie jusqu'à la position donnée et chaque entrée sera mise à 0.


Exemple:

for (var i = 0; i < ds_list_size(list); i++;)
    {
    ds_list_set(list, i, -1);
    }

Le code ci-dessus va ajouter la valeur stockée dans la variable "score" dans la liste indexée dans la variable "sc_list".
