ds_list_find_index

Syntaxe:

ds_list_find_index(id, val);


Argument La description
id L'identifiant de la liste à utiliser.
val La valeur à trouver


Retours: Real


La description

Avec cette fonction, vous pouvez vérifier la liste donnée pour une valeur et la position dans la liste pour cette valeur sera retournée. Notez que s'il y a plusieurs entrées dans la liste avec la même valeur, la position de l'une d'elles peut être retournée, et que si la valeur n'existe pas, alors -1 sera retourné. Notez que la valeur peut être un tableau, que vous pouvez vérifier avec la fonction is_array.


Exemple:

pos = ds_list_find_index(list, "Player1");

Le code ci-dessus vérifie la liste indexée dans la variable "list" pour la valeur "Player1" et stocke la position retournée dans la variable "pos".
