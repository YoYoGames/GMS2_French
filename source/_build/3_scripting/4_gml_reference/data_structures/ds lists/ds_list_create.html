ds_list_create

Syntaxe:

ds_list_create();


Retours: réel


La description

Cette fonction créera une nouvelle structure de données de liste et renverra la valeur d'index. Cette valeur doit être stockée dans une variable et utilisée dans tous les autres appels de fonction liés à la liste. Notez que si vous essayez d’accéder à une valeur de la liste avant qu’elle ne soit renseignée - c’est-à-dire que la liste est "vide", la valeur renvoyée peut alors être 0 car en interne, les premières entrées de la liste sont définies lors de leur création afin de minimiser les problèmes de performances. lors de l’ajout initial d’éléments à la liste (bien que le ds_list_size() la fonction retournera toujours 0 sur une liste nouvellement créée). Si vous souhaitez vous assurer que la liste est "vraiment" vide lors de la création, vous devez appeler ds_list_clear() après avoir créé la liste, ce qui signifie que toutes les valeurs renvoyées pour des emplacements de liste non remplis seront undefined.

IMPORTANT! Lorsque vous créez une structure de données, la valeur d'index identifiant celle-ci est un entier commençant à 0. Cela signifie que les structures de données de types différents peuvent avoir la même valeur d'index. Par conséquent, en cas de doute, vous devez utiliser ds_exists fonctionner avant d'y accéder. Notez également que les index sont réutilisés, de sorte qu'une valeur d'index de structure de données détruite peut être utilisée par une nouvelle valeur créée ultérieurement.


Exemple:

list = ds_list_create();

Cela créera une nouvelle liste et affectera son identifiant d'index à la variable d'instance "list".
