ds_stack_create

Crée une nouvelle pile et renvoie son identifiant.

Syntaxe:

ds_stack_create();


Retours: réel


La description

Cette fonction créera une nouvelle structure de données de pile et renverra la valeur d'index. Cette valeur doit être stockée dans une variable et utilisée dans tous les autres appels de fonction relatifs à la pile.

IMPORTANT! Lorsque vous créez une structure de données, la valeur d'index identifiant celle-ci est un entier commençant à 0. Cela signifie que les structures de données de types différents peuvent avoir la même valeur d'index. Par conséquent, en cas de doute, vous devez utiliser ds_exists fonctionner avant d'y accéder. Notez également que les index sont réutilisés, de sorte qu'une valeur d'index de structure de données détruite peut être utilisée par une nouvelle valeur créée ultérieurement.


Exemple:

stack = ds_stack_create();

Cela créera une nouvelle pile et affectera son identifiant d'index à la variable d'instance "pile".
