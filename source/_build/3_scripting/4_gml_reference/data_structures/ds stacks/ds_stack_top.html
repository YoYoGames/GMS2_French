ds_stack_top

Lit la valeur depuis le haut de la pile.

Syntaxe:

ds_stack_top(id);


Argument La description
id L'ID de la structure de données à lire.


Retourne: Réel ou Chaîne


La description

Cette fonction ne lit que la première valeur de la pile (celle qui est "en haut"). Il ne fera pas apparaître la valeur, ce qui signifie qu'il peut encore être lu à l'avenir par cette fonction ou le ds_stack_pop.


Exemple:

num = ds_stack_top(control_stack);

Le code ci-dessus va lire la valeur de la pile indexée dans la variable "control_stack" et stocker la valeur de retour dans la variable "num".
