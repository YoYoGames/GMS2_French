ds_grid_write

Transforme une structure de données en une chaîne et renvoie cette chaîne.

Syntaxe:

ds_grid_write(index);


Argument La description
index L'index de la grille à écrire.


Retourne: Chaîne


La description

Cette fonction peut être utilisée pour convertir le ds_grid donné en une chaîne, qui peut ensuite être stockée dans un fichier externe (par exemple). Vous pouvez lire la chaîne renvoyée à partir de cette fonction dans un ds_grid en utilisant la fonction ds_grid_read().

Remarque: la chaîne renvoyée n'est pas une chaîne lisible par un humain, mais plutôt une sauvegarde du contenu de la structure de données


Exemple:

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close();

Le code ci-dessus va ouvrir un fichier ini (en le créant s'il n'existe pas déjà), puis écrire le ds_grid donné en tant que chaîne dans ce fichier.
