ds_grid_set_region

Définit toutes les cellules de la région dans la grille avec l'ID donné, à la valeur indiquée (peut être un nombre ou une chaîne).

Syntaxe:

ds_grid_set_region(index, x1, y1, x2, y2, val);


Argument La description
index L'indice de la grille.
x1 La position x de la gauche de la région dans la grille.
y1 La position y du sommet de la région dans la grille.
x2 La position x de la droite de la région dans la grille.
y2 La position y du bas de la région dans la grille.
val La valeur pour définir les cellules de la région.


Retours: N / A


La description

Cette fonction peut être utilisée pour définir une région rectangulaire d'une grille donnée à une valeur spécifiée (qui peut être soit un réel ou une chaîne) comme illustré par l'image ci-dessous:


Exemple:

ds_grid_set_region(grid, 5, 5, 10, 10, 99);

Ceci placerait toutes les cellules dans la région de la grille indexée dans la variable "grille" de (5,5) à (10,10) à 99.
