ds_grid_destroy


La description

Cette fonction supprimera de la mémoire la structure de données de grille donnée, libérant les ressources qu’elle utilisait et supprimant toutes les valeurs qu’elle contenait. Cette fonction doit toujours être utilisée lorsque vous avez terminé d'utiliser ds_grid pour éviter les fuites de mémoire susceptibles de ralentir et de bloquer votre partie.

IMPORTANT! Lorsque vous créez une structure de données, la valeur d'index identifiant celle-ci est un entier commençant à 0. Cela signifie que les structures de données de types différents peuvent avoir la même valeur d'index. Par conséquent, en cas de doute, vous devez utiliser ds_exists fonctionner avant d'y accéder. Notez également que les index sont réutilisés, de sorte qu'une valeur d'index de structure de données détruite peut être utilisée par la suite, afin que nous recommandions de toujours définir la variable qui contenait l'index DS à -1 après la destruction.


Syntaxe:

ds_grid_destroy(index);

Argument La description
index Cet index de la grille à détruire.


Résultats:

N/A


Exemple:

if lives = 0
    {
    ds_grid_destroy(Wall_Grid);
    Wall_Grid = -1;
    room_goto(rm_Menu);
    }

Le code ci-dessus vérifiera la valeur de la variable globale intégrée "lives" et si elle vaut 0, il détruit le ds_grid indexé dans la variable "Wall_Grid" puis change de pièce.
