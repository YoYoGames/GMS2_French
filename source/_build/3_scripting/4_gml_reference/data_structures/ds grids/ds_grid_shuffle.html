ds_grid_shuffle


La description

Cette fonction peut être utilisée pour randomiser les positions de toutes les valeurs dans toutes les cellules d'une grille.

REMARQUE: Cette fonction permet de mélanger les cellules de la grille dans les mêmes positions à chaque nouvelle exécution du jeu, car GameMaker Studio 2 génère la même graine aléatoire initiale à chaque fois pour faciliter le débogage du code. Pour éviter ce comportement, utilisez randomise au début de votre partie. Cela n’est vrai que lors des tests et du débogage du jeu, car le package exécutable final ne montrera pas ce comportement et sera aléatoire à chaque lecture.


Syntaxe:

ds_grid_shuffle(index);

Argument La description
index L'index de la grille à mélanger.


Résultats:

N/A


Exemple:

ds_grid_shuffle(grid)

Cela prendrait toutes les valeurs de ds_grid indexées dans la variable "grille" et les mélangerait pour obtenir un ordre différent.
