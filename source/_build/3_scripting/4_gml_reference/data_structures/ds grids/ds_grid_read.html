ds_grid_read


La description

Cette fonction peut être utilisée pour convertir une chaîne créée précédemment par la fonction. ds_grid_write() retour dans un ds_grid. Ds_grid doit avoir été créé précédemment (voir l'exemple ci-dessous). Notez que si la structure de données a été créée avec les versions précédentes de GameMaker vous devez ajouter l’argument optionnel "legacy", en le définissant comme GameMaker true car le format de la chaîne a changé avec cette version.


Syntaxe:

ds_grid_read(index, string [, legacy]);

Argument La description
index L'index de la grille à lire.
string La chaîne à lire dans ds_grid.
legacy ( optional ) Peut être soit true ou false ou omis complètement.


Résultats:

N/A


Exemple:

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close()

Le code ci-dessus crée un fichier ds_grid basé sur la taille de la pièce (chaque carré de pixels 32x32 représente une cellule de grille), puis lit un ensemble de données de grille précédemment enregistré dans un fichier ini dans la nouvelle grille ds.
