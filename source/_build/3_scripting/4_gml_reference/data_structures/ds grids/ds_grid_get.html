ds_grid_get


La description

Cette fonction peut être utilisée pour obtenir la valeur (un nombre réel ou une chaîne) à partir de n’importe quelle cellule de la grille ds_grid donnée. Si vous transmettez des coordonnées de grille non valides à la fonction, la valeur renvoyée sera 0.


Syntaxe:

ds_grid_get(index, x, y);

Argument La description
index L'index de la grille.
x La position x de la cellule dont vous voulez trouver la valeur.
y La position y de la cellule dont vous voulez trouver la valeur.


Résultats:

Real or String


Exemple:

var xx, yy;
xx = irandom(ds_grid_width(grid) - 1);
yy = irandom(ds_grid_height(grid) - 1);
val = ds_grid_get(grid, xx, yy)

Le code ci-dessus sélectionne une cellule aléatoire à partir de la grille ds_grid indexée dans la variable "grid" et stocke sa valeur dans la variable "val".
