shader_enable_corner_id


Syntaxe:

shader_enable_corner_id(enable);


Argument La description
enable Activer ( true ) ou désactiver ( false ) cette fonction.


Résultats:

N/A


La description

Avec cette fonction, vous pouvez définir un état global pour tous les shaders utilisés où, lorsqu'il est activé, le shader "vole" 2 bits des valeurs de couleur d'entrée. Le premier provient du bit le plus bas de la valeur de couleur rouge et le second du bit le plus bas de la valeur de couleur bleue. Ces valeurs peuvent ensuite être récupérées dans le shader pour déterminer le sommet avec lequel vous traitez (ie: quel coin).

IMPORTANT! Cette fonction n'est pas disponible avec la licence d'évaluation du produit.


Exemple:

shader_enable_corner_id(true);

Le code ci-dessus permettra l'utilisation de bits de couleur pour l'identifiant de coin pour tous les shaders.
