xboxlive_user_is_signing_in


La description

Cette fonction n'est valide que lors de l'exportation à l'aide de la cible UWP pour la Xbox One et vous devez avoir coché l'option Activer XBox Live dans les options de jeu UWP. La fonction vérifiera si un utilisateur est en train de se connecter et de retourner true si tel est le cas et false autrement.


Syntaxe:

xboxlive_user_is_signing_in();


Renvoie: Boolean


Exemple:

if !xboxlive_user_is_signed_in()
    {
    if !xboxlive_user_is_signing_in()
        {
        xboxlive_show_account_picker();
        }
    }
else
    {
    global.GamerTag = xboxlive_gamertag_for_user();
    }

Le code ci-dessus vérifie si un utilisateur est connecté à XBox Live et s'il est connecté, le code recevra le gamertag pour l'utilisateur et le stockera dans une variable globale pour une utilisation future. S'ils ne sont pas connectés, le code vérifie s'ils sont en train de se connecter, auquel cas rien d'autre ne se produira, et s'ils ne le sont pas, le sélecteur de compte s'ouvrira, les invitant à signer. -dans.
