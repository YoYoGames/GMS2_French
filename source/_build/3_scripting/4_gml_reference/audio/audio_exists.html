audio_exists


La description

Cette fonction retourne si un son existe ( true ) ou pas ( false ). Notez que si l'index que vous recherchez n'a pas été initialisé précédemment, cette fonction provoquera une erreur car elle recherche des indices d'actifs inexistants. Le son à vérifier peut être une seule occurrence d'un son (l'index des sons individuels lus peut être stocké dans une variable lors de l'utilisation du audio_play_sound ou audio_play_sound_at fonctions) ou un actif sonore.


Syntaxe:

audio_exists(index);


Argument La description
index L'indice du son pour vérifier l'existence de.


Résultats:

Boolean


Exemple:

if audio_exists(global.Music)
   {
   audio_play_sound(global.Music, 0, true);
   }

Le code ci-dessus vérifie si un son existe. Si c'est le cas, il est configuré pour jouer en boucle.
