audio_get_type


La description

Lorsque vous ajoutez de l'audio à GameMaker Studio 2 il peut être diffusé en continu ou en mémoire. Si vous avez besoin de savoir si un indice sonore donné est pour l'audio diffusé ou non, vous pouvez utiliser cette fonction qui retournera 1 pour streamed, 0 pour le son en mémoire, et -1 s'il y a une erreur ou l'index ne pointe pas vers un ressource sonore valide.


Syntaxe:

audio_get_type(index);


Argument La description
index L'indice du son à vérifier.


Résultats:

Real


Exemple:

type = audio_get_type(snd_Music_1);

Le code ci-dessus vérifie le type d'audio indexé dans la variable "snd_Music_1" et stocke la valeur renvoyée dans la variable "type".
