audio_stop_recording


La description

Cette fonction arrêtera d’enregistrer sur le canal de l’enregistreur donné (l’index du canal est renvoyé lorsque vous appelez la fonction audio_start_recording ). Lorsque vous arrêtez l'enregistrement, aucun événement asynchrone d'enregistrement audio supplémentaire ne sera déclenché pour le canal de l'enregistreur donné. Vous devez donc normalement utiliser cette fonction dans l'événement asynchrone réel pour vous assurer que vous avez capturé toutes les données.

REMARQUE: la plupart des plates-formes, à l' exception de HTML5, prennent en charge l'enregistrement audio sous quelque forme que ce soit, mais cela ne signifie pas que tous les périphériques le permettent, même si la plate-forme le permet. Vous devez donc toujours vérifier que le audio_get_recorder_count() function renvoie une valeur supérieure à 0 pour vérifier que les dispositifs d’enregistrement sont disponibles avant d’utiliser les autres fonctions d’enregistrement.


Syntaxe:

audio_stop_recording(channel_index);


Argument La description
channel_index L'index du canal de l'enregistreur à arrêter.


Résultats:

N/A


Exemple:

audio_stop_recording(audio_channel);

Le code ci-dessus indique à GameMaker Studio 2 d'arrêter l'enregistrement sur l'index de canal audio donné.
