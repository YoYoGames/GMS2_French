audio_sync_group_debug


La description

Cette fonction peut être utilisée pour afficher des informations de débogage sur un groupe de synchronisation donné (l'index de groupe est la valeur renvoyée lors de la création du groupe à l'aide de la fonction. audio_create_sync_group() ). Vous pouvez appeler la fonction avec la valeur -1 pour désactiver la superposition.

Lorsque vous déboguez les groupes de synchronisation audio, la superposition indiquera quels groupes de synchronisation sont en cours de débogage

  1. Combien de canaux y a-t-il
  2. Combien de fois le groupe de synchronisation a échoué (cela signifie qu'il a essayé de faire trop de travail)
  3. Une liste des chaines
    • Le rouge signifie que cette chaîne ne fait actuellement aucun travail.
    • Jaune signifie que le canal est à l'état initial (pas encore en cours de décodage ou de lecture)
    • Vert signifie que la piste cherche / décode actuellement
    • Violet signifie que le canal est à court de données
  5. Une barre de profilage indiquant le travail effectué par le fil de décodage
    • bleu signifie qu'un fichier est lu pour la première fois
    • le rouge signifie que le fil lit un fichier pour savoir où lire
    • vert signifie que le fil est en train de décoder des données
    • le violet signifie que le fil dort jusqu'à ce qu'il ait besoin de fonctionner

Notez que pour le débogage audio général, vous avez également la fonction audio_debug().

REMARQUE: cette fonctionnalité n'est pas disponible pour la plate-forme cible HTML5.


Syntaxe:

audio_sync_group_debug(group_index);


Argument La description
group_index L'index de groupe à déboguer ou -1 à désactiver le débogage.


Résultats:

N/A


Exemple:

if (debug_mode)
   {
   audio_sync_group_debug(sg);
   }
else
   {
   audio_sync_group_debug(-1);
   }

Le code ci-dessus activera ou désactivera la superposition de débogage du groupe de synchronisation audio affichant le groupe de synchronisation indexé dans la variable "sg" selon que le jeu est exécuté ou non en mode débogage.
