audio_pause_sync_group


La description

Cette fonction met en pause le groupe de synchronisation donné s'il est en cours de lecture, l'index de groupe étant la valeur renvoyée lors de la création du groupe à l'aide de la fonction. audio_create_sync_group(). Cela n'arrête pas le son, et appeler audio_resume_sync_group(), commencera à jouer à partir de la même position où elle a été mise en pause.

REMARQUE: cette fonctionnalité n'est pas disponible pour la plate-forme cible HTML5.


Syntaxe:

audio_pause_sync_group(group_index);


Argument La description
group_index L'index du groupe à mettre en pause.


Résultats:

N/A


Exemple:

if (keyboard_check_pressed((ord)"P"))
   {
   global.Pause = !global.Pause
   if global.Pause
      {
      audio_pause_sync_group(sg);
      }
   else
      {
      audio_resume_sync_group(sg);
      }
   }

Le code ci-dessus vérifie que l'utilisateur appuie sur la touche "P". S'il est détecté, il bascule la variable "global.Pause", puis le met en pause ou reprend le groupe de synchronisation indexé dans la variable "sg".
