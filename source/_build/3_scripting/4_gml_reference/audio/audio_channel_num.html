audio_channel_num


La description

Avec cette fonction, vous pouvez définir le nombre de canaux audio disponibles pour la lecture audio dans GameMaker Studio 2. Cela signifie essentiellement que vous donnez le nombre de sons simultanés qui peuvent être joués à un moment donné, et que si le nombre dépasse la quantité, les sons de priorité inférieure sont arrêtés pour libérer un canal pour les sons ayant une priorité plus élevée. Vous pouvez utiliser cette fonction pour optimiser votre jeu pour les appareils car plus le nombre de canaux audio est faible, meilleure est la performance, mais gardez à l'esprit que cela peut également causer une coupure notable de certains sons si plusieurs sont joués simultanément.


Syntaxe:

audio_channel_num(num);


Argument La description
num Nombre de canaux audio disponibles (la valeur par défaut est 128).


Résultats:

N/A


Exemple:

switch (os_browser)
   {
   case browser_not_a_browser:
      switch (os_type)
         {
         case os_windows:
         case os_macos:
            audio_channel_num(200);
            break;
         default:
            audio_channel_num(64);
            break;
         }
      break;
   default:
      audio_channel_num(16);
      break;
   }

Le code ci-dessus vérifie la plate-forme sur laquelle s'exécute le jeu et modifie le nombre de canaux audio disponibles pour augmenter les performances.
