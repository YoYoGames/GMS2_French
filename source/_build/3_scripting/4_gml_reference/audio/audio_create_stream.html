audio_create_stream


La description

Avec cette fonction, vous pouvez créer un nouvel index sonore qui peut ensuite être utilisé dans les fonctions audio habituelles pour diffuser l'audio directement à partir d'une source de fichier OGG externe. La fonction nécessite le nom de fichier (qui peut être un fichier inclus, par exemple) et renverra le nouvel index sonore à utiliser. Notez que lorsque vous n’avez plus besoin du son, vous devez appeler la fonction audio_destroy_stream() avec l’index sonore pour le supprimer de la mémoire, sinon vous risquez une fuite de mémoire qui ralentira et finira par planter votre partie.

REMARQUE: cette fonctionnalité n'est pas disponible pour la plate-forme cible HTML5.
IMPORTANT! Cette fonction n'est pas disponible avec la licence d'évaluation du produit.


Syntaxe:

audio_create_stream(filename);


Argument La description
filename Le fichier (OGG uniquement) pour diffuser l'audio.


Résultats:

Sound Index


Exemple:

snd = audio_create_stream("Music/Track1.ogg");
audio_play_sound(snd, 0, true);

Le code ci-dessus crée un nouvel index sonore dans la variable "snd" à partir du fichier donné, puis lit ce son.
