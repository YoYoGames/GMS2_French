audio_falloff_set_model


La description

Pour ajouter plus de polyvalence au moteur audio, GameMaker Studio 2 vous permet de sélectionner le modèle de réduction qui convient à votre jeu. Ce modèle sera utilisé pour toutes les fonctions audio du jeu ou de l'application. Vous devez donc vous assurer que le modèle que vous choisissez est le bon, car chacune affectera la manière dont l'auditeur perçoit les sons que vous jouez via les émetteurs ou avec le récepteur. une fonction audio_play_sound_at.

Lorsque vous définissez l'atténuation dans votre jeu avec les fonctions appropriées, vous devez définir trois arguments, chacun étant adapté à un modèle spécifique et affectant la manière dont le son final est "entendu" par le joueur en fonction de la distance. que l'auditeur est de la source. Les trois arguments sont:

  • distance de référence - il s'agit de la distance entre l'auditeur et la distance sous laquelle le volume du son joué devrait normalement baisser de moitié avant d'être influencé par le facteur de réduction ou la distance maximale spécifiée.

  • distance maximale - définit la distance où il n'y aura plus d'atténuation du son source. Il peut s’agir du moment où le son n’est plus entendu ou du moment où le volume du son ne diminue plus en deçà du seuil minimal défini par le modèle choisi.

  • facteur d'atténuation - Le facteur d'atténuation est utilisé dans l'atténuation de la distance en fonction du modèle de distance inverse et définit le seuil minimal final pour un son avec atténuation.

Syntaxe:

audio_falloff_set_model(model);


Argument La description
model La constante utilisée pour définir le modèle de réduction.


Résultats:

N/A


Exemple:

audio_falloff_set_model(audio_falloff_exponent_distance_clamped);
audio_play_sound_at(snd_Waterfall, x, y, 0, 100, 300, 1, true, 1);

Le code ci-dessus définit le modèle de réduction, puis lit le son indexé dans la variable "snd_Waterfall", qui sera bouclée à la position de la pièce, avec une référence de réduction de 100, une distance de réduction de 300, un facteur de réduction de 1 et une faible priorité.
