audio_destroy_stream


La description

Si vous avez précédemment créé un flux audio à partir d'un fichier en utilisant la fonction audio_create_stream() et n'a plus besoin de ce son, vous pouvez appeler cette fonction pour le supprimer de la mémoire. Tout autre appel à l'index sonore après sa destruction donnera une erreur.

IMPORTANT! Cette fonction n'est pas disponible avec la licence d'évaluation du produit.


Syntaxe:

audio_destroy_stream(filename);


Argument La description
filename Le fichier à diffuser l'audio à partir de.


Résultats:

NA


Exemple:

audio_destroy_stream(snd);

Le code ci-dessus supprime de la mémoire le son indexé dans la variable "snd".
