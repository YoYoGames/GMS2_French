audio_emitter_get_listener_mask


La description

Cette fonction renvoie les données du masque de bits qui définissent les écouteurs audio dont un émetteur devrait jouer.


Syntaxe:

audio_emitter_get_listener_mask(emitterID);


Argument La description
emitterID L'identifiant unique de l'émetteur pour obtenir le masque de


Résultats:

Real


Exemple:

s_emit = audio_emitter_create();
if audio_emitter_get_listener_mask(s_emit) != global.PlayerMask
   {
   audio_emitter_set_listener_mask(snd, global.PlayerMask);
   }

Le code ci-dessus crée un émetteur puis vérifie les données du masque d'écoute, et s'il n'est pas identique à celui qui est stocké dans une variable globale, il définit le ou les écouteurs à lire à partir des données de masque stockées dans la variable globale.
