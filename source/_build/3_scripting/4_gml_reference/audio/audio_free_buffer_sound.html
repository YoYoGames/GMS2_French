audio_free_buffer_sound


La description

Avec cette fonction, vous pouvez libérer la valeur d'index de pointeur associée à l'ID de son. Les sons libérés ne seront pas disponibles pour la lecture, et si plusieurs instances du son sont jouées, elles seront toutes arrêtées. Notez qu'avant de pouvoir supprimer le tampon lui-même, vous devez d'abord libérer tous les ID de son associés.

IMPORTANT! Cette fonction n'est pas disponible avec la licence d'évaluation du produit.


Syntaxe:

audio_free_buffer_sound(index);


Argument La description
index L'indice du son tamponné à libérer.


Résultats:

N/A


Exemple:

audio_free_buffer_sound(soundID);

Le code ci-dessus libère le son bufferisé indexé dans la variable "soundID".
