audio_master_gain


La description

Avec cette fonction, vous pouvez définir la valeur absolue du volume global de tous les sons et de la musique. Il est basé sur une échelle linéaire allant de 0 (silencieux) à toute valeur supérieure à 0, bien que vous considériez normalement le volume maximal comme étant égal à 1. Tout élément supérieur à 1 peut être utilisé mais, en fonction du son utilisé et de la plate-forme en cours de compilation,, vous risquez d'obtenir une distorsion ou un écrêtage lors de la lecture du son. Cette fonction affectera le volume relatif de tous les sons et de la musique lus depuis votre jeu.


Syntaxe:

audio_master_gain(gain);


Argument La description
gain Valeur du volume global (0 à 1).


Résultats:

N/A


Exemple:

if keyboard_check(vk_up)
   {
   if vol < 1 vol += 0.05;
   audio_master_gain(vol);
   }
if keyboard_check(vk_down)
   {
   if vol > 0 vol -= 0.05;
   audio_master_gain(vol);
   }

Le code ci-dessus vérifie si les touches fléchées haut et bas sont enfoncées, ce qui augmente ou diminue la variable "vol". Cette variable est ensuite utilisée pour définir le gain global de tous les sons et de la musique du jeu.
