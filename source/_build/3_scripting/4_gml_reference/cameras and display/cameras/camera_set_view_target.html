camera_set_view_target


La description

Vous pouvez utiliser cette fonction pour définir la cible de suivi de la caméra de vue dans la pièce. Vous donnez la valeur d'ID de caméra unique (telle que renvoyée par les différents camera_create fonctions), puis donnez à l'instance cible ou à l'ID d'objet que vous souhaitez définir la vue de la caméra. Notez que si vous choisissez un ID d'objet et qu'il existe plusieurs instances de cet objet dans la pièce, GameMaker Studio 2 n'a aucun moyen de savoir quelle instance vous souhaitez suivre et il peut donc s'agir de l'une d'entre elles.


Syntaxe:

camera_set_view_target(camera_id, instance_id/object_id)


Arguments:

Argument La description
camera_id La valeur unique d'ID de caméra retournée lorsque vous avez créé la caméra
instance_id/object_id Instance ou objet pour avoir la cible de la caméra pour suivre


Résultats:

N/A


Exemple:

camera_set_view_target(view_camera[0], id);

Le code ci-dessus définit la cible de la caméra de vue pour la caméra assignée pour voir le port [0] à celui de l'instance exécutant le code.
