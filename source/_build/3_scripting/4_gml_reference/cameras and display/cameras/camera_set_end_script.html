camera_set_end_script


La description

Cette fonction peut être utilisée pour définir un script qui sera appelé à la fin de chaque image de jeu que la caméra est assignée à un port de vue visible et actif, après que tout ce qui a été rendu pour cette vue ait été rendu. Vous donnez la valeur d'ID de caméra unique (telle que renvoyée par les différents camera_create fonctions) et l'identifiant du script à appeler. L'ordre dans lequel les scripts attachés aux caméras et le rendu réel de la vue de la caméra sont les suivants:

  • Les caméras de tous les ports d'affichage visibles et actifs ont leur script de mise à jour appelé

  • Ensuite, pour chaque vue:
    • Le script de début de la caméra pour cette vue est appelé
    • Les événements de dessin sont exécutés pour cette vue
    • Le script final de la caméra est appelé
  • Passez à la vue visible active suivante et répétez...


Syntaxe:

camera_set_end_script(camera_id, script)


Arguments:

Argument La description
camera_id La valeur d'ID de caméra unique renvoyée lorsque vous avez créé la caméra.
script Le script pour exécuter chaque image de jeu


Résultats:

N/A


Exemple:

camera_set_end_script(view_camera[0], endCamera);

Le code ci-dessus définit le script de fin pour la caméra assignée à voir le port [0].
