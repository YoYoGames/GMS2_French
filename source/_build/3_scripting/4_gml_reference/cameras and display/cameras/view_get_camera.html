view_get_camera


La description

Cette fonction peut être utilisée pour récupérer la valeur d'ID de caméra unique pour la caméra assignée au port d'affichage donné (de 0 à 7). Si aucune caméra n'est assignée, la fonction retournera -1.


Syntaxe:

view_get_camera(view_port)


Arguments:

Argument La description
view_port Le port de vue à cibler (0 - 7)


Résultats:

Camera index (Real)


Exemple:

var cam = view_get_camera(0);
var cw = camera_get_view_width(cam);
var ch = camera_get_view_height(cam);
camera_set_view_pos(cam, mouse_x - (cw / 2), mouse_y - (ch / 2));

Le code ci-dessus récupérera l'identifiant de caméra pour la caméra assignée au port [0], puis changera sa position.
