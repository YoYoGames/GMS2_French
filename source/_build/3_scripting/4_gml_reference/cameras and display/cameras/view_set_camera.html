view_set_camera


La description

Cette fonction définit une caméra sur un port de vue spécifique. Vous donnez au port d'affichage à définir (de 0 à 7) et fournissez la valeur d'ID de caméra unique (telle que renvoyée par le camera_create fonctions ou lorsque vous utilisez view_get_camera() ). Si vous attribuez la valeur -1 à l'ID de caméra, vous supprimez une caméra du port d'affichage et notez que si ce port d'affichage est activé et visible, vous risquez d'obtenir des résultats imprévisibles.


Syntaxe:

view_set_camera(view_port, camera_id)


Arguments:

Argument La description
view_port Le port de vue à cibler (0 - 7)
camera_id La valeur unique d'ID de caméra retournée lorsque vous avez créé la caméra


Résultats:

N/A


Exemple:

var cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
view_set_camera(0, cam);

Le code ci-dessus créera une nouvelle caméra, puis assignera le port de visualisation [0].
