camera_set_view_size


La description

Vous pouvez utiliser cette fonction pour mettre à jour la taille de la caméra dans la pièce. Vous donnez la valeur d'ID de caméra unique (telle que renvoyée par les différents camera_create fonctions) puis donnez la largeur et la hauteur (en pixels) pour régler l'appareil photo.


Syntaxe:

camera_set_view_size(camera_id, width, height)


Arguments:

Argument La description
camera_id La valeur d'ID de caméra unique renvoyée lorsque vous avez créé la caméra.
width La largeur de la vue de la caméra en pixels
height La hauteur de la vue de la caméra en pixels


Résultats:

N/A


Exemple:

camera_set_view_size(view_camera[0], view_wport[0], view_hport[0]);

Le code ci-dessus définit la taille de la caméra de vue pour la caméra assignée au port de visualisation [0].
