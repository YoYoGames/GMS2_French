camera_set_view_speed


La description

Vous pouvez utiliser cette fonction pour mettre à jour la vitesse de la caméra dans la pièce. Vous donnez la valeur d'ID de caméra unique (telle que renvoyée par les différents camera_create fonctions), puis donnez à la vitesse x et y (horizontale et verticale) qu'elle doit se déplacer lorsqu'elle est définie pour suivre une instance donnée. La vitesse est calculée en pixels par pas et peut être réglée sur "-1" pour que la caméra bouge instantanément, mais si la caméra n'est pas réglée pour suivre une instance, les valeurs définies ici n'auront aucun effet visible.


Syntaxe:

camera_set_view_speed(camera_id, xspeed, yspeed)


Arguments:

Argument La description
camera_id La valeur d'ID de caméra unique renvoyée lorsque vous avez créé la caméra.
xspeed La vitesse (nombre de pixels par image de jeu) que la vue doit déplacer sur l'axe horizontal (x)
yspeed La vitesse (nombre de pixels par image de jeu) que la vue doit déplacer sur l'axe vertical (y)


Résultats:

N/A


Exemple:

camera_set_view_speed(view_camera[0], 5, 5);

Le code ci-dessus définit la vitesse de la caméra de vue pour la caméra assignée au port de visualisation [0].
