display_set_ui_visibility


La description

Cette fonction peut être utilisée pour afficher ou masquer l'interface utilisateur du système sur Android uniquement. La fonction nécessite que vous fournissiez un ou plusieurs indicateurs système sous forme de valeur entière. Lorsque vous utilisez plusieurs indicateurs, vous devez les fusionner en utilisant le "ou" au niveau du bit, comme illustré dans l'exemple ci-dessous. Vous pouvez trouver une liste des drapeaux du système Android ici.


Syntaxe:

display_set_ui_visibility(flags);

Argument drapeaux
flags Les indicateurs du Android à utiliser (sous forme de valeur entière).


Résultats:

N/A


Exemple:

var flags = 1024|4096;
display_set_ui_visibility(flags);

Le code ci - dessus utilisera les Android pour définir la visibilité de l'affichage des drapeaux du système « de SYSTEM_UI_FLAG_IMMERSIVE_STICKY » et « SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN ».
