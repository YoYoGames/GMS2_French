screen_save


La description

Avec cette fonction, vous pouvez enregistrer une capture d'écran du jeu telle qu'elle apparaît actuellement dans la fenêtre de jeu. Le fichier sera sauvegardé dans le répertoire de travail du jeu, qui se trouve dans Windows sous <drive>:/Users/YOURUSERNAME/AppData/Local/[Game Name]/ et sur Mac ce serait ~/Library/Application Support/[Game Name]/. L'image capturée sera le rendu final de la surface d'application du jeu ou (si la surface de l'application a été désactivée) il s'agira d'une image de tout l'affichage ou de la fenêtre. Il est important de noter que la fonction est conçue pour être appelée dans l' événement de fin de l'interface graphique Draw, ce qui devrait donner des résultats cohérents sur toutes les plateformes. Son utilisation dans tout autre événement risque de ne pas fonctionner correctement et de donner des résultats différents selon les cibles, voire les périphériques. Si vous avez besoin de plus de contrôle sur ce qui est sauvegardé, vous devriez utiliser la fonction screen_save_part.

REMARQUE: cette fonction ne fonctionne pas sur la cible HTML5.


Syntaxe:

screen_save(fname);


Argument La description
fname Le nom de fichier pour la capture d'écran à enregistrer


Résultats:

N/A


Exemple:

if keyboard_check_pressed(ord("S"))
   {
   screen_save(working_directory + "Screens\Screen_" + string(num++) + ".png")
   }

Le code ci-dessus prend une capture d'écran du jeu lorsque l'utilisateur appuie sur la touche "S". Cette capture d'écran est enregistrée dans un répertoire spécifique et numérotée à l'aide d'une variable qui est ensuite incrémentée.
