display_set_sleep_margin


La description

Cette fonction peut être utilisée pour définir la valeur de la marge de veille utilisée pour le rendu du rendu de votre jeu. Elle nécessite une valeur en millisecondes. Les valeurs par défaut sont les suivantes:

Plate-forme Millisecondes
les fenêtres 20
macOS 20
Ubuntu 20
HTML5 20
Android / Amazon Fire 4
iOS 4
Windows UWP 20
Xbox 20
PS4 20
Nintendo Switch 20


Notez que même si la méthode de synchronisation est configurée pour utiliser la synchronisation vsync, la marge de sommeil aura un effet sur le rendu du jeu. Vous devez donc faire attention lorsque vous définissez cette valeur. Pour plus d'informations sur la durée d'affichage, veuillez consulter display_set_timing_method().


Syntaxe:

display_set_sleep_margin(milliseconds);

Argument La description
milliseconds Le nombre de millisecondes pour la marge de sommeil


Résultats:

N/A


Exemple:

if display_get_timing_method() != tm_sleep
{
display_set_timing_method(tm_sleep);
if display_get_sleep_margin() != 20
    {
    display_set_sleep_margin(20);
    }
}

Le code ci-dessus vérifiera la méthode de synchronisation, puis s’il n’est pas défini sur tm_sleep il sera réglé sur cela et la limite de sommeil réglée sur 20.
