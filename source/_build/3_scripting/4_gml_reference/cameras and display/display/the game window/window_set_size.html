window_set_size

Définit la taille de la fenêtre en pixels.

Syntaxe:

window_set_size( w, h );


Argument La description
w La nouvelle largeur de la fenêtre.
h La nouvelle hauteur de la fenêtre.


Retours: N / A


La description

Avec cette fonction, vous pouvez changer la taille de la fenêtre de jeu. Si vous utilisez cette fonction pour changer la taille de la fenêtre de jeu, le contenu de la fenêtre sera mis à l'échelle avec un rapport de 1: 1 à la toile de fond qui gardera l'image claire et nette (bien que la fenêtre soit autre que la taille de la pièce ou de la vue). Par exemple, si vous avez une fenêtre de 200 x 150 pixels et que le lecteur étire la fenêtre en faisant glisser les bordures, la taille finale de la fenêtre peut être de 400 x 300 pixels, mais le dessin est toujours 200 x 150, ce qui donne une image floue comme le ratio de pixels est maintenant 1: 2. Toutefois, si vous utilisez cette fonction pour définir la taille de la fenêtre, le rapport 1: 1 est conservé, même si l'image est mise à l'échelle pour s'adapter à la nouvelle taille de la fenêtre.


Exemple:

if mouse_check_button_pressed(mb_left)
   {
   global.scale += 1;
   window_set_size(200 * global.scale, 150 * global.scale);
   }

Le code ci-dessus modifiera la taille de la fenêtre en fonction de la valeur de la variable "global.scale" lorsque le bouton de la souris est enfoncé.
