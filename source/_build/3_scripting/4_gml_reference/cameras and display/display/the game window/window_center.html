window_center

Centre la région en fonction de la zone visible ou du navigateur.

Syntaxe:

window_center();


Retours: N / A


La description

Avec cette fonction, vous pouvez centrer la fenêtre de jeu à l'écran lorsque le module cible est Windows, Ubuntu (Linux) ou MacOS, ou vous pouvez le centrer dans le navigateur si le module cible est HTML5. Cette fonction n'a aucun effet sur les autres appareils.

NOTE: Si vous avez redimensionné la fenêtre de jeu (par exemple, en passant de plein écran à fenêtré), et souhaitez centrer la nouvelle fenêtre, cette fonction doit être appelée dans une alarme [0] au moins une étape plus tard que la taille de la fenêtre changer, sinon cela ne fonctionnera pas correctement.


Exemple:

if !window_get_fullscreen()
   {
   window_center();
   }

Le code ci-dessus centrera la fenêtre de jeu dans l'affichage s'il n'est pas en mode plein écran.


Retour: La fenêtre de jeu
