window_has_focus

Renvoie si la fenêtre de jeu est active ou non.

Syntaxe:

window_has_focus();


Renvoie: Boolean


La description

Avec cette fonction, vous pouvez interroger l'état de la fenêtre (ou de l'onglet) et si elle perd le focus, la fonction retournera false sinon il reviendra true. Dans la plupart des cas, vous pouvez simplement utiliser os_is_paused Fonctionne pour tester ceci, mais dans certains cas très spécifiques (par exemple des jeux sur Chrome Apps) cette fonction ne se déclenchera pas, auquel cas vous devriez utiliser cette fonction à la place.

REMARQUE: Cette fonction est uniquement valide pour les modules HTML5, Windows, MacOS, Windows 8 (JS) et Tizen (JS).


Exemple:

if !window_has_focus()
   {
   scr_PauseGame();
   }

Le code ci-dessus va vérifier pour voir si la fenêtre de jeu est au point ou non, et si la fonction retourne false un script sera appelé.
