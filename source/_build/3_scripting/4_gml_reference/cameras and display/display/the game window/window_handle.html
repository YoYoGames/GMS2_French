window_handle

Renvoie l'identifiant de la fenêtre de jeu.

Syntaxe:

window_handle();


Retours: Pointer


La description

Avec cette fonction, vous pouvez obtenir la valeur interne de l'identifiant Windows (le HWND, un pointeur ). Cette fonction n'est vraiment utile que pour les rédacteurs qui ont besoin de window handle appeler API Windows dans le code DLL (le pointeur retourné doit être transtypé en une chaîne et ensuite dans le C ++ il suffit de le convertir en HWND). Le tableau ci-dessous montre les plateformes supportées avec ce qu'elles retournent:

Plate-forme Résultats
les fenêtres Fenêtre HWND
macOS Classe NSWindow
Ubuntu (Linux) Poignée XWindow
HTML5 ID de toile


Exemple:

win_id = windows_handle();

Le code ci-dessus va stocker l'identifiant de la fenêtre de jeu dans une variable.


Retour: La fenêtre de jeu
