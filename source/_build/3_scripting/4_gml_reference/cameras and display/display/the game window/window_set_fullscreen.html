window_set_fullscreen

Définit si le jeu est exécuté en plein écran ou non.

Syntaxe:

window_set_fullscreen(full);


Argument La description
full Que ce soit pour définir l'écran en plein écran (true) ou non (false).


Retours: N / A


La description

Avec cette fonction, vous pouvez définir la fenêtre de jeu en mode plein écran ( true ) ou pas ( false ). S'il vous plaît noter que pour la cible Mac OS, vous devez avoir décoché l'option "Démarrer en plein écran" et coché l'option "Autoriser le joueur à redimensionner la fenêtre de jeu" dans les options de jeu, sinon cette fonction échouera.


Exemple:

if mouse_check_button_pressed(mb_left)
   {
   if window_get_fullscreen()
      {
      window_set_fullscreen(false);
      }
   else
      {
      window_set_fullscreen(true);
      }
   }

Le code ci-dessus vérifie la pression d'un bouton de la souris, puis définit la fenêtre en plein écran si ce n'est pas déjà fait, ou la définit en fenêtre si elle l'est déjà.
