window_set_fullscreen

Définit si le jeu est exécuté en plein écran ou non.

Syntaxe:

window_set_fullscreen(full);


Argument La description
full Définir si l'écran doit être en plein écran (true) ou non (false).


Retours: N / A


La description

Avec cette fonction, vous pouvez configurer la fenêtre de jeu en mode plein écran ( true ) ou pas ( false ). Veuillez noter que pour la cible Mac OS, vous devez avoir décoché l'option "Démarrer en plein écran" et coché l'option "Autoriser le joueur à redimensionner la fenêtre de jeu" dans les Options de jeu, sinon cette fonction échouera. Notez également que cette fonction ne fonctionnera pas sur HTML5 à moins d'être ajoutée en tant que rappel "cliquable" (voir ici pour plus de détails).


Exemple:

if mouse_check_button_pressed(mb_left)
   {
   if window_get_fullscreen()
      {
      window_set_fullscreen(false);
      }
   else
      {
      window_set_fullscreen(true);
      }
   }

Le code ci-dessus vérifie si vous appuyez sur un bouton de la souris, puis définit la fenêtre en plein écran si ce n'est pas déjà fait, ou la définit avec window si c'est déjà fait.
