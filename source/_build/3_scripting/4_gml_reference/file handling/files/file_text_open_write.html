file_text_open_write

Syntaxe:

file_text_open_write(fname);


Argument La description
fname Le nom du fichier à écrire.


Retours: réel


La description

Cette fonction ouvre le fichier texte avec le nom de fichier indiqué uniquement en écriture (si le fichier n’existe pas, il est créé), renvoyant l’ id unique du fichier qui doit être stocké dans une variable car elle sera utilisée pour toutes les actions ultérieures. faire avec ce fichier. Notez que si le fichier ne peut pas être créé (à cause d'un nom de fichier illégal, par exemple), la fonction retournera -1.


REMARQUE: Vous ne pouvez ouvrir qu'un maximum de 32 fichiers à la fois. Vous devez également toujours fermer les fichiers lorsque vous avez terminé, car cela écrit les informations et libère la mémoire associée au fichier.

ATTENTION! Cette fonction risque de ne pas fonctionner GameMaker Studio 2 car GameMaker Studio 2 mis en bac à sable! Veuillez consulter la section sur les limites du système de fichiers pour plus d'informations.


Exemple:

var file;
file = file_text_open_write(working_directory + "level.txt");
file_text_write_string(file, level_data);
file_text_close(file);

Le code ci-dessus ouvre le fichier "level.txt" en écriture, puis écrit la chaîne stockée dans la variable "level_data" avant de refermer le fichier.
