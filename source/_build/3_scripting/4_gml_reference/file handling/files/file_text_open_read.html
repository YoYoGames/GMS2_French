file_text_open_read

Syntaxe:

file_text_open_read(fname);


Argument La description
fname Le nom du fichier à lire.


Retours: Real


La description

Cette fonction ouvre le fichier texte avec le nom de fichier indiqué pour la lecture uniquement, en retournant l'identifiant unique du fichier qui doit être stocké dans une variable car il sera utilisé pour toutes les autres actions à faire avec ce fichier. Si le fichier n'existe pas, la fonction renverra la valeur -1.


REMARQUE: Vous ne pouvez avoir qu'un maximum de 32 fichiers ouverts à la fois. Vous devriez également toujours fermer les fichiers lorsque vous avez terminé car cela libère la mémoire associée au fichier.

ATTENTION! Cette fonction peut ne pas fonctionner comme prévu car GameMaker Studio 2 est en sandbox! Veuillez consulter la section sur les limites du système de fichiers pour plus d'informations.


Exemple:

file = file_text_open_read(working_directory + "\level.txt");

Cela ouvrirait "level.txt" du même répertoire que le jeu et stockerait l'identifiant du fichier dans la variable "file".
