file_bin_read_byte

Syntaxe:

file_bin_read_byte(binfile);


Argument La description
binfile L'ID du fichier pour obtenir les données.


Retours: Real


La description

Cette fonction renvoie un octet de données à partir de la position actuelle dans le fichier avec l'ID de fichier donné. Vous fournissez la valeur d'ID de fichier, telle que renvoyée par la fonction file_bin_open.

REMARQUE: ces fonctions ne fonctionnent pas lorsque le module cible est HTML5.


Exemple:

file = file_bin_open('myfile.bin', 2);
data = file_bin_read_byte(file);
file_bin_close(file);

Cela ouvrirait un fichier du même répertoire que le jeu, et obtiendrait un octet de données avant de le refermer.
