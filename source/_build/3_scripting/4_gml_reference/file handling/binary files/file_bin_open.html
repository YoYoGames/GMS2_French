file_bin_open

Syntaxe:

file_bin_open(fname, mode);


Argument La description
fname Le nom du fichier à lire.
mode l'indicateur de ce qui peut être fait avec le fichier.


Retours: Real


La description

Cette fonction ouvrira le fichier binaire avec le nom indiqué. Le mode indique ce qui peut être fait avec le fichier:

  • 0 = lecture
  • 1 = écriture
  • 2 = lecture et écriture

Lorsque le fichier n'existe pas, il est créé et la fonction renvoie l'identifiant du fichier qui doit être utilisé dans les autres fonctions. Vous pouvez ouvrir plusieurs fichiers en même temps (32 max), mais n'oubliez pas de les fermer une fois que vous en avez fini avec eux.

ATTENTION! Cette fonction peut ne pas fonctionner comme prévu car GameMaker Studio 2 est en sandbox! Veuillez consulter la section sur les limites du système de fichiers pour plus d'informations.

REMARQUE: ces fonctions ne fonctionnent pas lorsque le module cible est HTML5.


Exemple:

file = file_bin_open('myfile.bin', 2);

Cela ouvrirait un fichier du même répertoire que le jeu et assignerait son index à la variable "file".
