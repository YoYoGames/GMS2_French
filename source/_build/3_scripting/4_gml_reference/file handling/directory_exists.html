répertoire_exists


La description

Cette fonction retournera true si le répertoire indiqué existe ou false si ça ne fait pas. Le nom spécifié doit inclure le chemin complet, pas un chemin relatif, et vous ne pouvez accéder à aucun répertoire à partir du lot de jeux car tous les jeux sont en mode bac à sable.

ATTENTION! Cette fonction risque de ne pas fonctionner GameMaker Studio 2 car GameMaker Studio 2 mis en bac à sable! Veuillez consulter la section sur les limites du système de fichiers pour plus d'informations.


Syntaxe:

directory_exists(dname)


Argument La description
dname Le nom du répertoire à rechercher.


Résultats:

Boolean


Exemple:

if directory_exists(working_directory + "Saves\")
   {
   file = file_find_first(working_directory + "Saves\*.doc", fa_readonly);
   }

Cela vérifiera si le répertoire spécifié existe, puis, le cas échéant, y retournera le premier "lecture seule" doc fichier trouvé.
