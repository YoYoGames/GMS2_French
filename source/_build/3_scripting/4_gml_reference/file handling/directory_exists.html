directory_exists

Syntaxe:

directory_exists(dname)


Argument La description
dname Le nom du répertoire à rechercher.


Renvoie: Booléen.


La description

Cette fonction retournera true si le répertoire indiqué existe ou false si ça ne fait pas. Le nom spécifié doit inclure le chemin d'accès complet, pas un chemin relatif et vous ne pouvez accéder à aucun répertoire à partir de l'ensemble de jeux, car tous les jeux sont en sandbox.


ATTENTION! Cette fonction peut ne pas fonctionner comme prévu car GameMaker Studio 2 est en sandbox! Veuillez consulter la section sur les limites du système de fichiers pour plus d'informations.


Exemple:

if directory_exists(working_directory + "\Saves\")
   {
   file = file_find_first(working_directory + "\Saves\*.doc", fa_readonly);
   }

Cela vérifiera pour voir si le répertoire spécifié existe alors, si c'est le cas, allez là et renvoyez le premier "lecture seule" doc fichier trouvé.
