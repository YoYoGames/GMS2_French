ini_write_string

Syntaxe:

ini_write_string(section, key, value);


Argument La description
section La section du.ini à écrire.
key La clé dans la section pertinente du.ini à écrire.
value La chaîne à écrire dans la destination pertinente.


Retours: N / A


La description

Vous pouvez utiliser cette fonction pour écrire une chaîne (texte) dans un fichier de données ini. Les fichiers Ini sont divisés en sections et chaque section est ensuite divisée en paires clé - valeur. Donc, un fichier ini typique ressemblerait à ceci:


Exemple:

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

Le code ci-dessus ouvre un fichier ini pour la lecture et l'écriture, puis écrit la chaîne stockée dans la variable globale "Name" dans la section "Save" avec la clé "Player" avant de refermer le fichier.
