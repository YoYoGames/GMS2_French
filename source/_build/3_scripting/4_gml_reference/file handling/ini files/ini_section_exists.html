ini_section_exists


La description

Cette fonction vérifie si une section existe dans l'ini actuellement ouvert. Ce n'est pas une vérification nécessaire pour éviter les erreurs car, lorsqu'une section n'existe pas, la lecture d'une section inexistante renverra simplement une valeur par défaut, mais il peut être utile de voir si un fichier ini a sauvegardé des données spécifiques.


Syntaxe:

ini_section_exists(section);


Argument La description
section La section du fichier.ini ouvert à vérifier.


Résultats:

Boolean


Exemple:

ini_open("savedata.ini");
if !ini_section_exists("save1")
   {
   global.savegame = "1";
   }
ini_close();

Cela définira la variable global.savegame sur "1" si "save1" N'EST PAS trouvé. Avec une boucle for, ceci pourrait être utilisé comme moyen de base de calculer le nombre de sauvegardes d’un jeu (par exemple, la création d’un système pour plusieurs sauvegardes simultanées d’un jeu).
