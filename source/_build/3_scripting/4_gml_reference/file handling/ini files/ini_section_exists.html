ini_section_exists

Syntaxe:

ini_section_exists(section);


Argument La description
section La section dans le fichier.ini ouvert à vérifier.


Renvoie: Boolean


La description

Cette fonction vérifie si une section existe dans l'ini actuellement ouvert. Ce n'est pas une vérification nécessaire pour éviter les erreurs car, lorsqu'une section n'existe pas, la lecture d'une section inexistante retournera simplement une valeur par défaut, mais il peut être utile de voir si un fichier ini a sauvegardé des données spécifiques.


Exemple:

ini_open("savedata.ini");
if !ini_section_exists("save1")
   {
   global.savegame = '1';
   }
ini_close();

Ceci placera la variable global.savegame à '1' si 'save1' n'est PAS trouvé. Avec une boucle for, cela pourrait être utilisé comme base de calcul du nombre de parties sauvegardées pour un jeu (c.-à-d. Créer un système pour plusieurs jeux de sauvegarde simultanés pour un jeu).
