ini_write_real

Syntaxe:

ini_write_real(section, key, value);


Argument La description
section La section du.ini à écrire.
key La clé dans la section pertinente du.ini à écrire.
value La valeur réelle à écrire à la destination pertinente.


Retours: N / A


La description

Vous pouvez utiliser cette fonction pour écrire une valeur (numérique) dans un fichier de données ini. Les fichiers Ini sont divisés en sections et chaque section est ensuite divisée en paires clé - valeur. Donc, un fichier ini typique ressemblerait à ceci:


Exemple:

score = 1000;
ini_open( 'savedata.ini' );
ini_write_real( 'save1', 'Score', score );
score2 = ini_read_real( 'save1', 'Score', 0 );
ini_close();

Ceci mettra le score à 1000, puis ouvrira 'savedata.ini' et écrira cette valeur à 'save1'> 'Score'. Ensuite, il mettra score2 à la valeur réelle sous 'save1'> 'score' dans celui-ci, 1000, et ferme à nouveau le.ini. S'il n'avait pas trouvé cette valeur, score2 aurait fini comme 0.
