ini_close


La description

Cette fonction doit être appelée au moment où vous avez fini de lire ou d'écrire dans un fichier ini ouvert. Si vous n'utilisez pas la fonction après avoir utilisé l'une des fonctions d'écriture ini, rien ne sera écrit sur le disque, car les informations sur le fichier sont conservées en mémoire jusqu'à ce que cette fonction soit appelée, ce qui force l'écriture. Si vous essayez d'ouvrir un ini sans en avoir préalablement fermé un autre (ou le même), vous obtiendrez également une erreur.

La fonction renverra également une chaîne contenant le fichier ini. Cette chaîne peut ensuite être enregistrée sur un serveur et / ou réutilisée avec la fonction ini_open_from_string recréer l'ini.


Syntaxe:

ini_close();


Résultats:

String


Exemple:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

Ceci ouvrira 'savedata.ini' et définira le score à la valeur sous la section "save1" avec la clé "score", puis refermera le fichier.ini. S'il n'y a pas de valeur sous "save1", "score" ou s'il n'y a pas de fichier "savedata.ini", le score sera mis à 0 et un nouveau fichier ini_file sera créé.
