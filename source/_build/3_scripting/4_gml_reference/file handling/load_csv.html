load_csv

Syntaxe:

load_csv(filename)


Argument La description
filename Le nom du fichier à ouvrir (sous forme de chaîne)


Renvoie: DS Grid Index (réel)


La description

Cette fonction chargera un fichier au format CSV et le convertira en une grille DS, en retournant la valeur d'ID unique pour la grille créée.


Exemple:

file_grid = load_csv("spreadsheet.csv");
var ww = ds_grid_get_width(file_grid);
var hh = ds_grid_get_height(file_grid);
var xx = 32;
var yy = 32;
for (var i = 0; i < ww; i++;)
    {
    for (var j = 0; j < hh; j++;)
        {
        draw_text(xx, yy, string(file_grid[# i, j]));
        yy += 32;
        }
    yy = 32;
    xx += 32;
    }

Le code ci-dessus ouvrira le fichier CSV donné et stockera la grille DS retournée dans la variable "file_grid". Cette grille est ensuite analysée dans un couple de for les lopins et les contenus dessinés à l'écran.
