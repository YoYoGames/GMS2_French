destruction_répertoire


La description

Cette fonction supprimera un répertoire portant le nom indiqué dans la zone de sauvegarde locale du bac à sable.

ATTENTION! Cette fonction risque de ne pas fonctionner GameMaker Studio 2 car GameMaker Studio 2 mis en bac à sable! Veuillez consulter la section sur les limites du système de fichiers pour plus d'informations.


Syntaxe:

directory_destroy(dname)


Argument La description
dname Le nom du répertoire à supprimer.


Résultats:

N/A


Exemple:

if directory_exists("DLC")
   {
   directory_destroy("DLC");
   }

Cela vérifiera si le répertoire spécifié existe dans le dossier de données local et, le cas échéant, il sera supprimé.
