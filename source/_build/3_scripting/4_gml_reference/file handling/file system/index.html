Fonctions du système de fichiers

Il existe dans GameMaker Studio 2 un certain nombre de fonctions qui vous permettent de manipuler les fichiers de différentes manières:

  • le fichier existe
  • file_delete
  • file_rename
  • file_copy
  • file_find_first
  • file_find_next
  • file_find_close
  • file_attributes

Les fonctions suivantes peuvent être utilisées pour changer les noms de fichier:

NOTE: Ces fonctions ne fonctionnent pas sur les fichiers réels! Ils traitent uniquement les chaînes liées aux différents éléments de chemin et doivent être utilisés conjointement avec les autres fonctions de fichier.
  • filename_name
  • filename_path
  • filename_dir
  • filename_drive
  • filename_ext
  • filename_change_ext

Enfin, il existe deux fonctions spéciales pour obtenir des noms de fichiers qui peuvent être utilisés pour charger et enregistrer des fichiers avec des filtres de type fichier:

NOTE: Ces fonctions sont Windows SEULEMENT
  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
ATTENTION! Cette fonction peut ne pas fonctionner comme prévu car GameMaker Studio 2 est en sandbox! Veuillez consulter la section sur les limites du système de fichiers pour plus d'informations.
