get_open_filename_ext

Syntaxe:

get_open_filename_ext(filter, fname, directory, caption);


Argument La description
filter Le filtre de type de fichier et / ou le nom.
fname Le nom de fichier suggéré à utiliser.
directory Le répertoire de départ de la recherche.
caption La légende de la fenêtre.


Retours: String.


La description

Cette fonction ouvre un dialogue et demande au lecteur un nom de fichier à ouvrir avec le filtre donné. Le filtre a la forme " nom1 | masque1 | nom2 | masque2 |... ", où le masque peut contenir les différentes options avec un point-virgule entre elles et vous pouvez également utiliser un "*" pour signifier n'importe quelle chaîne. Par exemple: " bitmaps | *.bmp; *. Wmf ", demanderait à l'utilisateur de sélectionner le nom d'un fichier bitmap dans le système, mais uniquement ceux qui possèdent les deux extensions spécifiées ( bmp et png ). cette fonction n'ouvre pas le fichier lui-même! Il ne renvoie qu'une chaîne avec le nom complet et le chemin du fichier. Si l'utilisateur appuie sur " Annuler ", une chaîne vide "" est renvoyée. Vous pouvez également ajouter un répertoire dans lequel lancer la recherche et une légende à afficher en haut de la fenêtre au lieu de celle par défaut.

Une chose importante à noter lors de l'utilisation de cette fonction est qu'elle vous accorde certaines autorisations pour ce fichier, pour la durée de votre partie. Ainsi, une fois que vous avez obtenu le chemin du fichier de l'utilisateur, vous pouvez y accéder encore et encore sans avoir à demander. Cependant, comme cette fonction est destinée à la lecture d'un fichier, les autorisations accordées ne sont que pour la lecture et l'enregistrement n'est pas autorisé par le système d'exploitation. Si vous avez besoin de lire et d’écrire dans le fichier, vous pouvez obtenir la permission en utilisant la fonction get_save_filename.

REMARQUE: cette fonction n’est valable que sur les cibles Windows et macOS.


Exemple:

var file;
file = get_open_filename_ext("text file|*.txt", "", working_directory, "Open a saved level");
if file != ""
   {
   file_text_open_read(file);
   }

Le code ci-dessus demandera à l’utilisateur de sélectionner un fichier texte à ouvrir à partir du working_directory puis vérifiez le nom et le chemin du fichier renvoyé et, s’il existe, ouvrira le fichier en lecture.
