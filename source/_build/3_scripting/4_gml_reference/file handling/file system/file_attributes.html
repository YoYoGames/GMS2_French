file_attributes

Syntaxe:

file_attributes(fname, attr);


Argument La description
fname Le nom du fichier à vérifier.
attr Les attributs à vérifier.


Renvoie: Booléen.


La description

Vous pouvez utiliser cette fonction pour vérifier les attributs d'un fichier donné. Vous pouvez ajouter les constantes suivantes pour voir le type de fichiers que vous voulez:

  • fa_readonly - Fichiers en lecture seule
  • fa_hidden - fichiers cachés
  • fa_sysfile - fichiers système
  • fa_volumeid - fichiers volume-id
  • fa_directory - répertoires
  • fa_archive - fichiers archivés

REMARQUE: il s'agit d'une fonction Windows uniquement.


Exemple:

if !file_attributes(file, fa_hidden) file_delete(file);

Cela vérifie un fichier pour voir s'il est caché ou non, et sinon il est supprimé.
