json_decode


La description

JSON (JavaScript Object Notation) est un format léger d’échange de données, facile à lire et à écrire, pour les utilisateurs comme pour les ordinateurs. Il est construit sur deux structures de base:

  • Une collection de paires nom / valeur, appelée un ds_map dans GameMaker Studio 2 mais également appelé "dictionnaire" ou "objet".

  • Une liste ordonnée de valeurs, appelée un ds_list dans GameMaker Studio 2 mais cela peut aussi être appelé "tableau" ou "séquence".

Avec cette fonction, vous pouvez décoder un morceau de JSON et le convertir en un fichier. ds_map, prêt à être utilisé dans GameMaker Studio 2. Si le JSON à décoder nécessite une hiérarchie de listes et de mappes dans la ds_map centrale, ceux-ci sont également décodés et créés pour vous, à l'aide des règles suivantes (notez que ces règles s'appliquent uniquement à la structure de niveau supérieur):

  • Json est une valeur unique - retourne un ds_map avec une seule entrée "default" qui est la valeur

  • Json est un tableau d'objets ou de valeurs - retourne un ds_map avec une seule entrée "default" qui est une liste de ds des objets ou des valeurs

  • Json est un objet - retourne un ds_map contenant les entrées de l'objet
REMARQUE: lors du décodage de tableaux, il existe une carte avec la clé "default" UNIQUEMENT lorsqu'un tableau est la structure de niveau supérieur et UNIQUEMENT pour ce tableau de niveau supérieur. Les listes internes décodent directement en ds_lists sans être incluses dans un ds_map.


Normalement, vous saurez quelles clés le JSON décode, mais sinon, vous pouvez utiliser le ds_map_size, ds_map_find_first et ds_map_find_next fonctions pour analyser la carte et obtenir les informations nécessaires.

REMARQUE: GameMaker Studio 2 crée les ds_maps et les listes nécessaires à partir du JSON. Pour le nettoyage, il vous suffit de supprimer la carte ou la liste de niveau supérieur. GameMaker Studio 2 supprimera automatiquement de la mémoire toutes les cartes et listes situées en dessous.


Syntaxe:

json_decode(string)


Argument La description
string La chaîne de format JSON que vous transmettez à la fonction de décodage


Résultats:

ds_map id or -1 if it fails


Exemple:

var resultMap = json_decode(requestResult);
var list = ds_map_find_value(resultMap, "default");
var size = ds_list_size(list);
for (var n = 0; n < ds_list_size(list); n++;)
   {
   var map = ds_list_find_value(list, n);
   var curr = ds_map_find_first(map);
   while (is_string(curr))
      {
      global.Name[n] = ds_map_find_value(map, "name");
      curr = ds_map_find_next(map, curr);
      }
   }
ds_map_destroy(resultMap);

Le code ci-dessus décodera une chaîne JSON et l’analysera pour générer un tableau global.
