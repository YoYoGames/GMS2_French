buffer_base64_encode

Syntaxe:

buffer_base64_encode(buffer, offset, size);


Argument La description
buffer L'index du tampon à utiliser.
offset La valeur de décalage de données.
size La taille du tampon.


Retourne: Chaîne


La description

Cette fonction va convertir les données du tampon donné en une chaîne codée au format base64. C'est un schéma d'encodage couramment utilisé qui est souvent utilisé pour tout support qui doit être stocké ou transféré sur Internet sous forme de texte, et rend la sortie illisible pour l'œil humain. Pour l'utiliser, vous devez spécifier un tampon déjà créé, la valeur de décalage (qui est le point dans le tampon où vous voulez commencer le codage) ainsi que la taille, en octets, de la mémoire tampon à encoder.


Exemple:

var b_str = buffer_base64_encode(buff, 0, buffer_get_size(buff))

Le code ci-dessus créera les données complètes stockées dans le tampon indexé par la variable "buff", et stockera la chaîne retournée dans la variable locale "b_str".
