tampon_compresse


Syntaxe:

buffer_compress(buffer, offset, size);


Argument La description
buffer L'index du tampon à compresser.
offset Le décalage dans le tampon à compresser (en octets).
size La taille de la zone tampon à compresser (en octets).


Retours: Index tampon


La description

Avec cette fonction, vous pouvez compresser une partie (ou la totalité) d'un tampon en utilisant la compression zlib. Vous fournissez l'ID du tampon à compresser (tel que retourné par buffer_create() ), le décalage dans la mémoire tampon à utiliser en octets et la taille des données de la mémoire tampon à compresser (également en octets). La fonction renverra une nouvelle valeur d'ID de mémoire tampon pour la mémoire tampon compressée ou une valeur inférieure à 0 en cas d'échec pour une raison quelconque. Cette fonction ne modifiera pas le tampon d'origine.


Exemple:

var srcBuff = buffer_create(1024, buffer_grow, 1);
buffer_write(srcBuff, global.DataString);
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff));
buffer_save(cmpBuff, "Player_Save.sav");
buffer_delete(srcBuff);
buffer_delete(cmpBuff);

Le code ci-dessus créera un tampon, puis le peuplera avec les données d'une chaîne. Ce tampon est ensuite compressé et enregistré, et les tampons source et compressé sont supprimés
