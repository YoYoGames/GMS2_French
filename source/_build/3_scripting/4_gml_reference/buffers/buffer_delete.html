buffer_delete

Syntaxe:

buffer_delete(buffer)


Argument La description
buffer L'index du tampon à supprimer.


Retours: N / A


La description

Avec cette fonction, vous pouvez supprimer un tampon précédemment créé en utilisant la fonction buffer_create de mémoire, libérer les ressources utilisées pour le créer et supprimer les données qu'il peut contenir actuellement.

Remarque: il est important de toujours supprimer toutes les ressources créées dynamiquement de la mémoire lorsque vous n'en avez plus besoin pour éviter les fuites de mémoire.


Exemple:

buffer_delete(player_buffer);

Le code ci-dessus va supprimer le tampon créé précédemment avec l'identifiant stocké dans la variable "player_buffer".
