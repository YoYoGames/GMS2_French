buffer_async_group_option

Syntaxe:

buffer_async_group_option(option, value);


Argument La description
option L'option à définir.
value La valeur à définir (peut être une chaîne ou réelle, selon l'option).


Retours: N / A


La description

Avec cette fonction, vous pouvez définir certaines options spécifiques à la plate-forme pour le groupe de tampons en cours de sauvegarde. Les options disponibles sont les suivantes:

  • " subtitle " ou " slottitle " - La valeur de cette option serait une chaîne, qui sera affichée à l'utilisateur lors de la gestion de ses données de sauvegarde dans le système d'exploitation. Ceci est important uniquement lors de la sauvegarde des données, pas lors du chargement à nouveau.

  • " showdialog " - La valeur de cette option est un booléen true ou false. Si réglé à true Lorsque vous enregistrez ou chargez, GameMaker Studio affiche l'interface utilisateur système, sinon une sauvegarde / chargement GameMaker sera effectuée sans interface utilisateur affichée. Vous n’avez normalement besoin de cela que si vous prenez en charge plusieurs emplacements de sauvegarde afin de permettre à l’utilisateur de choisir un emplacement, mais si vous ne supportez qu’un emplacement par utilisateur, définissez cette option sur false.

  • " savepadindex " - Pour cette option, vous spécifiez l' index du pad qui sauvegarde ou charge le lecteur. Le système écrit et lit les données du dossier de sauvegarde de ce lecteur.

  • " saveslotsize " - Cette option nécessite que vous spécifiiez la taille réelle en octets que vous voulez enregistrer (pour pouvoir buffer_seek et buffer_tell pour l'obtenir, par exemple). Notez qu'il n'est pas obligatoire de fournir cette valeur car toutes les sauvegardes se voient attribuer un espace minimum pré-assigné, ce qui correspond généralement à environ 10 Mo.

IMPORTANT - Cette fonction n’est actuellement valable que pour les modules cibles PSVita, PS4 et XBox One. Sur toutes les autres cibles, cela ne fera rien.


Exemple:

buffer_async_group_begin("save_folder_name");
buffer_async_group_option("showdialog", false);
buffer_async_group_option("slottitle", "Catch The Haggis Save");
buffer_async_group_option("subtitle", "All your haggis are saved here!");
save = buffer_save_async(buff, "Player_Save.sav", 0, 16384);
buffer_async_group_end();

Le code ci-dessus démarre un groupe de tampons, puis définit les options du groupe avant de définir 4 fichiers à enregistrer de manière asynchrone. La définition du groupe est alors terminée (à quel point la sauvegarde commencera).
