buffer_write

Syntaxe:

buffer_write(buffer, type, value)


Argument La description
buffer L'index du tampon dans lequel écrire.
type Le type de données à écrire dans la mémoire tampon (voir la liste des constantes ci-dessous).
value Les données à écrire.


Retours: N / A


La description

Cette fonction peut être utilisée pour écrire des données dans un tampon créé précédemment. Les données que vous écrivez doivent être en accord avec l'argument "type" de cette fonction, ce qui signifie que vous ne pouvez pas essayer d'écrire une chaîne sous la forme d'un entier non signé de 16 bits, par exemple. Les constantes suivantes peuvent être utilisées pour définir le type de données:

Constant La description
buffer_u8 Un entier non signé de 8 bits. C'est une valeur positive de 0 à 255.
buffer_s8 Un entier signé de 8 bits. Cela peut être une valeur positive ou négative de -128 à 127 (0 est considéré comme positif).
buffer_u16 Un entier non signé de 16 bits. C'est une valeur positive de 0 à 65 535.
buffer_s16 Un entier signé, 16 bits. Cela peut être une valeur positive ou négative de -32 768 à 32 767 (0 est considéré comme positif).
buffer_u32 Un entier non signé de 32 bits. C'est une valeur positive de 0 à 4 294 967 295.
buffer_s32 Un entier signé 32 bits. Il peut s’agir d’une valeur positive ou négative comprise entre -2 147 483 648 et 2 147 483 647 (0 étant considéré comme positif).
buffer_u64 Un entier non signé de 64 bits.
buffer_f16 Un float de 16 bits. Cela peut être une valeur positive ou négative dans la même plage qu'un entier signé de 16 bits. (Pas actuellement supporté!)
buffer_f32 Un float de 32 bits. Cela peut être une valeur positive ou négative dans la même plage qu'un entier signé 32 bits.
buffer_f64 Un float de 64 bits.
buffer_bool Une valeur booléenne. Ne peut être que 1 ou 0 ( true ou false )
buffer_string Une chaîne de n'importe quelle taille, finalisée avec un caractère final nul.
buffer_text Une chaîne de n'importe quelle taille, sans le caractère final final NULL.


Exemple:

buffer_seek(buff, buffer_seek_start, 0);
buffer_write(buff, buffer_s16, 0);
buffer_write(buff, buffer_s16, x);
buffer_write(buff, buffer_s16, y);

Le code ci-dessus trouve le début de la mémoire tampon avec l’id stocké dans la variable "buff". Il écrit une série de valeurs entières signées sur 16 bits.
