buffer_sizeof

Syntaxe:

buffer_sizeof(type);


Argument La description
type Le type de données à vérifier (voir la liste des constantes ici ).


Retours: N / A


La description

Cette fonction renverra la taille (en octets) de l'une des constantes de données données (listées ici ).


Exemple:

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

Le code ci-dessus calcule d'abord la taille du tampon à créer en multipliant le type de données 8bit non signé par 12 (puisque nous utiliserons le tampon pour contenir 12 morceaux de données), puis utilise cette valeur pour définir un tampon fixe à la bonne Taille.
