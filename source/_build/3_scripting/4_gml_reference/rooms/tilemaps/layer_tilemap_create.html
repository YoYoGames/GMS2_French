layer_tilemap_create

La description

Avec cette fonction, vous pouvez affecter une ressource de jeu de carreaux à une couche à utiliser comme fond de carte dans votre projet. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de layer_create() ) ou le nom du calque (sous forme de chaîne - cela aura un impact sur les performances), puis une position initiale (x, y) pour ajouter le fond de carte à la pièce, la ressource de mosaïque à utiliser, puis la largeur et la hauteur du tilemap dans les cellules (c'est-à-dire: une largeur de 20 et une hauteur de 10 créera un tilemap de 200 cellules large de 20 cellules et d'une hauteur de 10 cellules), la taille des cellules elles-mêmes étant définie par le jeu de mosaïques choisi. Il est à noter que vous ne pouvez pas placer de mosaïques à des positions négatives dans le tilemap. Par conséquent, toutes les mosaïques doivent être placées dans les espaces de cellules 0 à la largeur, 0 à la hauteur.


Syntaxe:

layer_tilemap_create(layer_id, x, y, tile set, width, height)


Arguments:

Argument La description
layer_id La valeur d'ID unique du calque à cibler (ou le nom du calque sous forme de chaîne)
x La position x du tilemap dans la pièce
y La position y du tilemap dans la pièce
tile set L'indice d'ensemble de tuiles à utiliser
width Le tilemap de largeur (en cellules)
height La hauteur du tilemap (en cellules)


Résultats:

tilemap element ID (Real)


Exemple:

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32);

Le code ci-dessus crée un nouveau calque et y ajoute un nouvel élément tilemap, en définissant la position du tilemap sur (0,0), ainsi que le jeu de mosaïques à utiliser ainsi que la largeur et la hauteur du tilemap.
