room_speed


La description

Cette variable maintient la vitesse de course de toutes les pièces (et du jeu) dans les images de jeu par seconde. Notez que ce n'est pas le FPS (images par seconde) mais plutôt le nombre d'étapes de jeu que GameMaker Studio 2 essayera de maintenir chaque seconde.

IMPORTANT! Cette variable est conservée uniquement pour Legacy Support et ne doit pas être utilisée car elle ne définit plus la vitesse pour une seule pièce, mais pour TOUTES les pièces du jeu. Pour changer la vitesse de jeu, vous devriez plutôt utiliser la fonction game_set_speed.


Syntaxe:

room_speed;


Résultats:

Real


Exemple:

alarm[0] = room_speed * 10;

Le code ci-dessus utilise room_speed pour activer une alarme pendant 10 secondes. L'utilisation de la variable de cette manière garantit que l'alarme fonctionne pendant 10 secondes, quelle que soit la vitesse définie pour la pièce.
