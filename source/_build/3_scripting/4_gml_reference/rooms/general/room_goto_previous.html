room_goto_previous


La description

Avec cette fonction, vous pouvez faire en sorte que votre jeu passe au précédent, comme indiqué dans l'arbre des ressources au moment de la compilation du jeu. Si cette pièce n'existe pas, une erreur sera lancée et le jeu sera forcé de se fermer. Notez que la pièce ne changera pas jusqu'à la fin de l'événement où la fonction a été appelée, donc tout code après que celui-ci a été appelé sera toujours exécuté si dans le même événement.


Syntaxe:

room_goto_previous();


Résultats:

N/A


Exemple:

if room_exists(room_previous(room)) room_goto_previous();

Le code ci-dessus vérifiera s'il y a une autre pièce avant celle en cours et si c'est le cas, elle ira dans cette pièce.
