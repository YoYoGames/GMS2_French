room_get_name


La description

Cette fonction peut être utilisée pour renvoyer le nom de la pièce spécifiée sous forme de chaîne. S'il vous plaît noter que ceci est seulement une chaîne et ne peut pas être utilisé pour référencer la pièce directement - pour cela vous auriez besoin de l' index de la pièce. Vous pouvez cependant utiliser cette chaîne pour obtenir l' index de la pièce en utilisant la chaîne retournée avec la fonction asset_get_index().


Syntaxe:

room_get_name(index);


Arguments:

Argument La description
index L'index de la pièce pour vérifier le nom de.


Résultats:

String


Exemple:

var roomname = room_get_name(room);
draw_text(32, 32, roomname);

Le code ci-dessus obtiendra le nom de la pièce actuelle et le dessinera sur l'écran.
