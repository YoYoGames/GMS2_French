room_set_viewport


La description

Avec cette fonction, vous pouvez définir les propriétés du port de vue pour n'importe quelle pièce de votre jeu, à l'exception de celle en cours. Vous fournissez l'index de la pièce pour définir le port d'affichage, l'index du port d'affichage (de 0 à 7), puis si le port d'affichage doit être visible au début de la pièce ou non (défini sur true pour rendre le port visible, et false autrement). Après cela, vous définissez les coordonnées x et y (correspondant à la position en haut à gauche du port d'affichage), puis la largeur et la hauteur en pixels.


Syntaxe:

room_set_viewport(rm, vind, vis, xport, yport, wport, hport);


Arguments:

Argument La description
rm L'index de la pièce à définir
vind L'index du port de vue à définir
vis La visibilité du port de vue ( true est visible, false is invisible )
xport La position x pour le port de vue dans la pièce
yport La position y du point de vue dans la pièce
wport La largeur (en pixels) du port de vue
hport La hauteur (en pixels) du port de vue


Résultats:

N/A


Exemple:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_viewport(global.myroom, 0, true, 0, 0, 640, 480);

Cela définira les propriétés du port d'affichage [0] dans la pièce indexée dans la variable "global.myroom".
