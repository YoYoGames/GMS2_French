room_goto


La description

Cette fonction vous permet d'accéder à n'importe quelle pièce de votre projet de jeu, qu'elle soit créée à l'aide de code ou dans l'arborescence des ressources. Vous fournissez la valeur unique de l'ID de pièce (stockée dans la variable pour le nom de la pièce, ou en tant que variable renvoyée par la fonction room_add. Notez que l'appel de cette fonction ne change pas instantanément les pièces, et la pièce ne changera pas jusqu'à la fin du cadre de jeu actuel (ce qui signifie que tout code après cette fonction sera toujours exécuté, comme certains événements). Cette fonction déclenche également l'événement Room End.


Syntaxe:

room_goto(index);


Arguments:

Argument La description
index L'index de la pièce à visiter.


Résultats:

N/A


Exemple:

switch (global.rm)
   {
   case 0: room_goto(rm_level1); break;
   case 1: room_goto(rm_level2); break;
   case 2: room_goto(rm_level3); break;
   }

Le code ci-dessus vérifiera une variable globale et changera de pièce en fonction de la valeur qu'il contient.
