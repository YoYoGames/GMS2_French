room_persistent


La description

Cette variable peut être utilisée pour obtenir et pour définir le drapeau persistant pour la pièce actuelle. Si défini sur true la pièce est considérée comme persistante, auquel cas chaque fois que vous quittez la pièce et revenez à nouveau, l'état des instances dans cette pièce aura été maintenu. Cependant, s'il est marqué comme false, chaque fois que vous revenez dans la pièce, il sera réinitialisé à son état initial. Vous devriez noter qu'une pièce persistante utilise beaucoup plus de mémoire qu'une pièce normale et il n'est pas recommandé d'en avoir trop dans votre jeu.


Syntaxe:

room_persistent;


Résultats:

Boolean


Exemple:

if lives < 1
   {
   room_persistent = false;
   room_goto(rm_start);
   }

Le code ci-dessus vérifie la variable "lives" et s'il est inférieur à 1, il va définir la persistance de la pièce à false puis changez de chambre.
