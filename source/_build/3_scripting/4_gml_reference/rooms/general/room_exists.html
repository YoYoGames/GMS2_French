room_exists


La description

Avec cette fonction, vous pouvez vérifier et voir si la pièce que vous spécifiez existe ou non. Cette fonction prend l' index de la pièce (un nombre réel) et non le nom de la pièce (une chaîne). Cette fonction est la plus utile lorsque vous créez dynamiquement des pièces en utilisant la fonction room_add, mais vous pouvez également utiliser les variables en lecture seule room_first et room_last ou les fonctions room_next et room_previous pour obtenir un index de pièce spécifique, ou fournir une variable qui a stocké l'index de toute autre pièce de votre jeu.


Syntaxe:

room_exists(index);


Arguments:

Argument La description
index L'index de la pièce à vérifier.


Résultats:

Boolean


Exemple:

if room_exists(global.rm[0])
   {
   room_goto(global.rm[0]);
   }

Le code ci-dessus vérifie si la pièce indexée dans le tableau "global.rm []" existe et si elle le fait, elle va dans cette pièce.
