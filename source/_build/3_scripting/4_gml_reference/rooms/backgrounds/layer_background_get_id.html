layer_background_get_id


La description

Cette fonction peut être utilisée pour récupérer la valeur d'ID unique de l'élément d'arrière-plan sur un calque. Vous fournissez l'ID de couche (que vous obtenez lorsque vous utilisez le nom de la couche avec layer_get_id() ) et la fonction renvoie la valeur d'ID associée à l'élément d'arrière-plan sur la couche. Notez que cette fonction est spécialement conçue pour être utilisée avec des arrière-plans qui ont été ajoutés dans l'EDI, comme si vous ajoutiez un arrière-plan à un calque via le code en utilisant la fonction layer_background_create, alors il retournera l'identifiant unique pour l'élément d'arrière-plan ajouté.


Syntaxe:

layer_background_get_id(layer_id)


Arguments:

Argument La description
layer_id La valeur d'ID unique de la couche à cibler


Résultats:

Background element ID value (Real)


Exemple:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_sprite(back_id, bck_Trees_Winter);

Le code ci-dessus obtiendra l'ID de couche pour le calque nommé "Background_trees", puis l'utilisera pour obtenir l'ID de l'élément d'arrière-plan sur ce calque. Cet ID est ensuite utilisé pour modifier l'image-objet de l'élément.
