layer_background_create


La description

Avec cette fonction, vous pouvez affecter une ressource d'image-objet à un calque à utiliser comme arrière-plan dans votre projet. Vous fournissez l'ID de couche (que vous obtenez lorsque vous créez la couche en utilisant layer_create() ) ou le nom du calque (en tant que chaîne - cela aura un impact sur les performances) et un index d'image-objet (qui sera le nom de l'image-objet comme indiqué dans l'arborescence), et il sera ajouté à la couche. La fonction renvoie la valeur d'ID unique de l'élément, qui peut ensuite être utilisée dans d'autres fonctions de couche pour les arrière-plans.


Syntaxe:

layer_background_create(layer_id, sprite)


Arguments:

Argument La description
layer_id La valeur d'ID unique de la couche à cibler (ou le nom de la couche sous forme de chaîne)
sprite L'index de sprite à utiliser


Résultats:

Background element ID (Real)


Exemple:

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

Le code ci-dessus crée un nouveau calque et y ajoute un nouvel élément d'arrière-plan, en définissant une image-objet comme image de fond utilisée.
