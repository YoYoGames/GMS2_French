layer_background_alpha


La description

Cette fonction contrôle l'alpha (transparence) de l'image-objet en arrière-plan. Vous donnez l'ID d'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan en utilisant layer_background_create() ou lorsque vous utilisez la fonction layer_background_get_id() ), puis définissez la valeur alpha à utiliser. Alpha peut être compris entre 0 (entièrement transparent) et 1 (entièrement opaque) avec la valeur alpha par défaut pour l'élément d'arrière-plan étant 1. Notez que si la couche à laquelle l'élément d'arrière-plan a été affecté n'est pas visible - ou l'élément lui-même invisible - vous ne verrez aucune différence avec cette fonction tant que la couche ou l'élément n'a pas été rendu visible à nouveau.


Syntaxe:

layer_background_alpha(background_element_id, alpha);


Arguments:

Argument La description
background_element_id La valeur d'ID unique de l'élément d'arrière-plan à modifier
alpha L'alpha pour l'image-objet d'arrière-plan, de 0 à 1 (la valeur par défaut est 1)


Résultats:

N/A


Exemple:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_alpha(back_id, random(1));

Le code ci-dessus obtient la valeur ID de l'arrière-plan assigné à la couche "Background_sky", puis définit son alpha à une valeur aléatoire entre 0 et 1.
