layer_background_speed


La description

Cette fonction peut être utilisée pour définir la vitesse d'une image-objet d'animation affectée à un élément d'arrière-plan. Vous donnez l'ID d'élément d'arrière-plan (que vous obtenez lorsque vous créez un élément d'arrière-plan à l'aide de layer_background_create() ou quand vous utilisez la fonction layer_background_get_id() ), puis définissez la vitesse d'animation. Si l'EDI a défini la vitesse d'animation du sprite utilisé comme image par image, la valeur de la vitesse indiquée est un multiplicateur (1 étant la valeur par défaut). Si vous définissez cette valeur sur 0.5, la vitesse d'animation sera deux fois plus petite. le doubler Cependant, si l'EDI a défini la vitesse d'animation du sprite utilisé comme image par seconde, la valeur de la vitesse donnée est une valeur littérale en images par seconde. Par conséquent, définir cette valeur sur 5, par exemple, définira l'animation en boucle. 5 images par seconde pendant le jeu. Si l'élément d'arrière-plan n'a pas d'image-objet attaché ou si l'image-objet utilisée n'a pas de sous-image, cette fonction n'aura aucun effet.


Syntaxe:

layer_background_speed(background_element_id, speed);


Arguments:

Argument La description
background_element_id La valeur d'identifiant unique de l'élément d'arrière-plan à modifier
speed Le multiplicateur de vitesse / valeur à utiliser pour l'élément


Résultats:

N/A


Exemple:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_speed(back_id, 0.5);

Le code ci-dessus va obtenir l'ID de la couche nommée "Background_water", puis l'utiliser pour obtenir l'ID de l'élément d'arrière-plan de cette couche. Cet identifiant est ensuite utilisé pour modifier la vitesse de l'image de l'élément.
