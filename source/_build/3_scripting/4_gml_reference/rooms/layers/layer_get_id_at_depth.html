layer_get_id_at_depth


La description

Vous pouvez utiliser cette fonction pour obtenir les ID de toutes les couches affectées à une profondeur spécifique. Vous donnez la profondeur à vérifier et la fonction retournera un tableau avec une ou plusieurs entrées selon qu'il existe ou non des couches à la profondeur donnée. S'il n'y a pas de couches à la profondeur donnée, le tableau aura une seule entrée à la position [0] avec la valeur -1, mais s'il y a des couches à la profondeur, une entrée sera faite dans le tableau pour chaque couche trouvée - la valeur d'entrée sera l'identifiant unique d'une couche.


Syntaxe:

layer_get_id_at_depth(depth)


Arguments:

Argument La description
depth La profondeur à partir de laquelle vérifier et récupérer les identifiants de couche.


Résultats:

Array


Exemple:

var a = layer_get_id_at_depth(0);
if a[0] != -1
    {
    for (var i = 0; i < array_length_1d(a); i++;)
        {
        layer_destroy(a[i]);
        }
    }

Le code ci-dessus récupère les données sur les couches avec une profondeur de 0. Une vérification est effectuée pour voir s'il existe des couches à cette profondeur, et s'il y en a, le tableau retourné est analysé et chacune des couches trouvées est détruite.
