layer_get_element_layer


La description

Vous pouvez utiliser cette fonction pour obtenir l' ID de couche sur lequel l'élément donné est activé. Vous fournissez la valeur de l'ID de l'élément unique (par exemple, renvoyé la fonction qui a créé l'élément ou l'éditeur de pièce) et la fonction renvoie l'ID unique du calque sur lequel l'élément est trouvé. Si l'ID d'élément donné n'est pas valide, la fonction renverra -1.


Syntaxe:

layer_get_element_layer(element_id)


Arguments:

Argument La description
element_id La valeur ID unique de l'élément pour obtenir la couche de


Résultats:

Layer ID or -1 if the element is invalid (ie: does not exist)


Exemple:

element = layer_get_element_layer(asset_1);

Le code ci-dessus obtient l'identifiant de couche pour l'élément avec l'ID stocké dans la variable "asset_1" et le stocke dans une variable.
