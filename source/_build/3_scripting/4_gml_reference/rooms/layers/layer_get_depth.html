layer_get_depth


La description

Vous pouvez utiliser cette fonction pour obtenir la valeur de profondeur associée à un calque donné. Vous fournissez l'ID de la couche (que vous obtenez lorsque vous créez la couche à l'aide de layer_create() ) ou le nom du calque (sous forme de chaîne - cela aura un impact sur les performances) et la fonction retournera la profondeur de la couche sous forme de nombre réel. Notez que la profondeur est définie comme étant le "plus éloigné" de la caméra et le plus "rapproché" de la caméra. Ainsi, si vous avez trois calques avec des profondeurs de -100, 0, 100, les calques seront dessinés dans l’ordre 100, 0, -100, de sorte que la couche "supérieure" (c'est-à-dire la plus proche de la vue de la caméra et ainsi dessinée par-dessus tout le reste) sera la couche avec la profondeur -100.

L'image suivante montre un schéma du fonctionnement de la profondeur pour les calques:

Notez que si vous fournissez une valeur d'ID de couche non valide, vous obtiendrez une erreur.


Syntaxe:

layer_get_depth(layer_id)


Arguments:

Argument La description
layer_id La valeur d'identifiant unique du calque pour obtenir la profondeur de (ou le nom du calque sous forme de chaîne)


Résultats:

Real


Exemple:

if layer_get_depth(global.temp_layer) != -100
   {
   layer_destroy(global.temp_layer);
   global.temp_layer = layer_create(-100);
   }

Le code ci-dessus vérifie la profondeur d'un ID de couche stocké dans une variable globale. S'il ne correspond pas à -100, il détruit la couche et la recrée à une profondeur de -100.
