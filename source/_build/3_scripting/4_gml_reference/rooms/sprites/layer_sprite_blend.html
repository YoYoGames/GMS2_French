layer_sprite_blend


La description

Cette fonction contrôle le fondu (ou la "teinte") du sprite de l’image-objet et la valeur par défaut est -1 (ce qui représente la constante c_white, qui peut également être utilisé). Toute autre valeur (y compris les constantes de couleur internes telles que c_red, ou c_aqua ) mélangera la couleur spécifiée avec l’image-objet originale. Vous donnez l'ID d'élément sprite (que vous obtenez lorsque vous créez un élément sprite en utilisant layer_sprite_create() ou quand vous utilisez la fonction layer_sprite_get_id() ), puis définissez la couleur de fusion à utiliser. Ci-dessous, vous pouvez voir un exemple d'image-objet qui a été mélangé avec différentes couleurs: Veuillez noter que vous devez essayer de limiter le mélange sur la plate-forme HTML5 (sauf si vous utilisez WebGL), car chaque image-objet mélangée doit être mise en cache séparément. Par conséquent, le fait d'avoir plusieurs images-objets mélangées peut nuire aux performances (vous pouvez également définir la taille du cache à l'aide de la fonction sprite_set_cache_size() ).


Syntaxe:

layer_sprite_blend(sprite_element_id, blend);


Arguments:

Argument La description
sprite_element_id La valeur d'ID unique de l'élément de l'image-objet à modifier
blend La couleur à mélanger avec l’image-objet de l’image-objet (valeur par défaut: c_white )


Résultats:

N/A


Exemple:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_blend(spr_id, c_gray);

Le code ci-dessus obtient l'ID de l'image-objet "Nuages" affecté à la couche "Asset_sky", puis la colore en une couleur.
