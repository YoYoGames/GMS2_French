physics_particle_set_density

Définissez la densité de toutes les particules dans une simulation physique.

Syntaxe:

physics_particle_set_density(density)


Argument La description
density La densité de la fixation des particules.


Retours: N / A


La description

Avec cette fonction, vous pouvez définir la densité des particules dans une simulation physique. La définition de la densité de la particule aura un impact direct sur son inertie ainsi que sur la façon dont elle réagit aux collisions, donc si vous faites une petite particule avec une densité élevée, elle aura une très grande masse, mais si vous définissez une grande particule avec une faible densité, elle aura une masse beaucoup plus petite. Cette fonction a une portée globale, en ce sens qu'elle va changer la densité non seulement pour les nouvelles particules créées après le changement, mais aussi pour celles déjà présentes dans la simulation.


Exemple:

physics_particle_set_radius(15);
physics_particle_set_density(0.5);
physics_particle_set_damping(1);
physics_particle_set_gravity_scale(1);

Le code ci-dessus définira les propriétés de base pour toutes les particules dans la simulation.
