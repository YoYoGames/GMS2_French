phy_dynamic

Retourne si l'instance est dynamique ou non.

Syntaxe:

phy_dynamic;


Renvoie: Boolean


La description

Une instance dynamique est celle qui est entièrement simulée dans le monde de la physique et cette variable en lecture seule retournera true si l'instance en cours de vérification est entièrement simulée ou false si ce n'est pas le cas.


Exemple:

if other.phy_dynamic
   {
   with (other)
      {
      var dir;
      dir = point_direction(x, y, other.x, other.y);
      physics_apply_impulse(x, y, x + lengthdir_x(100, dir), y + lengthdir_y(100, dir));
      }
   }

Le code ci-dessus crée une nouvelle instance et la définit ensuite comme une "puce" dans le monde de la physique avant de lui donner une impulsion massive le long de l'axe des ordonnées.
