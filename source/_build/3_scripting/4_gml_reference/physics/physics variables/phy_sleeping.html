phy_sleeping

Renvoie l'état actif de l'instance.

Syntaxe:

phy_sleeping;


Renvoie: Boolean


La description

Cette variable en lecture seule renvoie si l'instance est actuellement "en veille" ( true ) ou pas ( false ), Une instance «dormante» est une instance qui n'est activement engagée dans aucune simulation physique. GameMaker Studio 2 mettra les objets en veille pour sauvegarder les cycles de simulation lorsqu'une instance est au repos et non en collision avec une autre instance.


Exemple:

if phy_sleeping
   {
   instance_destroy();
   }

Le code ci-dessus vérifie si l'objet est activement simulé et si ce n'est pas le cas, il est détruit.
