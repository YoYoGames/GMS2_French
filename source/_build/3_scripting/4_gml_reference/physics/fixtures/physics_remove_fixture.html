physics_remove_fixture

Cette fonction supprime un appareil d'une instance.

Syntaxe:

physics_remove_fixture(id, fixture)


Argument La description
id L'ID de l'instance pour supprimer le projecteur de
fixture L'ID de l'appareil à supprimer de l'instance


Retours: N / A


La description

Cette fonction supprime (ou "désolidarise") un appareil d'une instance ou d'instances. Il nécessite l'unique "id" de l'appareil lié (tel que renvoyé par la fonction physics_fixture_bind et il supprimera toutes les propriétés physiques actuellement définies pour l'instance, ce qui vous permettra de redéfinir un nouvel appareil et de le lier à l'instance. De cette façon, vous pouvez modifier les propriétés physiques des instances sans avoir à les détruire et à les recréer.


Exemple:

physics_remove_fixture(id, my_fix);

Le code ci-dessus va supprimer le fixture avec le "id" stocké dans la variable "my_fix" de l'instance.
