physics_set_density

Cette fonction définit la valeur de densité pour un appareil lié.

Syntaxe:

physics_set_density(fixture, density)


Argument La description
fixture l'identifiant du luminaire relié
density la nouvelle valeur de densité à appliquer


Retours: N / A


La description

Lorsque vous liez un projecteur à une instance en utilisant physics_fixture_bind cela retourne un "id" pour l'appareil lié. Vous pouvez utiliser cet identifiant pour définir la valeur de densité de l'appareil lié, et non l'appareil "de base", à tout moment en utilisant cette fonction.


Exemple:

var density = physics_get_density(fix_id);
physics_set_density(fix_id, density - 0.1);

Le code ci-dessus obtient la valeur de densité actuelle pour les propriétés physiques liées de l'instance, puis les définit sur une valeur différente.
