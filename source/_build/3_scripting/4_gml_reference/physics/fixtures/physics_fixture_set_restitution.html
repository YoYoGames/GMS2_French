physique_fixture_set_restitution

Cette fonction définit la valeur de restitution d'un appareil.

Syntaxe:

physics_fixture_set_restitution(fixture, restitution)


Argument La description
fixture L'index de l'appareil
restitution La restitution du luminaire (généralement entre 0 et 1)


Retours: N / A


La description

En physique, la restitution est définie comme «le retour d'un objet ou d'un système à son état initial après déformation élastique», mais comme les appareils de GameMaker Studio 2 sont des corps rigides et ne peuvent pas être déformés, la restitution est vraiment une façon de dire "Bouncy" l'appareil est. Ce paramètre affecte la valeur de rebondissement d'un objet lorsqu'il entre en collision avec d'autres objets et dépend d'autres forces agissant sur l'instance telles que la gravité et la friction. Il est généralement compris entre 0 et 1 (des valeurs supérieures peuvent être utilisées). mais peut donner des résultats imprévisibles). Voici une illustration de la façon dont cela fonctionne:


Exemple:

physics_fixture_set_restitution(fix_Ball, 0.9);

Le code ci-dessus fixera la restitution de l'appareil indexé dans "fix_ball" à 0.9.
