physics_fixture_set_edge_shape

Cette fonction définit la forme de l'appareil à celle d'un cercle avec le rayon spécifié.

Syntaxe:

physics_fixture_set_edge_shape(fixture, local_x1, local_y1, local_x2, local_y2)


Argument La description
fixture l'index de l'appareil
local_x1 commencer x position pour le bord
local_y1 commencer la position y pour le bord
local_x2 position x fin pour le bord
local_y2 End y Position pour le bord


Retours: N / A


La description

Cette fonction définit une forme d'appareil "en bordure". Une forme d'arête est simplement une ligne qui va générer une collision quand d'autres fixtures la survolent, et peut être très utile pour générer (par exemple) un terrain, ou pour créer des bordures autour d'une pièce. La position du bord est définie en utilisant l' espace local, c'est-à-dire que la position x / y de l'instance est considérée (0,0), cela doit être pris en compte lors de la création (dans l'exemple de code ci-dessous été placé à (0,0) dans la pièce pour éviter les complications).


Exemple:

var xx = 0;
var y1 = room_height - 100;
var y2 = room_height - 50 - irandom(100);
for (var i = 0; i < 10; i++;)
   {
   var fix = physics_fixture_create();
   physics_fixture_set_edge_shape(fix, xx, y1, xx + 50, y2);
   physics_fixture_bind(fix, id);
   physics_fixture_delete(fix);
   xx += 50;
   y1 = y2;
   y2 = room_height - 50 - irandom(100);
   }

Le code ci-dessus va créer une ligne de luminaires "edge" avec une variété de hauteurs sur toute la longueur de la pièce.
