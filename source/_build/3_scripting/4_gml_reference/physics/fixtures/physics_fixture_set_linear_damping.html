physics_fixture_set_linear_damping

Cette fonction définit la valeur d'amortissement linéaire d'un appareil.

Syntaxe:

physics_fixture_set_linear_damping(fixture, damping)


Argument La description
fixture l'index de l'appareil
damping l'amortissement de l'appareil, généralement entre 0 et 1


Retours: N / A


La description

L'amortissement est utilisé pour réduire la vitesse du monde physique des instances, et diffère du frottement en ce que la friction ne se produit que lorsque deux instances avec des appareils sont en contact. L'amortissement est également beaucoup moins cher à simuler que le frottement, mais veuillez noter que l'amortissement ne remplace pas le frottement; les deux effets peuvent et doivent être utilisés ensemble. Voici une image pour illustrer l'amortissement:

Les paramètres d'amortissement doivent être compris entre 0 et l'infini, 0 signifiant pas d'amortissement et infini signifiant amortissement complet. Normalement, vous utiliserez une valeur d'amortissement comprise entre 0 et 1, mais vous pouvez utiliser n'importe quelle valeur non négative si nécessaire.


Exemple:

physics_fixture_set_linear_damping(fix_Ball, 0.1);

Le code ci-dessus va régler l'amortissement linéaire de l'appareil indexé dans "fix_ball" à 0.1.
