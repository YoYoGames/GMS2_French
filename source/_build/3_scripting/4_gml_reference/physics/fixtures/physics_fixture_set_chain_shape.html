physics_fixture_set_chain_shape

Cette fonction définit la forme de l'appareil à celle d'une chaîne.

Syntaxe:

physics_fixture_set_chain_shape(fixture, loop)


Argument La description
fixture L'index de l'appareil
loop Que ce soit pour fermer la chaîne ( true ) ou pas ( false ).


Retours: N / A


La description

Cette fonction définit une forme de "chaîne". Un montage de chaîne est composé d'un certain nombre de points, qui sont ensuite connectés ensemble en utilisant des formes de bord pour les joindre. La fonction elle-même prend l'index (ID) de l'appareil et peut dire à l'appareil de faire une boucle ou non. Réglage de ceci à true connectera le dernier point au premier point avec un bord, tout en le réglant sur false ne le fera pas. Notez que cette fonction prépare le projecteur à accepter les points requis pour créer la chaîne, et ceux-ci doivent être ajoutés après avoir appelé cette fonction en utilisant physics_fixture_add_point, comme vous le feriez pour construire un polygone.

Lorsque vous créez un montage en chaîne, vous devez lui donner au moins deux points, mais vous n'êtes pas limité au nombre de points suivants que vous pouvez ajouter pour constituer l'appareil fini.


Exemple:

var xx = -100;
var yy = room_height / 2;
var fix = physics_fixture_create(); physics_fixture_set_chain_shape(fix, false);
for (var i = 0; i < 10; i++;)
   {
   physics_fixture_add_point(fix, xx, yy);
   xx += 50 + random(150);
   yy = (room_height / 2) - 250 + irandom(500);
   }
physics_fixture_bind(fix, id);

Le code ci-dessus va créer une ligne de luminaires "edge" avec une variété de hauteurs sur toute la longueur de la pièce.
