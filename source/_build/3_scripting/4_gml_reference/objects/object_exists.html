object_exists

Retourne si un objet avec l'index donné existe ou non.

Syntaxe:

object_exists(obj);


Argument La description
obj L'index de l'objet à vérifier.


Renvoie: Boolean


La description

Cette fonction renvoie si un objet avec l'index spécifié existe ou non. Notez que cela vérifie si un objet est présent dans l'arbre des ressources, et pas réellement dans la salle de jeux. Pour cela, vous devriez utiliser la fonction instance_exists.


Exemple:

if object_exists(obj_temp)
   {
   with (obj_Temp) scr_add_event();
   }

Le code ci-dessus vérifie la variable "obj_temp" pour s'assurer qu'il stocke l'index d'un objet et ensuite, s'il le fait, il fait exécuter un script à l'objet.
