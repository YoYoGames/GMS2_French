object_get_parent

Renvoie le parent de l'objet spécifié.

Syntaxe:

object_get_parent(obj);


Argument La description
obj L'index de l'objet à vérifier


Retours: Real


La description

Cette fonction vous donnera l'object_index de tout parent qui a été assigné à l'objet spécifié, ou alors retourner -100 pour montrer que l'objet n'a aucun parent assigné, ou -1 si l'objet en cours de vérification n'existe pas. Pour plus d'informations sur les parents, consultez la section sur l' éditeur d'objets.


Exemple:

par = object_get_parent(object_index);

Le code ci-dessus obtiendra le parent de l'index d'objet pour l'instance exécutant le code et stockera la valeur de retour dans la variable "par".
