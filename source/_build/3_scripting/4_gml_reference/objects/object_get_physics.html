object_get_physics

Renvoie si un objet est marqué comme activé physique ou non.

Syntaxe:

object_get_physics(obj);


Argument La description
obj L'index de l'objet à vérifier.


Renvoie: Boolean


La description

Cette fonction vous indiquera si l'objet que vous vérifiez a été marqué comme "physique activé", renvoyant true ou non, retour false.


Exemple:

if object_get_physics(object_index)    {    phy_active = true;    }

Le code ci-dessus va vérifier l'instance qui l'exécute pour voir si l'objet à partir duquel il a été créé est activé pour la physique, et si c'est le cas, il active la simulation physique pour l'instance.
