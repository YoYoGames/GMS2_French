object_get_sprite

Renvoie le sprite_index de l'objet spécifié.

Syntaxe:

object_get_sprite(obj);


Argument La description
obj L'index de l'objet à vérifier


Retours: réel


La description

Cette fonction vous dira si l'objet que vous vérifiez a un sprite ou non, et si c'est le cas, il retournera l'index de cette image-objet, ou -1 si ce n'est pas le cas. S'il vous plaît noter que ce n'est pas une fonction d'instance! Vous pouvez avoir un objet sans sprite alors qu'une instance de ce même objet peut en avoir une et vice-versa, ou bien ils peuvent même avoir des sprites différents. Vous pouvez définir un sprite d'instances individuelles à l'aide du sprite_index variable d'instance.


Exemple:

if sprite_index != object_get_sprite(object_index) sprite_index = object_get_sprite(object_index);

L'exemple ci-dessus vérifie l'index d'image-objet de l'instance par rapport à l'image-objet de l'objet-index de l'instance. Si elles ne sont pas les mêmes, il affectera le même sprite que celui de l'index d'objet à l'instance.
