event_perform

Effectue un événement donné de l'instance appelante.

Syntaxe:

event_perform(type, numb);


Argument La description
type Le type d'événement à effectuer. Voir ci-dessous.
numb Constante ou valeur d'événement spécifique, le cas échéant (sinon, utilisez simplement 0).


Retours: N / A


La description

Cette fonction effectuera l'événement spécifié, avec l'argument désigné, pour l'instance qui exécute le code. Il y a beaucoup d'options ici qui permettent une simulation complète de tous les événements possibles, mais notez que ceci exécute littéralement tout le code de l'événement et que le jeu ne modifiera rien pour le faire se déclencher manuellement, par exemple si vous choisissez d'utiliser un clavier. appuyez sur événement, l'événement sera déclenché mais la touche correspondante ne sera pas reconnue comme ayant été appuyée. Ou si vous effectuez un événement d'alarme, le compteur d'alarmes ne sera pas mis à -1 mais continuera plutôt à décompter.

Exemple étendu

Pour vous aider à comprendre comment cette fonction peut être utilisée, reportez-vous aux exemples suivants:

event_perform(ev_keypress, ord("W"));

Ceci exécuterait l'événement associé à la touche "W" du clavier en appuyant sur la touche de vérification (sans générer d'appuyer sur le clavier).

event_perform(ev_step, ev_step_begin);

Ceci exécutera l'événement Begin Step (s'il est appelé à partir de l'un des événements d'étape, le code de l'événement begin step sera exécuté deux fois).

event_perform(ev_create, 0);

Ceci effectuerait l'événement Create de l'instance.
