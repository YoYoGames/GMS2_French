event_action

L'index de l'action en cours d'exécution.

Syntaxe:

event_action;


Retours: Real


La description

Cette variable en lecture seule a renvoyé l'index de l'action en cours d'exécution, commençant à 0 sur les versions précédentes de GameMaker. Cependant, il s'agit maintenant d'une fonction obsolète dans GameMaker Studio 2. Il a été laissé pour le support hérité seulement, et retournera toujours 0 car toutes les actions sont concaténées ensemble pour améliorer la vitesse d'exécution.


Exemple:

num = event_action;

Le code ci-dessus stocke le numéro d'action en cours dans la variable "num".
