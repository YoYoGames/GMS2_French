object_get_mask

Renvoie le mask_index de l'objet spécifié.

Syntaxe:

object_get_mask(obj);


Argument La description
obj L'index de l'objet à vérifier


Retours: réel


La description

Cette fonction vous dira si l'objet que vous vérifiez a un index de masque ou non, et si c'est le cas, il retournera l'index de ce masque, ou -1 s'il ne l'est pas. S'il vous plaît noter que ce n'est pas une fonction d'instance! Vous pouvez avoir un objet sans masque alors qu'une instance de ce même objet peut en avoir une et vice-versa, ou même des masques différents. Vous pouvez définir un index de masque d'instances individuelles à l'aide du mask_index variable d'instance.


Exemple:

if mask_index != object_get_mask(object_index) mask_index = object_get_mask(object_index);

L'exemple ci-dessus vérifiera l'index du masque de l'instance par rapport au masque de l'objet object_index de l'instance. Si elles ne sont pas identiques, le même masque sera attribué à celui que l'index d'objet a pour l'instance.
