object_set_visible

Définit la visibilité de l'objet donné.

Syntaxe:

object_set_visible( index, vis );


Argument La description
index L'indice de l'objet à changer.
vis La nouvelle visibilité de l'objet (true = visible, false = invisible).


Retours: N / A


La description

Avec cette fonction, vous pouvez marquer un objet spécifique comme visible ou non (pour plus d'informations sur l'indicateur "visible", voir Définition d'objets ). Cela signifie que toutes les occurrences de cet objet créées après la modification du visible seront créées avec ce nouvel état, alors que les instances déjà présentes dans la pièce ne seront pas affectées.

S'il vous plaît noter que ce n'est pas une fonction d'instance! Vous pouvez définir le drapeau visible des instances individuelles en utilisant visible variable et ainsi avoir dix instances toutes marquées comme visibles même si l'objet en question a l'indicateur mis à false dans l'EDI, et même si vous changez le drapeau visible de l'objet à vrai en utilisant cette fonction, toutes les instances qui sont actuellement dans la pièce resteront tels qu'ils étaient, et seules les instances créées après l'appel de la fonction commenceront avec le marquage visible comme vrai.


Exemple:

if !object_get_visible(obj_Block) object_set_visible(obj_Block, true);

Le code ci-dessus vérifie le drapeau visible de l'objet "obj_Block" et s'il est faux, il est changé pour être marqué comme vrai.
