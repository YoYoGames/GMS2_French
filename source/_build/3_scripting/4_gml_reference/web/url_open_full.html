url_open_full

Syntaxe:

url_open_full(url, target, options);


Argument La description
url L'URL (adresse du site Web) vers laquelle créer un lien.
target C'est la zone cible pour ouvrir l'URL (voir description).
options Options de navigateur standard (voir description).


Retours: N / A


La description

Cela ouvrira l'URL spécifiée sur le navigateur du périphérique cible choisi ou, si vous utilisez le module HTML5, dans le navigateur actuellement ouvert. Le paramètre "cible" que vous spécifiez est identique à la valeur "nom" standard de JavaScript lorsque vous utilisez la commande open() méthode (sachez que tout sauf _self peut entraîner le blocage du navigateur ou demander à l'utilisateur s'il le permet) et les "options" sont identiques au paramètre JavaScript "specs" permettant de contrôler les propriétés de la nouvelle fenêtre / tab devrait afficher (tous les navigateurs ne peuvent pas supporter toutes les fonctionnalités).

Les cibles valides sont:

"_self": Ouvre le document lié dans le même cadre que celui sur lequel vous avez cliqué (valeur par défaut).
"_blank": ouvre le document lié dans une nouvelle fenêtre ou un nouvel onglet (cela ne fonctionnera pas si les fenêtres contextuelles sont bloquées par l'utilisateur. Dans ce cas, vous pouvez utiliser le clickable_* fonctions à la place).
"_parent": ouvre le document lié dans le cadre parent.
"_top": ouvre le document lié dans le corps entier de la fenêtre.

Les options valides sont:

"height = [px]": La hauteur de la fenêtre, la valeur minimale étant 100.
"width = [px]": La largeur de la fenêtre, la valeur minimale étant 100.
"left = [px]": La position de gauche de la fenêtre.
"top = [px]": position la plus haute de la fenêtre (IE uniquement).
"location = [boolean]": Affiche ou non le champ d'adresse (la valeur par défaut est 1).
"menubar = [boolean]": Affiche ou non la barre de menus (la valeur par défaut est 1).
"resizable = [boolean]": Indique si la fenêtre est redimensionnable (la valeur par défaut est 1).
"scrollbars = [boolean]": Affiche ou non les barres de défilement (la valeur par défaut est 1).
"status = [boolean]": Ajout ou non d'une barre d'état (la valeur par défaut est 1).
"titlebar = [boolean]": Affiche ou non la barre de titre. Ceci est ignoré sauf si l'application appelante est une application HTML ou une boîte de dialogue sécurisée (la valeur par défaut est 1);
"toolbar = [boolean]": Affiche ou non la barre d'outils du navigateur (la valeur par défaut est yes).


Exemple:

url_open_full( 'http://yoyogames.com', '_blank', 'resizable=0, height=200, scrollbars=0');

Cela ouvrirait la page YoYo Games accueil de YoYo Games dans une nouvelle fenêtre non redimensionnable, d'une hauteur de 200 pixels et sans barres de défilement.
