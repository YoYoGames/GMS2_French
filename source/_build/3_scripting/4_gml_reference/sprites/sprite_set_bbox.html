sprite_set_bbox


La description

Cette fonction peut être utilisée pour définir les valeurs du cadre de sélection pour une image-objet. Vous fournissez l'indice de sprite à utiliser, puis les valeurs de gauche, haut, droite et bas pour les positions du cadre de sélection. Les positions sont des valeurs absolues, où la position (0, 0) correspond au coin supérieur gauche de l'image-objet, quel que soit le décalage de l'image-objet, les pixels "vides" que cette image-objet peut avoir ou l'endroit où elle est dessinée. pièce.

REMARQUE: Cette fonction affecte la ressource de l'image-objet , de sorte que toutes les instances ultérieures avec cette image-objet auront le même cadre de sélection.


Syntaxe:

sprite_set_bbox(ind, left, top, right, bottom);

Argument La description
ind L'index de l'image-objet sur lequel définir le cadre de sélection.
left Le côté gauche du cadre de sélection
top Le haut de la boîte englobante.
right Le côté droit du cadre de sélection
bottom Le fond du cadre de sélection.


Résultats

N/A


Exemple:

if sprite_get_bbox_mode(sprite_index) == 2
    {
    left = irandom(sprite_width / 2);
    right = irandom((sprite_width / 2) + irandom(sprite_width / 2));
    top = irandom(sprite_height / 2);
    bottom = irandom((sprite_height / 2) + irandom(sprite_height / 2));
    sprite_set_bbox(sprite_index, left, top, right, bottom);
    }

Le code ci-dessus vérifiera le mode boîte englobante de l’image-objet assignée à la sprite_index et s'il est réglé sur manual sa boîte de sélection sera alors modifiée.
