sprite_flush_multi


La description

Avec cette fonction, vous pouvez supprimer un nombre quelconque de pages de texture pour les sprites donnés de la mémoire de texture. Vous fournissez les index des images-objets sous forme de tableau et les pages de texture auxquelles elles sont attribuées seront supprimées de la mémoire de texture. Notez que si l'une des pages de texture est utilisée ailleurs dans la salle (par exemple, une image-objet ou un arrière-plan, etc.), vous risquez d'obtenir un impact mineur sur les performances, car la page est immédiatement rechargée dans la mémoire. prises lors de l'utilisation de cette fonction. Notez que la fonction retournera -1 si flush n'est pas pris en charge pour les ressources choisies ou si la plate-forme cible est HTML5, ou elle renverra 0 si tout a fonctionné correctement.


Syntaxe:

sprite_flush_multi(array);

Argument La description
array Tableau avec les indices de sprite à effacer


Résultats:

Int (-1 or 0)


Exemple:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_flush_multi(spr_a);

Le code ci-dessus crée un tableau où chaque élément contient un index de sprite. Ce tableau est ensuite utilisé pour effacer ces textures de sprite de la mémoire.
