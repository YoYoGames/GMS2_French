sprite_get_yoffset


Syntaxe:

sprite_get_yoffset(index);

Argument La description
index L'index du sprite pour trouver le yoffset de.


Résultats

Real


La description

Lorsque vous définissez une image-objet dans l'éditeur d'images-objets, vous devez définir l' origine de cette image-objet. C'est le point où le sprite sera "attaché" ou "dessiné" lorsqu'il est utilisé avec une instance. Cette fonction renvoie le décalage relatif pour l'axe des y de l'origine en se basant sur le coin supérieur gauche étant la position 0,0, avec un + y étant bas et un -y étant supérieur à cela. Exemple:

if y - sprite_get_yoffset(sprite_index) < 0
   {
   y = sprite_get_yoffset(sprite_index);
   }

Le code ci-dessus garantit qu'une instance est conservée dans la pièce en fonction de l'image-objet (afin que l'image-objet soit toujours visible).
