sprite_get_name


Syntaxe:

sprite_get_name(index);

Argument La description
index L'index de l'image-objet pour obtenir le nom de.


Résultats

String


La description

Cette fonction renverra le nom en tant que chaîne de l'image-objet spécifiée. Ce nom est celui qui a été spécifié pour l'image-objet dans l'arborescence des ressources de la GameMaker Studio principale de GameMaker Studio 2. Veuillez noter que ceci est seulement une chaîne et ne peut pas être utilisé pour référencer l'image-objet directement - pour cela vous auriez besoin de l' index d'image - objet. Vous pouvez toutefois utiliser cette chaîne pour obtenir l' index d'image - objet à l'aide de la chaîne renvoyée avec la fonction asset_get_index().


Exemple:

str = sprite_get_name(sprite_index);

Le code ci-dessus obtiendra le nom de l'index d'image-objet pour l'instance exécutant le code et stockera la valeur de retour dans la variable "str".
