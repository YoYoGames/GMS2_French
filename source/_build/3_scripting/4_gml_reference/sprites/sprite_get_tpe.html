sprite_get_tpe


Syntaxe:

sprite_get_tpe(sprite, index);

Argument La description
sprite L'index de l'image-objet pour trouver l'entrée de la page de texture.
index L'index d'image du sprite.


Résultats

Real


La description

Cette fonction renvoie une valeur spéciale qui se rapporte à la position de l'image-objet et de l'image secondaire sur la page de texture utilisée par votre jeu. De cette façon, vous pouvez passer la valeur aux extensions pour une utilisation en dehors du jeu compilé, en contournant les fonctions de dessin de GameMaker Studio 2 et en permettant l'utilisation du sprite dans le contenu DOM qui peut ensuite être dessiné n'importe où dans la fenêtre.. Cette fonction est particulièrement intéressante pour ceux qui souhaitent créer des boutons et autres médias interactifs en dehors de l'élément canvas de GameMaker Studio 2 sur leur page hôte en utilisant la fonction clickable_add.

NOTE: Cette fonction est pour HTML5 seulement.


Exemple:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

Le code ci-dessus crée un bouton DOM à la position (32, 32) de la page sur laquelle le canevas de jeu s'exécute. Le bouton utilise l'image-objet référencée à partir de la page de texture comme "spr_MS_Home" et lorsque vous cliquez dessus, le bouton ouvre une nouvelle fenêtre pour l'URL spécifiée et les propriétés définies pour cette fenêtre.
