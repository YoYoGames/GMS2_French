sprite_get_bbox_top


Syntaxe:

sprite_get_bbox_top(ind);

Argument La description
ind L'index du sprite à vérifier.


Résultats

Real


La description

Cette fonction renvoie la position relative du haut du cadre de sélection du sprite. Cette valeur est donnée en tant que valeur relative basée sur le coin supérieur gauche de l'actif d'image-objet de base étant (0,0). c'est la même valeur que celle que vous pouvez trouver dans l'éditeur de sprite pour les propriétés du masque de collision. L'image ci-dessous montre comment il est calculé:


Exemple:

var ww, hh;
ww = sprite_get_bbox_left(sprite_index) - sprite_get_bbox_right(sprite_index);
hh = sprite_get_bbox_bottom(sprite_index) - sprite_get_bbox_top(sprite_index);

Le code ci-dessus calcule la largeur et la hauteur du masque de collision en fonction des positions latérales relatives du cadre de sélection.
