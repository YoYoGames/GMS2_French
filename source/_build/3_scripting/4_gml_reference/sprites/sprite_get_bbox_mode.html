sprite_get_bbox_mode


La description

Cette fonction renverra le "mode" actuel pour les calculs du cadre de sélection. Vous fournissez l'index de l'image-objet de l'image-objet à vérifier, et la fonction renvoie l'une des constantes illustrées ci-dessous.


Syntaxe:

sprite_get_bbox_mode(ind);

Argument La description
ind L'index du sprite à vérifier.


Résultats

Constant

Constant La description
bboxmode_automatic Automatique - Le cadre de sélection est calculé automatiquement en fonction du paramètre de tolérance défini pour l'image-objet.
bboxmode_fullimage Image complète - Le cadre de sélection sera configuré pour utiliser toute la largeur et la hauteur de l'image-objet, quelles que soient la tolérance et les pixels "vides".
bboxmode_manual Manuel - Le cadre de sélection a été défini manuellement sur les valeurs définies par l'utilisateur (dans l'éditeur de sprite ou à l'aide de la fonction sprite_set_bbox() )


Exemple:

if sprite_get_bbox_mode(sprite_index) != 0
    {
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
    }

Ce qui précède vérifie le mode bbox du sprite actuel et, dans le cas contraire automatic, alors il la met à cette valeur.
