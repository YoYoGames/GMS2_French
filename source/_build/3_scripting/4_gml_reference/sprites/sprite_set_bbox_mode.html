sprite_set_bbox_mode


La description

Cette fonction peut être utilisée pour définir le mode boîte de sélection pour une image-objet. Vous fournissez l'index de sprite et le mode à utiliser, qui doit être l'une des constantes suivantes:

Constant La description
bboxmode_automatic Automatique - Le cadre de sélection est calculé automatiquement en fonction du paramètre de tolérance défini pour l'image-objet.
bboxmode_fullimage Image complète - Le cadre de sélection sera configuré pour utiliser toute la largeur et la hauteur de l'image-objet, quelles que soient la tolérance et les pixels "vides".
bboxmode_manual Manuel - Le cadre de sélection est défini manuellement sur des valeurs définies par l'utilisateur, ce qui nécessite l'utilisation de la fonction. sprite_set_bbox()
REMARQUE: Cette fonction affecte la ressource de l'image-objet , de sorte que toutes les instances ultérieures associées à cette image-objet auront le même mode de cadre de sélection.


Syntaxe:

sprite_set_bbox_mode(ind, mode);

Argument La description
ind L'index du sprite pour changer le mode de.
mode Le mode à définir (une constante).


Résultats

N/A


Exemple:

if sprite_get_bbox_mode(sprite_index) != 0
    {
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
    }

Ce qui précède vérifie le mode bbox du sprite actuel et, dans le cas contraire automatic, alors il est réglé sur cette valeur.
