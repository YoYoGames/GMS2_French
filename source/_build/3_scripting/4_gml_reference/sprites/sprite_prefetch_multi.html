sprite_prefetch_multi


Syntaxe:

sprite_prefetch_multi

Argument La description
array Tableau avec les indices de sprites à récupérer


Résultats:

N/A


La description

Cette fonction peut être utilisée pour pré-extraire (placer dans la mémoire de texture) un certain nombre de pages de texture qui contiennent les sprites donnés. Vous fournissez un tableau rempli avec les index d'image-objet (tels que définis lors de la création de l'élément d'image-objet) et les pages de texture sur lesquelles ils se trouvent seront chargées en mémoire.

REMARQUE: il y a un impact sur les performances lorsque la texture est chargée dans la mémoire de texture sur la plupart des périphériques. Il est donc recommandé d'appeler sprite_preftech sur tout graphique requis au début d'un niveau pour éviter les décrochages.


Exemple:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_prefetch_multi(spr_a);

Le code ci-dessus crée un tableau dans lequel chaque élément contient un index d'image-objet. Ce tableau est ensuite utilisé pour placer ces textures de sprites en mémoire.
