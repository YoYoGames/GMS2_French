sprite_prefetch


La description

Cette fonction peut être utilisée pour pré-extraire (placer dans la mémoire de texture) une page de texture avec l’image-objet donnée. Vous fournissez l'index de l'image-objet tel que défini lors de la création de l'actif de l'image-objet. La page de texture sur laquelle il se trouve sera chargée en mémoire. Notez que la fonction retournera -1 si prefetch n'est pas pris en charge pour la ressource choisie ou si la plate-forme cible est HTML5, ou elle renverra 0 si tout a fonctionné correctement.

REMARQUE: La performance est affectée par le chargement de la texture dans la mémoire de texture sur la plupart des périphériques. Il est donc recommandé d'appeler sprite_prefetch sur tous les graphiques requis au début d'un niveau pour éviter les blocages.


Syntaxe:

sprite_prefetch(ind)

Argument La description
ind L'index du sprite à récupérer


Résultats:

Int (-1 or 0)


Exemple:

sprite_prefetch(spr_Player_Aura);

Le code ci-dessus placera l'image-objet référencée dans une mémoire de texture prête à l'emploi.
