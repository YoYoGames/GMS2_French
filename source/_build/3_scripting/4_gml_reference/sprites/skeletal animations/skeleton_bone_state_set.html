skeleton_bone_state_set


Syntaxe:

skeleton_bone_state_set(bone, map);

Argument La description
bone Le nom (sous forme de chaîne) de l'os.
map Ds_map (créé précédemment) qui stocke les données de l'os.


Résultats:

N/A


La description

Votre animation squelettique est composée d'un certain nombre "d'os", que vous aurez définis et nommés dans votre programme d'animation. Cette fonction peut être utilisée pour définir certaines données de l'os nommé à tout moment. Notez que ces données définiront la pose actuelle pour le squelette, en fonction du jeu d'animation utilisé, et que la fonction nécessite une création préalable. ds_map, qui devrait avoir les clés suivantes et leurs valeurs requises:

  • "x": La position locale x de l'os par rapport à l'os parent.

  • "y": Position y locale de l'os par rapport à l'os parent.

  • "angle": rotation locale de l'os par rapport à l'os parent.

  • "xscale": L'échelle horizontale locale de l'os.

  • "yscale": L'échelle verticale locale de l'os.

  • "parent": Le nom (une chaîne) de l'os parent.

Cette fonction est fournie afin que vous puissiez "intercepter" les données d'animation et les modifier avant qu'elles ne soient dessinées à l'écran. Vous souhaitez donc les utiliser dans l'événement Other - Animation Update, cet événement étant déclenché juste avant le Draw. Événements.

Notez que vous pouvez utiliser la même carte que vous avez remplie à l'aide de la touche skeleton_bone_state_get fonction, même si elle contient les clés et les valeurs "Monde" supplémentaires. Puisque celles-ci se réfèrent à des valeurs en lecture seule, les définir avec cette fonction n'aura aucun effet.

IMPORTANT! Cette fonction n'est pas disponible avec la licence d'évaluation du produit.


Exemple:

var map = ds_map_create();
skeleton_bone_state_get("head", map);
var xx = ds_map_find_value(map, "worldX");
var yy = ds_map_find_value(map, "worldY");
var deltax = mouse_x - (x + xx);
var deltay = mouse_y - (y + yy);
var angle = -radtodeg(arctan2(deltay, deltax));
ds_map_replace(map, "angle", angle);
skeleton_bone_state_set("head", map);
ds_map_destroy(map);

Le code ci-dessus crée un fichier ds_map, puis le remplit avec les données de l'os nommé "head". Il extrait ensuite la position mondiale de l'os et utilise ces données pour définir "l'angle" de l'os afin de suivre la position de la souris dans le jeu.
