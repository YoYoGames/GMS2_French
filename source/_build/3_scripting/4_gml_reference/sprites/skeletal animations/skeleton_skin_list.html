skeleton_skin_list


Syntaxe:

skeleton_skin_list(sprite, list);

Argument La description
sprite L'index sprite de l'animation squelettique Spine pour obtenir la liste.
list L'ID de la liste DS à remplir avec les noms d'animation.


Résultats:

N/A


La description

Avec cette fonction, vous pouvez remplir un (pré-créé) ds_list avec tous les noms des skins inclus dans le sprite d'animation squelettique. Les noms seront des chaînes et peuvent ensuite être utilisés dans les autres fonctions d'habillage d'animation squelette pour ces types d'images-objets.

IMPORTANT! Cette fonction n'est pas disponible avec la licence d'évaluation du produit.


Exemple:

var list = ds_list_create();
skeleton_skin_list(sprite_index, list);
var num = ds_list_size(list);
skeleton_skin_set(list[| irandom(num - 1));
ds_list_destroy(list);

Le code ci-dessus crée une liste DS puis le remplit avec les noms de skin. Un aléatoire est alors choisi et appliqué à l'instance avant que la liste ne soit détruite.
