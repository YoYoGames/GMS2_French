skeleton_attachment_get


Syntaxe:

skeleton_attachment_get(slot);

Argument La description
slot Le nom de l'emplacement (une chaîne) pour obtenir la pièce jointe.


Résultats:

String


La description

Un sprite d'animation squelettique peut avoir d'autres sprites ajoutés en pièces jointes, ces sprites étant ajoutés à un slot nommé (le nom est donné lorsque vous créez le slot d'attachement dans votre programme d'animation) et ils seront dessinés avec l'animation du sprite actuel. Avec cette fonction, vous pouvez obtenir le nom (en tant que chaîne) de la pièce jointe pour l'emplacement donné de l'image-objet actuellement affectée. Notez que les images-objets attachées sont référencées via leur chaîne de nom comme attribué dans Spine ou lorsque vous avez appelé skeleton_attachment_create.

IMPORTANT! Cette fonction n'est pas disponible avec la licence d'évaluation du produit.


Exemple:

if skeleton_attachment_get("slot_leftHand") == ""
   {
   skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
   }

Le code ci-dessus vérifie le nom de pièce jointe actuellement attribué pour l'emplacement "slot_leftHand" et si une chaîne vide est renvoyée, une nouvelle image-objet est attachée.
