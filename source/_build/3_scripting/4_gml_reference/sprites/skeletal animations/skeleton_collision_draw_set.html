skeleton_collision_draw_set


Syntaxe:

skeleton_collision_draw_set(flag);

Argument La description
flag Mis à true pour activer le dessin, et false pour l'éteindre.


Résultats:

N/A


La description

Avec cette fonction, vous pouvez activer ( true ) ou désactivé ( false ) dessiner les données de collision pour le sprite d'animation squelettique en cours utilisé par l'instance. Si cette option est activée, le cadre de sélection et le masque de collision précis seront dessinés en tant que contours autour du sprite.

IMPORTANT! Cette fonction n'est pas disponible avec la licence d'évaluation du produit.


Exemple:

if debug_mode == true
   {
   skeleton_collision_draw_set(true);
   }

Le code ci-dessus vérifie si le jeu est exécuté en mode débogage et si c'est le cas, il bascule les données de collision squelettique pour l'instance à afficher.
