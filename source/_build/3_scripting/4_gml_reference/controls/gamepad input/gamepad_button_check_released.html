gamepad_button_check_released

Retourne si un bouton de gamepad donné a été libéré.

Syntaxe:

gamepad_button_check_released(device, button);


Argument La description
device Quel périphérique de jeu "slot" à vérifier.
button Quelle constante du bouton de gamepad pour vérifier.


Renvoie: Boolean


La description

Cette fonction retournera true ou false selon que le bouton de la manette est détecté comme ayant été libéré ou non. Notez que cette fonction ne se déclenchera qu'une fois pour le bouton au moment où il a été libéré. Pour qu'il se déclenche à nouveau, le bouton doit d'abord être pressé puis relâché une fois de plus.


Exemple:

if gamepad_button_check_released(0, gp_select)
   {
   audio_play_sound(snd_Button, 0, false);
   global.Pause = !global.Pause;
   }

Le code ci-dessus détectera si le bouton "select" de la manette connectée au dispositif "slot" 0 a été enfoncé ou non et basculera la variable globale "Pause".
