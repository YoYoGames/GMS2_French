gamepad_get_guid


La description

Avec cette fonction, vous pouvez récupérer le GUID de la manette de jeu connectée à l’index de logement donné. Vous fournissez l'emplacement de la manette de jeu à vérifier (de 0 à 11) et la fonction renverra l'une des chaînes suivantes:

  • " none " si aucun GUID n'est disponible ou si aucune manette de jeu n'est connectée
  • " index de périphérique hors de portée " si l'index de l'emplacement de la manette de jeu est hors de portée
  • La chaîne du GUID, quelque chose comme "050000005e040000fd020000ffff3f00", si un manette de jeu est attribué à la fente et qu'il peut obtenir le GUID correspondant.

Cette fonction serait généralement utilisée en conjonction avec gamepad_get_description() générer une partie de la chaîne SDL requise pour remapper les contrôleurs sous Windows, Android et macOS.


Syntaxe:

gamepad_get_guid(index);

Argument La description
index Quel index "slot" de gamepad à vérifier


Résultats:

String


Exemple:

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

Le code ci-dessus obtient le GUID et les chaînes de description de nom, puis les concatène et stocke la chaîne finale dans une variable globale pour la référence de suture.
