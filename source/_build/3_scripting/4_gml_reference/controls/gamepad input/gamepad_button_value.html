gamepad_button_value


La description

Avec cette fonction, vous pouvez obtenir la valeur actuelle d'un bouton analogique, comprise entre 0 et 1, où 0 correspond à aucune pression et 1 correspond à une pression maximale. Vous indiquez l’index des emplacements de la manette de jeu à vérifier, ainsi qu’une constante de bouton (comme indiqué ici ) ou un nombre entier compris entre 0 et gamepad_button_count() -1.


Syntaxe:

gamepad_button_value(device, button);

Argument La description
device Quel périphérique "gamepad" contrôler?
button Quelle constante de bouton de la manette de jeu rechercher?


Résultats:

Real


Exemple:

speed = gamepad_button_value(0, gp_shoulderrb) * 4;

Le code ci-dessus utilise la valeur de déclenchement analogique du gamepad branché sur le "slot" 0 du périphérique pour définir la vitesse de l'instance.
