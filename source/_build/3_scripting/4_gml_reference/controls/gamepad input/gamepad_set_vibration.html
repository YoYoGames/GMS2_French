gamepad_set_vibration

Contrôle les vibrations de l'appareil

Syntaxe:

gamepad_set_vibration(device, left_motor, right_motor);


Argument La description
device Quel périphérique de jeu "slot" à vérifier.
left_motor La quantité de vibrations du moteur gauche de 0 à 1.
right_motor La quantité de vibrations du moteur droit de 0 à 1.


Retours: N / A


La description

Avec cette fonction, vous pouvez régler la vibration des moteurs de la manette, l'un ou l'autre des moteurs utilisant une valeur de 0 (pas de vibration) à 1 (vibration complète). Notez qu'il n'y a pas de limite de temps pour cette fonction, vous devrez donc utiliser une variable ou une alarme pour désactiver la vibration (mettre les moteurs à 0) après un certain temps sinon la manette de jeu continuera à vibrer indéfiniment.

REMARQUE: Cette fonction est actuellement disponible uniquement pour les modules cibles Windows, PS4 et Xbox One standard.


Exemple:

if lives = 0
   {
   gamepad_set_vibration(0, 1, 1);
   alarm[0] = room_speed / 2;
   }

Le code ci-dessus serait utilisé (par exemple) dans un événement de collision pour faire vibrer la manette de jeu dans "slot" 0 pendant une demi-seconde, avec l'alarme qui est utilisée pour l'éteindre à nouveau une fois ce temps passé.
