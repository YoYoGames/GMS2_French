gamepad_set_vibration


Syntaxe:

gamepad_set_vibration(device, left_motor, right_motor);

Argument La description
device Quel périphérique "gamepad" contrôler?
left_motor La quantité de vibration du moteur gauche de 0 à 1.
right_motor La quantité de vibration du moteur droit de 0 à 1.


Résultats:

N/A


La description

Avec cette fonction, vous pouvez régler la vibration des moteurs de la manette de jeu, l’un ou l’autre moteur utilisant une valeur de 0 (pas de vibration) à 1 (vibration maximale). Notez que cette fonction n’est pas limitée dans le temps, vous devrez donc utiliser une variable ou une alarme pour désactiver les vibrations (régler les moteurs à 0) après un temps donné, sinon la manette de jeu continuera à vibrer indéfiniment.

REMARQUE: Cette fonction est actuellement disponible uniquement pour les modules cibles standard Windows, PS4 et Xbox One.


Exemple:

if lives = 0
   {
   gamepad_set_vibration(0, 1, 1);
   alarm[0] = room_speed / 2;
   }

Le code ci-dessus serait utilisé (par exemple) dans un événement de collision pour faire vibrer la manette de jeu dans le "logement" 0 pendant une demi-seconde, l’alarme configurée étant utilisée pour la désactiver une fois ce délai écoulé.
