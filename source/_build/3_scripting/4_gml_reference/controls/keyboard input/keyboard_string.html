keyboard_string

Retient une chaîne des dernières touches enfoncées.

Syntaxe:

keyboard_string;


Retourne: Chaîne


La description

Cette variable contient une chaîne contenant le dernier (au plus) 1024 caractères tapés sur le clavier. Cette chaîne ne contiendra que les caractères imprimables tapés, mais elle répondra correctement à l'appui sur la touche d'effacement arrière en effaçant le dernier caractère. Cette variable n'est pas en lecture seule et vous pouvez la modifier, par exemple pour la définir sur "" (une chaîne vide) si vous l'avez déjà manipulée, ou utilisez les fonctions de chaîne pour la manipuler.


Exemple:

if string_length(keyboard_string) > 15
   {
   keyboard_string = string_copy(keyboard_string, 1, 15);
   }

Le code ci-dessus limitera la longueur de la chaîne de clavier à 15 caractères, supprimant ceux qui dépassent cette limite en copiant les quinze premiers caractères dans la variable.


Retour: Entrée au clavier
