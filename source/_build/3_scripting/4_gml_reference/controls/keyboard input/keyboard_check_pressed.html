keyboard_check_pressed


La description

Avec cette fonction, vous pouvez vérifier si une touche a été enfoncée ou non. Contrairement à la keyboard_check fonction, cette fonction ne sera exécutée qu’une fois à chaque fois que vous appuierez sur la touche. Par conséquent, pour qu’elle se déclenche à nouveau, vous devez tout d’abord relâcher la touche, puis à nouveau. La fonction prendra une valeur de code d'activation telle que retournée par la fonction ord() (lettres majuscules de A à Z ou des chiffres de 0 à 9), ou l' un des vk_* constantes répertoriées sur la page principale de saisie au clavier.


Syntaxe:

keyboard_check_pressed(key);

Argument La description
key La clé pour vérifier l'état pressé de.


Résultats:

Boolean


Exemple:

if keyboard_check_pressed(vk_anykey)
   {
   room_goto_next();
   }

Le code ci-dessus passera à la pièce suivante si le joueur appuie sur l'une des touches du clavier (agissant comme une invite "Appuyez sur n'importe quelle touche pour continuer").
