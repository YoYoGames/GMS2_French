keyboard_get_map


La description

Parfois, vous souhaiterez peut-être obtenir le code ASCII pour une clé mappée (pour voir si elle est déjà mappée, par exemple), ce que cette fonction retourne.


Syntaxe:

keyboard_get_map(key);

Argument La description
key C’est la clé que vous souhaitez obtenir la valeur mappée


Résultats:

Real


Exemple:

if keyboard_get_map(ord("A")) = ord("A")
   {
   keyboard_set_map(ord("A"), keyboard_lastkey);
   }

L'exemple de code ci-dessus vérifie d'abord si "A" a été mappé sur une autre clé. Sinon, il le mappera sur la dernière clé sur laquelle l'utilisateur a appuyé.
