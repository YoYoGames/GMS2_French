virtual_key_show

Affiche une clé virtuelle créée sur l'écran de l'appareil.

Syntaxe:

virtual_key_show(index);


Argument La description
index L'index de la clé virtuelle à afficher.


Retours: N / A


La description

Une fois que vous avez créé votre clé virtuelle pour vos appareils, vous devrez peut-être déboguer et tester le bon positionnement et le bon fonctionnement. Pour cela, vous appelez cette fonction, en utilisant l'index de la clé virtuelle que vous voulez voir (précédemment créé et stocké en utilisant virtual_key_add ) qui dessinera un rectangle coloré pour représenter la clé sur l'écran. Une fois que vous êtes satisfait des choses, il est recommandé de créer et de placer votre propre graphique dans sa zone.

REMARQUE: Le rectangle en cours de dessin sera affecté par les options de fusion, de couleur et d'alpha définies par les fonctions de dessin correspondantes.


Exemple:

if global.Debug
   {
   virtual_key_show(global.Left);
   }
else
   {
   virtual_key_hide(global.Left);
   }

Le code ci-dessus vérifie la variable globale "global.Debug" et si elle vérifie vrai alors la clé virtuelle indexée dans la variable "global.Left" sera dessinée sur l'écran, et si elle est fausse alors la clé sera cachée.
