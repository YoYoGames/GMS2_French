mouse_clear


La description

Cette fonction effacera l'état de tous les boutons de la souris en cours d'utilisation. Cela signifie que les contrôles pour leur maintien / relâchement ne reviendront pas true jusqu'à ce que le joueur relâche les boutons et les réprime. Vous fournissez le bouton de la souris pour vérifier l'une des constantes suivantes:

Constant La description
mb_left Le bouton gauche de la souris
mb_middle Le bouton du milieu de la souris (ceci peut ne pas être valable pour toutes les plates-formes cibles)
mb_right Le bouton droit de la souris
mb_none Pas de bouton de la souris
mb_any L'un des boutons de la souris


Syntaxe:

mouse_clear(button);


Argument La description
button Quel bouton de souris constante à vérifier.


Résultats:

Boolean


Exemple:

mouse_clear(mb_any);

Le code ci-dessus effacera l'état bas de tous les boutons de la souris.
