Entrée de la souris


L'entrée de la souris est acceptée sur toutes les plates-formes (sur les appareils mobiles, elle est acceptée comme une simple touche Si vous devez utiliser le multi-touch, vous devez utiliser les fonctions spécifiques du périphérique ) et un certain nombre de constantes être pressé. Ces constantes sont indiquées dans le tableau suivant:

Constant La description
mb_left Le bouton gauche de la souris
mb_middle Le bouton du milieu de la souris (ceci peut ne pas être valable pour toutes les plates-formes cibles)
mb_right Le bouton droit de la souris
mb_none Pas de bouton de la souris
mb_any L'un des boutons de la souris


Les fonctions suivantes existent pour les contrôles standard du bouton de la souris:

  1. bouton de la souris
  2. mouse_check_button
  3. mouse_check_button_pressed
  4. mouse_check_button_released
  5. mouse_clear
  6. mouse_last_button
  7. mouse_wheel_up
  8. mouse_wheel_down
  9. mouse_x
  10. mouse_y

Notez qu'il existe également un ensemble de fonctions de fenêtre liées à l'utilisation de la souris sur les cibles de bureau:

  1. window_mouse_get_x
  2. window_mouse_get_y
  3. window_mouse_set
  4. window_view_mouse_get_x
  5. window_view_mouse_get_y
  6. window_views_mouse_get_x
  7. window_views_mouse_get_y
