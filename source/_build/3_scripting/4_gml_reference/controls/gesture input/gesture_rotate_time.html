gesture_rotate_time


La description

Cette fonction est utilisée pour définir le temps pendant lequel une paire de touches doit tourner dans une direction cohérente pour qu'un mouvement de départ de rotation soit déclenché. Le temps est mesuré en secondes et a une valeur par défaut de 0.16s.

IMPORTANT! Actuellement, pour un jeu à 60fps, vous ne pouvez le régler que sur une seconde maximum, sinon aucune rotation ne sera détectée. Cela augmentera pour un framrate inférieur, par exemple un jeu de 30fps peut avoir un temps maximum de 2 secondes.


Syntaxe:

gesture_rotate_time();


Résultats:

Real (inches)


Exemple:

if gesture_get_rotate_time() != 0.1
    {
    gesture_rotate_time(0.1);
    }

Le code ci-dessus vérifie si le temps de rotation pour les gestes est fixé à 0,1 seconde et si ce n'est pas le cas, il le définit à cette valeur.
