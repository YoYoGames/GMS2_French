Saisie gestuelle


Les fonctions d' entrée de périphérique conviennent parfaitement pour traiter de simples effleurements à l'écran ou pour obtenir une orientation de périphérique, mais vous souhaiterez parfois que vos projets détectent des éléments plus complexes, comme pouvoir faire glisser ou déplacer une instance dans une pièce.. Ces fonctions peuvent être réalisées à l'aide des fonctions de l'appareil, mais pour simplifier les choses, GameMaker Studio 2 propose une gamme complète d' événements de gestes permettant de détecter les doubles taps, les glissements, les glissements, les pincements et les rotations, à la fois pour une instance et pour un niveau global. renvoyer des informations sur l'événement. Les fonctions répertoriées ici sont toutes conçues pour vous permettre de mieux contrôler comment et quand ces événements seront déclenchés.

IMPORTANT! Ces fonctions ne fonctionneront pas sur la plate-forme cible HTML5. Ils ne donneront pas d'erreur s'ils sont utilisés, mais seront simplement ignorés.


Les sections suivantes répertorient toutes les différentes fonctions de geste:

Les fonctions suivantes peuvent être utilisées pour obtenir les différentes valeurs utilisées pour calculer le glissement, le glissement, le pincement, etc.

  1. gesture_get_drag_time
  2. gesture_get_drag_distance
  3. gesture_get_flick_speed
  4. gesture_get_double_tap_time
  5. gesture_get_double_tap_distance
  6. gesture_get_tap_count
  7. gesture_get_pinch_distance
  8. gesture_get_pinch_angle_towards
  9. gesture_get_pinch_angle_away
  10. gesture_get_rotate_time
  11. gesture_get_rotate_angle

Les fonctions suivantes peuvent être utilisées pour définir les différentes valeurs utilisées pour calculer le glissement, le glissement, le pincement, etc.

  1. gesture_drag_time
  2. gesture_drag_distance
  3. gesture_flick_speed
  4. gesture_double_tap_time
  5. gesture_double_tap_distance
  6. gesture_tap_count
  7. gesture_pinch_distance
  8. gesture_pinch_angle_towards
  9. gesture_pinch_angle_away
  10. gesture_rotate_time
  11. gesture_rotate_angle
