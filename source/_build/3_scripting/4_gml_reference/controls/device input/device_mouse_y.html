device_mouse_y

Renvoie la position y de l'appareil lorsque vous appuyez dessus.

Syntaxe:

device_mouse_y(device);


Argument La description
device L'appareil (de 0 à n ) en cours de vérification.


Retours: Real


La description

Cette fonction renvoie la position y d'un contact sur l'appareil. Si vous l'utilisez sur les modules HTML5 ou PC et Mac, cette valeur est mise à jour en permanence, tant que l'appareil (généralement une souris) est branché, mais pour les appareils mobiles, il ne sera mis à jour que lorsque l'écran est en cours. touché et notez que le nombre maximal de touches pouvant être détectées dépendra beaucoup de l'appareil utilisé et du système d'exploitation utilisé.

Exemple:

if device_mouse_check_button(0, mb_left)
   {
   if device_mouse_y(0) > y-32 && device_mouse_y(0) < y+32
      {
      pressed = true;
      }
   else
      {
      pressed = false;
      }
   }

Le code ci-dessus vérifie si l'appareil est pressé et si c'est le cas, il interroge alors la position y du dispositif pour voir si elle est dans les paramètres. Si c'est le cas, la variable "pressée" true, autrement, il le met à false.
