device_mouse_check_button_pressed

Retourne si un bouton d'appareil donné a été pressé.

Syntaxe:

device_mouse_check_button_pressed(device, button);


Argument La description
device L'appareil (de 0 à n ) en cours de vérification.
button Le bouton de l'appareil en cours de vérification.


Renvoie: Boolean


La description

Cette fonction renvoie true ou false selon que l'appareil que vous spécifiez a été "touché" (cliqué) ou non. L'argument de l' appareil fait référence au numéro de touche, qui peut aller de 0 à n (le nombre maximum de touches pouvant être détectées dépendra beaucoup de l'appareil utilisé et du système d'exploitation utilisé). Cette fonction n'est déclenchée qu'une seule fois par l'action de pression réelle, et les constantes standard de mb_left et mb_right peut être utilisé pour vérifier l'un ou l'autre bouton.

Exemple:

if device_mouse_check_button_pressed(0, mb_left)
{
press=true;
}

Le code ci-dessus vérifie si le périphérique a été enfoncé et définit une variable sur true si c'est le cas.
