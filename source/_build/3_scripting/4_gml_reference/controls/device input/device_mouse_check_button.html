device_mouse_check_button

Renvoie si un bouton de périphérique donné est actuellement arrêté.

Syntaxe:

device_mouse_check_button(device, button);


Argument La description
device L'appareil (de 0 à n ) en cours de vérification.
button Le bouton de l'appareil en cours de vérification.


Renvoie: Boolean


La description

Cette fonction renvoie true (1) ou false (0) selon que l'appareil que vous spécifiez est utilisé ou non. Maintenant, l' appareil fait référence au numéro de la souris, qui peut être compris entre 0 et 4 et cet événement est déclenché lorsque l'écran tactile d'un appareil mobile est touché. Ainsi, vous pouvez tester jusqu'à n touches d'écran différentes (le maximum le nombre de touches pouvant être détectées dépendra beaucoup de l'appareil utilisé et du système d'exploitation utilisé). Cette fonction met constamment à jour chaque étape et les constantes standard de mb_left et mb_right peut être utilisé pour vérifier l'un ou l'autre bouton.

Exemple:

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
room_goto(rm_Menu);
}

Le code ci-dessus vérifie si les deux touches sont maintenues enfoncées en même temps et si elles le font ensuite, elles vont dans une autre pièce.
