device_mouse_dbclick_enable

Définit si un double clic gauche doit être détecté en tant que mb_right ou non.

Syntaxe:

device_mouse_dbclick_enable(bool);


Argument La description
bool Définissez la détection de double-clic sur ( true ) ou désactivé ( false ).


Renvoie: Boolean


La description

Cette fonction peut être utilisée pour configurer l'appareil afin de détecter une double pression mb_left (bouton gauche de la souris) en tant que mb_right (bouton droit de la souris) appuyez ou non. Par défaut, il est réglé sur true, ce qui signifie que chaque fois que l'utilisateur appuie deux fois sur l'écran de l'appareil rapidement et consécutivement, la valeur de retour est la même que si le bouton droit de la souris avait été cliqué. Quand ceci est activé, le premier tap sera détecté comme mb_left, et le second comme mb_right, donc assurez-vous que tout code que vous utilisez en tient compte.


Exemple:

if os_type == os_windows || os_type == os_mac
   {
   device_mouse_dbclick_enable(false);
   }

Le code ci-dessus vérifie si l'appareil qui exécute le jeu est un PC Windows ou un Mac et si c'est l'un d'entre eux, il désactive la fonction de double frappe.
