device_mouse_check_button_released

Retourne si un bouton de périphérique donné a été libéré.

Syntaxe:

device_mouse_check_button_released(device, button);


Argument La description
device L'appareil (de 0 à n ) en cours de vérification.
button Le bouton de l'appareil en cours de vérification.


Renvoie: Boolean


La description

Cette fonction renvoie true ou false selon que le toucher de l'appareil que vous spécifiez a été libéré ou non. L'argument device se réfère au numéro de touche de l'appareil, qui peut être compris entre 0 et n et notez que le nombre maximal de touches détectables dépendra beaucoup du périphérique utilisé et du système d'exploitation utilisé. Cette fonction est uniquement déclenchée une fois par l'action de libération réelle et les constantes standard de mb_left et mb_right peut être utilisé pour vérifier l'un ou l'autre bouton.

Exemple:

if device_mouse_check_button_released(0, mb_left)
{
press=false;
}

Le code ci-dessus vérifie si le périphérique a été enfoncé et définit une variable sur false si c'est le cas.
