device_is_keypad_open

Vérifie l'appareil pour voir si un clavier est ouvert ou non.

Syntaxe:

device_is_keypad_open


Renvoie: Boolean


La description

Cela fait une vérification de l'appareil pour un clavier et si un est disponible, il retourne true sinon ça revient false. Veuillez noter que cette fonction est principalement utilisée avec Android appareils Android.

Pour les utilisateurs disposant d'un Sony Xperia Play, il existe une configuration de bouton / carte clé dans GameMaker Studio 2, vous pouvez donc utiliser les constantes de clavier vk_up, vk_down, vk_left, vk_right pour les touches du joypad et Triangle est ord("T"), Square est ord("S"), Circle est vk_alt + vk_backspace alors que le bouton retour est simplement vk_backspace, Cross est vk_space, Select est vk_return et Start est vk_rshift, le L Trigger est ord("L") et R Trigger est ord("R").


Exemple:

if device_is_keypad_open()
   {
   global.Setting = 2;
   }
else
   {
   global.Setting = 1;
   }

Le code ci-dessus vérifie un clavier puis modifie une variable globale en fonction de la valeur renvoyée.
