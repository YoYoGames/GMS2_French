font_replace_sprite


La description

Avec cette fonction, vous pouvez utiliser une "bande d'image-objet" pour remplacer un élément de police d'image-objet créé précédemment, chaque sous-image étant un symbole ou une lettre individuel. L'index est le numéro de carte UTF8 du premier caractère à utiliser et les plages à utiliser sont identiques à celles d'une police normale, commençant par "!". mais rappelez-vous que vous ne devez inclure que ce dont vous avez besoin et que vous devriez utiliser le ord fonction pour obtenir la valeur UTF8 correcte pour la lettre de départ.

Vous pouvez également définir l'espacement proportionnel de la police ( true ) ou pas ( false ), où une police proportionnelle est espacée en fonction de la largeur réelle des lettres (la lettre "i" prend donc moins de place que la lettre "w", par exemple), tandis qu'une police non proportionnelle sera espacée en fonction du largeur de l'image, comme une police monospace. Enfin, vous pouvez définir l’espace à laisser entre chaque lettre lors de l’écriture, ce qui peut être n’importe quelle valeur entière, 0 ne représentant pas d’espace (les lettres se "touchent" si elles sont proportionnelles).

Il est également intéressant de noter qu'il ne peut y avoir aucun espace dans la carte des caractères, de sorte que si vous utilisez par exemple "." et '0' mais pas '/', vous devez quand même inclure '/' ou laisser une sous-image vierge entre les deux caractères utilisés. Syntaxe:

font_replace_sprite(ind, spr, first, prop, sep);


Argument La description
ind L'index de la police à remplacer (c'est-à-dire que toutes les références à cette ressource au-delà de cette fonction feront référence à la nouvelle police, pas à la police d'origine).
spr Le sprite pour ajouter une police basée sur.
first L'index du premier caractère disponible.
prop Indique si les espaces autour de chaque lettre doivent être supprimés (si la valeur est false, la police créée sera monospace, sinon ce sera comme la plupart des polices classiques).
sep L'espace entre chaque lettre, 0 signifiant que la droite d'une lettre touche juste la gauche de la suivante.


Résultats:

N/A


Exemple:

font_replace_sprite(global.Font, working_directory + "Sprites\spr_Font.png", ord("!"), true, 2);

Le code ci-dessus remplacera l'actif de police indexé dans la variable globale "Police" par un nouvel actif de police à partir d'un fichier image externe.
