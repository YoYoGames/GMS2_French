Polices


GameMaker Studio 2 dispose d'un certain nombre de fonctions qui peuvent être utilisées pour obtenir des informations sur les ressources de polices incluses, ainsi qu'un ensemble supplémentaire de fonctions qui peuvent être utilisées pour modifier, ajouter ou supprimer des polices de votre jeu au moment de l'exécution. En particulier la fonction font_add() peut être utilisé pour ajouter des polices non standard, comme les polices glyphes asiatiques, dans *.ttf format.

Il convient de noter que si vous ajoutez une police à votre jeu qui n'a pas les glyphes requis par le texte que vous souhaitez écrire, le caractère 9647 (▯) est utilisé pour remplacer ces glyphes manquants lors du rendu dans le dessiner l'événement. Donc, si votre police n'a pas, par exemple, le symbole °, alors écrire 90 ° produira en réalité 90▯.

Les fonctions suivantes peuvent être utilisées pour obtenir des informations sur les ressources de polices:

  • font_get_name
  • font_get_fontname
  • font_get_first
  • font_get_last
  • font_get_italic
  • font_get_bold
  • font_get_size
  • font_get_texture
  • font_get_uvs

Les fonctions suivantes peuvent être utilisées pour changer et manipuler des polices pendant une partie:

  • font_set_cache_size
  • font_exists
  • font_add
  • font_add_sprite
  • font_add_sprite_ext
  • font_replace_sprite
  • font_replace_sprite_ext
  • font_delete
  • font_texture_page_size
