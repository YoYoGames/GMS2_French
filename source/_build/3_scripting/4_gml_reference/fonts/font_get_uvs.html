font_get_uvs


La description

Cette fonction renvoie un tableau avec les coordonnées UV de la texture de la police sur la page de texture, en remplissant le tableau avec les valeurs suivantes: [0] = left, [1] = top, [2] = right et [3] = bottom. Cette valeur peut ensuite être utilisée dans d'autres fonctions de dessin, en particulier dans le dessin général lors de l'utilisation de primitives ainsi que dans les fonctions Shader.


Syntaxe:

font_get_uvs(font);


Argument La description
font L'index de la police à utiliser.


Résultats:

Array


Exemple:

var tex = font_get_uvs(fnt_Main);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

Le code ci-dessus va stocker les coordonnées UV pour l'arrière-plan donné dans un tableau local, puis affecter les valeurs aux variables d'instance.
