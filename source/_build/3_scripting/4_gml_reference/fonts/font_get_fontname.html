font_get_fontname


La description

Avec cette fonction, vous pouvez obtenir le nom réel du système de l'actif de police sélectionné. Cette fonction renvoie une chaîne et non un index, et le nom retourné dépendra de la police utilisée. Par exemple, vous pouvez avoir un actif de police appelé "fnt_Main" dans l' GameMaker Studio 2 ressources de GameMaker Studio 2, et la police elle-même peut être la police système Arial de Windows. Dans ce cas, la fonction retournera "Arial" car c'est le nom du système de la police. Si vous avez besoin du nom tel qu'il apparaît dans l'arborescence des ressources, vous devez utiliser font_get_name.


Syntaxe:

font_get_fontname(ind);


Argument La description
ind Index de la police à vérifier.


Résultats:

String


Exemple:

fnt_Name = font_get_fontname(font0);

Le code ci-dessus obtiendra le nom du système d'une ressource de police et le stockera comme une chaîne dans la variable "fnt_Name".
