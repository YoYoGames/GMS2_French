font_get_name


La description

Cette fonction peut être utilisée pour obtenir le nom (en tant que chaîne) qui a été donné à la police lors de son ajout en tant qu'actif à l' GameMaker ressources de GameMaker Studio 2. S'il vous plaît noter que ce n'est qu'une chaîne et ne peut pas être utilisé pour référencer la police directement - pour cela, vous auriez besoin de l' index de police. Vous pouvez cependant utiliser cette chaîne pour obtenir l' index de police en utilisant la chaîne retournée avec la fonction asset_get_index().


Syntaxe:

font_get_name(ind);


Argument La description
ind Index de la police à vérifier.


Résultats:

String


Exemple:

fnt_Name = font_get_name(font0);

Le code ci-dessus obtiendra le nom d'une ressource de police telle qu'elle apparaît dans l'arbre de ressources et la stockera sous la forme d'une chaîne dans la variable "fnt_Name".
