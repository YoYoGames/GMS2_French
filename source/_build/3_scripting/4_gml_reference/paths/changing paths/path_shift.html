path_shift

Décale le chemin donné de n'importe quel nombre de pixels horizontalement et verticalement.

Syntaxe:

path_shift(index, xshift, yshift);


Argument La description
index L'indice du chemin à changer.
xshift Le nombre de pixels horizontaux pour décaler le chemin. Négatif = gauche, positif = droit.
yshift Le nombre de pixels verticaux pour déplacer le chemin. Négatif = haut, positif = bas.


Retours: N / A


La description

Avec cette fonction, vous pouvez déplacer un chemin le long des axes horizontal et vertical. Veuillez noter que ceci n'aura aucun effet visible dans le jeu si l'instance suivant le chemin décalé le suit relativement (plutôt qu'absolument) car la position du chemin est alors ignorée. Cette fonction modifie la ressource de chemin réelle, et affectera donc de façon permanente la façon dont le chemin est utilisé par toutes les instances dans le jeu à partir du moment où la fonction est utilisée jusqu'à la fin de la partie.


Exemple:

path_shift(mypath, 50, 50);

Cela changerait path0 50 pixels à droite et 50 pixels à la hausse.
