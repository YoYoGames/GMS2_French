path_add

Ajoute un chemin vide et renvoie son index.

Syntaxe:

path_add();


Renvoie: Real.


La description

Avec cette fonction, vous pouvez créer un chemin dans GameMaker Studio 2 sans utiliser l'éditeur de chemin. cette fonction retournera l'index du chemin qui devrait être stocké dans une variable et utilisé comme référence pour ce chemin à partir de ce moment. S'il vous plaît noter que le chemin créé est vide, c'est à dire: il n'a pas de points définis, de sorte que vous devrez alors utiliser les autres fonctions disponibles pour ajouter des points au chemin ou utiliser mp_grids pour générer le chemin. Une fois que vous avez fini d'utiliser le chemin, ou que vous souhaitez en créer un nouveau et stocker son index dans la même variable, vous devez d'abord supprimer l'ancien chemin avec path_delete pour éviter les fuites de mémoire qui peuvent éventuellement provoquer un crash de votre jeu.

IMPORTANT! Cette fonction n'est pas disponible avec la licence d'évaluation du produit.



Exemple:

global.newpath = path_add();

Cela va créer un nouveau chemin et affecter son index à global.newpath.
