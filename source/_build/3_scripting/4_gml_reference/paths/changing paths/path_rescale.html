path_rescale

Échelle le chemin donné par rapport à son centre.

Syntaxe:

path_rescale(index, xscale, yscale);


Argument La description
index L'indice du chemin à l'échelle.
xscale Que multiplier l'échelle horizontale actuelle par. L'échelle par défaut est 1.
yscale Que multiplier l'échelle verticale actuelle par. L'échelle par défaut est 1.


Retours: N / A


La description

Cette fonction peut être utilisée pour redimensionner le chemin donné le long des deux axes (vertical ou horizontal), en déplaçant fondamentalement chacun des points de trajectoire vers une nouvelle position correspondant à cette échelle autour du centre du chemin. Cette fonction modifie la ressource de chemin réelle, et affectera donc de façon permanente la façon dont le chemin est utilisé par toutes les instances dans le jeu à partir du moment où la fonction est utilisée jusqu'à la fin de la partie.


Exemple:

path_rescale(path0, 3, 3);

Cela redimensionnerait path0 à trois fois sa taille d'origine.
