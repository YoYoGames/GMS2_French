path_append

Joint deux chemins ensemble.

Syntaxe:

path_append(index, path);


Argument La description
index Le nouvel index du chemin.
path L'index du chemin qui sera ajouté à 'index'.


Retours: N / A


La description

Avec ce code, vous pouvez ajouter un chemin à un autre, les assembler efficacement, et pour que cela fonctionne, les deux chemins doivent déjà exister (soit dans l'éditeur de chemin, soit en utilisant path_add ).

Remarque: cela ne supprimera pas le chemin étant ajouté à partir des ressources. Il est toujours là, seulement il n'y a plus de points et si vous ne souhaitez plus l'utiliser, vous devez l'enlever avec la fonction path_delete.


Exemple:

path_append(mypath, mppath); path_delete(mppath);

Le code ci-dessus ajoute les données de chemin indexées dans la variable "mppath" au chemin indexé dans la variable "mypath" puis supprime le "mppath" de la mémoire.
