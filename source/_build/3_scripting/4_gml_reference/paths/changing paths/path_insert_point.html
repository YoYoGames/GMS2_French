path_insert_point

Insère un point dans le chemin donné à une position définie.

Syntaxe:

path_insert_point(index, n, x, y, speed);


Argument La description
index L'index du chemin pour insérer le point dans.
n Le point de définition pour insérer le nouveau point AVANT.
x La coordonnée x (relative au chemin) du nouveau point.
y La coordonnée y (relative au chemin) du nouveau point.
speed Le facteur de vitesse du point.


Retours: N / A


La description

Avec cette fonction, vous pouvez insérer un nouveau point dans un chemin (le chemin peut avoir été créé dans l'éditeur de chemin ou via le code en utilisant path_add ). Le point sera ajouté dans le chemin avant le point "n" spécifié dans la fonction.


Exemple:

path_insert_point(mypath, 0, 50, 50, 100);

Cela va insérer un point au tout début du chemin indexé dans la variable "mypath", en (50,50), avec un facteur de vitesse de 100%.
