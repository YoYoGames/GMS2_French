path_rotate

Fait pivoter le chemin donné par rapport à son centre.

Syntaxe:

path_rotate(index, angle);


Argument La description
index L'index du chemin à retourner.
angle L'angle de rotation des degrés.


Retours: N / A


La description

Vous pouvez utiliser cette fonction pour faire pivoter un chemin donné autour de son centre. Souvenez-vous que dans GameMaker Studio 2 (sauf si vous utilisez la physique), les angles sont calculés dans le sens inverse des aiguilles d'une montre. Faire pivoter le chemin de 90 degrés tournera donc le chemin vers la gauche. Cette fonction modifie la ressource de chemin réelle, et affectera donc de façon permanente la façon dont le chemin est utilisé par toutes les instances dans le jeu à partir du moment où la fonction est utilisée jusqu'à la fin de la partie.


Exemple:

path_rotate(path0, 90);

Cela ferait pivoter le path0 de 90 degrés dans le sens antihoraire.
