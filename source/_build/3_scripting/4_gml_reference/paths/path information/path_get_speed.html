path_get_speed

Renvoie le facteur de vitesse de la position donnée d'un chemin donné.

Syntaxe:

path_get_speed(ind, pos);


Argument La description
index L'index du chemin à vérifier.
pos Jusqu'où peut aller le chemin à parcourir? Entre 0 (début) et 1 (fin), peut être décimal.


Renvoie: Real.


La description

Cette fonction renvoie le facteur de vitesse de n'importe quelle position sur un chemin. La position doit être une valeur comprise entre 0 et 1 (vous pouvez utiliser path_position, par exemple, si vous avez besoin de la position actuelle de l'instance) avec 0 étant la position de départ et 1 étant la fin. La valeur retournée sera le facteur par lequel le réel path_speed est en train d'être défini. Par exemple, sur un chemin avec seulement deux points, 0 et 1, où le facteur de vitesse du point 0 est 100 et le facteur de vitesse du point 1 est 0, la fonction path_get_speed(path, 0.5) retournera 50.


Exemple:

speed = path_get_speed(path0, 0.5) / 20;

Ceci placera la vitesse de l'instance appelante au facteur de vitesse du point exactement à mi-chemin de path0, divisé par 20. Donc si le facteur de vitesse trouvé est le 100 par défaut, il fixera la vitesse de l'instance appelante à 5.
