path_get_point_speed

Renvoie le paramètre de vitesse du point de définition donné d'un chemin donné.

Syntaxe:

path_get_point_speed(index, n);


Argument La description
index L'index du chemin à vérifier.
n L'identité du point à vérifier.


Renvoie: Real.


La description

Avec cette fonction, vous pouvez obtenir la vitesse du point (tel que défini dans l' éditeur de chemin ou lorsque vous ajoutez dynamiquement un point de chemin en utilisant path_add_point() ) exprimé en pourcentage. Ainsi, si vous avez un point de chemin défini à 50 dans l'éditeur de chemin, cette fonction renvoie 50 lorsqu'elle est utilisée.


Exemple:

if path_get_point_speed(c_path, 0) != 100
   {
   path_change_point(c_path, 0, path_get_point_x(c_path, 0), path_get_point_x(c_path, 0), 100);
   }

le code ci-dessus vérifie le point de chemin "0" du chemin indexé par "c_path" pour voir si la vitesse n'est pas égale à 100. Si ce n'est pas le cas, la vitesse de ce point est fixée à 100.
