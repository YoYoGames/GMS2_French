path_exists

Retourne si un chemin avec l'index donné existe ou non.

Syntaxe:

path_exists(index);


Argument La description
index L'index du chemin à vérifier.


Renvoie: Booléen.


La description

Cette fonction renvoie si un chemin avec l'index donné existe ou non. Notez que si vous vérifiez l'existence d'un chemin dans une variable qui n'a pas encore été déclarée, une erreur sera générée.


Exemple:

if path_exists(path0)
   {
   path_start(path0, 4, path_action_reverse, 0);
   }

Cela commencera path0, s'il existe.
