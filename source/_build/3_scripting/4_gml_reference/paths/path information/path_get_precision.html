path_get_precision

Renvoie la précision du chemin donné.

Syntaxe:

path_get_precision(index);


Argument La description
index L'index du chemin à vérifier.


Renvoie: Real.


La description

Cette fonction retourne avec quelle précision le chemin donné a été "lissé", et sera une valeur entière de 1 à 8. Bien que vous puissiez obtenir (et définir) cette valeur pour un chemin en ligne droite, elle n'aura aucune influence sur la façon dont instance utilise le chemin car il n'est pertinent que lorsque le type de chemin est défini sur "lisse".


Exemple:

if path_get_kind(mypath)
   {
   if path_get_precision(mypath) !=8 path_set_precision(mypath, 8);
   }

Le code ci-dessus vérifie le chemin indexé dans "mypath" pour voir de quel type il s'agit et s'il est lisse, il vérifie alors la précision du lissage. Si elle est inférieure à 8, elle est alors définie sur 8.
