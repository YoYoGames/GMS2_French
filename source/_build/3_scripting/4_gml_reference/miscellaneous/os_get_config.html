os_get_config

Renvoie le nom de la configuration actuelle.

Syntaxe:

os_get_config()


Retourne: Chaîne


La description

Cette fonction renvoie le nom (en tant que chaîne) de la configuration actuellement sélectionnée pour votre jeu. Pour plus d'informations sur les configurations, veuillez consulter la section Utilisation avancée: Configurations.


Exemple:

if os_get_config() = "Free_Version"
   {
   ads_enable((room_width / 2) - 100, 0, 200, 75, 60);
   }

Le code ci-dessus va vérifier pour voir quelle configuration est utilisée et si c'est celui appelé "Free_Version", les publicités seront activées dans le jeu.
