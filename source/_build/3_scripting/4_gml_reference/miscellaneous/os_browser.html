os_browser

Cette variable contient une constante différente en fonction du navigateur dans lequel le jeu est actuellement exécuté.

Syntaxe:

os_browser


Retours: Constant


La description

Cette variable contient l'un des constants GameMaker: Studio doit vous indiquer le navigateur dans lequel vous exécutez le jeu (le cas échéant). Les constantes suivantes peuvent être retournées:

Constant La description
browser_not_a_browser Le jeu n'est pas joué dans un navigateur
browser_unknown Navigateur inconnu
browser_ie Internet Explorer
browser_ie_mobile Internet Explorer sur un appareil mobile
browser_firefox Mozilla Firefox
browser_chrome Google Chrome
browser_safari Safari
browser_safari_mobile Safari sur un appareil mobile
browser_opera Opéra
browser_tizen Navigateur de l'appareil mobile Tizen
browser_windows_store Windows 8 App

Exemple:

if os_browser == browser_not_a_browser
   {
   global.Config = 0;
   }
else
   {
   global.Config = 1;
   }

Le code ci-dessus vérifie si le jeu s'exécute dans un navigateur et définit une variable globale à une valeur en fonction du résultat de la vérification.
