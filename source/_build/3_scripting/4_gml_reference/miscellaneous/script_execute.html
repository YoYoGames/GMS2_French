script_execute

Syntaxe:

script_execute(scr, arg0, arg1, arg2, ...);


Argument La description
scr L'ID du script que vous souhaitez appeler
arg0... arg14 Les différents arguments que vous souhaitez transmettre au script


Retours: N / A


La description

Il peut arriver que vous souhaitiez passer un script à une instance de manière dynamique, et pour ces cas, nous fournissons le script_execute une fonction. Il exécutera une ressource de script créée par l'utilisateur et transmettra jusqu'à 15 arguments à ce script. Pour plus d'informations sur les scripts, voir Scripts.


Exemple:

script_execute(choose(scr_Up, scr_Down, scr_Left, scr_Right), irandom(5));

L'exemple de code ci-dessus utilisera script_execute pour choisir l'une des quatre ressources de script créées par l'utilisateur (dans le cas présent, liées au mouvement) et lui transmettre un entier aléatoire comme argument0.
