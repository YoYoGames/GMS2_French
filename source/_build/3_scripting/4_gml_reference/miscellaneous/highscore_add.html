highscore_add

Syntaxe:

highscore_add(str, numb);


Argument La description
str La chaîne (nom) pour attribuer le score à
numb Le nombre (score) à ajouter.


Retours: N / A


La description

Avec cette fonction, vous pouvez ajouter un nom et un score à la liste des meilleurs scores globaux internes. Il n'est pas nécessaire de vérifier la valeur pour voir si elle est suffisamment élevée pour entrer dans la liste des partitions, car GameMaker Studio 2 stockera uniquement les valeurs supérieures à la dixième position enregistrée.


Exemple:

if lives = 0
   {
   highscore_add(global.Name, score);
   score = 0;
   room_goto(rm_MainMenu);
   }

Le code ci-dessus vérifiera les vies et si elles sont mises à 0, il ajoutera le score actuel et la chaîne contenue dans la variable globale "Name" dans la liste des meilleurs scores, avant de finalement mettre le score à 0 et envoyer le joueur à la pièce indexée dans "rm_MainMenu".
