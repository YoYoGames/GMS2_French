paramètre_count

Syntaxe:

parameter_count();


Retours: Real


La description

Les paramètres de ligne de commande sont les commandes supplémentaires que vous pouvez ajouter à un exe pour modifier le mode d'exécution du programme. Vous pouvez trouver le nombre de paramètres pour le jeu actuel en utilisant cette fonction, où le premier paramètre a l'index 1 et le dernier a l'index retourné par la fonction (une valeur de 0 est spécial sur le fait qu'il s'agit du nom de fichier du jeu exécutable, y compris le chemin). Il convient de noter que cette fonction fonctionnera sur la plate-forme HTML5, en récupérant les paramètres url.


Exemple:

p_num = parameter_count();

Le code ci-dessus va stocker le nombre de paramètres de ligne de commande qui ont été utilisés pour le jeu dans la variable "p_num".
