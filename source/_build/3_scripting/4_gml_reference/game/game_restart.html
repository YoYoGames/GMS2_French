game_restart

Syntaxe:

game_restart();


Retours: N / A


La description

Avec cette fonction, vous pouvez redémarrer le jeu. C'est essentiellement la même chose que de lancer le jeu pour la première fois et ainsi l' événement de départ de jeu sera déclenché, ainsi que l' événement de fin de partie. Cependant, il convient de noter que les variables globales ne seront pas réinitialisées à moins d'être explicitement codées en tant que telles - par exemple, la variable globale intégrée score ne commencera pas à zéro après un redémarrage du jeu s'il a déjà été modifié dans le jeu. C'est également le cas pour l'état GPU (donc si vous avez défini la couleur de tirage ou l'alpha, par exemple, il restera à la valeur modifiée) et pour la vitesse de jeu. Notez également que si une ressource de l'arborescence des ressources a été modifiée au moment de l'exécution dans le jeu, par exemple si vous modifiez l'origine d'une ressource d'image-objet ou déplacez la position d'une ressource-chemin, elle ne sera pas non plus réinitialisée.


Exemple:

if keyboard_check_pressed(ord("R")) game_restart();

Cela relancerait le jeu lorsque le joueur appuierait sur la touche "R".
