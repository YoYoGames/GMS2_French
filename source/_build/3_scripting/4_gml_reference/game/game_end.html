Fin du jeu


Syntaxe:

game_end();


Retours: N / A


La description

Avec cette fonction, vous pouvez mettre fin au jeu (et l’ événement de fin de jeu sera déclenché). Cela ne se produira pas instantanément, mais plutôt à la fin de l'étape en cours. Ainsi, tout code que vous avez dans la même étape après l'appel de cette fonction sera toujours exécuté. Veuillez noter que cette fonction a les restrictions suivantes:

  • Sur les Android (et Amazon Fire ), appelez game_end() va pousser l'application en arrière-plan, mais ne ferme pas l'application. Cela doit être fait par l'utilisateur.

  • Sur iOS, cela ne fera rien et échouera silencieusement.

  • Sous UWP, XBoxOne, PS4 et Nintendo Switch, le jeu se bloque et ne doit donc pas être utilisé.

  • En HTML5, le jeu se terminera, mais l'utilisateur laissera un regard vide sur le canevas vide de la page Web. Il n'est donc pas recommandé de l'utiliser sur cette plate-forme cible.

  • Sous Windows, Linux et Mac, la fonction termine simplement le jeu et ferme la fenêtre de jeu (un événement de fin de jeu sera déclenché).


Exemple:

if keyboard_check_pressed(vk_escape) game_end();

Cela mettrait fin au jeu si le joueur appuyait sur la touche "échap".
