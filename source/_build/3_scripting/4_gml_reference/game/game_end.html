Fin du jeu

Syntaxe:

game_end();


Retours: N / A


La description

Avec cette fonction, vous pouvez terminer le jeu (et l' événement de fin de partie sera déclenché). Cela n'arrivera pas instantanément, mais plutôt à la fin de l'étape en cours, donc tout code que vous avez dans la même étape après que cette fonction a été appelée sera toujours exécuté. Veuillez noter que cette fonction a les restrictions suivantes:

  • Il signalera une erreur sur les modules Windows UWP et iOS, car il est contraire aux conditions de leurs marchés respectifs.
  • Il échouera silencieusement sur le module HTML5.
  • Il fonctionnera sur Windows (y compris Steam Workshop), Windows Phone, Android, Linux et Mac.


Exemple:

if keyboard_check_pressed(vk_escape) game_end();

Cela mettrait fin à la partie si le joueur appuie sur la touche "escape".
