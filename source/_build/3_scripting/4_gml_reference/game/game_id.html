identifiant de jeu

Syntaxe:

game_id


Retours: Real


La description

Cette variable en lecture seule renvoie l'identifiant unique du jeu que vous avez créé. Vous pouvez l'utiliser si vous avez besoin d'un nom de fichier unique ou de tout autre élément qui nécessite quelque chose pour identifier votre jeu uniquement. Cela peut être défini dans les options de jeu.


Exemple:

ini_open("Score.ini");
ini_write_real("Scores","0",score + game_id);
ini_close();

Le code ci-dessus effectue un cryptage très basique sur la partition en y ajoutant le game_id avant de l'enregistrer. En le relisant dans le jeu, vous devez déduire game_id pour obtenir la valeur correcte à nouveau.
