game_set_speed


La description

Cette fonction peut être utilisée pour définir la vitesse de jeu. Vous pouvez définir cela de deux manières - en images par seconde (FPS) ou en microsecondes par image (MPF) - en utilisant l'une des deux constantes suivantes:

Constant La description
gamespeed_fps Définit la vitesse de jeu en utilisant des images par seconde.
gamespeed_microseconds Définit la vitesse de jeu en microsecondes par image.


Ainsi, par exemple, une vitesse de jeu de 30 images par seconde correspond à 33333 microsecondes par image de jeu. Cette fonction serait alors exprimée par cette fonction comme suit:

game_set_speed(30, gamespeed_fps);

ou:

game_set_speed(33333, gamespeed_microseconds);


Syntaxe:

game_set_speed(speed, type);

Argument La description
speed La nouvelle vitesse de jeu (en FPS ou en MPF).
type Le type de méthode utilisée pour définir la vitesse de jeu (voir les constantes ci-dessus).


Résultats:

N/A


Exemple:

if os_browser == browser_not_a_browser
    {
    game_set_speed(60, gamespeed_fps);
    }
else
    {
    game_set_speed(30, gamespeed_fps);
    }

Le code ci-dessus vérifie si le jeu est exécuté dans un navigateur et définit la vitesse de jeu en conséquence en tant que valeur FPS.
