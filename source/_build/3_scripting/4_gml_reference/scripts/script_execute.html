script_execute


Syntaxe:

script_execute(scr, arg0, arg1, arg2..., etc...);

Argument La description
scr le nom du script que vous voulez appeler
arg0, arg1, arg2..., etc... Les différents arguments que vous voulez transmettre au script


Résultats:

Will depend on the return value from the script being called


La description

Il y a des moments où vous voudrez peut-être passer un script à une instance de manière dynamique, et pour ces cas, nous fournissons le script_execute fonction. Il exécutera une ressource de script créée par l'utilisateur et lui transmettra les arguments donnés. Pour plus d'informations sur les scripts, voir L'éditeur de script. Si le script a une valeur de retour, alors cette fonction retournera également cette valeur lors de l'appel.


Exemple:

script_execute(choose(scr_Up, scr_Down, scr_Left, scr_Right), irandom(5));

L'exemple de code ci-dessus utilisera script_execute choisir l'une des quatre ressources script créées par l'utilisateur (dans ce cas, liées au mouvement) et lui transmettre un entier aléatoire comme argument0.
