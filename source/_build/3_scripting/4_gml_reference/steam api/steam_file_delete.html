steam _file_delete

Syntaxe:

steam_file_delete(filename);


Argument La description
filename Le nom de la suppression du fichier.


Retours: Real


La description

Cette fonction va supprimer le fichier donné du Steam Cloud. La fonction retournera une valeur de 0 si elle échoue pour une raison quelconque et une valeur supérieure à 0 si elle réussit.


Exemple:

if steam_file_exists("Save.txt")
   {
   steam_file_delete("Save.txt");
   }

Le code ci-dessus va vérifier pour voir si un fichier existe, et si c'est le cas, il supprime le fichier du Steam Cloud.
