steam _create_leaderboard

Syntaxe:

steam_create_leaderboard(lb_name, sort_method, display_type);


Argument La description
lb_name Le nom du leaderboard que vous créez (une chaîne).
sort_method La méthode de tri des entrées de classement (voir les constantes listées ci-dessous).
display_type La façon d'afficher le leaderboard à l'utilisateur (voir les constantes listées ci-dessous).


Retours: Real


La description

Avec cette fonction, vous pouvez créer un nouveau leaderboard pour votre jeu. Le premier argument est une chaîne qui définit le nom de votre leaderboard, et ce nom doit être utilisé dans tous les autres appels de fonction relatifs au leaderboard en cours de création. Vous pouvez ensuite définir la méthode de tri (voir les constantes listées ci-dessous) ainsi que la façon dont les informations sont affichées (encore une fois, l'une des constantes listées ci-dessous). La fonction retournera une valeur de -1 si elle ne parvient pas à créer le leaderboard ou une valeur supérieure à -1 si elle a réussi.

NOTE: Si vous avez déjà créé un leaderboard avec le même nom (soit par le code ou via votre page Steam pour le jeu) cette fonction sera ignorée!

Constante de tri La description
lb_sort_none Pas de tri. L'information sera affichée "telle quelle".
lb_sort_ascending Triez le classement dans l'ordre croissant.
lb_sort_descending Triez le classement dans l'ordre décroissant.
Afficher la constante La description
lb_disp_none Afficher le classement "tel quel".
lb_disp_numeric Afficher le classement en affichage numérique
lb_disp_time_sec Afficher les valeurs du classement en temps, avec la valeur de base étant secondes.
lb_disp_time_ms Affiche les valeurs du classement en temps, avec la valeur de base en millisecondes.


Exemple:

steam_create_leaderboard("Game Times", lb_sort_ascending, lb_disp_time_sec);

Le code ci-dessus va créer un classement appelé "Game Times", et le définir pour afficher les résultats dans l'ordre croissant et avec un affichage en secondes.
