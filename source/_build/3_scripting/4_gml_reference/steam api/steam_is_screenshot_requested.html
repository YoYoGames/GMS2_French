steam _is_screenshot_requested

Syntaxe:

steam_is_screenshot_requested();


Renvoie: Boolean


La description

Cette fonction interrogera l'API Steam pour voir si la touche permettant de prendre une capture d'écran du jeu a été pressée. La fonction ne retournera que true pour une étape (tick du jeu) si la touche a été pressée, et reviendra false à tous les autres moments.

NOTE: ceci ne prend pas une capture d'écran pour vous. Ceci indique seulement que la touche a été pressée et que vous devez utiliser les fonctions de GameMaker Studio 2 screen_save ou screen_save_part pour enregistrer une copie locale du fichier à télécharger.


Exemple:

if steam_is_screenshot_requested()
   {
   var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
   screen_save(file)
   steam_send_screenshot(file, window_get_width(), window_get_height());
   global.scrn_num += 1;
   }

Le code ci-dessus va interroger l'API Steam pour une demande de capture d'écran et si cela a été, un nom unique pour le fichier image sera généré, une capture d'écran sera prise et le fichier sera envoyé à la page Communauté Steam pour l'utilisateur.
