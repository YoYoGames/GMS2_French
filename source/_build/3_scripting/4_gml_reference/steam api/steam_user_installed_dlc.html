steam _user_installed_dlc

Syntaxe:

steam_user_installed_dlc(dlc_id);


Argument La description
dlc_id L'identifiant unique du DLC à vérifier (un entier).


Renvoie: Boolean


La description

Si votre jeu a été créé pour le DLC, vous pouvez utiliser cette fonction pour vérifier et voir si l'utilisateur l'a installé avant d'accéder aux fichiers qui lui sont associés. La fonction retourne true si le contenu du lecteur est installé, et false si l'utilisateur ne le fait pas, mais notez que l'utilisateur doit également posséder le DLC, vous devez donc utiliser la fonction supplémentaire de steam_user_owns_dlc pour vérifier qu'il est également possédé avant de l'utiliser.


Exemple:

global.Level_Max = 100;
if steam_user_owns_dlc(10354)
   {
   if steam_user_installed_dlc(10354)
      {
      global.Level_max = 200;
      }
   }

Le code ci-dessus vérifiera si l'utilisateur a acheté et installé le DLC avec l'ID 10354, et si c'est le cas, définissez une variable globale à une valeur différente.
