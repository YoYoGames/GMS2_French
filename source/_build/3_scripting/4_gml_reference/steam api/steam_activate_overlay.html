steam _activate_overlay

Syntaxe:

steam_activate_overlay(overlayindex);


Argument La description
overlayindex L'index de page de l'interface utilisateur de l'API Steam à afficher (voir les constantes répertoriées ci-dessous).


Retours: N / A


La description

La superposition Steam est une partie de l'interface utilisateur Steam qui peut être activée au-dessus de presque tous les jeux lancés via Steam. Il permet à l'utilisateur d'accéder à sa liste d'amis, son navigateur Web, son chat et ses achats de contenu téléchargeable. La touche par défaut pour un utilisateur d'accéder à la superposition pendant un jeu est SHIFT + TAB, mais vous pouvez également afficher n'importe quelle page de la superposition en utilisant cette fonction. Il prend l'une des six constantes énumérées ci-dessous:

Constant La description
ov_friends La page des amis de l'utilisateur actuel
ov_community La page de la communauté pour votre jeu
ov_players La page montrant les autres qui jouent au même jeu ou que vous avez récemment joué avec
ov_settings Les paramètres de superposition du client Steam
ov_gamegroup Ouvre le navigateur Web Steam Community au groupe de jeu officiel pour ce jeu
ov_achievements La page des succès pour votre jeu


Exemple:

var key = keyboard_lastkey;
switch (key)
   {
   case vk_f1: steam_activate_overlay(ov_friends); break;
   case vk_f2: steam_activate_overlay(ov_community); break;
   case vk_f3: steam_activate_overlay(ov_players); break;
   case vk_f4: steam_activate_overlay(ov_settings); break;
   case vk_f5: steam_activate_overlay(ov_gamegroup); break;
   case vk_f6: steam_activate_overlay(ov_achievements); break;
   }

Le code ci-dessus interroge la dernière touche du clavier enfoncée et, s'il s'agit de l'une des touches de fonction de 1 à 6, elle ouvre la page correspondante de la superposition Steam.
