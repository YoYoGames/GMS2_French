steam _download_friends_scores

Syntaxe:

steam_download_friends_scores(lb_name);


Argument La description
lb_name Le nom du classement dont vous téléchargez les partitions (une chaîne de caractères).


Retours: Real


La description

Avec cette fonction, vous ne pouvez récupérer que les scores du classement appartenant aux personnes marquées comme "amis" dans le client Steam. Donc, si votre classement compte 200 entrées, et que 50 d'entre elles sont vos amis, cette fonction ne récupérera que les 50 résultats. Le nom du classement est une chaîne qui a été définie lorsque vous avez créé le classement en utilisant la fonction steam_create_leaderboard, et la fonction renverra une valeur qui peut ensuite être utilisée pour identifier le rappel dans l' événement Steam Async, ou elle renverra -1 si elle a échoué.


Exemple:

score_get = steam_download_friends_scores("Game Scores");

Cela enverra une requête au serveur Steam pour les scores des amis utilisateurs du leaderboard donné et stockera l' identifiant asynchrone de la requête dans la variable "score_get". Cela sera ensuite traité dans l' événement Steam Async, comme indiqué dans l'exemple étendu pour steam_download_scores.
