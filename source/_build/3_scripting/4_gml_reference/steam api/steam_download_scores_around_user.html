steam _download_scores_around_user

Syntaxe:

steam_download_scores_around_user(lb_name, range_start, range_end);


Argument La description
lb_name Le nom du classement dont vous téléchargez les partitions (une chaîne de caractères).
range_start La position de départ dans le classement.
range_end La position de fin dans le classement.


Retours: Real


La description

Cette fonction est utilisée pour récupérer les entrées du classement par rapport à l'entrée actuelle des utilisateurs. le range_start paramètre est le nombre d'entrées à récupérer avant l'entrée actuelle des utilisateurs, et le range_end paramètre est le nombre d'entrées après l'entrée de l'utilisateur actuel, et l'entrée de l'utilisateur actuel est toujours incluse dans les résultats. Par exemple, si l'utilisateur actuel est le numéro 5 sur un classement donné, alors définir la plage de départ à -2 et la plage de fin à 2 renverra 5 entrées: 3 à 7. S'il n'y a pas assez d'entrées dans le classement avant ou après l'entrée de l'utilisateur, Steam ajustera la gamme pour essayer de retourner le nombre d'entrées demandées. Par exemple, si l'utilisateur est # 1 sur le classement, le début est défini sur -2 et la fin sur 2, Steam renvoie les 5 premières entrées dans le classement.


Exemple:

score_get = steam_download_scores_around_user("Game Scores", -4, 5);

Cela enverra une requête au serveur Steam pour une gamme de 10 scores du classement "Game Scores", centrée sur le joueur et stockera l' identifiant asynchrone de la requête dans la variable "score_get". Cela sera ensuite traité dans l' événement Steam Async, comme indiqué dans l'exemple étendu pour steam_download_scores.
