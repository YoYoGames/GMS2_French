xboxlive_stats_delete_stat


La description

Cette fonction peut être utilisée pour supprimer une statistique du gestionnaire de statistiques pour l'ID utilisateur donné. Vous fournissez l'ID utilisateur tel que renvoyé par (par exemple) la fonction xboxlive_get_user(), puis la chaîne de statistiques à supprimer. Ceci efface la valeur de la statistique et la supprime du gestionnaire de statistiques, ce qui signifie qu'elle ne sera plus renvoyée par les fonctions. xboxlive_stats_get_stat_names ou xboxlive_stats_get_stat.

La fonction retournera -1 s'il y avait une erreur ou si l'ID utilisateur est invalide, ou toute autre valeur si la fonction a été appelée avec succès

IMPORTANT! Cette fonction est uniquement valide lors de l'exportation à l'aide de la cible UWP pour la Xbox One et vous devez donc cocher l'option Activer XBox Live dans les options de jeu UWP.


Syntaxe:

xboxlive_stats_delete_stat(user_id, stat);

Argument La description
user_id Le pointeur d’ID utilisateur pour supprimer la stat de
stat La statistique à supprimer (une chaîne)


Résultats:

Real


Exemple:

for(var i = 0; i < xboxlive_get_user_count(); i++;)
    {
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_delete_stat(user_id[i], "HighScore");
    }

Le code ci-dessus parcourt les utilisateurs connectés et supprime la stat spécifiée du gestionnaire de stat pour chacun d'entre eux.
