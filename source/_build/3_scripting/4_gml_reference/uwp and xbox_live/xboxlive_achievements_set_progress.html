xboxlive_achievements_set_progress


La description

Cette fonction peut être utilisée pour mettre à jour l'avancement d'une réalisation. Vous fournissez l'ID utilisateur tel que renvoyé, par exemple, par la fonction xboxlive_get_user, puis la chaîne de réalisation (il s'agit de l'ID de réussite tel qu'il a été attribué dans le Centre de développement XDP / Windows lors de sa création). Enfin, vous définissez la valeur de progression sur une valeur comprise entre 0 et 100. Notez que le système d’exécution refusera les mises à jour inférieures à la valeur de progression actuelle.


Syntaxe:

xboxlive_achievements_set_progress(user_id, achievement, progress);

Argument La description
user_id L'ID utilisateur de l'utilisateur pour définir la réalisation de
achievement La réalisation à définir (une chaîne)
progress La valeur de progression à définir (de 0 à 100)


Résultats:

N/A


Exemple étendu:

Voici un exemple étendu d'utilisation de cette fonction. Pour commencer, appelez-le dans un événement comme le démarrage ou la création d'une pièce:

xboxlive_stats_get_social_leaderboard(user_id, "GlobalTime", 20, 1, false, false);

Le code ci-dessus serait appelé pour obtenir une liste de toutes les positions mondiales du classement pour le jeu et générera un rappel d'événement social asynchrone que nous traiterions de la manière suivante:

var _progress = (global.Level / global.LevelMax) * 100;
xboxlive_achievements_set_progress(user_id, "Game_Completed", _progress);

Le code ci-dessus crée une valeur en pourcentage basée sur certaines variables globales et l'utilise ensuite pour définir la progression des performances pour la réalisation "Game_Completed".
