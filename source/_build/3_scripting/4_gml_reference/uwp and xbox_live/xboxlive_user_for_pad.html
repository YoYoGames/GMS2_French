xboxlive_user_for_pad


La description

Cette fonction renvoie le pointeur d’ID utilisateur associé à la valeur d’index de la manette de jeu donnée, ou pointer_null si aucun utilisateur n'y est associé. Notez que cette fonction ne doit être utilisée qu'avec les index de la manette de jeu, et non avec xboxlive_get_user_count().

IMPORTANT! Cette fonction est uniquement valide lors de l'exportation à l'aide de la cible UWP pour la Xbox One et vous devez donc cocher l'option Activer XBox Live dans les options de jeu UWP.


Syntaxe:

xboxlive_user_for_pad(pad_index);

Argument La description
pad_index L'index (un entier) de la fente de la manette de jeu à vérifier


Résultats:

Pointer


Exemple:

for(var i = 0; i < 11; ++i;)
    {
    var u_id = xboxlive_user_for_pad(i);
    if u_id != pointer_null
        {
        pad_name[i] = xboxlive_gamedisplayname_for_user(u_id);
        }
    else
        {
        pad_name[i] = "Press Play";
        }
    }

Le code ci-dessus parcourt tous les emplacements de pad disponibles et vérifie si un pointeur ID utilisateur y est associé. Si tel est le cas, un tableau est défini avec le nom d'affichage de l'utilisateur. Sinon, il contient un texte par défaut.
