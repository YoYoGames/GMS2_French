xboxlive_sprite_add_from_gamerpicture


La description

Avec cette fonction, vous pouvez obtenir une image de joueur pour un pointeur d’ID utilisateur donné. La fonction fonctionne de manière asynchrone et déclenche un événement asynchrone Image Loaded pour vous informer que la fonction est terminée (comme avec les appels HTML5 / URL à sprite_add() ). L’événement Asynchrone Image Loaded aura une carte DS stockée dans la variable intégrée. async_load. La carte contient les informations suivantes:

  • " nom de fichier ": le chemin complet du fichier que vous avez demandé.
  • " id ": l'ID de la ressource que vous avez chargée. Ce sera la même chose que la variable à laquelle vous avez affecté la ressource.
  • " status ": Retourne une valeur inférieure à 0 en cas d'erreur.
REMARQUE: lorsque vous chargez dynamiquement une image-objet dans votre jeu au moment de l'exécution, vous devez vous rappeler de le supprimer à nouveau à l'aide de sprite_delete() lorsque vous n'en avez plus besoin, sinon il y a un risque de fuite de mémoire qui ralentira et finira par planter votre partie.
IMPORTANT! Cette fonction est uniquement valide lors de l'exportation à l'aide de la cible UWP pour la Xbox One et vous devez donc cocher l'option Activer XBox Live dans les options de jeu UWP.


Syntaxe:

xboxlive_sprite_add_from_gamerpicture(user_id, imagesize, xorig, yorig);

Argument La description
user_id L'ID utilisateur (un pointeur) pour obtenir l'image du joueur pour
imagesize Taille en pixels du sprite à renvoyer
xorig Indiquer la position x de l'origine dans le sprite
yorig Indiquer la position y de l'origine dans le sprite


Résultats:

Real


Exemple:

var _a = 0;
var _num = xboxlive_get_user_count();
for (var i = 0; i < _num; ++i;)
    {
    var _uid = xboxlive_get_user(i);
    if _uid != pointer_null
        {
        global.UserName[_a] = xboxlive_gamedisplayname_for_user(_uid);
        global.UserScore[_a] = xboxlive_gamerscore_for_user(_uid);
        global.UserRep[_a] = xboxlive_reputation_for_user(_uid);
        global.UserAvatar[_a] = xboxlive_sprite_add_from_gamerpicture(_uid, 256, 0, 0);
        ++a;
        }
    }

Le code ci-dessus parcourt les utilisateurs connectés et stocke leurs données de joueur dans diverses baies globales.
