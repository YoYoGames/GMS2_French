xboxlive_stats_set_stat_string


La description

Cette fonction peut être utilisée pour définir une statistique sur une chaîne pour l'ID utilisateur donné. Vous fournissez l'ID utilisateur tel que renvoyé par (par exemple) la fonction xboxlive_get_user(), puis la chaîne de statistiques à définir ( si la chaîne de statistiques n’existe pas déjà, une nouvelle statistique sera créée et définie sur la valeur donnée ), puis la chaîne sur laquelle définir la statistique. Notez que le nom de la statistique doit être alphanumérique uniquement, sans symboles ni espaces, mais que la valeur de chaîne que vous définissez peut comporter des espaces et des caractères spéciaux.

Lors de la définition de la chaîne de statistique, toute chaîne précédente sera remplacée. Par conséquent, il vous appartient de déterminer si la chaîne de statistique doit être mise à jour ou non en comparant la chaîne de statistiques actuelle à la nouvelle avant de la définir.

La fonction retournera -1 s'il y avait une erreur ou si l'ID utilisateur n'est pas valide, ou toute autre valeur si la fonction a été appelée avec succès.

IMPORTANT! Cette fonction est uniquement valide lors de l'exportation à l'aide de la cible UWP pour la Xbox One et vous devez donc cocher l'option Activer XBox Live dans les options de jeu UWP.


Syntaxe:

xboxlive_stats_set_stat_string(user_id, stat, value);

Argument La description
user_id Le pointeur d’ID utilisateur pour définir la statistique pour
stat La statistique à définir (une chaîne)
value La valeur pour définir la stat sur (une chaîne)


Résultats:

Real


Exemple:

xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName);

Le code ci-dessus définit la stat "LevelName" sur la chaîne stockée dans la variable globale donnée.
