xboxlive_gamedisplayname_for_user


La description

Avec cette fonction, vous pouvez récupérer le nom d’affichage - sous forme de chaîne - à partir du pointeur d’ID utilisateur indiqué. Notez que si l'utilisateur est local, la valeur renvoyée est simplement une chaîne de son nom complet, mais s'il s'agit d'un utilisateur distant, la chaîne sera son gamertag, qui peut ensuite être utilisé, par exemple, pour la mise en sourdine / réactivation du son. bavarder.

IMPORTANT! Cette fonction est uniquement valide lors de l'exportation à l'aide de la cible UWP pour la Xbox One et vous devez donc cocher l'option Activer XBox Live dans les options de jeu UWP.


Syntaxe:

xboxlive_gamedisplayname_for_user(user_id);

Argument La description
user_id Le pointeur d'identification de l'utilisateur à vérifier.


Résultats:

String


Exemple:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
    {
    user_id[i] = xboxlive_get_user(i);
    user_name[i] = xboxlive_gamedisplayname_for_user(user_id[i]);
    }

Le code ci-dessus obtient le pointeur d'identification d'utilisateur pour chaque compte d'utilisateur disponible, puis stocke le pointeur avec le nom d'affichage dans deux tableaux.
