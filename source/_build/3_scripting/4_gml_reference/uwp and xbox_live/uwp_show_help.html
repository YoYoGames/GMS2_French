uwp_show_help


La description

Cette fonction lance l’application d’aide Xbox One qui affiche à son tour le fichier d’aide de votre jeu (fichier au format HTML que vous avez précédemment chargé sur le portail de développeur XBox). Notez qu'un ID utilisateur est spécifié lors de l'appel de cette fonction afin que les informations de localisation de l'utilisateur puissent être utilisées lors de son affichage. Vous pouvez récupérer l'ID utilisateur avec les fonctions xboxlive_get_user(), xboxlive_get_activating_user(), ou uwp_license_trial_user.

REMARQUE: cette fonction n’est applicable que lors de la soumission de jeux via XBox Live pour la XBox One à l’aide du package UWP.


Syntaxe:

uwp_show_help(user_id);

Argument La description
user_id Le pointeur d'utilisateur pour ouvrir le fichier d'aide.


Résultats:

Boolean


Exemple:

if gamepad_button_check_pressed(0, gp_start)
    {
    if uwp_license_trial_version()
        {
        var _uid = uwp_license_trial_user;
        uwp_show_help(_uid);
        }
    }

Le code ci-dessus vérifie si un bouton est enfoncé, puis vérifie si l'application est en cours d'exécution avec une licence d'évaluation. Si c'est le cas, il obtient l'ID utilisateur pour la licence et ouvre le fichier d'aide pour cet utilisateur.
