xboxlive_user_is_guest


La description

Cette fonction retournera true si le pointeur d’ID utilisateur donné est un utilisateur invité et false s'ils ne le sont pas. Vous pouvez obtenir un pointeur d’ID utilisateur avec la fonction xboxlive_get_user().

IMPORTANT! Cette fonction est uniquement valide lors de l'exportation à l'aide de la cible UWP pour la Xbox One et vous devez donc cocher l'option Activer XBox Live dans les options de jeu UWP.


Syntaxe:

xboxlive_user_is_guest(user_id);

Argument La description
user_id Le pointeur d'identification de l'utilisateur à vérifier.


Résultats:

Boolean


Exemple:

if xboxlive_user_is_guest(user_id[0])
    {
    global.user_name[0] = "GUEST";
    }
else
    {
    global.user_name[0] = xboxlive_gamedisplayname_for_user(user_id[0]);
    }

Ce qui précède stocke le pointeur d’activation de l’ID utilisateur dans une variable globale.
