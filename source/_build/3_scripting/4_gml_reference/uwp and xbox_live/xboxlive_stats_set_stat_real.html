xboxlive_stats_set_stat_real


La description

Cette fonction peut être utilisée pour définir la valeur d’une statistique pour l’ID utilisateur donné. Vous fournissez l'ID utilisateur tel que renvoyé par (par exemple) la fonction xboxlive_get_user(), puis la chaîne de statistiques à définir ( si la chaîne de statistiques n’existe pas déjà, une nouvelle statistique sera créée et définie sur la valeur donnée ) et une valeur (un réel) pour définir la statistique. Notez que le nom de la statistique doit être alphanumérique uniquement, sans symboles ni espaces.

Lorsque vous définissez la valeur stat, toute valeur précédente sera remplacée. Par conséquent, il vous appartient de déterminer si la valeur stat doit être mise à jour ou non (c.-à-d. Vérifier que le score le plus élevé est réellement le plus élevé) en le comparant à la valeur stat actuelle. avec le nouveau avant de le régler.

La fonction retournera -1 s'il y avait une erreur ou si l'ID utilisateur n'est pas valide, ou toute autre valeur si la fonction a été appelée avec succès.

IMPORTANT! Cette fonction est uniquement valide lors de l'exportation à l'aide de la cible UWP pour la Xbox One et vous devez donc cocher l'option Activer XBox Live dans les options de jeu UWP.


Syntaxe:

xboxlive_stats_set_stat_real(user_id, stat, value);

Argument La description
user_id Le pointeur d’ID utilisateur pour définir la statistique pour
stat La statistique à définir (une chaîne)
value La valeur pour définir la stat sur (un réel)


Résultats:

Real


Exemple:

if game_over == true
    {
    if xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100
        {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
        }
    }

Le code ci-dessus vérifie une variable et si elle est true, il vérifiera alors la valeur de la statistique "PercentDone". Si cette valeur est inférieure à 100, une valeur est ensuite générée et l'état défini sur cette valeur.
