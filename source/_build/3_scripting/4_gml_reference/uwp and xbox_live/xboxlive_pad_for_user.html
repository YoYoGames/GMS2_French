xboxlive_pad_for_user


La description

Les utilisateurs de la Xbox peuvent avoir différents pads assignés à leur ID utilisateur, chacun remplissant un "emplacement". Ces slots sont numérotés à partir de 0 et peuvent contenir n’importe quel index de pad, ainsi l’emplacement 0 pourrait être attribué au pad 3 et l’emplacement 1 au pad 2. Cette fonction peut être utilisée pour obtenir le pad affecté à un slot donné. et est particulièrement utile si un utilisateur laisse une manette de jeu sans entrée pendant un certain temps, car il s’éteindra lui-même. Vous pouvez ainsi vérifier tout ID utilisateur à l’aide de cette fonction pour vous assurer qu’il dispose d’une manette de jeu active. ne réagissez pas ensuite en conséquence, par exemple en mettant votre jeu en pause. Si le pad est actif, sa valeur d'index sera retournée, mais sinon, la fonction retournera -1.

Notez que si un pad de n'importe quel slot devient inactif, les slots suivants "glissent" dans la position vacante. Par conséquent, si vous avez trois pads actifs affectés aux slots 0, 1 et 2, et que le pad de la slot 0 est inactif, les slots 1 et to deviendra maintenant les slots 0 et 1, donc le slot 0 devrait toujours contenir un pad actif à moins que tous les pads ne soient désactivés (dans ce cas, il retournera -1).

IMPORTANT! Cette fonction est uniquement valide lors de l'exportation à l'aide de la cible UWP pour la Xbox One et vous devez donc cocher l'option Activer XBox Live dans les options de jeu UWP.


Syntaxe:

xboxlive_pad_for_user(user_id, slot);

Argument La description
user_id L'ID utilisateur (un pointeur) à vérifier
user_id L'index (un entier) de la fente de la manette de jeu à vérifier


Résultats:

Pointer


Exemple:

if xboxlive_pad_for_user(user_id, 0) == -1
    {
    for(var i = 0; i < gamepad_get_device_count(); i++;)
        {
        var u_id = xboxlive_user_for_pad(i);
        if u_id == -1
            {
            async_ID = xboxlive_pad_for_user(i, u_id);
            }
        }
    }

Le code ci-dessus vérifie l'emplacement 0 pour le pointeur d'identifiant d'utilisateur donné pour voir si des manettes de jeu ont été attribuées et si ce n'est pas le cas, l'une d'entre elles est affectée.
