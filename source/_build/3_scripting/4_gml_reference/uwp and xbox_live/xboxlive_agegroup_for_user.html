xboxlive_agegroup_for_user


La description

Cette fonction renverra l’une des quatre constantes (illustrées ci-dessous) pour indiquer la tranche d’âge attribuée au pointeur d’ID utilisateur spécifié.

IMPORTANT! Cette fonction est uniquement valide lors de l'exportation à l'aide de la cible UWP pour la Xbox One et vous devez donc cocher l'option Activer XBox Live dans les options de jeu UWP.


Syntaxe:

xboxlive_agegroup_for_user(user_id);

Argument La description
user_id L'ID utilisateur (un pointeur) à vérifier


Résultats:

Constant

Constant La description
xboxlive_agegroup_unknown Le groupe d'âge de l'utilisateur est inconnu
xboxlive_agegroup_child Le groupe d'âge des utilisateurs est compris entre 8 et 12 ans
xboxlive_agegroup_teen Le groupe d'âge des utilisateurs est compris entre 13 et 17 ans (inclus, et entre 13 et 19 ans en Corée du Sud)
xboxlive_agegroup_adult Le groupe d’âge des utilisateurs est de 18 ans ou plus (ou de 20 ans et plus en Corée du Sud).


Exemple:

var _age = xboxlive_agegroup_for_user(xboxlive_get_activating_user());
if _age != xboxlive_agegroup_adult
    {
    global.ContentControl = true;
    }
else global.ContentControl = false;

Le code ci-dessus vérifie le groupe d’âge de l’activateur et définit une variable globale sur true ou false en fonction de la constante retournée.
