xboxlive_stats_get_stat_names


La description

Cette fonction peut être utilisée pour extraire toutes les statistiques définies du gestionnaire de statistiques pour un utilisateur donné. Vous fournissez l'ID utilisateur tel que renvoyé par (par exemple) la fonction xboxlive_get_user() et la fonction renverra un tableau de chaînes contenant les statistiques pour l'utilisateur. Si une erreur survient ou si l'utilisateur n'a pas de statistiques, le tableau sera toujours renvoyé mais aura un nombre d'éléments égal à 0.

IMPORTANT! Cette fonction est uniquement valide lors de l'exportation à l'aide de la cible UWP pour la Xbox One et vous devez donc cocher l'option Activer XBox Live dans les options de jeu UWP.


Syntaxe:

xboxlive_stats_get_stat_names(user_id);

Argument La description
user_id Le pointeur d’ID utilisateur pour obtenir les noms de statistiques de


Résultats:

Array


Exemple:

var _stat_str = xboxlive_stats_get_stat_names(user_id);
for (var i = 0; i < array_length_1d(_stat_str); ++i;)
    {
    xboxlive_stats_delete_stat(user_id, _stat_str[i]);
    }

Le code ci-dessus récupère toutes les statistiques définies pour le pointeur ID utilisateur stocké dans la variable donnée, puis parcourt le tableau et supprime la statistique spécifiée du gestionnaire de statistiques.
