uwp_is_constrained


La description

Cette fonction retournera true si le jeu est exécuté en mode ressources limitées ou false si vous courez normalement. Le mode ressources limitées peut être activé lorsque le jeu n'a pas le focus, et dans ce mode, les ressources du processeur graphique ont été réduites. La réponse à cela devrait normalement être de mettre le jeu en pause (étant donné que l'utilisateur n'a plus le contrôle). En outre, la désactivation de tout effet particulièrement éprouvant peut être une bonne idée si un ralentissement est observé dans ce mode. Notez que contrairement au uwp_is_suspending() fonction, cette fonction retournera true continuellement tant que le jeu est en mode contraint (et pas une seule fois quand il se produit).


Syntaxe:

uwp_is_constrained();


Résultats:

Boolean


Exemple:

if xboxliveis_constrained()
    {
    if !instance_exists(obj_PauseMenu)
        {
        instance_create(0,0,obj_PauseMenu);
        }
    }

Le code ci-dessus vérifie si l'application est contrainte et, le cas échéant, crée une instance d'objet (utilisée pour suspendre le jeu).
