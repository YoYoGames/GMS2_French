date_inc_day

Renvoie une date / heure après l'ajout d'un nombre de jours donné.

Syntaxe:

date_inc_day(date, amount);


Argument La description
date Le datetime à ajouter.
amount Le nombre de jours (doit être un nombre entier) à ajouter.


Retours: Real


La description

Avec cette fonction, vous pouvez incrémenter une valeur datetime donnée d'un nombre spécifique de jours et retourner la nouvelle valeur datetime.


Exemple:

mynewdatetime = date_inc_day(date_current_datetime(), 365);

Cela définirait "mynewdatetime" à la date actuelle mais avec 365 jours ajoutés.


Retour: Date et heure
