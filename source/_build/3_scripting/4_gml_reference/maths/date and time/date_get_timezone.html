date_get_timezone

Définit le fuseau horaire de base à utiliser pour toutes les fonctions de date et d'heure.

Syntaxe:

date_get_timezone();


Retours: Constant


La description

Obtient le fuseau horaire de base utilisé pour toutes les autres fonctions de date et d'heure. Ce fuseau horaire peut être local (défini par le système) ou UTC et la fonction renvoie l'une des constantes suivantes:

Constant La description
timezone_local utiliser le fuseau horaire local défini par le système
timezone_utc utiliser le temps universel coordonné


Exemple:

if date_get_timezone() != timezone_utc
   br>    date_set_timezone(timezone_utc);
   }

Ce code vérifie le réglage du fuseau horaire de base du jeu et s'il ne l'est pas, il le modifie.


Retour: Date et heure
