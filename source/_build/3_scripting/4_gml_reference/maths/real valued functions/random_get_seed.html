random_get_seed


La description

Pour générer un nombre aléatoire, GameMaker Studio 2 commence par un numéro de graine aléatoire. Avec cette fonction, vous pouvez récupérer cette graine, qui peut ensuite être stockée pour reproduire une série spécifique d’événements aléatoires. Par exemple, supposons que vous ayez créé un jeu procédural, mais que vous souhaitiez que le joueur puisse revenir aux zones précédemment découvertes. Eh bien, au début de chaque zone, vous pouvez stocker la graine aléatoire avec cette fonction. Ainsi, chaque fois que vous devez y retourner, vous pouvez utiliser random_set_seed pour définir la valeur de départ comme auparavant et ainsi forcer GameMaker Studio 2 à reproduire le même ensemble de fonctions "aléatoires" que celui utilisé précédemment.


Syntaxe:

random_get_seed();


Résultats:

Real


Exemple:

ini_open("Save.ini")
ini_write_real("Levels", string(level), random_get_seed());
ini_close();

Le code ci-dessus va récupérer la graine aléatoire et la stocker dans un fichier ini.
