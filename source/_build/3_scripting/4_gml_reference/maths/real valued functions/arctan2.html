arctan2

Calcule arctan (y / x) et renvoie un angle dans le quadrant correct.

Syntaxe:

arctan2(y, x);


Argument La description
y La coordonnée y à calculer.
x La coordonnée x à calculer.


Retours: Real


La description

Cette fonction renvoie la tangente inverse d'un angle y / x, où y = côté opposé du triangle et x = côté adjacent du triangle. Contrairement à arctan (x), la fonction arctan2 (y, x) est valable pour tous les angles et peut donc être utilisée pour convertir un vecteur en un angle sans risque de division par zéro, et renvoie également un résultat dans le quadrant correct.

NOTE: La valeur renvoyée est en radians et non en degrés.


Exemple:

val = arctan2(1, 1);

Cela va régler val à l'angle correct, dans ce cas 0,79.
