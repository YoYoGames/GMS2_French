log10

Renvoie la base de journal 10 de n.

Syntaxe:

log10(n);


Argument La description
n La valeur d'entrée.


Retours: Real


La description

Cette fonction est similaire à la log2(n) fonction, seulement que log10(n) obtient le nombre de 10 que nous devons multiplier ensemble pour obtenir n. Alors log10(100) renvoie 2, en multipliant 10 par lui-même donne 100, log10(1000) renvoie 3 comme 10x10x10 = 1000 etc... Le nombre ne doit pas être divisible par dix, mais cela retournera quand même une valeur appropriée, par exemple si nous avons log10(101) nous obtenons 2.00432137... combien de fois vous pouvez diviser 101 par 10.


Exemple:

logval = log10(num);

Le code ci-dessus obtient le log10 de la valeur stockée dans "num".
