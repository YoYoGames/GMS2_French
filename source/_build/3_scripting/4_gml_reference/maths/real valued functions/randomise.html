randomiser


La description

Cette fonction définit le germe sur une valeur aléatoire. Si vous devez conserver une valeur constante sur plusieurs exécutions d’un jeu, vous devriez utiliser random_set_seed(). Veuillez noter que, lorsque vous utilisez les fonctions de nombre aléatoire dans GameMaker Studio 2, la valeur initiale est toujours la même, ce qui facilite GameMaker Studio le suivi des erreurs et le débogage. Si vous souhaitez tester avec de vrais aléatoires, appelez cette fonction au début de votre partie. La fonction renverra la nouvelle valeur de départ randomisée (un entier non signé de 32 bits).


Syntaxe:

randomise();
// or
randomize();


Résultats:

Integer (unsigned 32 bit value)


Exemple:

randomise();

Le code ci-dessus va randomiser la graine.
