plage_aléatoire


La description

Cette fonction renvoie un nombre aléatoire entre la plage spécifiée et cette valeur de retour n'a pas besoin d'être un entier. Par exemple, random_range(20,50) retournera un nombre aléatoire de 20 à 50, mais la valeur peut être un nombre réel tel que 38,65265. Les nombres réels peuvent également être utilisés comme arguments d'entrée.

REMARQUE: cette fonction renverra la même valeur à chaque nouvelle exécution du jeu car GameMaker Studio 2 génère la même graine aléatoire initiale à chaque fois pour faciliter le débogage du code. Pour éviter ce comportement, utilisez randomise au début de votre partie.


Syntaxe:

random_range(n1, n2);


Argument La description
n1 La limite inférieure de la plage à partir de laquelle le nombre aléatoire sera sélectionné.
n2 Le haut de la plage à partir de laquelle le nombre aléatoire sera sélectionné.


Résultats:

Real


Exemple:

score += random_range(500, 600);

Cela ajoutera entre 500 et 600, au score total.
