Arctan

Renvoie la tangente inverse de x.

Syntaxe:

arctan(x);


Argument La description
x L'angle (en radians) pour retourner la tangente inverse de.


Retours: Real


La description

Renvoie la tangente inverse de x. Cela acceptera n'importe quel nombre car, contrairement à tan, arctans asymptotes sont sur l'axe des y, donc cela signifie simplement que vous ne recevrez jamais un nombre supérieur à pi / 2 ou inférieur à -pi / 2.


Exemple:

val = arctan( 0 );

Cela va mettre val à 0.
