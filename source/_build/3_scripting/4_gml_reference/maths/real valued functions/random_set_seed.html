random_set_seed


La description

Pour générer un nombre aléatoire, GameMaker Studio 2 commence par un numéro de graine aléatoire. Avec cette fonction, vous pouvez attribuer à cette valeur initiale une valeur connue et ainsi "forcer" le résultat de tous les événements aléatoires à la même valeur à chaque exécution du programme. Par exemple, cette fonction peut être utilisée conjointement avec random_get_seed pour créer du contenu généré de manière procédurale et enregistrer les résultats sans avoir de grandes sauvegardes (vous ne sauvegardez que le germe, vous n'avez besoin de rien d'autre). Si vous avez besoin de résultats vraiment aléatoires pour tout, vous devriez utiliser le randomise() une fonction.

REMARQUE: Bien que cette valeur initiale donne des résultats cohérents sur chaque plate-forme cible, les résultats peuvent varier d'une plate-forme à l'autre en raison du mode de fonctionnement différent de chaque cible.


Syntaxe:

random_set_seed(val);


Argument La description
val La graine à définir.


Résultats:

N/A


Exemple:

if debug random_set_seed(1);

Le code ci-dessus définit la graine aléatoire sur 1 uniquement si la variable "debug" est vraie.
