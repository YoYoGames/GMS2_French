exp

La fonction exponentielle

Syntaxe:

exp(n);


Argument La description
n Le pouvoir de e.


Retours: Real


La description

C'est la fonction power (e, n), où e est approximativement 2.718281828 (aussi connu sous le nom de nombre d'Euler ), et n est le nombre de fois qu'il devrait être multiplié par lui-même. Cela dit essentiellement "combien de croissance puis-je obtenir après n unités de temps (et 100% de croissance continue)?". Alors exp(3) retournerait 20,08, ce qui signifie que, après 3 unités de temps, nous avons 20,08 fois le montant que nous avons commencé avec.


Exemple:

val = exp(2);

Ceci mettra val à e ^ 2, autour de 7.4.
