instance_place_list


La description

Avec cette fonction, vous pouvez vérifier la position d'une collision avec toutes les instances d'un objet à l'aide du masque de collision de l'instance qui exécute le code pour la vérification. En l'utilisant, vous demandez effectivement à GameMaker Studio 2 de déplacer l'instance vers le nouvel GameMaker Studio 2, de rechercher une collision, puis de revenir en arrière et de vous dire si une collision a été détectée ou non. Cela fonctionnera pour des collisions précises, mais uniquement si l'instance et les occurrences de l'objet soumis à la vérification disposent de masques de collision précis sélectionnés. Sinon, seules les collisions du cadre de sélection sont appliquées. La fonction remplira une liste DS (créée précédemment) avec l'unique id les valeurs de toutes les instances de l'objet considérées comme étant en conflit avec l'instance appelante, et vous avez la possibilité d'ordonner la liste en fonction de la distance entre l'origine de l'instance effectuant la vérification et l'origine des instances trouvées en collision. Notez que la fonction accepte également le mot clé spécial. all Dans ce cas, toutes les instances trouvées en conflit seront répertoriées. La fonction renvoie le nombre d'instances trouvées, ou 0 si aucune n'est trouvée.

Notez que les coordonnées x / y données seront arrondies au nombre entier le plus proche avant que la vérification ne soit effectuée. Par conséquent, si ce n'est pas ce dont vous avez besoin (ou si vous utilisez un produit GameMaker hérité), vous devez floor les coordonnées dans le chèque: instance_place_list(floor(x), floor(y), obj_Enemy, list, true).


Syntaxe:

instance_place_list(x, y, obj, list, ordered);

Argument La description
x La position x pour vérifier les instances.
y La position y pour vérifier les instances.
obj L'objet à vérifier pour les instances de.
list La liste DS à utiliser pour stocker les ID des instances en conflit.
ordered Si les instances doivent être classées par distance ou non de l'origine.


Résultats:

Int (The number of instances found to be in collision)


Exemple:

var _list = ds_list_create();
var _num = instance_place_list(x, y, obj_Enemy, _list, false);
if _num > 0
    {
    for (var i = 0; i < _num; ++i;)
        {
        instance_destroy(_list[| i]);
        }
    }
ds_list_destroy(_list);

Le code ci-dessus recherchera une collision avec toutes les instances trouvées à la position des instances appelantes. Celles-ci seront ajoutées à une liste DS, qui sera ensuite bouclée pour détruire chacune des instances en conflit.
