direction


La description

Toutes les instances de GameMaker Studio 2 possèdent certaines propriétés "intégrées" que vous pouvez utiliser et définir pour contrôler leur aspect et leur comportement. Direction est l'une de ces propriétés et peut être utilisée pour définir la direction du mouvement de l'instance lorsque l'instance a un speed autre que 0. Notez que les directions dans GameMaker Studio 2 sont généralement calculées comme 0 ° étant à droite, 90 ° étant en hausse, 180 ° étant à gauche et 270 ° étant vers le bas.


Syntaxe:

direction;


Résultats:

Real


Exemple:

if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

Le code ci-dessus va changer la direction du mouvement de l'instance en fonction de la touche (gauche ou droite) pressée.
