timeline_speed


La description

Normalement, à chaque étape, la position dans la ligne de temps est augmentée de 1, mais vous pouvez changer cette valeur en réglant cette variable sur une valeur différente. Vous pouvez utiliser des nombres réels (comme 0.5 ou 2.4 par exemple) et si la valeur est supérieure à un, plusieurs moments peuvent se produire dans le même pas de temps (ils seront tous exécutés dans le même ordre que pour la ligne de temps, aucune action ne sera ignorée).

Pour plus d'informations sur les délais, voir ici.


Syntaxe:

timeline_speed;


Résultats:

Real


Exemple:

if !timeline_running
   {
   timeline_running = true;
   timeline_speed = 0.5;
   }

Le code ci-dessus va vérifier et voir si l'instance exécute sa ligne de temps associée et si ce n'est pas le cas, il commencera la ligne de temps à la moitié de la vitesse.
