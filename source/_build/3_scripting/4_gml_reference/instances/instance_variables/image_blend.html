image_blend


La description

Cette variable contrôle la "teinte" de l'image-objet de l'instance et la valeur par défaut est -1 (mais peut également être c_white ). Toute autre valeur (y compris les constantes de couleur internes comme c_red, ou c_aqua ) fusionnera la couleur spécifiée avec l'image-objet d'origine. Notez que pour que les modifications de cette variable soient visibles, l'instance ne doit comporter aucun événement draw (et donc GameMaker Studio 2 dessine l'image-objet par défaut) ou être dessiné en utilisant l'une des fonctions de dessin étendues comme draw_self() ou draw_sprite_ext(). Notez que vous devez essayer de limiter la fusion sur les plates-formes Windows 8 (JavaScript) ou HTML5 (à moins d'utiliser WebGL), car chaque image-objet fusionnée doit être mise en cache séparément, ce qui peut affecter les performances (vous pouvez également définir taille du cache en utilisant la fonction sprite_set_cache_size ).


Syntaxe:

image_blend;


Résultats:

Real


Exemple:

image_blend = make_colour_hsv(255, 255, random(255));

Le code ci-dessus définit l'image_blend de l'image-objet définie pour l'occurrence sur une couleur aléatoire.
