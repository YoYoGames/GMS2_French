timeline_index


La description

Cette variable contient l'index de la ligne de temps actuellement associée à l'instance. Vous pouvez définir cette option sur une ligne de temps particulière pour l'utiliser ou la définir sur -1 pour arrêter l'utilisation d'une ligne de temps pour l'instance (si aucune ligne de temps n'est définie pour l'instance, -1 est également renvoyé). Notez que cela ne démarre pas la ligne de temps - pour cela utiliser la variable timeline_running.

Pour plus d'informations sur les délais, voir ici.


Syntaxe:

timeline_index;


Résultats:

Real (Resource index value)


Exemple:

if timeline_exists(global.tl)
   {
   timeline_index = global.tl;
   }

Le code ci-dessus assignera à l'instance exécutant le code une ligne de temps indexée dans la variable "global.tl" si cette ligne de temps existe.
