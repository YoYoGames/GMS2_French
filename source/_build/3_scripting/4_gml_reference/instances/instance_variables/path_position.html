path_position


La description

Cette fonction peut être utilisée pour obtenir ou pour définir la position d'une instance le long d'un chemin. La valeur est normalisée de 0 à 1, donc si vous le mettez à, par exemple, 0.5, l'instance sera déplacée exactement au milieu du chemin.

Pour les fonctions relatives aux chemins, voir ici.


Syntaxe:

path_position;


Résultats:

Real


Exemple:

path_position = random(1);

Le code ci-dessus définit l'instance à une position aléatoire n'importe où sur le chemin actuel.
