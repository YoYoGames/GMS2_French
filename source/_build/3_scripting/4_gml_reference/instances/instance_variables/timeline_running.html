timeline_running


La description

Cette variable contient l'état actuel de la ligne de temps affectée et retournera true s'il est en cours d'exécution et false si ce n'est pas le cas. Vous pouvez également définir cette variable sur true ou false pour démarrer et arrêter la ligne de temps à tout moment. il convient de noter qu'une ligne de temps arrêtée n'est pas réinitialisée, et ainsi la recommencer plus tard la démarrera à partir du moment exact où elle a été arrêtée.

Pour plus d'informations sur les délais, voir ici.


Syntaxe:

timeline_running;


Résultats:

Boolean


Exemple:

if !timeline_running
   {
   timeline_position = 0;
   timeline_running = true;
   }

Le code ci-dessus va vérifier si l'instance exécute une ligne de temps, et si ce n'est pas le cas, alors il réinitialise la ligne de temps affectée pour commencer au premier moment, puis le démarre.
