sprite_width


La description

Cette variable en lecture seule renvoie la largeur de l'image-objet qui a été affectée à l'instance. Cette largeur est renvoyée en pixels et dépendra du image_xscale. Si vous avez besoin de la largeur non mise à l'échelle, utilisez sprite_get_width.

Pour les fonctions relatives aux sprites, voir ici.


Syntaxe:

sprite_width;


Résultats:

Real


Exemple:

if sprite_width != sprite_get_width(sprite_index)
   {
   image_xscale = 1;
   }

Le code ci-dessus vérifie la largeur de l'image-objet affectée à l'instance exécutant le code par rapport à la largeur de la ressource d'image-objet et, si elle n'est pas la même, réinitialise image_xscale à 1,
