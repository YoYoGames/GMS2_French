path_index


La description

Toutes les ressources de GameMaker Studio 2 ont une valeur d'identification unique path_index est une variable en lecture seule qui contient la valeur d'index pour une ressource de chemin donnée. Si l'instance n'a pas de chemin affecté, la variable sera définie sur -1.

Pour les fonctions relatives aux chemins, voir ici.


Syntaxe:

path_index;


Résultats:

Real (Resource index value)


Exemple:

if path_index = -1
   {
   path_start(pth_enemy3, 4, path_action_reverse, 0);
   }

Le code ci-dessus vérifie si un chemin a été assigné à l'instance, et sinon il commence un nouveau chemin (en l'assignant au path_index en même temps).
