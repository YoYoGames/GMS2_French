image_angle


La description

Cette valeur définit l'angle (rotation) de l'image-objet et est mesurée en degrés, avec la droite étant 0º, en haut étant 90º, gauche étant 180º et descendant étant 270º. Définissez cette variable sur 0 pour réinitialiser l'image-objet à dessiner comme cela a été défini dans l'éditeur de sprites. S'il vous plaît noter que les modifications de cette variable soient visibles, l'instance doit avoir soit aucun cas de tirage au sort (et donc GameMaker Studio 2 sera par défaut le sprite) ou être aspirés en utilisant l' une des fonctions de dessin LIKE draw_self() ou draw_sprite_ext().


Syntaxe:

image_angle;


Résultats:

Real


Exemple:

image_angle = point_direction(x, y, mouse_x, mouse_y);

Le code ci-dessus fera pivoter l'image-objet de l'instance pour toujours pointer sur la position de la souris.
