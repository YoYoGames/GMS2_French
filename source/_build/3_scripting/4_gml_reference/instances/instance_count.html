instance_count


La description

Avec cette variable en lecture seule, vous pouvez obtenir le nombre de toutes les instances actives qui se trouvent dans la pièce. Cela inclura l'instance exécutant le code, mais n'inclut pas les instances qui ont été désactivées en utilisant le instance_deactivate les fonctions. Notez que cette fonction ne vous donnera que le nombre d'instances au début de l'étape, donc toute modification apportée aux instances de la pièce après le démarrage de l'étape ne sera pas prise en compte.


Syntaxe:

instance_count;


Résultats:

Real


Exemple:

if (instance_count < 100)
   {
   var dif = 100 - instance_count;
   while (--dif > 0)
      {
      instance_create_layer(random(room_width), random(room_height), "Effects", obj_Star);
      }
   }

Le code ci-dessus crée plusieurs instances de l'objet "obj_Star" jusqu'à ce que le nombre total d'instances atteigne 100.
