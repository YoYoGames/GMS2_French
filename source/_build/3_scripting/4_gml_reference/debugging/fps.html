fps


La description

Dans GameMaker Studio 2, deux méthodes principales permettent de déterminer la vitesse d'exécution de votre jeu. le game speed (comme spécifié dans les options de jeu) et fps (images par seconde). Ces valeurs sont souvent confuses, mais fondamentalement, il y a le nombre d'étapes de jeu que GameMaker Studio 2 est censé accomplir en une seconde, tandis que l'autre (le fps) est le nombre d'étapes CPU que GameMaker Studio 2 complète dans un seconde jusqu'à une valeur maximale de la vitesse de la pièce elle-même. Pour obtenir le vrai fps, c'est à dire. le nombre réel de pas de CPU par étape de jeu, utilisez le fps_real variable.

Cette variable en lecture seule renvoie le fps en cours en tant que valeur entière. S'il vous plaît noter que la fonction ne sera mise à jour qu'une fois à chaque étape de votre jeu et peut donc sembler "sauter" d'une valeur à l'autre, mais c'est tout à fait normal.


Syntaxe:

fps


Résultats:

Real


Exemple:

if debug_mode
   {
   draw_text(32, 32, "FPS = " + string(fps));
   }

Le code ci-dessus va vérifier pour voir si le jeu est en mode débogage et si c'est le cas, il affichera le fps actuel sur l'écran.
