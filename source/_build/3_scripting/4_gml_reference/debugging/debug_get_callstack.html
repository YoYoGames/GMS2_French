debug_get_callstack


La description

Cette fonction génère un tableau de chaînes, dans lequel le script actuel est répertorié en premier. Le format exact de la chaîne variera en fonction de la plate-forme cible choisie, mais le nom du script / de l'événement y sera principalement associé, suivi du signe deux-points:, puis du numéro de ligne, comme suit:

"gml_Script_script2:1"
"gml_Script_script1:32"
"gml_Script_script0:64"
"gml_Object_object0_Create_0:1"


Syntaxe:

debug_get_callstack()


Résultats:

Array


Exemple:

if debug_mode
    {
    if keyboard_check(vk_escape)
        {
        var _a = debug_get_callstack();
        for (var i = 0; i < array_length_id(_a); ++i;)
            {
            show_debug_message(_a[i]);
            }
        }
    }

Le code ci-dessus vérifie si le mode de débogage est activé et si c'est le cas, il vérifie si une clé iOS est maintenue enfoncée. Si c’est le cas, alors la pile d’appels en cours est sortie sur la console
