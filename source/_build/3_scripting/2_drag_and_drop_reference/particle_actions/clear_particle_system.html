Système de particules transparent


La description

Avec cette action, vous pouvez effacer le système de particules donné à son état par défaut, en supprimant tous les émetteurs (il n'est pas nécessaire d'appeler l'action Destroy Particle Emitter comme cela est pris en charge automatiquement) et effacer toutes les particules de l'écran. Vous fournissez la valeur d'ID unique pour le système à effacer, telle que renvoyée lorsque vous avez créé le système avec l'action Créer un système de particules. Il faut faire attention en utilisant cette action car elle détruit tous les émetteurs associés au système de particules - cela signifie que toute instance dans la pièce qui utilise un émetteur peut donner une erreur car cet émetteur n'existe plus, alors assurez-vous que l'action est seulement appelé quand vous ne créerez pas de nouvelles particules à partir de cet émetteur (pour plus d'informations sur les émetteurs de particules, voir Créer un émetteur de particules ).

NOTE: Cette action n'effacer les particules visibles dans la salle, mais il ne supprime pas les propriétés de particules ni les supprimer de la mémoire. Pour cela, vous devez utiliser l'action Détruire le type de particule.


Syntaxe de l'action:


Arguments:

Argument La description
System La valeur d'identifiant unique pour le système à détruire


Exemple:

Le code de bloc d'action ci-dessus effacera toutes les particules de l'écran si le joueur a 0 ou moins de vies, puis redémarre la pièce.
