Set Sprite de particules


La description

Cette action peut être utilisée pour définir l'image-objet qu'une particule peut utiliser lors du dessin. Vous fournissez la valeur d'ID unique pour le type à définir l'image-objet (comme renvoyé lorsque vous avez créé le type avec l'action Créer un type de particule ) et la ressource d'image-objet à utiliser. Si l'image-objet possède des sous-images, la particule s'anime à travers les images à la vitesse définie dans l'éditeur d'image pour l'image-objet. Notez également que si vous définissez une valeur de couleur pour la particule (à l'aide de l'action Définir la couleur des particules ), cette couleur sera mélangée avec l'image-objet utilisée, tout comme la fusion d'images sur une occurrence fusionnera une couleur avec l'image-objet. blanc pour ne pas mélanger).

Remarque: Si vous utilisez cette action, vous ne devez pas utiliser l'action Définir la forme des particules car vous ne pouvez avoir qu'une image-objet ou une forme, mais pas les deux.


Syntaxe de l'action:


Arguments:

Argument La description
Type ID La valeur d'ID du type à définir l'image-objet


Exemple:

Le code de bloc d'action ci-dessus crée un nouveau type de particule et affecte sa valeur d'ID unique à une variable globale. Il procède ensuite à définir toutes les propriétés pour le type de particule.
