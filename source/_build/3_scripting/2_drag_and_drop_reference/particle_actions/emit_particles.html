Emettre des particules


La description

Cette action est utilisée pour créer des particules à partir d'un émetteur. Vous indiquez l'ID du système de particules auquel appartient l'émetteur (renvoyé par l'action Create Particle System ), puis l'ID de l'émetteur (renvoyé par l'action Create Particle Emitter ) et enfin le type de particule à créer (tel que renvoyé par le action Créer un type de particule ). Vous pouvez alors sélectionner "Burst" ou "Stream" les particules de l'émetteur, ainsi que le nombre de celles que vous voulez créer.

Lorsque vous choisissez de diffuser des particules, vous n'avez pas besoin de placer cette action dans un événement de jeu comme l'événement d'étape ou l'événement Dessiner, mais appelez simplement une fois dans l'événement de création (par exemple) et l'émetteur jeu jusqu'à ce que vous appelez à nouveau cette action pour l'arrêter (en réglant la valeur de comptage sur 0). Si vous choisissez d'éclater les particules, vous pouvez les appeler dans n'importe quel type d'impact et les particules seront créées une fois.


Syntaxe de l'action:


Arguments:

Argument La description
System L'ID du système dans lequel se trouve l'émetteur
Emitter L'identifiant de l'émetteur à utiliser
Type L'ID du type de particule à créer
Event Le type d'événement pour les particules, soit "Burst" ou "Stream"
Count Le nombre de particules à éclater ou à diffuser
