Mettre à jour le système de particules


La description

Cette action forcera le système de particule donné à mettre à jour la position, la couleur, alpha, etc... Vous fournissez la valeur d'ID unique pour le système à mettre à jour (tel que renvoyé lorsque vous avez créé le système avec l'action Create Particle System ). Lorsque vous appelez l'action, le système donné mettra à jour une trame de jeu, ce qui signifie que vous pouvez, par exemple, appeler cette action dans une boucle Repeat et mettre à jour le système de particules plusieurs fois dans un même cadre de jeu. avoir un effet qui nécessite la présence de particules dès le début, comme la neige ou la pluie). Vous pouvez également utiliser cette action lorsque le système de particules a été mis en pause à l'aide du Pause Particle System pour mettre à jour manuellement les particules dans le système.

Arguments:

Argument La description
System La valeur d'identifiant unique pour le système à mettre à jour
