Détruire l'émetteur de particules


La description

Cette action va "détruire" l'émetteur de particules donné, c'est à dire: libérer la mémoire utilisée par l'émetteur de particules. Vous fournissez l'ID du système de particules auquel l'émetteur appartient (tel que renvoyé par l'action Créer un système de particules ), puis l'ID émetteur réel (tel que renvoyé par l'action Créer un émetteur de particules ). Cette action doit être appelée chaque fois que vous n'avez plus besoin d'un émetteur de particules dans votre jeu, ou lorsque vous souhaitez recréer l'émetteur de particules (par exemple, juste avant d'appeler un redémarrage de jeu ). Notez que si le système de particules auquel l'émetteur a été affecté n'a pas été marqué comme persistant, vous n'avez pas besoin d'appeler cette action, car l'émetteur sera automatiquement nettoyé à la fin de la pièce avec le système de particules.


Syntaxe de l'action:


Arguments:

Argument La description
System La valeur d'ID du système qui contient l'émetteur
Emitter La valeur d'identification de l'émetteur à détruire


Exemple:

Le code de bloc d'action ci-dessus vérifiera une variable d'instance et s'il est supérieur à 100, il détruira l'émetteur assigné à l'instance et ira ensuite dans la pièce suivante.
