Régler la vitesse des particules


La description

Cette action est utilisée pour définir la vitesse des particules, qui peut être statique ou augmenter ou diminuer avec le temps. Vous fournissez la valeur d'ID unique pour le type à définir la taille de (tel que renvoyé lorsque vous avez créé le type avec l'action Créer un type de particule ), puis donnez une plage de vitesses de départ initiales, puis la quantité d'augmentation et enfin le "wiggle". La vitesse de création d'une particule est calculée en pixels par trame de jeu, de sorte qu'un minimum de 2 et un maximum de 5 émettent des particules se déplaçant entre 2 et 5 pixels par trame de jeu.

L'option permettant d'augmenter la vitesse d'une particule vous permet d'entrer une valeur qui sera ajoutée à la valeur de la vitesse de base pour chaque image de la durée de vie de la particule. Si vous définissez cette valeur sur, par exemple, 0,1 ajoutera 0,1 à la vitesse de chaque image de jeu. Notez que vous pouvez également utiliser ici des valeurs négatives pour ralentir la particule, bien que la vitesse ne soit jamais inférieure à 0. Notez que la diminution de vitesse minimale acceptée est de -0,01 et que l' incrément minimal est de 0,01.

Enfin, vous pouvez définir l'option "wiggle". C'est une valeur qui sera générée aléatoirement entre 0 et la valeur que vous donnez puis ajoutée ou soustraite à la vitesse à chaque étape de la durée de vie des particules, entraînant une fluctuation de la vitesse de l'image dessinée. De toute évidence, les valeurs plus élevées sont plus prononcées que les valeurs plus petites, et cette valeur peut même être négative, la plage maximale se situant entre -20 et 20.


Syntaxe de l'action:


Arguments:

Argument La description
Type La valeur d'ID unique pour le type de particule pour définir la taille de
Min La vitesse minimale pour le type de particule (par défaut 0,1)
Max La vitesse maximale pour le type de particule (par défaut 1)
Increment La valeur à ajouter au type de particule accélère chaque trame de jeu (peut être négative, minimum +/- 0.01)
Wiggle La valeur maximale à ajouter ou soustraire de façon aléatoire à la vitesse des particules par pas (+/- 20).

Le code de bloc d'action ci-dessus crée un nouveau type de particule et affecte sa valeur d'ID unique à une variable globale. Il procède ensuite à définir toutes les propriétés pour le type de particule.
