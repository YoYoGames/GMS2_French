Éclater les particules


La description

Cette action vous permet d'éclater des particules à tout moment sans dépendre d'un émetteur. Vous fournissez la valeur d'ID unique du système pour créer les particules (comme indiqué lors de la création du système avec l'action Create Particle System ) et l'ID unique du type de particule à créer (tel que renvoyé lorsque vous avez créé le type de particule avec action Créer un type de particule ), ainsi que la position x / y pour les créer à et le nombre à créer. La position peut être une position absolue dans la pièce, ou vous pouvez la marquer comme étant une position relative par rapport à l'instance qui appelle l'action.

Vous pouvez appeler cette action chaque image de jeu pour répliquer l'effet de flux d'un émetteur, ou vous pouvez l'appeler une fois dans un événement (comme l'événement Destroy) pour simplement faire éclater les particules.

Arguments:

Argument La description
System La valeur d'identification du système de particules à utiliser
Type La valeur d'ID du type de particule à éclater
X La position x pour éclater les particules à
Y Ils positionnent pour éclater les particules à
Count Le nombre de particules à éclater
