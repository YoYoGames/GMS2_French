Définir la vie des particules


La description

Cette action définira la durée de vie du type de particule donné. Vous devez d'abord spécifier la valeur ID du type de particule (qui est la valeur renvoyée lorsque vous avez créé la particule à l'aide de l'action Create Particle Type ), puis définir la durée de vie minimale et maximale de la particule dans les images de jeu. Par exemple, si votre jeu a une image par seconde de 60, le fait de régler ici les deux valeurs de durée de vie à 60 signifie que les particules dureront une seconde à l'écran avant de disparaître. Si vous utilisez deux valeurs différentes, chaque particule générée aura une durée de vie aléatoire dans le minimum et le maximum donnés. Notez que toutes les valeurs doivent être des entiers positifs.

IMPORTANT! Définir des durées de vie très longues et ensuite émettre des centaines de particules peut causer des problèmes sur des machines moins puissantes et sur des appareils mobiles. Soyez prudent lorsque vous réglez la durée de vie des particules pour maximiser l'effet visuel, mais minimisez le coût de la performance, et n'utilisez qu'avec parcimonie des particules de très longue durée de vie.


Syntaxe de l'action:


Arguments:

Argument La description
Type ID La valeur d'ID du type global pour définir la durée de vie de
Min Life La durée de vie minimale d'une particule (dans les cadres de jeu)
Max Life La durée de vie maximale d'une particule (dans les cadres de jeu)


Exemple:

Le code de bloc d'action ci-dessus crée un nouveau type de particule et affecte sa valeur d'ID unique à une variable globale. Il procède ensuite à définir toutes les propriétés pour le type de particule.
