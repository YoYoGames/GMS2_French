Définir la particule alpha


La description

Cette action définit l'alpha pour quand vous créez la particule ainsi que l'alpha au cours de sa durée de vie. Vous devez d'abord spécifier la valeur de l'ID du type de particule (qui doit être identique à la valeur renvoyée lorsque vous avez créé la particule à l'aide de l'action Créer un type de particule ), puis vous pouvez définir les valeurs alpha à utiliser.

Vous pouvez choisir trois alphas à utiliser, et la particule se fondra entre eux pendant la durée de vie de la particule. Par exemple, si une particule a 0.5, 1 et 0 comme valeurs alpha, et une durée de vie de 60 images de jeu, alors quand elle est créée (frame 0), elle sera à 0.5 alpha, puis elle passera vers un alpha complet de 1 (frame 30), avant de devenir invisible avec un alpha de 0 (frame 60).


Syntaxe de l'action:


Arguments:

Argument La description
Type ID La valeur d'ID unique pour le type de particule pour définir la couleur de
Start L'alpha initial à utiliser
Middle Le entre alpha à utiliser
End L'alpha final à utiliser

Le code de bloc d'action ci-dessus crée un nouveau type de particule et affecte sa valeur d'ID unique à une variable globale. Il procède ensuite à définir toutes les propriétés pour le type de particule.
