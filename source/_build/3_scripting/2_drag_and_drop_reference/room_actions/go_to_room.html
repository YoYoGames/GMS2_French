Aller dans la chambre


La description

Cette action mettra fin à la pièce actuelle et ira dans la pièce spécifiée par l'index d'entrée. La pièce doit exister ou sinon une erreur sera donnée, et si vous utilisez la même pièce que la pièce actuelle, cela aura le même effet que l'utilisation de Restart Room. Notez que la pièce ne changera pas jusqu'à la fin de l'événement ou du script où l'action a été appelée, ainsi toute action après celle-ci sera toujours exécutée.


Syntaxe de l'action:


Arguments:

Argument La description
Room La pièce où aller


Exemple:

Le code de bloc d'action ci-dessus récupère une variable globale et vérifie ensuite si la valeur qu'il contient est inférieure ou égale à 0. Si c'est le cas, le jeu ira dans une autre pièce.
