Redémarrer la pièce


La description

Cette action va redémarrer la pièce actuelle. Notez que la salle ne redémarrera pas avant la fin de l'événement ou du script où l'action a été appelée, de sorte que tout code après cette action sera toujours exécuté.


Syntaxe de l'action:


Exemple:

Le code de bloc d'action ci-dessus vérifie la frappe d'une touche et, si une action est détectée, la pièce est redémarrée.
