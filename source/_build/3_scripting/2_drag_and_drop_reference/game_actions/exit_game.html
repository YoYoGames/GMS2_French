Quitter le jeu


La description

Avec cette action, vous pouvez terminer le jeu, déclenchant l' événement Autre - Fin de partie. Notez que cela ne se produit pas au moment où l'action est appelée, mais plutôt à la fin de l'événement ou du script en cours, donc toutes les actions que vous avez faites après cette action seront toujours exécutées jusqu'à la fin de l'événement / script.

Notez que cette action comporte les restrictions suivantes:

  • Il signale une erreur sur la cible iOS car elle est contraire aux conditions de leur marché.
  • Il n'aura aucun effet sur les cibles HTML5 et Windows UWP.
  • Il fonctionnera sur Windows (y compris les jeux construits pour Steam ), Android, Ubuntu (Linux) et Mac.
