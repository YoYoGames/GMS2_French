Gamepad Obtenir l'axe


La description

Vous pouvez utiliser cette action pour obtenir la valeur des différents axes d'une manette donnée. Vous fournissez l'index du gamepad (c'est le nombre de "slot" qu'occupe une manette de jeu) et sélectionnez l'axe à vérifier à partir du bâton gauche ou droit, horizontal ou vertical. La valeur retournée sera comprise entre -1 et 1 pour chacun des axes horizontaux et verticaux disponibles: -1 est à gauche et 1 est à droite pour l'axe horizontal, et -1 est à la hausse et 1 est à la baisse pour l'axe vertical. Cette valeur retournée sera stockée dans la variable cible que vous fournissez pour une utilisation future, que vous pouvez marquer comme variable "Temp", ce qui signifie que l'action créera une nouvelle variable temporaire (locale) pour contenir la valeur de retour pour vous, comme montré dans l'exemple ci-dessous.

Arguments:

Argument La description
Gamepad L'index du gamepad.
Axis L'axe à vérifier (stick gauche / stick droit, horizontal / vertical)
Target La variable cible pour stocker la valeur renvoyée dans.
