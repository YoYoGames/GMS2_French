Gamepad Get Trigger


La description

Vous pouvez utiliser cette action pour obtenir la valeur des différents déclencheurs d'une manette donnée. Vous fournissez l'index du gamepad (c'est le nombre de "slot" qu'occupe une manette) et sélectionnez le trigger à vérifier (à gauche ou à droite). La valeur retournée sera entre 0 et 1, où 0 est sans pression sur la gâchette et 1 pression complète. Cette valeur retournée sera stockée dans la variable cible que vous fournissez pour une utilisation future.


Syntaxe de l'action:


Arguments:

Argument La description
Gamepad L'index du gamepad.
Trigger Le déclencheur à vérifier (gauche / droite)
Target La variable cible pour stocker la valeur renvoyée dans.


Exemple:

Le code de bloc d'action ci-dessus crée une variable locale (temporaire) et l'utilise ensuite pour stocker la valeur actuelle du déclencheur de gauche. Si la valeur de déclenchement est supérieure à 0,5, une instance est créée à la position de l'instance appelante.
