Définir la zone morte de l'Axe du Gamepad


La description

Cette action peut être utilisée pour définir la "zone morte" de l'axe du manche de la manette. Vous spécifiez l'index de la manette à définir, puis définissez une valeur de 0 à 1 et si la quantité d'entrée est inférieure à la valeur donnée, l'axe de manette est considéré comme étant à 0. Notez qu'il s'agit d'un paramètre global qui affectera tous les axes du bâton connecté.


Syntaxe de l'action:


Arguments:

Argument La description
Gamepad L'index du gamepad.
Deadzone La valeur de deadzone (0 - 1)
