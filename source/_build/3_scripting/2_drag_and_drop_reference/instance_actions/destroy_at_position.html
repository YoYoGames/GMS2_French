Détruire à la position


La description

Ce bloc d'action détruira toutes les instances trouvées au point donné, en appelant l' événement Destroy puis l' événement de nettoyage pour toutes les instances trouvées. Notez que cette action est basée sur le masque de collision des instances, et si une partie du masque survole le point cible, la fonction détruira cette instance (si les instances n'ont pas d'image-objet ou de masque appliqué, elles seront détruites). ne pas être détruit).


Syntaxe de l'action:


Arguments:

Argument La description
x La position x dans la pièce à utiliser.
y La position y dans la pièce à utiliser.


Exemple:

Le code de bloc d'action ci-dessus vérifie une pression de touche (la touche d'espace) et, s'il est détecté, il détruira toutes les instances trouvées à la position relative des instances.
