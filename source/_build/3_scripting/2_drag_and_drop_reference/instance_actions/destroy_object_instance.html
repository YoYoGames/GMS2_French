Détruire l'instance


La description

Vous pouvez appeler cette action chaque fois que vous souhaitez "détruire" une instance, ce qui la supprimera de la pièce. Notez que lorsque vous détruisez une instance, son événement Destroy est appelé immédiatement après l'action qui appelle la destruction. Notez également que, bien que l'événement Destroy soit exécuté à l'instant où l'action est appelée, l'instance n'est pas immédiatement supprimée du jeu et le code contenu dans l'événement actuel sera exécuté jusqu'à la fin, et uniquement lorsque l'événement en cours sera terminé. est terminée, l’instance sera supprimée du jeu et l’ événement de nettoyage sera alors appelé.
