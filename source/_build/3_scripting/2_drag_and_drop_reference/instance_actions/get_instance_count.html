Obtenir le nombre d'instances


La description

Avec cette action, vous pouvez trouver le nombre d'instances d'un objet donné actif dans la pièce. Vous pouvez sélectionner n'importe quel objet de la liste de ressources ou vous pouvez utiliser le mot-clé spécial all pour obtenir le nombre de toutes les instances actives. Vous devez également fournir une variable cible pour renvoyer la valeur afin que vous puissiez l'utiliser dans d'autres actions.


Syntaxe de l'action:


Arguments:

Argument La description
Object Le nom de l'objet à vérifier.
Target La variable à cibler pour la valeur de retour.


Exemple:

Le code de bloc d'action ci-dessus crée une variable locale (temporaire) et lui assigne le nombre d'instances de l'objet "obj_Squirrel". Cette valeur est ensuite vérifiée et l'instance est détruite si le nombre d'instances est supérieur à 100.
