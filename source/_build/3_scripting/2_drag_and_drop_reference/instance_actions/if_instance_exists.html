Si l'instance existe


La description

Cette action peut être utilisée pour vérifier si une instance d'un objet spécifique existe dans la salle de jeu, ou si une instance spécifique existe elle-même. Vous donnez l'objet à partir de l'explorateur d'actifs, et si une instance existe dans la pièce, l'action retournera true sinon il reviendra false. Notez que vous pouvez utiliser une variable ici avec l'ID unique pour une instance spécifique (par exemple renvoyée par l'action Créer une instance d'objet ) ou même la valeur d'ID d'instance renvoyée par l'éditeur de pièce lorsque vous avez ajouté l'instance.

Si vous marquez l'argument "Non", l'action vérifie si aucune instance n'existe et si aucune n'est trouvée, elle retourne true et si même un existe, il reviendra false.
