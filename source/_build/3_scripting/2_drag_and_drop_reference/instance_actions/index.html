Bibliothèque d'actions d'instance


La bibliothèque d'actions Instance vous permet de créer et de détruire des instances, ainsi que de modifier leurs propriétés et d'utiliser leurs alarmes. Les actions disponibles sont toutes listées ci-dessous:

Créer une instance d'objet
Détruire l'instance d'objet
Détruire à la position
Changer l'instance d'objet
Régler l'alarme
Obtenir une alarme
Obtenir le nombre d'instances
Appelez un événement utilisateur
Définir la variable d'instance
Obtenir la variable d'instance
Définir Sprite
Définir la rotation d'instance
Définir la vitesse d'animation
Définir l'échelle d'instance
Définir l'instance Alpha
Définir la couleur de l'instance
Si l'instance existe
Appelez l'événement parent
