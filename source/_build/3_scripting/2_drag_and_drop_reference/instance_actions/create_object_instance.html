Créer une instance d'objet


La description

Avec cette action, vous pouvez créer une instance de n'importe quel objet de l'arbre des ressources. Vous donnez le nom de l'objet et une position x / y dans la pièce où il doit être créé. Vous pouvez cocher les indicateurs "relatifs" pour positionner l'instance par rapport à l'instance exécutant les blocs de code d'action. Chaque fois que vous créez une instance dans le jeu, elle reçoit une valeur ID unique avec laquelle vous pouvez ensuite identifier l'instance dans d'autres actions. Cette action vous permet donc de définir une variable "cible" pour contenir cette valeur d'ID (comme indiqué dans le exemple ci-dessous). Notez que la cible peut être marquée comme une variable temporaire (locale), qui sera créée pour l'action et peut être utilisée dans toutes les actions suivantes.

Arguments:

Argument La description
Object Le nom de l'objet pour créer une instance de.
x La position x dans la pièce pour créer l'instance (peut être relative à l'instance appelante).
y La position y dans la pièce pour créer l'instance (peut être relative à l'instance d'appel)
Target La variable à cibler pour la valeur de retour.
