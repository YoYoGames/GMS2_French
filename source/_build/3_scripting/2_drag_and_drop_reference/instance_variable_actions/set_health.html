Définir la santé


La description

Avec cette action, vous pouvez créer ou définir une variable d'instance pour l'état de santé. Vous fournissez la valeur pour définir la variable d'intégrité ou pour l'ajouter à la variable d'intégrité (si vous cochez la case relative, la valeur sera ajoutée à la valeur "d'intégrité" actuelle). Notez que si vous souhaitez utiliser l'action Draw HealthBar, cette valeur doit être comprise entre 0 et 100. Dans le cas contraire, elle peut correspondre au montant dont vous avez besoin. Notez également que contrairement aux variables d'instance régulières, le nom de la variable "health" est stocké en interne. Pour récupérer la valeur du score, vous devez utiliser l'action Get Health.


Syntaxe de l'action:


Arguments:

Argument La description
Health Le montant pour définir l'état de santé (ou pour ajouter à la santé si le parent est coché)
