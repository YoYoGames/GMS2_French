Renommer le fichier


La description

Avec cette action, vous pouvez renommer n'importe quel fichier qui a été enregistré par votre jeu. Vous donnez le nom du fichier à modifier (sous la forme d'une chaîne), puis le nouveau nom (également sous forme de chaîne et en incluant l'extension de fichier), et le fichier sera renommé. Notez que vous ne pouvez renommer aucun fichier inclus dans un fichier inclus avec votre jeu (voir la section sur le système de fichiers pour plus d'informations).


Syntaxe de l'action:


Arguments:

Argument La description
Filename Le nom (sous forme de chaîne) du fichier à renommer
New Name Le nouveau nom (sous forme de chaîne) pour le fichier


Exemple:

Le code de bloc d'action ci-dessus vérifiera si le fichier "checkpoint.sav" existe et si c'est le cas, il vérifie si le fichier "checkpoint_OLD.sav" existe. Si ce fichier existe également, il est supprimé, puis le fichier "checkpoint.sav" d'origine est renommé "checkpoint_OLD.sav". Enfin, un tampon est sauvegardé sous "checkpoint.sav" (essentiellement, nous sauvegardons un fichier tampon sauvegardé chaque fois que nous le sauvegardons, de sorte qu'il y a toujours un enregistrement "en cours" et un "ancien" fichier de sauvegarde).
