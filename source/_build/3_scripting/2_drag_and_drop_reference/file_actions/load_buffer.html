Charger le tampon


La description

Avec cette action, vous pouvez charger un tampon précédemment enregistré avec l'action Enregistrer le tampon. L'action créera un nouveau tampon, chargera le fichier de données dans celui-ci et retournera la valeur de l'identifiant unique pour identifier le tampon de la variable. Vous indiquez le nom du fichier à charger (sous forme de chaîne) et une variable pour stocker l'ID de tampon renvoyé. Notez que si vous marquez la variable comme variable locale temporaire, vous devez vous assurer de supprimer le tampon créé par cette fonction avant la fin du script ou de l'événement d'action, sinon vous perdrez la référence et obtiendrez une fuite de mémoire lorsque la variable est supprimée.


Syntaxe de l'action:


Arguments:

Argument La description
Filename Le nom (en tant que chaîne) du fichier à ouvrir
Target La variable cible pour contenir la valeur d'ID unique pour le tampon créé en charge


Exemple:

Le code de bloc d'action ci-dessus crée une variable pour contenir les données du tampon, puis vérifie si un fichier de sauvegarde du tampon a déjà été créé. Si le fichier existe, il est chargé dans la variable de tampon que nous avons créée, mais si elle n'existe pas, elle est créée, écrite et sauvegardée, prête à être chargée la prochaine fois.
