Lire le fichier Ini


La description

Avec cette action, vous pouvez lire une donnée à partir d'un fichier Ini. Vous devez avoir préalablement ouvert le fichier Ini pour la lecture / écriture en utilisant l'action Open Ini File, puis avec cette action vous donnez le type de données à lire (soit un nombre réel ou une chaîne entre ""), ainsi que le en-tête de section et la clé dans cette section pour obtenir les données. Vous pouvez également fournir une valeur par défaut qui peut être soit un nombre réel d'une chaîne, et le type doit coïncider avec le "Type" défini dans l'action. Cette action par défaut sera renvoyée par l'action si l'un des événements suivants se produit:

  • Le fichier Ini que vous avez ouvert n'existe pas
  • L'en-tête de section que vous référencez n'existe pas
  • La clé dont vous essayez d'obtenir des données n'existe pas

Dans l'une des circonstances ci-dessus, vous obtiendrez la valeur par défaut renvoyée plutôt qu'une valeur du fichier Ini. Quelle que soit la valeur renvoyée, elle sera stockée dans la variable cible que vous spécifiez, qui peut également être créée à ce moment si elle est signalée comme variable locale temporaire.

IMPORTANT! essayer de lire des données à partir d'un fichier Ini qui n'a pas été ouvert précédemment en utilisant l'action Open Ini File provoquera une erreur sérieuse et fera planter votre jeu.


Syntaxe de l'action:


Arguments:

Argument La description
Type Le type de données à lire, soit string (entre guillemets ""), ou un real number
Section La section du fichier Ini à lire (sous forme de chaîne)
Key La clé dans la section à lire (en tant que chaîne)
Default La valeur par défaut à renvoyer si le fichier, la section ou la clé n'existe pas
Target La variable cible pour conserver la valeur de retour de l'action
