Fermer le fichier Ini


La description

Cette action fermera le fichier Ini actuellement ouvert. Cette action doit être appelée au moment où vous avez fini de lire ou d'écrire dans un fichier Ini ouvert (les fichiers Ini sont ouverts à l'aide de l'action Ouvrir le fichier Ini ). Si vous n'utilisez pas cette action après avoir utilisé l'action Écrire dans un fichier Ini, rien ne sera écrit sur le disque car les informations du fichier sont conservées en mémoire jusqu'à ce que cette action soit appelée, à quel moment elle est prise en mémoire et écrite le fichier sur le disque. Notez également que si vous essayez d'ouvrir un fichier Ini sans en avoir préalablement fermé un autre (ou le même), vous obtiendrez une erreur.

Exemple:

Le code de bloc d'action ci-dessus ouvrira un fichier ini à utiliser, puis récupérera la valeur associée à la clé "name" sous l'entête "Player". Si le nom retourné correspond à la valeur du nom par défaut (ex: le fichier, la section ou la clé n'existe pas ), le fichier reçoit une variable globale avant d'être fermée (et d'écrire les nouvelles données sur le disque).
