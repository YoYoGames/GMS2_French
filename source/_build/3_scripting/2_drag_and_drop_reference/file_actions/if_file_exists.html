Si le fichier existe


La description

Avec cette action, vous pouvez vérifier si un fichier avec le nom donné existe. Vous sélectionnez le fichier à vérifier (comme une piqûre et y compris l'extension) et la fonction sera retrun true si cela existe, et false si ça ne fait pas. Notez que cela retournera true si le fichier existe dans le jeu en tant que fichier inclus ou si le fichier a été sauvegardé pendant l'exécution du jeu (pour plus d'informations, reportez-vous à la section sur le système de fichiers ).
