Ouvrir le fichier Ini


La description

Avec cette action, vous pouvez ouvrir un fichier Ini pour la lecture et / ou l'écriture. Si le fichier Ini n'existe pas à l'emplacement que vous vérifiez, GameMaker Studio 2 peut en créer un, mais uniquement si vous y écrivez des données. Si vous n'avez lu que les informations du fichier Ini, les valeurs par défaut de la fonction read seront renvoyées, mais le fichier Ini ne sera pas créé.

Veuillez noter que vous ne pouvez ouvrir qu'un seul fichier Ini à la fois et n'oubliez pas d'utiliser Close Ini File une fois que vous avez fini de lire / écrire depuis le fichier Ini car les informations ne sont pas stockées sur le disque (il est stocké dans mémoire jusqu'à la fermeture du fichier).


Syntaxe de l'action:


Arguments:

Argument La description
Filename Le nom - comme une chaîne et avec le .ini extension - du fichier à ouvrir


Exemple:

Le code de bloc d'action ci-dessus ouvrira un fichier ini à utiliser, puis récupérera la valeur associée à la clé "name" sous l'entête "Player". Si le nom retourné correspond à la valeur du nom par défaut (ex: le fichier, la section ou la clé n'existe pas ), le fichier reçoit une variable globale avant d'être fermée (et d'écrire les nouvelles données sur le disque).
