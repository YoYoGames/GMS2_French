Enregistrer le tampon


La description

Avec cette action, vous pouvez enregistrer un fichier contenant les données d'un tampon. Vous fournissez l'ID de tampon unique (tel que renvoyé lorsque vous avez créé le tampon avec Create Buffer ) et donnez un nom au fichier (en tant que chaîne, avec l'extension de votre choix) et le tampon sera écrit dans ce fichier prêt à être chargé à nouveau en utilisant Load Buffer.


Syntaxe de l'action:


Arguments:

Argument La description
Buffer La valeur unique de l'ID de tampon du tampon à enregistrer
Filename Le nom (sous forme de chaîne) du fichier à enregistrer


Exemple:

Le code de bloc d'action ci-dessus crée une variable pour contenir les données du tampon, puis vérifie si un fichier de sauvegarde du tampon a déjà été créé. Si le fichier existe, il est chargé dans la variable de tampon que nous avons créée, mais si elle n'existe pas, elle est créée, écrite et sauvegardée, prête à être chargée la prochaine fois.
