Obtenir la variable de vue


La description

Cette action vous permet d'obtenir la valeur de certaines variables intégrées liées à un port de vue donné. Vous sélectionnez la variable de port de vue à récupérer, puis donnez le port de vue à la cible (de 0 à 7) et enfin la variable cible pour contenir la valeur renvoyée (ceci peut être marqué comme étant une variable locale temporaire). Le port d'affichage est la zone de l'écran qui sera utilisée pour dessiner une vue de caméra donnée (telle que configurée dans l'éditeur de pièce), et les différentes variables que vous pouvez obtenir de cette manière sont décrites dans le tableau ci-dessous:

Variable La description
Caméra La valeur renvoyée sera la valeur d'identifiant unique de la caméra à assigner au port (voir ici pour plus d'informations).
Visibilité Cela retournera une valeur de true ou false où true est que le port de vue est visible (c'est-à-dire qu'il est dessiné à l'écran), et false ce qui signifie que rien n'est dessiné.
Voir Port X Coordinate C'est la position X du port de vue dans la fenêtre de jeu.
Voir Port Y Coordinate C'est la position Y du port de vue dans la fenêtre de jeu.
Voir la largeur du port C'est la largeur (en pixels) du port de vue.
Afficher la hauteur du port C'est la hauteur (en pixels) du port de vue.
Afficher l'ID de surface de port Cela renverra la valeur d'ID unique pour la surface assignée au port de visualisation ou -1 si aucune surface n'a été affectée.

Arguments:

Argument La description
Variable La variable de vue intégrée pour obtenir
View Le port de vue à cibler (de 0 à 7)
Target La variable cible pour contenir la valeur retournée (peut être marquée comme une variable locale temporaire)
