Choisir


La description

Avec cette action, vous pouvez donner un certain nombre d'options différentes (cela peut être un nombre réel, une chaîne, un ID de ressource ou une constante) à partir de laquelle l'action en choisira une à retourner. Vous pouvez ajouter des options supplémentaires en cliquant sur l'icône plus à côté de l'action, et donner une autre valeur qui pourrait être retournée. La valeur retournée sera alors stockée dans la variable cible qui peut avoir été créée précédemment ou peut être une nouvelle variable temporaire (si vous cochez la case "Temp"). Vous pouvez ajouter des variables supplémentaires dans la même action en cliquant sur l'icône plus à côté de l'action, et donner un autre nom de variable et une valeur.

Arguments:

Argument La description
Option L'option qui peut être retournée par l'action choose.
