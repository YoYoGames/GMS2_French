Chaîne au numéro


La description

Cette action peut convertir une chaîne en un nombre. Vous fournissez la chaîne à modifier, puis vous donnez une variable cible pour renvoyer le nombre réel (cette variable peut être marquée comme variable locale temporaire). Notez que si la chaîne a des caractères autres que des nombres et un point décimal, elle ne sera pas convertie, bien que si la chaîne commence par un nombre, ceux-ci seront convertis et le reste de la chaîne sera exclu. Il est important de noter que lors de la transmission de décimales dans cette action, la chaîne retournée sera arrondie à deux décimales seulement. Voici un tableau des valeurs d'entrée et ce à quoi vous vous attendiez.

Chaîne Valeur retournée
"123" 123
"123 Hello World" 123
"Hello World" 0 (par défaut si la chaîne ne peut pas être convertie, 0 sera retourné)
"Hello World 123" 0
"123.45678" 123,46


Syntaxe de l'action:


Arguments:

Argument La description
Value La chaîne à convertir
Target La variable cible pour contenir la valeur renvoyée


Exemple:

Le code de bloc d'action ci-dessus initialise une variable globale, puis ouvre un ini fichier. Une chaîne est lue dans le fichier et stockée dans une variable temporaire avant d'être convertie en un nombre réel et affectée à la variable globale. le ini est alors fermé.
