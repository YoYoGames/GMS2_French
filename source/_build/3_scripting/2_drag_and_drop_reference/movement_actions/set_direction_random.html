Définir la direction au hasard


La description

Cette action est utilisée pour définir la direction du mouvement de l'instance vers une ou plusieurs directions aléatoires choisies parmi les 8 points cardinaux. Vous pouvez cliquer sur une ou plusieurs des 8 flèches directionnelles de l'action et l'action en choisira une au hasard pour l'utiliser comme direction de l'instance. Si vous ne sélectionnez aucune des flèches, l'action n'aura aucun effet sur la direction actuelle de l'instance.

Exemple:

Le code de bloc d'action ci-dessus génère une valeur aléatoire comprise entre 0 et 10 et la stocke dans une variable temporaire (locale). Cette variable est ensuite vérifiée et si elle est supérieure à cinq, alors un ensemble de directions aléatoires est choisi et s'il est inférieur à 5, alors un autre ensemble est choisi.
