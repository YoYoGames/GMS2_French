La description

Cette action est utilisée pour déplacer une instance de sa position actuelle vers la position où elle a été créée à l'origine. La position d'origine où une instance est créée est stockée dans deux variables d'instance appelées xstart et ystart. Ces variables peuvent être utilisées dans n'importe quelle autre action et peuvent également être réglées pour changer la position de départ enregistrée.
