Direction du point de consigne


La description

Cette action est utilisée pour définir une direction à partir d'un vecteur formé par la position actuelle de l'instance et la position donnée. Vous indiquez les positions x et y sur "point" à - elles peuvent être signalées par rapport à la position de l'instance - et la direction de l'instance à ce point sera utilisée pour l'instance.


Syntaxe de l'action:


Arguments:

Argument La description
x La composante x du vecteur de direction.
y La composante y du vecteur de direction.


Exemple:

Le code de bloc d'action ci-dessus vérifie si une instance de l'objet "obj_Player" existe dans la pièce et, si c'est le cas, définit la direction de l'instance pour pointer vers la position x / y de "obj_Player".
