Définir la vitesse


La description

Cette action est utilisée pour définir la vitesse de déplacement de l'instance. Vitesse calculée comme le nombre de pixels qu'une instance doit déplacer dans une seule image de jeu (étape). Par défaut, la valeur de la vitesse sera ajoutée au vecteur de direction - donc si vous réglez la vitesse sur 2 et la direction sur 45 °, l'instance se déplacera vers le haut et 2 pixels à droite de chaque image de jeu - mais vous pouvez également cliquer sur le direction et sélectionnez la vitesse horizontale ou verticale. Choisir l'un ou l'autre ne définira la vitesse que pour les vecteurs horizontaux ou verticaux. Notez que vous pouvez marquer la valeur comme étant relative à la vitesse actuelle, ce qui ajoutera ou soustraira la valeur à la vitesse réelle de l'instance, et vous pouvez également fournir des nombres négatifs, ce qui la fera bouger dans la direction opposée au courant valeur de direction.


Syntaxe de l'action:


Arguments:

Argument La description
speed La valeur de la vitesse (peut être négative).


Exemple:

Le code de bloc d'action ci-dessus vérifie la présence d'une souris intermédiaire appuyez sur l'instance et si une est détectée, elle définit la direction vers le bas (270 °) et la vitesse vers 1 plus une valeur aléatoire de 0 à 1.
