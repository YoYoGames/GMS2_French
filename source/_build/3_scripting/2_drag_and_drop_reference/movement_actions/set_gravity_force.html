Définir la force de gravité


La description

Cette action est utilisée pour définir la force de gravité agissant sur une instance. Chaque instance peut avoir une force de gravité (et une direction) différente de toutes les autres. La force appliquée est une valeur en pixels par trame de jeu, et est cumulative, c'est-à-dire: elle est ajoutée à chaque pas, donc si vous avez une vitesse de 2 et une gravité de 1, la vitesse augmentera de 1 vous voulez des valeurs très basses ici. Notez que vous pouvez entrer des valeurs négatives pour la gravité et que l'instance sera "tirée" dans la direction opposée à la direction de gravité.

Arguments:

Argument La description
force La force de gravité à appliquer (peut être négative)
