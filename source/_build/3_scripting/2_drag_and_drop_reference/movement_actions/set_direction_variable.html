Définir la variable de direction


La description

Cette action est utilisée pour définir une direction pour l'instance qui n'est pas fixée à des incréments de 45 °. Vous fournissez la valeur de la direction de 0 ° à 360 ° - où le droit est 0 °, augmente de 90 °, gauche est de 180 ° et baisse de 270 ° - et vous pouvez également définir le drapeau relatif pour ajouter / soustraire la valeur que vous donnez la direction actuelle.


Syntaxe de l'action:


Arguments:

Argument La description
Direction Direction de 0 ° à 360 ° (les valeurs en dehors de la plage seront enveloppées).
