La description

L'action "else" n'est utilisée que conjointement avec If Variable (ou toute autre action "if"), et est un moyen de définir un ensemble d'actions à effectuer si l'évaluation "if" renvoie false. Par exemple, vous pouvez avoir une action "if" qui vérifie si la variable de santé des joueurs est inférieure ou égale à 0, et si cela est vrai, vous détruisez l'instance. Cependant, si vous voulez effectuer un ensemble differnt d'actions de code si l'état de santé du joueur n'est pas inférieur à 0, vous pouvez ajouter un "else" dans, puis spécifier les actions à effectuer pendant que le if évalue à false.
