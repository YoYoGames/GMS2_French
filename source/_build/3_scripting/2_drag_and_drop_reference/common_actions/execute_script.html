Exécuter le script


La description

Cette action peut être utilisée pour appeler un script personnalisé. Vous donnez la ressource de script à appeler et fournissez tous les arguments dont elle a besoin, c'est-à-dire: toutes les valeurs (chaînes, entiers, index de ressources, etc...) que vous souhaitez entrer dans le script pour pouvoir les utiliser. Vous pouvez ajouter des arguments supplémentaires pour le script en cliquant sur l'icône plus à côté de l'action.

Certains scripts peuvent également renvoyer une valeur pour une utilisation ultérieure dans la liste des blocs d'actions. Ainsi, vous pouvez également spécifier la variable cible qui doit stocker cette valeur de retour, qui peut avoir été créée précédemment ou être temporaire (si vous vérifiez la "Temp" case à cocher). Les champs target et argument peuvent être laissés vides s'ils ne sont pas requis.


Syntaxe de l'action:


Arguments:

Argument La description
Script Le nom du script à appeler.
Argument0... Argument3 Les différents arguments (valeurs) qui doivent être passés au script (les arguments inutilisés peuvent être laissés vides)
Target Le nom de la variable à cibler pour les valeurs renvoyées (peut être laissé vide)
