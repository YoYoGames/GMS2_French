Obtenir la variable globale


La description

Cette action permet de récupérer la valeur d'une variable globale et de l'affecter à une autre instance ou variable temporaire (locale). La variable à laquelle vous attribuez la valeur globale peut être celle qui a été créée précédemment ou vous pouvez cocher la case "Temp" et donner un nouveau nom de variable pour que l'action crée une nouvelle variable temporaire (locale) spécifiquement pour contenir le retour valeur.

Notez que vous pouvez ajouter des variables supplémentaires dans la même action en cliquant sur l'icône plus à côté de l'action, et en donnant un autre nom de variable globale et un nom de variable locale temporaire. Pour plus d'informations sur les variables, consultez la section Variables et étendue des variables.

Temp Créez une variable temporaire (locale) pour conserver la valeur renvoyée (désactivée par défaut)
