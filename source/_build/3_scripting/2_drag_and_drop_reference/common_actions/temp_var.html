Déclarer une variable temporaire


La description

Cette action est utilisée pour déclarer et attribuer une valeur initiale à une variable temporaire (également appelée variable locale ), qui est une variable qui n'existe que pour la durée de l'événement ou du script dans lequel elle a été déclarée. Les variables temporaires sont très utiles pour stocker des valeurs qui ne sont nécessaires que pour une seule opération et éviter que vos objets ne soient "encombrés" par des variables ne pouvant être utilisées qu'une seule fois. Ils sont également très utiles pour décomposer des opérations complexes ou pour définir des valeurs dans d'autres instances lorsque vous modifiez l'étendue de l'action. En effet, les variables déclarées comme temporaires ne sont pas étendues à une instance particulière, mais à l'événement ou au script lui-même. Par conséquent, si vous créez une variable temporaire puis modifiez l'étendue de l'action à l'aide de l'action Appliquer à..., vous pouvez toujours vérifiez la variable temporaire sans problème et sans changer son étendue d'action.

Il est à noter que lorsque nous parlons de "valeurs", nous ne parlons pas uniquement de valeurs numériques, car une variable peut être une chaîne, un pointeur, un ID de ressource ou tout autre élément qu'une fonction peut renvoyer ou utiliser, même une expression.. Voir la section sur les types de données pour plus d'informations.

Notez que de nombreuses actions ont une case à cocher "Temp" qui vous permet de créer une nouvelle variable locale temporaire pour contenir la valeur de retour de l'action. Cette variable est accessible dans le script ou l'événement de l'action comme vous le feriez par la suite avec toute autre variable.. Notez également que les variables locales temporaires sont étendues au script ou à l'événement dans lequel elles ont été créées. Elles peuvent donc être utilisées par d'autres instances du même bloc lors de l'utilisation de Appliquer à.

Vous pouvez également ajouter des variables supplémentaires dans la même action en cliquant sur l'icône plus. à côté de l'action et en donnant un autre nom de variable locale temporaire et sa valeur. Pour plus d'informations sur les variables, veuillez vous reporter à la section Variables et portée des variables.
