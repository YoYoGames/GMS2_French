Définir une mosaïque au pixel


La description

Avec cette action, vous pouvez définir une mosaïque sur un calque donné à un nouvel index. Le calque est spécifié à partir du nom de calque donné (une chaîne, utilisée pour nommer le calque dans l'éditeur de pièces), puis vous donnez une position x et y pour obtenir la vignette à définir. La position x et y peut être signalée comme étant relative à la position de l'instance appelante, et la tuile étant définie sera celle dans laquelle la position donnée se situe dans la pièce et l'ensemble de tuiles utilisé sera celui assigné à la couche. dans l'éditeur de la salle. Enfin, vous fournissez l'index des tuiles pour définir la tuile (les tuiles sont indexées de gauche à droite de haut en bas, avec 0 en haut à gauche). Notez que vous pouvez toujours définir l'index de mosaïque sur 0 pour effacer la mosaïque.


Syntaxe de l'action:


Arguments:

Argument La description
Layer La couche pour définir la mosaïque
x La position x à vérifier
y La position y à vérifier
Tile Le nouvel index de la tuile à utiliser


Exemple:

Le code de bloc d'action ci-dessus vérifie la pression d'un bouton gauche de la souris et, s'il est détecté, il obtient l'indice de la cellule de tuile dans laquelle se trouve la position de la souris. Cet index de tuile est alors vérifié et s'il n'est pas 0, il est mis à 0.
