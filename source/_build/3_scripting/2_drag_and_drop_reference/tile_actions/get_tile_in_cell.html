Obtenez des carreaux dans la cellule


La description

Avec cette action, vous pouvez obtenir un index de tuiles sur un calque donné. La couche est spécifiée à partir du nom de couche donné (une chaîne, utilisée pour nommer la couche dans l'éditeur de pièces), puis vous donnez la ligne et la colonne de la cellule dans la couche pour obtenir l'index de la mosaïque. Enfin, vous fournissez la variable cible pour stocker l'index de tuiles retourné. Notez que la variable cible peut être marquée comme une variable temporaire (locale) qui peut être utilisée jusqu'à la fin de l'événement.


Syntaxe de l'action:


Arguments:

Argument La description
Layer La couche pour obtenir la tuile de
Column La colonne (de gauche à droite) de la cellule
Row La rangée (de haut en bas) de la cellule
Target La variable cible pour la valeur de retour


Exemple:

Le code de bloc d'action ci-dessus crée une boucle pour passer par la ligne supérieure de la couche de pièce donnée et vérifie si la mosaïque indexée dans la cellule de la colonne est définie sur 0. Si ce n'est pas l'index de la mosaïque pour cette cellule est définie à 0 et la variable temporaire (locale) est incrémentée avant que la boucle effectue un autre jeu de contrôles.
