Défaut


La description

Cette action créera un nouveau cas par défaut dans un commutateur. Le cas par défaut est celui qui sera effectué si la valeur d'entrée ne correspond à aucune autre constante de casse. Les actions qui sont ajoutées à un cas par défaut doivent être placées sur le côté de l'action, tout comme vous ajouteriez une action à un "si":

Exemple:

Le code de bloc d'action ci-dessus obtient la valeur stockée dans une variable globale, puis la compare aux différents cas possibles. Si l'un des cas est le même que la valeur, alors l'action dans ce cas sera effectuée, sinon le cas suivant sera évalué. Si aucun cas ne correspond à la valeur d'entrée, le cas par défaut sera exécuté à la place.
