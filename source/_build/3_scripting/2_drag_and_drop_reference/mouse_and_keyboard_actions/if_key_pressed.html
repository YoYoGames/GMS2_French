Si la touche est pressée


La description

Avec cette action, vous pouvez interroger le clavier pour voir si une touche est en cours d'appui. L'action retournera true chaque fois que la touche du clavier est enfoncée (et maintenue) enfoncée, ou false sinon, si vous cliquez sur le not modificateur, vous pouvez ensuite vérifier si la touche n'est pas enfoncée, c'est-à-dire: l'action retournera true alors qu'aucune touche n'est pressée et false s'il y a. Si vous n'avez besoin de vérifier qu'une seule touche, utilisez l'action si la touche est enfoncée.

Notez que pour ajouter des actions dans le bloc "if", elles doivent être placées sur le côté de l'action, comme indiqué dans l'image ci-dessous:
Toutes les actions abandonnées ailleurs seront effectuées après le bloc "if".


Syntaxe de l'action:


Arguments:

Argument La description
Key Le nom de la clé à vérifier.
Not Annulez le contrôle (true devient false et vice versa)


Exemple:

Le code de bloc d'action ci-dessus interroge l'état du clavier à chaque pas et si une touche est maintenue enfoncée, elle vérifie la pression initiale sur la touche. Si le contrôle est true lors de la première pression vers le bas, un son est joué, puis, tandis que la touche est maintenue enfoncée, la fusion des sprites est changée en rouge. Si la touche n'est pas enfoncée, une vérification est effectuée sur la touche pour réinitialiser la couleur de fusion au blanc.
