Définir la vitesse de la timeline


La description

Cette action peut être utilisée pour définir la vitesse à laquelle une ligne de temps se déplacera à travers les moments qu'elle contient. Normalement, dans chaque image de jeu, vous souhaitez que la position dans la timeline soit augmentée de 1, mais vous pouvez modifier cette valeur en définissant cette action sur une valeur différente. La valeur que vous choisissez sera traitée comme un multiplicateur. Par exemple, une valeur de 2 fera tourner la ligne de temps à deux fois la vitesse, tandis qu'une valeur de 0,5 la fera tourner à la moitié de la vitesse. Ce que cela signifie pour les moments est que si la valeur est supérieure à un, plusieurs moments peuvent se produire dans le même cadre de jeu (aucun moment n'est jamais ignoré), et si la vitesse est inférieure à un, alors après un moment, Par exemple, si vous réglez la vitesse de la timeline sur 0.1, par exemple, vous aurez un moment sur une image de jeu, puis 9 images de jeu sans moments, et sur la dixième le prochain moment sera couru.

Si vous définissez une chronologie pour commencer à utiliser les actions Définir la chronologie de l'instance ou Définir l'état de la timeline mais ne définissez pas la vitesse, le scénario prendra par défaut 1 image par image de jeu.


Syntaxe de l'action:


Arguments:

Argument La description
Speed La vitesse - en tant que multiplicateur - pour l'exécution de la timeline (la valeur par défaut est 1)

Le code de bloc d'action ci-dessus affectera une chronologie à l'instance appelant les actions ainsi que le moment initial à 10, puis la vitesse de la timeline à 2.
