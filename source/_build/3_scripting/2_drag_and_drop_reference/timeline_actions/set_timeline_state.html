Définir l'état de la chronologie


La description

Cette action peut être utilisée pour démarrer, arrêter, mettre en pause ou reprendre une chronologie. Le scénario doit d'abord avoir été défini pour l'instance à l'aide de l'action Définir le scénario de l'instance, puis vous pouvez utiliser cette action à tout moment pour modifier son état de fonctionnement parmi les options suivantes

Option La description
Début Lancez la timeline assignée à l'instance (si la timeline a été mise en pause, la position de départ sera de nouveau positionnée sur 0).
CV Reprendre une chronologie en pause (si la chronologie a été arrêtée et non en pause cela ne fera rien).
En pause Mettre en pause la chronologie assignée à l'instance (pendant la pause, les moments d'une ligne de temps ne progresseront pas avec les images de jeu).
Arrêtez Arrête le scénario assigné à l'instance.
