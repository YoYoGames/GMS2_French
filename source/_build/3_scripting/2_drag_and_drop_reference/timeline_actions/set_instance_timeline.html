Définir la chronologie de l'instance


La description

Cette action attribue un scénario à partir de l'arborescence des ressources à l'instance. Vous donnez la chronologie à assigner, puis vous pouvez choisir d'avoir la boucle de la ligne de temps quand elle atteint le dernier moment (l'image suivante après le dernier moment sera bouclée au premier moment de la ligne de temps) ainsi que démarrer la ligne de temps immédiatement ou non. Si vous ne démarrez pas immédiatement le scénario, vous pouvez utiliser l'action Définir l'état du scénario pour le démarrer ultérieurement dans un autre événement ou script.


Syntaxe de l'action:


Arguments:

Argument La description
Timeline La ressource timleine à affecter à une instance
Loop Indicateur pour que la boucle de la chronologie se termine après le dernier moment (la valeur par défaut est marquée)
Start Timeline Indicateur pour que la chronologie commence immédiatement ou non (la valeur par défaut est marquée)
