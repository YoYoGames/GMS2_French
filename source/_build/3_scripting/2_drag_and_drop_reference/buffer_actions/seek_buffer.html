Rechercher tampon


La description

Cette action peut être utilisée pour déplacer (rechercher) une position dans un tampon. Vous fournissez la valeur unique de l'ID de tampon comme renvoyée par l'action Créer un tampon, puis la valeur de décalage en octets vers laquelle se déplacer. La valeur de décalage sera appliquée à partir du début du tampon, donc l'appel de cette fonction déplace la position de lecture / écriture au début du tampon puis déplace le nombre d'octets donnés et positionne la nouvelle position de lecture / écriture là. Cependant, si vous marquez le décalage comme relatif, il déplacera la position de lecture / écriture vers l'avant (ou vers l'arrière si le décalage est une valeur négative) à partir de la position actuelle.


Syntaxe de l'action:


Arguments:

Argument La description
Buffer L'index du tampon (stocké dans une variable)
offset Le décalage (en octets) par rapport à la position de recherche actuelle
