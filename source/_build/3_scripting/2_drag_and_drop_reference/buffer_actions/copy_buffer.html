Copier le tampon


La description

Cette action peut être utilisée pour copier toutes les données stockées dans un tampon à un autre. Lors de l'utilisation de deux tampons et de la copie de l'un à l'autre, les deux tampons doivent avoir été préalablement créés à l'aide de l'action Create Buffer et fournir l'ID de tampon unique renvoyé en tant qu'arguments from / to.

REMARQUE: vous ne pouvez pas copier dans le même tampon.


Syntaxe de l'action:


Arguments:

Argument La description
Buffer L'index de tampon (stocké dans une variable) à copier à partir de
Copy To L'indice de tampon (stocké dans une variable) à copier


Exemple:

Le code de bloc d'action ci-dessus va créer un nouveau tampon et stocker son identifiant dans une variable temporaire. Il vérifie ensuite si le buffer indexé dans la variable globale donnée existe, et si c'est le cas, il est copié dans le tampon temporaire et la position de lecture / écriture est remise au début, et si elle n'existe pas, alors il est créé.
