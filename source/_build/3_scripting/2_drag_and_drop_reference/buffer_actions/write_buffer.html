Ecrire un tampon


La description

Cette action peut être utilisée pour écrire des données dans un tampon. Vous fournissez l'ID de tampon unique à écrire (renvoyé par l'action Create Buffer ), puis la valeur à écrire, qui peut être une valeur réelle ou une valeur chaîne, mais notez que toutes les valeurs sont stockées sous forme de chaînes. buffer (en utilisant Read Buffer ), vous devrez peut-être utiliser l'une des actions de type de données pour le convertir d'une chaîne à la valeur d'origine. Gardez également à l'esprit que les tampons sont lus et écrits de manière séquentielle et que l'écriture dans un tampon déplacera la position «rechercher» le long d'un tampon, ce qui signifie que pour écrire diverses données, vous appelez simplement le nombre de fois requis.
