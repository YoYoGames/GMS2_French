Obtenir l'élément de liste à


La description

Avec cette action, vous pouvez récupérer la position d'index d'un élément dans la liste donnée. Vous fournissez la variable qui stocke l'index de liste (renvoyé par l'action Créer une liste ) et la position d'index dans la liste pour obtenir la valeur de l'élément, ainsi qu'une variable cible pour stocker la valeur de l'élément retourné (qui peut être marqué comme une variable locale temporaire à utiliser jusqu'à la fin du script ou de l'événement).

NOTE: Si vous donnez une position qui est en dehors de la taille de liste donnée (c'est-à-dire: position 11 dans une liste de 10 valeurs) alors l'action retournera undefined. Vous pouvez vérifier cela à l'aide de l'action Si non défini.


Syntaxe de l'action:


Arguments:

Argument La description
List L'index (stocké dans une variable) de la liste à supprimer de
Index La position de l'index dans la liste pour obtenir l'élément de
Target La variable cible pour stocker la valeur renvoyée

Une fois la boucle de répétition terminée, la liste est libérée de la mémoire.
