Effacer la grille


La description

Avec cette action, vous pouvez effacer une structure de données de grille en une seule valeur (chaque cellule de la grille sera définie sur la valeur donnée). Vous fournissez la variable qui contient l'index de grille (tel que renvoyé par l'action Create Grid ), puis donnez la valeur à laquelle effacer la grille.


Syntaxe de l'action:


Arguments:

Argument La description
Grid L'index (stocké dans une variable) de la grille pour effacer
Value La valeur pour effacer la grille à


Exemple:

Le code de bloc d'action ci-dessus efface la structure de données de grille donnée pour contenir 0 dans chaque cellule si une variable globale est inférieure ou égale à 0.
