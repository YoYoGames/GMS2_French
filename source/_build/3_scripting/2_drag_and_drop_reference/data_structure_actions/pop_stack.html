Pop Stack


La description

Avec cette action, vous pouvez extraire une valeur du haut d'une structure de données de la pile. Vous indiquez la variable qui contient l'index de la pile (renvoyée par l'action Create Stack ), puis la variable cible qui doit contenir la variable renvoyée (qui peut être marquée comme variable locale temporaire).


Syntaxe de l'action:


Arguments:

Argument La description
Stack L'index (stocké dans une variable) de la pile à utiliser
Target La variable cible pour contenir la valeur renvoyée


Exemple:

Le code de bloc d'action ci-dessus vérifie si la pile donnée est vide, et si ce n'est pas le cas, il supprime la valeur supérieure et la stocke dans une variable temporaire. Ceci est ensuite utilisé pour définir une alarme, tandis que si la pile est vide, 10 valeurs aléatoires sont poussées sur la pile.
