Obtenir le nombre d'éléments de la liste


La description

Vous pouvez utiliser cette action pour connaître le nombre d'entrées dans une liste. Vous fournissez la variable qui contient l'index de liste (tel que renvoyé par l'action Créer une liste ) et la variable cible pour stocker le nombre d'éléments renvoyés dans la liste (qui peut être marqué comme une variable locale temporaire à utiliser jusqu'à la fin du script ou événement).


Syntaxe de l'action:


Arguments:

Argument La description
List L'index (stocké dans une variable) pour obtenir le nombre d'éléments de
Target La variable cible pour stocker le nombre d'éléments renvoyés


Exemple:

Le code de bloc d'action ci-dessus obtient la taille de la liste référencée dans la variable globale, puis utilise une boucle de répétition pour la parcourir et détruire les instances avec les identifiants stockés dans la liste. Une fois la boucle de répétition terminée, la liste est libérée de la mémoire.
