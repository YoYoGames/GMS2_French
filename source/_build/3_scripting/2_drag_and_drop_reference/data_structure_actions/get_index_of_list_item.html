Obtenir l'index de l'élément de liste


La description

Avec cette action, vous pouvez récupérer la position d'index d'un élément dans la liste donnée. Vous fournissez la variable qui stocke l'index de liste (tel que renvoyé par l'action Créer une liste ) et la valeur que vous souhaitez trouver la position d'index dans la liste, ainsi qu'une variable cible pour stocker la position d'index retournée pour l'élément (qui peut être marqué comme une variable locale temporaire à utiliser jusqu'à la fin du script ou de l'événement). Notez que si la valeur que vous recherchez n'existe pas dans la liste, il n'y aura pas de position d'index de liste à retourner, et donc la valeur de retour sera -1.


Syntaxe de l'action:


Arguments:

Argument La description
List L'index (stocké dans une variable) de la liste à supprimer de
Value La valeur pour vérifier la liste pour
Target La variable cible pour stocker l'index renvoyé

S'il existe, l'élément est supprimé de la liste et l'instance est détruite.
