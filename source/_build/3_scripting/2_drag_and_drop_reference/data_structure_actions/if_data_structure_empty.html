Si la structure de données est vide


La description

Avec cette action, vous pouvez vérifier si la structure de données donnée est vide ou non. Vous devez d'abord choisir le type de structure de données à vérifier (soit une liste, une carte ou une pile - pas de grille car ils ne peuvent même pas être vides) et ensuite fournir la variable qui contient l'index de structure de données (tel que renvoyé par les actions Créer une liste, Créer une carte et Créer une pile ). L'action retournera true si la structure de données est vide, et false si ce n'est pas le cas. Notez que vous pouvez marquer "non" dans l'action pour vérifier si une structure de données contient des données en vérifiant si elle n'est pas vide.
