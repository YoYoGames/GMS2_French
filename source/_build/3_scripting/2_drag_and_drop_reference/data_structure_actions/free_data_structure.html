Structure de données gratuite


La description

Cette action libère la structure de données de la mémoire et doit être appelée chaque fois qu'une structure de données n'est plus nécessaire, par exemple dans le Destroy même d'une instance ou l'événement Room End, etc... Libérer une structure de données signifie qu'aucune autre action peut être utilisé pour le référencer car il n'existe plus, bien que vous puissiez également le vérifier en utilisant l'action Si la structure de données existe.

Notez que si la structure de données n'a pas d'index affecté à une variable de portée globale, elle ne sera pas transmise d'une pièce à l'autre. Par exemple, si vous créez une instance de joueur et qu'une structure de données de liste est affectée à une variable dans cette instance, la liste devra être libérée si le joueur est détruit ou si la pièce change avec l'instance de joueur (sauf si le joueur instance est marquée comme persistante), de sorte que l'instance de lecteur possède deux actions de structure de données libre pour ces deux événements. Il est important que toutes les structures de données créées soient détruites de cette manière lorsqu'elles ne sont plus nécessaires pour éviter les fuites de mémoire qui peuvent causer des problèmes de performances ou même provoquer un crash de votre jeu.
