Si la structure de données existe


La description

Avec cette action, vous pouvez vérifier une variable pour voir si elle indexe une structure de données ou non. Vous devez d'abord choisir le type de structure de données à vérifier (soit une liste, une carte, une grille ou une pile) et ensuite fournir la variable contenant l' index de structure de données (tel que renvoyé par les actions Créer une liste, Créer une carte, Créer une grille et Créer une pile ). L'action retournera true si la structure de données existe, et false si ça ne fait pas. Notez que vous pouvez marquer "non" dans l'action pour vérifier si une variable ne contient pas un index de structure de données.
