Retirer de la liste


La description

Cette action peut être utilisée pour supprimer un élément d'index de la liste donnée. Vous fournissez la variable qui stocke l'index de la liste (renvoyé par l'action Créer la liste ) et la position de l'index dans la liste à supprimer, où la position de l'index est comprise entre 1 et (longueur de la liste -1). Cette action ne renvoie rien alors si vous avez besoin de la valeur à la position de l'index, vous devez utiliser Get List Item At avant de supprimer l'index.


Syntaxe de l'action:


Arguments:

Argument La description
List L'index (stocké dans une variable) de la liste à supprimer de
Index L'index dans la liste à supprimer


Exemple:

Le code de bloc d'action ci-dessus vérifie la présence d'une collision à la position de l'instance et, si elle est trouvée, la valeur ID unique de l'instance est stockée dans une variable temporaire puis vérifiée pour voir si elle existe dans la structure de données. S'il existe, l'élément est supprimé de la liste et l'instance est détruite.
