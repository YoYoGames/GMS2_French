Jouer l'audio


La description

Cette action définit le son donné. Vous sélectionnez le son à partir de l'explorateur de ressources, puis définissez si vous souhaitez que le son soit bouclé ou non (un son en boucle sera lu encore et encore jusqu'à ce qu'il soit arrêté) en réglant le drapeau "Loop".


Syntaxe de l'action:


Arguments:

Argument La description
Sound La ressource sonore à jouer
Loop Que ce soit pour boucler le son ou non (désactivé par défaut)


Exemple:

Le code de bloc d'action ci-dessus vérifie la pression d'un bouton de la souris et, s'il est détecté, il vérifie une variable globale pour voir si elle n'est pas vraie. Si ce n'est pas vrai, alors un son est joué (et mis en boucle) et la variable globale définie sur true, sinon le son est arrêté et la variable globale est définie sur false.
