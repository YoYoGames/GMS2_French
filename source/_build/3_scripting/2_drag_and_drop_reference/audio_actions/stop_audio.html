Arrêter l'audio


La description

Cette action peut être utilisée pour arrêter la lecture d'un son. Vous sélectionnez le son dans l'explorateur de ressources et toutes les instances de ce son cessent de jouer (donc si vous avez réglé le même son pour qu'il joue dans trois événements différents, alors tous s'arrêteront).


Syntaxe de l'action:


Arguments:

Argument La description
Sound La ressource sonore pour arrêter


Exemple:

Le code de bloc d'action ci-dessus vérifie la pression d'un bouton de la souris et, s'il est détecté, il vérifie une variable globale pour voir si elle n'est pas vraie. Si ce n'est pas vrai, alors un son est joué (et mis en boucle) et la variable globale définie sur true, sinon le son est arrêté et la variable globale est définie sur false.
