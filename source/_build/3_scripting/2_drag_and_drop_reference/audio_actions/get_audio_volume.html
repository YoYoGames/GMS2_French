Obtenir le volume audio


La description

Cette action renvoie le volume du son donné. Vous choisissez le son dans l'explorateur d'actifs et vous donnez une variable cible pour renvoyer la valeur du volume (vous pouvez marquer ceci comme étant une variable locale temporaire). La valeur retournée sera entre 0 et 1, où 0 est silencieux et 1 est plein volume.


Syntaxe de l'action:


Arguments:

Argument La description
Sound La ressource sonore pour obtenir le volume de
Target La variable cible pour stocker le volume renvoyé


Exemple:

Le code de bloc d'action ci-dessus vérifie une variable globale pour voir si elle est true ou fausse. Si c'est true alors le volume du son donné est récupéré et stocké dans une variable locale temporaire. La valeur est ensuite vérifiée pour voir si elle n'est pas égale à 0.5, et si ce n'est pas le cas, alors le volume du son est fixé à 0.5. Si la variable globale évalue à false, alors le son a son volume réglé à 1.
