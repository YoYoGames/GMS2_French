Définir le volume principal


La description

Avec cette action, vous pouvez régler le volume pour l'ensemble du système sonore. La valeur du volume que vous donnez peut être comprise entre 0 (silencieux) et 1 (plein volume) et l'échelle est linéaire, de telle sorte qu'une valeur de 0,5 serait égale à la moitié du volume. Notez que cette action affectera toutes les instances de tous les sons joués actuellement dans la pièce et affectera toutes les instances futures de tous les sons joués. Cette action est également indépendante du volume défini pour chaque son individuel, donc si vous avez un son réglé pour jouer à 0,5 volume, puis réglez le volume principal sur 0,5, le son jouera à 0,25 volume.


Syntaxe de l'action:


Arguments:

Argument La description
Volume Le nouveau volume principal pour tous les sons (de 0 à 1, par défaut 1)


Exemple:

Le code de bloc d'action ci-dessus vérifie une variable globale pour voir si elle est true ou fausse. Si c'est true le volume principal du système audio est alors récupéré et stocké dans une variable locale temporaire. La valeur est ensuite vérifiée pour voir si elle n'est pas égale à 0,5, et si ce n'est pas le cas, alors le volume principal du système audio est fixé à 0,5. Si la variable globale évalue à false, le volume principal est réglé sur 1.
