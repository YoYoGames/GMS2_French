Obtenir la longueur audio


La description

Cette action peut être utilisée pour récupérer la longueur d'un son dans votre jeu. Vous fournissez la ressource sonore pour obtenir la longueur et une variable cible pour renvoyer la valeur aussi (ou vous pouvez marquer la cible comme une variable temp pour que l'action crée la variable à utiliser jusqu'à la fin de l'événement ou du script). La longueur sera renvoyée à la variable cible (qui peut être rendue temporaire) en secondes et peut être une valeur décimale.


Syntaxe de l'action:


Arguments:

Argument La description
Sound La ressource sonore pour obtenir la longueur de
Target La variable cible pour stocker la valeur de retour


Exemple:

Le code de bloc d'action ci-dessus obtient la longueur de lecture de la ressource sonore donnée et la stocke dans une variable locale temporaire. Ceci est ensuite utilisé pour générer une valeur aléatoire entre 0 et la longueur du son, qui est ensuite stockée dans une variable locale temporaire différente. Cette nouvelle valeur aléatoire est ensuite utilisée pour définir la position de départ du son et le son est alors joué.
