Dessiner Ellipse


La description

Cette action dessine une ellipse à une position donnée dans la pièce. Vous donnez la position en haut à gauche et la position en bas à droite de la zone dans laquelle l'ellipse doit "entrer", et l'ellipse sera dessinée entre eux. Les positions peuvent être soit une position absolue dans la pièce, soit une position relative à l'instance appelant l'action, et vous pouvez également définir si l'ellipse peut être remplie ou soulignée en cochant la case de remplissage en bas.

REMARQUE: cette action est uniquement destinée à être utilisée dans les divers événements de dessin et ne dessinera rien si elle est utilisée ailleurs.


Syntaxe de l'action:


Arguments:

Argument La description
Left La position gauche pour commencer à dessiner à partir de
Top La meilleure position pour commencer à dessiner à partir de
Right La bonne position pour terminer le dessin de
Bottom La position inférieure pour terminer le dessin de
