Dessiner la bibliothèque d'actions


La bibliothèque d'actions de dessin est l'emplacement où vous pouvez trouver les actions requises pour dessiner des sprites, du texte ou des formes, ainsi que définir certaines propriétés de dessin. La plupart de ces actions sont uniquement destinées à être utilisées dans les divers événements Draw d'un objet et peuvent ne pas fonctionner si elles sont utilisées en dehors de l'événement Draw. Les exceptions à ceci sont les actions d' ensemble, qui peuvent être ajoutées à n'importe quel événement et affecteront tous les dessins pour toutes les instances après.

Il est important de noter que si vous ajoutez des actions dans l' événement Draw principal d'un objet, il ne dessinera pas l'image-objet qui a été assignée à l'instance sauf si vous lui avez explicitement GameMaker Studio 2 de dessiner, en utilisant une action comme Draw Soi -même Fondamentalement, GameMaker Studio 2 dessine par défaut toute image-objet affectée à une instance, uniquement s'il n'y a rien d'autre dans l'événement Draw.
