Dessiner une ligne


La description

Cette action trace une ligne à une position donnée dans la pièce. Vous donnez la position initiale x / y et ensuite la position finale x / y et la ligne sera tracée entre eux en utilisant la couleur de tirage actuelle. Les positions peuvent être soit une position absolue dans la pièce, soit une position relative à l'instance appelant l'action.

REMARQUE: cette action est uniquement destinée à être utilisée dans les divers événements de dessin et ne dessinera rien si elle est utilisée ailleurs.


Syntaxe de l'action:


Arguments:

Argument La description
x La position de départ x
y La position y de départ
x2 La position x de finition
y2 La position finale y


Exemple:

Le code de bloc d'action ci-dessus dessine une ligne rose dans le haut de la pièce.
