Dessiner le score d'instance


La description

Avec cette action, vous pouvez dessiner la valeur de score de l'instance. Lorsque vous utilisez une action de score dans GameMaker Studio 2 l'instance qui l'appelle aura une nouvelle variable d'étendue d'instance appelée score ajoutée. Cette variable peut ensuite être vérifiée et définie comme toute autre variable. Dans le cas de cette action, la variable sera dessinée à l'écran à la position donnée, avec tout texte de légende que vous souhaitez fournir. Le texte de légende sera dessiné précédant la valeur du score et s'il s'agit d'un texte, il doit être entouré de "", et les deux seront formatés en utilisant la police, la couleur et l'alignement actuels.

REMARQUE: cette action est uniquement destinée à être utilisée dans les divers événements de dessin et ne dessinera rien si elle est utilisée ailleurs.


Syntaxe de l'action:


Arguments:

Argument La description
Caption La chaîne à dessiner avec la valeur de score (précédente et peut être laissée vide)
x La position x pour dessiner le score à
y La position y pour tracer le score à
