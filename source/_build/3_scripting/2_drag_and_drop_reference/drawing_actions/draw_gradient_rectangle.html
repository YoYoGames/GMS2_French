Dessiner un rectangle de dégradé


La description

Cette action dessine un rectangle à une position donnée dans la pièce, en utilisant un ensemble de couleurs de fusion pour créer un dégradé. Vous donnez la position en haut à gauche et la position en bas à droite et le rectangle sera tracé entre eux, et la position peut être soit une position absolue dans la pièce, soit une position par rapport à l'instance appelant l'action. Vous définissez ensuite les couleurs à fusionner à partir de chacun des coins et pouvez également définir si le rectangle peut être rempli ou souligné en cochant la case de remplissage en bas.

REMARQUE: cette action est uniquement destinée à être utilisée dans les divers événements de dessin et ne dessinera rien si elle est utilisée ailleurs.
