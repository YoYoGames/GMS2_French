Définir l'alignement du texte


La description

Cette action définit l'alignement des polices pour toutes les actions de texte de dessin suivantes. Vous pouvez définir l'alignement horizontal comme étant à gauche, au centre ou à droite et l'alignement vertical sur haut, milieu ou bas et le texte sera aligné par rapport à la position x / y de l'action de dessin, comme illustré dans les images ci-dessous:

Le texte est aligné horizontalement à gauche
Le texte est aligné horizontalement au centre
Le texte est aligné horizontalement à droite
Le texte est aligné verticalement vers le haut
Le texte est aligné verticalement au milieu
Le texte est aligné verticalement vers le bas

Notez que ceci définira l'alignement à utiliser pour tous les appels de texte suivants, et peut donc être appelé, par exemple, une fois au début du jeu - dans tous les cas, pas seulement l'événement de dessin - et toutes les actions de dessin utiliseront l'alignement défini. Cependant, si vous avez besoin d'aligner différentes actions de texte, vous devrez appeler cette action avant chaque action de texte que vous souhaitez aligner différemment.


Syntaxe de l'action:


Arguments:

Argument La description
H.Align La valeur d'alignement horizontal (Gauche, Centre, Droite)
V.Align La valeur d'alignement vertical (Haut, Moyen, Bas)
