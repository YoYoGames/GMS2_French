Définir l'alignement du texte


La description

Cette action définira l'alignement de la police pour toutes les actions de texte de dessin suivantes. Vous pouvez définir l'alignement horizontal à gauche, au centre ou à droite et l'alignement vertical en haut, au milieu ou en bas et le texte sera aligné par rapport à la position x / y de l'action de dessin, comme indiqué dans les images ci-dessous.:

Le texte est aligné horizontalement à gauche
Le texte est aligné horizontalement au centre
Le texte est aligné horizontalement à droite
Le texte est aligné verticalement vers le haut
Le texte est aligné verticalement au milieu
Le texte est aligné verticalement vers le bas

Notez que cela définira l'alignement à utiliser pour tous les appels de texte de dessin ultérieurs, et peut donc être appelé, par exemple, une fois au début du jeu - dans tous les cas, pas seulement l'événement de dessin - et toutes les actions de texte de dessin utiliseront l'alignement du jeu. Toutefois, si vous souhaitez aligner plusieurs actions de texte dessinées de différentes manières, vous devez appeler cette action avant chaque action de texte dessiné que vous souhaitez aligner différemment.


Syntaxe d'action:


Arguments:

Argument La description
H.Align La valeur d'alignement horizontal (gauche, centre, droite)
V.Align La valeur d'alignement vertical (Haut, Milieu, Bas)
