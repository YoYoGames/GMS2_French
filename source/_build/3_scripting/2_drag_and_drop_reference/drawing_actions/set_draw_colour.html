Définir la couleur de dessin


La description

Avec cette action, vous pouvez définir la couleur à utiliser pour toutes les formes (rectangle / ellipse / ligne) ou pour tout texte dessiné ultérieurement. Notez que cela ne doit pas être appelé avant chaque action de tirage, et ne peut être défini qu'une seule fois au début de la partie. Par exemple, si tout votre texte va être jaune, vous pouvez appeler cette action dans l'événement de création de la première pièce du jeu, et tout le texte (et les formes) sera alors dessiné en jaune à partir de là. Cependant, si vous devez utiliser plusieurs couleurs différentes, vous devrez l'appeler avant chaque texte ou forme qui nécessite une couleur différente.

L'action a une case à cocher en bas pour Utiliser l'alpha. Cela permet à la composante alpha de la couleur d'être utilisée pour définir l'alpha de tirage (à peu près la même chose qu'en utilisant Set Draw Alpha.) Si vous ne voulez pas inclure le composant alpha et avoir juste une couleur unie, cela peut être désactivé ( c'est activé par défaut).
