Dessiner Sprite


La description

Cette action va dessiner un sprite quelque part dans la pièce. Vous spécifiez le sprite, la position (soit une position absolue dans la pièce, soit une position relative par rapport à l'instance faisant le dessin) et le cadre d'animation de l'image-objet, s'il a plusieurs images (notez que les images sont numérotées de 0 ). Si vous avez une image-objet animée et que vous voulez qu'elle dessine les images comme si elles avaient été affectées à l'objet, utilisez -1 pour l'index d'image ou la variable d'instance intégrée image_index, mais notez que si l'instance n'a aucune image-objet assignée, elle ne s'anime pas, et si l'image-objet affectée a un nombre différent d'images à dessiner, l'image-objet sera dessinée avec des images supplémentaires ou des images manquantes.

IMPORTANT: Cette action ne dessine pas l'image-objet transformée, même si vous avez modifié le mélange ou l'échelle de l'image. Pour cela voir l'action Draw Sprite Transformed.
REMARQUE: cette action est uniquement destinée à être utilisée dans les divers événements de dessin et ne dessinera rien si elle est utilisée ailleurs.


Syntaxe de l'action:


Arguments:

Argument La description
Sprite Le sprite à dessiner
x La position x à dessiner à
y La position y pour dessiner à
Frame Index L'index de cadre à dessiner
