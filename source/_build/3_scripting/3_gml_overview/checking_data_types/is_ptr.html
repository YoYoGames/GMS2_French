is_ptr


La description

Cette fonction renvoie si une variable donnée est un pointeur ou non. Dans certains cas, vous voulez vérifier et voir quel type de données contient une variable dans GameMaker Studio 2 et c'est à ce moment-là que vous utiliserez cette fonction. Ça revient true ou false selon que la valeur est un pointeur ou non. Pour plus d'informations sur les types de données, voir ici.


Syntaxe:

is_ptr(n);


Arguments:

Argument La description
n L'argument à vérifier.


Résultats:

Boolean


Exemple:

if !is_ptr(val)
   {
   show_debug_message("Not a valid texture!");
   }

Le code ci-dessus vérifie la variable "val" pour voir s'il contient un pointeur et si ce n'est pas le cas, il affiche un message dans la console de débogage.
