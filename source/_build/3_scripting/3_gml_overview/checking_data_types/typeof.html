Type de


La description

Cette fonction renvoie le type de données d'une variable donnée sous la forme d'une chaîne. Les valeurs de retour possibles sont répertoriées dans le tableau ci-dessous:

Chaîne La description
number La variable contient un nombre réel (virgule flottante)
string La variable contient une chaîne
array La variable référence un tableau
bool La variable contient un booléen ( true / false )
int32 La variable contient un entier de 32 bits
int64 La variable contient un entier de 64 bits
ptr La variable contient un pointeur
undefined La variable est indéfinie
null La variable contient une valeur nulle (ceci ne devrait pas être vu normalement)
vec3 La variable contient un vecteur de 3 valeurs
vec4 La variable contient un vecteur de 4 valeurs
unknown La valeur est inconnue. Cela ne devrait jamais être vu et signifie que quelque chose s'est mal passé au niveau le plus fondamental, comme un écrasement de la mémoire


Syntaxe:

typeof(variable);


Arguments:

Argument La description
variable La variable pour obtenir le type de données.


Résultats:

String (see table, above)


Exemple:

data_str = typeof(global.ExtensionInput);

Le code ci-dessus obtient le type de données détenu par la variable globale donnée et renvoie la chaîne à une variable d'instance.
