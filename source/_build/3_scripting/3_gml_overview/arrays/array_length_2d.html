array_length_2d


La description

Avec cette fonction, vous pouvez obtenir la longueur (nombre d'entrées) d'une deuxième dimension d'un tableau. Vous fournissez le numéro d’entrée de la première dimension et la fonction renvoie le nombre d’entrées de la deuxième dimension du tableau (pour rechercher la longueur de la première dimension, utilisez la fonction array_height_2D ). La fonction retournera 0 si la variable donnée n'est pas un tableau ou 1 si la variable est un tableau 1D (car il y a encore 1 ligne).


Syntaxe:

array_length_2d(array_index, n);


Arguments:

Argument La description
array_index L'index du tableau à vérifier.
n L'entrée du tableau pour obtenir la longueur de.

Résultats:

Real


Exemple:

for (var i = 0; i < array_height_2d(a); ++i;)
   {
   for (var j = 0; j < array_length_2d(a, i); ++j;)
      {
      a[i, j] = -1;
      }
   }

Le code ci-dessus parcourt un tableau 2D et définit chaque entrée sur -1.
