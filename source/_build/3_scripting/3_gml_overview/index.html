GameMaker langage GameMaker


Cette section du manuel contient toutes les informations nécessaires pour comprendre et utiliser le langage de script GML de GameMaker Studio 2. Le code dans cette langue peut être utilisé pour créer vos jeux et est ajouté aux objets depuis l' éditeur d'objets, normalement lorsque vous avez choisi de créer un nouveau GameMaker langue GameMaker, bien qu'il puisse également être utilisé avec DnD™ (Drag and Drop ). Ci-dessous vous pouvez voir une image typique d'un objet avec l'éditeur de code ouvert sur un événement: Chaque événement possède son propre onglet dans l'éditeur et vous pouvez à tout moment ajouter, modifier ou supprimer du code (pour plus d'informations sur les événements, reportez-vous à la section Événements d'objet ). Le code lui-même doit avoir une structure de base et peut contenir des indices de ressources, des variables, des fonctions, des expressions, des mots-clés... etc... qui sont tous expliqués dans les sections ci-dessous. Si vous êtes novice en matière de programmation ou de passage de DnD™, il est recommandé de commencer par la page Structure de code de base, puis de les lire tous, en testant le code de chacune des sections de GameMaker Studio 2.
