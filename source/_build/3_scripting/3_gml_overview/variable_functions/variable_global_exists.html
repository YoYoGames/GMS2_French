variable_global_exists


La description

Avec cette fonction, vous pouvez vérifier si une variable de portée globale existe ou non. Vous fournissez le nom de la variable globale à vérifier en tant que chaîne (voir l'exemple de code ci-dessous) et la fonction retournera true si une variable globale avec le nom donné existe ou false autrement.

REMARQUE: Cette fonction est principalement utilisée avec les scripts de compatibilité pour les projets importés et le code converti pour les actions de glisser-déposer.


Syntaxe:

variable_global_exists(name);


Arguments:

Argument La description
name Le nom de la variable globale à vérifier (sous forme de chaîne)

Résultats:

Boolean


Exemple:

if !variable_global_exists("enemy_num")) global.enemy_num = instance_number(obj_Enemey_Parent);

Le code ci-dessus va vérifier pour voir si la variable globale appelée "enemy_num" existe et si ce n'est pas le cas, il est créé et définir une valeur.
