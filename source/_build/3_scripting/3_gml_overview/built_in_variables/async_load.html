async_load


Syntaxe:

async_load;


Retours: Real


La description

Cette variable a une portée globale et est utilisée pour contenir une carte DS lorsqu'elle est utilisée dans les événements asynchrones, et -1 dans tous les autres cas. Le contenu réel de la mappe DS dépendra du type d'événement asynchrone dans lequel elle est créée et de la fonction utilisée pour déclencher l'événement. Par conséquent, reportez-vous aux pages individuelles correspondant à ces événements.


Exemple:

if ds_map_find_value(async_load, "id") == back
   {
   if ds_map_find_value(async_load, "status") >= 0
      {
      background_index[0] = back
      }
   }

Le code ci-dessus est extrait de l'événement Asynchrnous Image Loaded et vérifie d'abord l'ID du async_load Carte DS qui a été remplie, puis vérifiez l'état du rappel. Si la valeur est supérieure ou égale à 0 (succès de la signalisation), le résultat du rappel sera utilisé pour définir l'index d'arrière-plan de l'image nouvellement chargée.
