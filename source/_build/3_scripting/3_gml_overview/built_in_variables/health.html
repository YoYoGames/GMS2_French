santé (obsolète)


Syntaxe:

health;


Retours: Real


La description

Cette variable a une portée globale et est utilisée pour contenir une valeur numérique généralement utilisée pour l'état de santé du joueur. Gardez à l'esprit que ce n'est pas forcément toujours le cas - juste parce que cela s'appelle «santé» ne signifie pas qu'il doit être utilisé pour la santé... il peut contenir n'importe quelle valeur réelle ou entière que vous souhaitez pour votre jeu.

IMPORTANT! Cette variable est uniquement incluse pour prendre en charge les projets hérités des versions précédentes de GameMaker et ne doit pas être utilisée dans de nouveaux projets car elle est obsolète et dans les futures versions ou mises à jour, elle peut être entièrement supprimée.


Exemple:

if place_meeting(x, y, obj_Enemy)
   {
   health -= 10;
   if health <= 0 instance_destroy();
   }

Le code ci-dessus vérifie l'instance actuelle pour voir s'il y a une collision avec l'instance de l'objet indexée dans "obj_Enemy", et s'il y en a, il supprime 10 points de la variable d'intégrité globale puis vérifie cette valeur pour voir si elle est inférieur ou égal à 0. Si c'est le cas, l'instance se détruit elle-même.
